当我们穿过的时候，你可以看到它被锁在脸上，如果我们放大这个部分, 你可以看到我们需要在 H1-J1 区域内画青色。所以，如果我们回到我们的 RotoPaint 节点，一半混合网格 (设置 其中 到 0.15 在溶解节点属性中)，并在该部分绘制，我们现在应该看到我们的血液出现在大约正确的地方。所以我们将在 H1 和 J1 之间画青色，如果我们切换到脸，我们可以看到它大约是正确的。然而，这个设置仍然不理想，所以我将切换到一个双浏览器设置，现在我可以更互动地看到我在做什么。

所以现在我有了我的第一个绘画笔画，我现在可以移除网格，回到 RotoPaint，在这个区域工作 ( 油漆 和 涂抹 )。现在，当我们在镜头中移动时，因为我们已经在几何图形上画画，通过相机，我们的血液随着头部移动。然而，由于模型的不准确性，出现了滑点。为了解决这个问题，我们需要对所有这些绘画笔划的位置进行动画处理。为此，我们将创建一个层，选择绘制笔划，并将它们移动到该层中。现在，在 变换 Tab，你可以一次动画他们的位置。所以我将设置一个关键帧，进一步向下移动，并调整位置。通过少量的校正，我们能够让血液锁定在眼睛上。如果我们通过并继续检查它在哪里发生并重新调整它，我们最终会得到一个更准确的结果。在这里，你可以看到油漆笔划实际上是如何在 2D 板上移动的。
