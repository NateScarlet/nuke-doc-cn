Getting Started with Nuke

If you're new to Nuke or simply need a refresher on the basics, we recommend you start with the below tutorials. They familiarize you with the interface and the product family, and help you perform the most common tasks in Nuke. See:

• Step Up to Nuke Training Series - 11 videos that cover everything from an interface overview to 3D in Nuke.

• The Nuke Product Family - Tutorials that give you more information on the different Nuke products available to you.

• Animation in Nuke - Tutorials that teach you how to work with animated parameters in Nuke.
