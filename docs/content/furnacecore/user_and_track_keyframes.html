User Keyframes and Track Keyframes

User keyframes are keyframes you set manually. When a user keyframe has been set on the current frame, this is indicated by a red key that appears on the on-screen wire tool. Track keyframes are keyframes set by F_WireRemoval’s tracker.

User keyframes override track keyframes on the same frame. This has the following consequences:

• Once you set a user keyframe, the track keyframe on that frame will be lost.

• When you track a wire, existing user keyframes will not be replaced by track keyframes.
