什么是全局运动估计？

全局运动估计 (GME) 是一种尝试用简单的四角销将一幅图像映射到另一幅图像的技术。这不同于局部运动估计 (LME)，它试图找到图像中每个像素在另一个图像中的位置。GME 比 LME 计算便宜得多，但是给你的图像信息更少。然而，对于各种应用来说，它仍然非常强大。

使用插件的参数，你可以告诉 GME 引擎期望什么类型的运动。这可以是以下任何一项的组合:

1。 翻译 -允许四个角翻译相同的量,
2. 旋转 -允许角落围绕它们的中心旋转,
3. 规模 -允许角定义的区域的大小改变,
4. 透视 -允许角处的角度改变，因此它们定义的区域不再是矩形。

你允许的运动类型越多，运动估计就越昂贵。对于许多场景，旋转和平移就足够了。

GME 效应有一个精度控制，它控制铸造厂的 GME 引擎计算全局运动的工作量。通常，这越高，估计越好，但越昂贵。

GME 的局限性

如上所述，全局运动估计只需计算一个四角引脚来将一幅图像转换到另一幅图像。这意味着 GME 不能用于匹配两幅有严重视差、非常复杂的前景运动、物体变化等的图像。

思考 GME 能做什么的最好方法是，如果你能用四角销做，它就可以; 如果你不能，它就不能。然而，GME 会让手脚逐帧匹配的痛苦消失。
