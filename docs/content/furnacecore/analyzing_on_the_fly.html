在飞机上分析

要对齐两个快照并实时计算对齐，请执行以下操作:

1。 查找同一场景的两个快照，但具有稍微不同的相机运动和前景对象。选择 图像 > 阅读 加载这两个镜头。
2. 选择 FurnaceCore > F_Align .
3. 将其中一个快照连接到源 ( Src ) 输入 F_Align 和其他参考 ( 参考 ) 输入。查看 F_Align 的输出。

的 Src 将立即重新定位，以便与 参考 拍摄时不需要任何分析。

4. 根据两个镜头之间的确切差异，您可能需要启用 规模 和/或 透视 切换以获得一个体面的对齐。
5. 您可能还需要重新定位 分析区域 取决于前景细节的差异。然而，把它留在默认位置对大多数镜头都很有效。
6. 要查看两个剪辑的对齐程度，请按 M 在节点图上插入合并节点。连接合并节点的 一个 输入到 F_Align 和 B 输入到参考图像。查看合并节点的输出。然后，调整 混合 在合并控件中使用滑块，以在 F_Align 的输出和要匹配的引用剪辑之间溶解。
