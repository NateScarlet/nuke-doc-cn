参数

此插件的参数如下所述。

分析 -这是一个按钮，它将触发对输入剪辑的分析，并计算一个角销。中断分析不会删除已经计算好的角销键。

分析期间渲染 -如果设置，此切换将导致效果更新时间线，并在查看器中渲染新分析的帧，以便您可以看到效果的进度。这样做会在一定程度上降低分析速度，因此关闭此操作以加快一般分析。

清晰的分析 -按下这个按钮将删除角销参数中的所有关键帧，如果你觉得有必要，可以强制重新分析。

分析范围 -这控制了任何分析将被运行的帧范围。它可以是以下之一:

• 指定范围 -这将看看参数 分析开始 和 分析停止 对于要分析的框架范围,

• 源剪辑范围 -从输入剪辑的长度自动确定要分析的帧范围。

• 当前帧 -分析仅发生在当前帧上。这对于纠正分析整个剪辑时可能发生的任何错误非常有用。

分析开始 -从 if 分析的第一帧 分析 范围 设置为 指定范围 .

分析停止 -从 if 分析的最后一帧 分析范围 设置为 指定范围 .

规模 -指示计算的角销是否可以包括缩放因子的切换。

旋转 -指示计算的角销是否可以包括旋转的切换。

翻译 -指示计算的角销是否可以包括 x 和 y 中的翻译的切换。

透视 -指示计算的角销是否可以包括透视变换的切换。

分析区域 -这是分析计算四角销的区域。当执行任何形式的帧锁定时，这尤其有用，在这种情况下，转到锁定帧，查看参考剪辑，并将框定位在要锁定的区域上。

分析区域 BL -控制分析区域左下角的位置。

分析区域 TR -控制分析区域右上角的位置。

高级 -较少使用的细化控制。

精度 -这控制了时间/准确性的权衡。这越高，分析就越慢，但是你有更好的结果的可能性。

过滤 -设置过滤质量。

• 低 -低质量，但渲染速度快。

• 介质 -使用双线性过滤器。这提供了良好的结果，并且比高过滤更快地渲染。

• 高 -使用 sinc 滤波器来插入像素，从而进行更清晰的修复。这给出了最好的结果，但需要更长的时间来处理。

反转 -如果设置，则在渲染过程中使用计算的四角销的倒数。

四角销 -分析过程中计算的角销:

左下 -左下角的别针。

右下 -右下角的别针。

左上 -左上角的别针。

右上方 -右上角的别针。
