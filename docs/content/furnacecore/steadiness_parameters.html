参数

此插件的参数如下所述。

分析 -这是一个按钮，它将触发对输入夹的分析，并计算一个角销。中断分析不会删除已经计算好的角销键。

分析期间渲染 -如果设置，此切换将导致效果更新时间线并渲染新分析的帧，以便您可以看到效果的进度。这样做会在一定程度上降低分析速度，因此关闭此操作以加快一般分析。

清晰的分析 -按下这个按钮将删除角销参数中的所有关键帧，如果你觉得有必要，可以强制重新分析。

分析范围 -这控制了任何分析将被运行的帧范围。它可以是以下之一:

• 指定范围 -这将看看参数 分析开始 和 分析停止 对于要分析的框架范围,

• 源剪辑范围 -从输入剪辑的长度自动确定要分析的帧范围。

分析开始 -从 if 分析的第一帧 分析 范围 设置为 指定范围 .

分析停止 -从 if 分析的最后一帧 分析范围 设置为 指定范围 .

规模 -指示计算的角销是否可以包括缩放因子的切换。

旋转 -指示计算的角销是否可以包括旋转的切换。

翻译 -指示计算的角销是否可以包括 x 和 y 中的翻译的切换。

透视 -指示计算的角销是否可以包括透视变换的切换。

分析区域 -这是分析计算四角销的区域。当执行任何形式的帧锁定时，这尤其有用，在这种情况下，转到锁定帧，查看参考剪辑，并将框定位在要锁定的区域上。

分析区域 BL -控制分析区域左下角的位置。

分析区域 TR -控制分析区域右上角的位置。

高级 -较少使用的细化控制。

模式 -此参数控制 F_Steadiness 是在保持整个相机运动的同时平滑镜头，还是将镜头锁定到单个帧以完全移除相机运动。它可以设置为:

• 增量锁定 -在这种情况下，它将计算将每个帧带到锁定帧的引脚。这将通过从锁定帧到每个帧的工作来计算 pin，以增量方式计算每个帧之间的 GME，并累积它以创建拐角 pin。

• 绝对锁定 -这还计算了将每个帧带到锁定帧的 pin。然而，它通过直接从相关帧到锁定帧进行 GME 来做到这一点。

• 平滑 -在这种情况下，我们正在平滑镜头的一系列帧 平滑 参数。这是默认值。它可以用来保持相机的整体运动，但是可以抚平尖锐的肿块和踢腿。

增量锁在某些情况下工作得更好，而绝对锁在其他情况下工作得更好。然而，绝对锁在工作时通常更准确。例如，如果您想在帧 0 和帧 100 之间获得锁，请尝试首先使用绝对锁。如果绝对锁定不起作用，可能是因为帧 0 和帧 100 太不同。在这种情况下，如果从帧 0 到帧 100 有逐渐的空间进展，增量锁会有所帮助。

平滑 -这控制了平均运动的帧范围 模式 设置为 平滑 。默认值为 10。

锁定框架 -这将控制将被锁定到何时的帧 模式 设置为任一锁定模式。默认情况下，此值设置为 0。请注意，F_Steadiness 的帧 0 是帧 1 NukeX 。因此，如果您在查看器中查看帧 3，并希望将该帧用作锁定帧，则需要输入 2 作为 锁定框架 价值。

精度 -这控制了时间/准确性的权衡。这越高，分析就越慢，但是你有更好的结果的可能性。

过滤 -设置过滤质量。

• 低 -低质量，但渲染速度快。

• 介质 -使用双线性过滤器。这提供了良好的结果，并且比高过滤更快地渲染。

• 高 -使用 sinc 滤波器来插入像素，从而进行更清晰的修复。这给出了最好的结果，但需要更长的时间来处理。

反转 -如果设置，则在渲染过程中使用计算的四角销的倒数。 这在锁定模式下效果最好，并且可以用来跟踪静态锁定板回到镜头中。

自动刻度 -要平滑或移除相机运动，F_Steadiness 将转换并旋转源剪辑中的帧。这在图像周围留下黑色像素。Auto Scale 参数允许您通过向上缩放输出图像来填充边缘的黑色间隙。值为 1 (默认值) 将图像缩放到不可见黑色，而值为 0 将禁用缩放并使黑色边缘保持不变。自动比例使用必要的最小比例来移除边缘的黑色间隙，以尽可能多地保留细节。

四角销 -分析过程中计算的角销:

左下 -左下角的别针。

右下 -右下角的别针。

左上 -左上角的别针。

右上方 -右上角的别针。
