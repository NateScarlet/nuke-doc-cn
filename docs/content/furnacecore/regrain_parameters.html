参数

此插件的参数描述如下:

谷物类型 -选择谷物是否从 谷物 图像 ( 从谷物夹 ) 或来自一套标准股票。

• 预设库存 -谷物特征从提供的胶片中取样。提供 2 K 、 4 K 、光圈校正和非光圈校正库存。尽管包括标准库存，但建议您尽可能从您试图匹配的电影库存中取样。

• 从谷物夹 -样品和重建的谷物特征从 谷物 输入。

预设库存 -胶片库存谷物特征从何时取样 谷物类型 已设置为 预设库存 。供应普通富士和柯达库存。曝光可以在下方、上方，或者，如果留空，可以校正非光圈。大小是 2k 或 4k 像素。例如, FUJIF500 2 K 是指从富士胶片 500 胶片的 2k 平板上取样的颗粒特征进行无孔径校正。

谷物量 -调整纹理的亮度。将其设置为 0 表示不添加任何颗粒。

晶粒尺寸 -调整颗粒的大小。值越大，颗粒越大，越柔软。

输出 -设置是渲染结果还是测试图像。

• 结果 -显示 源 应用纹理的图像。

• 粒板 -显示应用谷物的测试图像。该测试图像是由输入图像的一部分组成的，该部分由从应用纹理的图像中采样的均匀纯色包围。如果内部区域与外部区域无法区分，那么你有一个很好的谷物样本。

分析 -这是一个按钮，将触发对输入剪辑的分析。如果分析谷物的输入剪辑已更改，但您不想移动分析区域以触发重新分析，请按下此按钮。每当输入剪辑更改时，您将在查看器中看到以下警告:

分析谷物的夹子已经改变了。按 Analyse 或移动分析区域以重新分析谷物。

分析区域 -一个选择框，标记用于分析纹理的图像区域 谷物类型 设置为 从谷物夹 。框架的这一部分必须不包含图像细节，只有纹理。

分析区域 BL -控制分析区域左下角的位置。

分析区域 TR -控制分析区域右上角的位置。

分析框架 -设置要从中采样谷物的框架。

纹理颜色空间 -这告诉 F_ReGrain 谷物样本剪辑是在什么颜色空间，当谷物起源。正确设置此设置可确保在采样之前，纹理不会被任何颜色空间转换夸大。

高级 -下的参数 高级 允许谷物的详细调整。

进程红色 -打开这个来处理红色通道。

红色金额 -设置红色通道中纹理的亮度。

红色尺寸 -调整红色通道中颗粒颗粒的大小。

过程绿色 -打开这个来处理绿色通道。

绿色量 -设置绿色通道中颗粒的亮度。

绿色尺寸 -调整绿色通道中颗粒颗粒的大小。

过程蓝色 -打开这个来处理蓝色通道。

蓝色量 -设置蓝色通道中纹理的亮度。

蓝色尺寸 -调整蓝色通道中颗粒颗粒的大小。

谷物响应 -下的参数 谷物响应 允许添加的颗粒量随着图像亮度的变化而变化。

应用谷物 -这控制了纹理样本重新应用于图像的颜色空间。

通常，这应该设置为 纹理颜色空间 以确保最准确的娱乐。

你可能想在一些外观或特殊情况下覆盖这个。

• Cineon / SRGB / 线性 -谷物样品将应用于指定的空间。

• 纹理颜色空间 -粮食样品将在太空集 纹理颜色空间 下拉菜单，在谷物样本部分。

低增益 -调整 lowlights 中谷物的增益。

中增益 -调整中间色调中颗粒的增益。

高增益 -在高光中调整谷物的增益。

使用采样响应 -打开此选项可将纹理的亮度缩放为纹理图像亮度的函数。

抽样反应组合 -此控制通常设置为 1。减少它会降低响应曲线的影响，直到在 0 时，它们对输出没有影响。此参数仅在 使用采样响应 是。

样品颗粒响应 -按此键从当前帧更新响应曲线。多个印刷机累积谷物响应，而不是每次重置。此参数仅在 使用采样响应 是。

复位谷物响应 -按此键将谷物曲线重置为默认 (平坦) 响应。此参数仅在 使用采样响应 是。

绘制响应 -覆盖查看器左下角的响应曲线。此参数仅在 使用采样响应 是。
