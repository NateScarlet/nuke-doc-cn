使用预采样的标准谷物类型

如果你没有图像来采样谷物，你也可以从各种预采样的标准谷物类型中进行选择。执行以下操作:

1。 选择 图像 > 阅读 加载要添加颗粒的序列。
2. 确保您的工作是完全分辨率，而不是代理分辨率。F_ReGrain 在代理解析时不起作用。(见 代理决议 )。
3. 选择 FurnaceCore > F_ReGrain .
4. 将您希望具有 grain 的序列连接到 F_ReGrain 的源 ( Src ) 输入。
5. 设置 谷物类型 到 预设库存 .
6. 尝试使用不同的谷物类型 预设库存 下拉菜单。包括 2 K 、 4 K 、光圈校正和非光圈校正股票。可以使用选择和调整单个颜色通道 高级 参数。
