制造一个场景

我们前面提到的场景节点创建了一个可以被摄像机和渲染节点看到多个物体的地方。如果你只有一个物体，你不需要一个场景节点，但是其中的乐趣在哪里？场景节点可以真正进入 Nuke 处理大量 3D 信息的能力，你应该知道如何使用它。

设置场景的步骤

1。 在 “设置 3D 系统” 节点树中，围绕 Nuke_sign.jpg 节点和 Card1 节点来选择它们。

2. 新闻 Ctrl + C (Mac 用户按 Cmd + C ) 复制选定的节点或选择 编辑> 复制 从右击菜单中。
3. 新闻 Ctrl + V (Mac 用户按 Cmd + V ) 或选择 编辑> 粘贴 插入节点的副本。新闻 Ctrl + V 或 Cmd + V 再次插入第二个副本，然后按如下所示排列节点。

4. 现在有多张卡片，你需要一个场景节点来创建一个空间，渲染节点可以一次看到所有卡片。
5. 单击节点图中的空白区域以取消选择所有节点。右键单击并选择 3D> 场景 插入 “场景 1” 节点。
6. 拖动 场景 1 节点到 Obj/scn 用于在之间插入场景 1 节点的连接器 Card1 和 ScanlineRender1 .

7. 连接 Obj/scn 连接器从 ScanlineRender1 节点到 场景 1 节点。连接每个 卡 节点到 场景 1 节点。

8。 双击上的 Card1 节点以打开其控制面板。在查看器中，您将看到第一张卡片的变换句柄。
9. 将卡移动到不同的位置并旋转。对于这个例子，你把它放在哪里并不重要。
10. 打开的控制面板 Card2 并将其卡片移动到场景中的不同位置。
11。 打开 Card3 控制面板和移动卡，也。

你可以切换到 2D 视图来查看结果，但是为什么不直接通过相机查看呢？在 “视图” 下拉菜单旁边，可以看到将 3D 视图锁定到相机的按钮。

12. 从视图下拉菜单中，选择 3D ( V ) 切换到三维透视视图。然后单击 “将视图锁定为 3D 相机” 按钮。

13。 关闭 “锁定选定相机的 3D 视图” 按钮。在本教程的其余部分，您不需要它。
