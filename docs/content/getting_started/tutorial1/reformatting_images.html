重新格式化图像

在中创建的元素 Nuke 脚本，如背景和渐变，自动继承全局格式，这就是你想要的这个项目。但是，导入的图像不符合项目设置，必须重新格式化。

使图像符合项目格式

1。 单击的 “读取” 节点 发动机.v01 剪辑以选择它。
2. 单击鼠标右键并选择 转换> 重新格式化 .
3. 对节点图中的所有读取节点重复步骤 1 和 2。
4. 将鼠标指针移动到查看器上，然后按键盘数字 ( 1 , 2 ,和 3 ) 在连接的图像之间切换。

每个图像现在都应该符合项目格式。

如果在项目设置中更改传递格式，则设置为 “root.format” 的所有元素也将更改为新的项目设置。如果您在将图像读入项目时忽略了重新格式化图像，则图像将保留其原始格式，独立于项目设置。
