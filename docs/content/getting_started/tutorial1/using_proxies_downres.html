使用代理和 “向下搜索”

代理是您打算创建的最终图像的低分辨率版本。对于许多合成任务，低分辨率版本可以帮助您更快地工作。然后，当您准备创建最终输出时，关闭代理模式并返回完整的 res 版本。

全分辨率。

代理解析。

Nuke 可以生成代理的飞行，根据 规模 或 格式 你的图像。您选择下的方法 编辑 > 项目设置 .

要切换在 “项目设置” 下定义的代理解析，请使用查看器上的 “代理” 按钮。或者，使用 “向下分辨率” 按钮来降低单个观众的显示分辨率。向下分辨率按钮在完全分辨率和代理模式下都有效。

激活代理模式

1。 单击节点图上的鼠标右键，然后选择 编辑 > 项目设置 .
2. 确保查看器窗口已打开。
3. 按下按键切换代理模式, Ctrl + P .

查看器中的标签表示您现在处于代理模式。

4. 将鼠标指针移动到查看器上，然后按几次加号 (+) 键以放大。
5. 新闻 Ctrl + P 几次在高分辨率和代理模式之间切换。
6. 继续之前，按 Ctrl + P 切换回全分辨率。

激活 “Down-Res”

1。 选择 1:4 从 “向下分辨率” 下拉菜单将显示分辨率更改为 25% 的全分辨率。

随着分辨率的降低, Nuke 计算和显示图像所需的时间更少。

2. 将 “down-res” 设置更改回 1:1 ,这是活动分辨率的 100%。

如果关闭代理模式，您应该恢复到完全分辨率。如果启用代理模式，则显示分辨率为代理的 100%。
