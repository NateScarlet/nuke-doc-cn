彩色校正图像

颜色校正和过滤器可以帮助您整合元素以获得更好的复合效果。在我们的示例中，您希望仅将校正限制在前景元素上，因此您将在合并节点之前插入一个颜色校正节点。

1。 选择 Reformat1 节点。然后，在节点图上单击鼠标右键并选择 颜色> 曝光 。这将插入 Exposure1 节点。

2. 假设你想调整红色通道的值。将鼠标指针移动到查看器窗口上，然后按 R 显示红色通道。
3. 在 Exposure1 控制面板中，取消选中 帮派 滑块。这允许您调整单个颜色通道。

4. 拖动 红色 滑块来调整颜色值。当你完成后，按 R 在浏览器上显示所有频道。

曝光节点按预期工作，但结果不太壮观。颜色变化太均匀了。如果有办法限制 -- 或者，事实上, 面具- 颜色校正，也许我们会看到一个更好的复合。嗯.
