用导航打开主题

你在这里:

在窗户内导航

节点图面板看起来很小，尤其是当节点树增长时。诚然，你已经知道如何调整大小和撕掉窗户，但是迟早你会耗尽展示房地产。是时候学习一些导航控件了，这些控件可以帮助您在 核武器 。尝试以下导航控件:

平移您的视图

• Windows/Linux: 按 Alt 键和鼠标左键，拖动鼠标指针穿过节点图。

• Mac OS X: 按下 选项 (alt) 键和鼠标左键，拖动鼠标指针穿过节点图。

拖动鼠标时，可以平移节点图的视图。

缩放或放大视图

• Windows/Linux: 按时 Alt 和鼠标中键，拖动鼠标指针穿过节点图。

• Mac OS X: 按时 选项 (alt) 和鼠标中键，拖动鼠标指针穿过节点图。

向右拖动，您将放大。向左拖动，您将缩小。

• 键盘放大/缩小。点击加号 ( ) 放大的关键。点击减号键 ( - ) 缩小。

使用节点图概述

• 当节点树超出窗口边界时，节点图的右下角会出现一个导航框。在框内拖动着色矩形，您将快速平移到节点树的另一个视图。

在窗口中构建视图

• 按字母 F 在键盘上，将节点树的全部内容放在节点图的边界内。

节点图的导航控件也在我们议程上的下一个窗口中工作，即查看器。
