在 Windows 上许可非商用 Nuke

订阅许可不同于传统的节点锁定或浮动许可证，因为单个许可证或权利在任何授权设备上有效，最多可达到权利的最大激活次数。

• 一个 权利 表示在一定数量的设备上运行一定时间的铸造产品的权利。

• 一个 授权设备 是可识别的设备，如桌面计算机，可以在其上激活权利。

例如，如果一个 权利 对于 Nuke 有五个激活，你可以使用 Nuke 在五个独立 授权设备 同时。如果你想激活另一个设备，你必须停用现有的设备，但是你可以随时激活和停用设备。

要开始使用 非商用 Nuke ,请按照以下步骤操作:

1。 使用我们网站上的有效电子邮件地址创建铸造账户, 在这里 .
2. 启动 Nuke 在非商业模式下描述 在 Windows 上启动 .

一个 许可 对话框显示，类似于常规许可。 非商用 Nuke 是免费的，但你的权利只包含两个激活。

3. 单击 授权设备 .
4. 输入您的帐户电子邮件地址和密码，然后单击 授权设备 .
5. 在您的主目录中创建订阅许可证: C: \ Users \ <username> \ FoundryLicensing \ <SystemID>

注意: 替换 <用户名> 和 <SystemID> 分别使用设备的当前用户和 MAC 地址。

许可证看起来像这样: C58edf7e-17ab-435b-8d8a-b3a9b347ab11.lic

6. 安装许可证后，单击 启动 开始使用 Nuke .

注意: 在 Windows 上，用户名包含非 ASCII 字符会出现已知问题，导致许可失败。如果显示类似以下内容的许可错误:
无法创建订阅许可证目录: C: \ Users \ zoe è hern á ndez \ FoundryLicensing \
尝试使用 license 环境变量将许可证目录更改为备用位置。请参阅 设置环境变量 欲了解更多信息。

7. 如果需要停用权利或取消对设备的授权，请导航到 帮助 > 许可证 然后，单击:

• 停用 Nuke 收回你的一项权利

• 取消授权设备 收回您在该设备上的现有铸造权利，并停止其他权利的运行，或者

• 取消授权所有设备 收回与您帐户相关的所有设备上现有的代工授权，并停止运行其他授权。
