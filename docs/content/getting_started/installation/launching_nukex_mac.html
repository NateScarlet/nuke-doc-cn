在 Mac OS X 上启动 NukeX

执行以下操作之一:

• 单击 NukeX Dock 图标。

• 打开 核武器 应用程序目录 (默认情况下, /应用程序/ 核武器 8.0v6 / )，然后双击 NukeX 图标 (或列表项)。

• 打开终端，导航到 /应用程序/ 核武器 8.0v6 / 核武器 8.0v6 。应用程序/内容/MacOS ,然后输入 。/ 核武器 8.0v6 -- Nukex .

注意: 在 Mac OS X 上，你不应该将 NukeX 捆绑包从 核武器 捆绑包位于，因为这可以防止 NukeX 正常工作。
