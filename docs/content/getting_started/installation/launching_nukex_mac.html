Launching NukeX on Mac OS X

Do one of the following:

• Click the NukeX dock icon.

• Open the Nuke application directory (by default, /Applications/Nuke8.0v6/), and double-click the NukeX icon (or list item).

• Open a terminal, navigate to /Applications/Nuke8.0v6/ Nuke8.0v6.app/Contents/MacOS, and enter ./Nuke8.0v6 --nukex.

NOTE:  On Mac OS X, you shouldn’t move the NukeX bundle from the folder where the Nuke bundle is located, as this prevents NukeX from working correctly.
