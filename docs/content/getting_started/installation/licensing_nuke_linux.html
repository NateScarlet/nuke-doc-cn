以下许可方法可用:

• 激活密钥 -激活密钥允许您在购买后的稍后时间，在您需要许可证的机器上激活和生成实际的产品许可证密钥。

它们为节点锁定许可证和浮动许可证提供，并在使用产品的 许可 使用 “激活产品” 页面对话框或联机: Www.thefoundry.co.uk/支持/许可/激活-产品/

• 节点锁定许可证 -这些可以用于在一台机器上许可应用程序。它们不在不同的机器上工作，如果你需要它们，你必须转移你的许可证。

节点锁定许可证 (有时称为未计数许可证) 不需要安装其他许可软件。

• 浮动许可证 -也称为计数许可证，使应用程序能够在任何联网的客户端机器上工作。浮动许可证放在服务器上，并被锁定到该服务器上的唯一号码。

服务器上的浮动许可证需要在服务器上安装额外的软件来管理许可证，并将其分发给需要许可证的客户端工作站。该软件被称为铸造许可工具 (FLT)，可以从我们的网站免费下载。

• 订阅许可证 -订阅许可不同于传统的节点锁定或浮动许可证，因为单个许可证或权利在任何授权设备上有效，直到权利的最大激活次数。

有关订阅许可的详细信息，请参阅 在 Linux 上许可 Nuke 非商业性 .

获取许可证

要获得许可证，您需要计算机的系统 ID (有时称为主机 ID 或 rlmhostid)。为了让你知道系统 id号是什么样子，这里有一个例子: 000ea641d7a1。

注意: 请记住，对于浮动许可证，您需要许可证服务器的系统 ID，而不是您打算运行应用程序的机器。

有许多方法可以找到机器的系统 ID:

• 在没有许可证的情况下启动应用程序，单击 地位 ,然后向下滚动错误报告，直到看到系统 ID。

• 免费下载铸造许可工具 (FLT) Www.thefoundry.co.uk/支持/许可/ 然后在终端 shell 中运行以下命令:

/Usr/local/foundry/LicensingTools7.0/bin/systemid

当您知道您的系统 ID 时，您可以申请铸造产品的许可证:

• 从铸造厂的销售部门 Sales@thefoundry.co.uk

• 从我们网站上的产品页面，如 Www.thefoundry.co.uk/products/nuke-product-family

• 通过在没有许可证的情况下启动应用程序并选择:

• 买 核武器 -直接打开 web 浏览器到我们的网站购买许可证。

• 试试 核武器 -显示 15 天的试用许可证下载屏幕。输入您的铸造厂帐户详细信息或创建一个新帐户，并按照屏幕上的说明接收试用许可证。

注意: 默认情况下，如果您安装了临时许可证，应用程序将在启动时显示一个对话框，提醒您剩余的天数。如果要禁用此行为，可以将 FN_DISABLE_LICENSE_DIALOG 环境变量设置为 1，以抑制关于即将到期的许可证的警告消息。请参阅 环境变量 欲了解更多信息。

安装许可证

在安装许可证之前启动应用程序时，许可对话框会显示一个错误，通知您没有可用的许可证。安装过程取决于您请求的许可证类型:

• 许可证文件 -如果您请求许可证文件，通常 铸造.lic ,此选项允许您浏览到文件位置并自动安装。请参阅 从磁盘安装许可证 欲了解更多信息。

• 激活密钥或许可证文本 -如果您通过电子邮件请求激活密钥或许可证，此选项允许您将密钥或许可证文本粘贴到许可对话框中，然后将许可证安装在正确的目录中。请参阅 安装激活密钥或许可证文本 欲了解更多信息。

• 浮动许可证 -如果您请求浮动许可证向多个客户端计算机提供许可证，则此选项允许您输入提供客户端许可证的服务器地址。

注意: 您必须在许可证服务器上安装浮动许可证和其他软件才能使用此选项。

请参阅 安装浮动许可证 欲了解更多信息。
