在 Mac OS X 上发射 Nuke 辅助

• 单击 “Nuke Assist dock” 图标。

• 打开 Nuke 应用程序目录 (默认情况下, /应用程序/ Nuke 8.0v6 / )，然后双击 Nuke 辅助 8.0v6 图标 (或列表项)。

• 使用终端，导航到 /应用程序/ Nuke 8.0v6 / Nuke 8.0v6 。应用程序/内容/MacOS ,然后输入

。/ Nuke 8.0v6 -- Nukeassist

注意: 在 Mac OS X 上，您不应该将 Nuke Assist 捆绑包从其原始安装文件夹中移动，因为这阻止了它正常工作。
