Launching Nuke Assist on Windows

Do one of the following:

• Double-click the Nuke Assist icon on the Desktop.

• Select Nuke Assist8.0v6 from Start > All Programs > The Foundry > Nuke8.0v6.

• Using a command prompt, navigate to the Nuke application directory (by default, /Program Files/Nuke8.0v6), and enter
nuke8.0v6 --nukeassist
