在 Linux 上卸载 Nuke

要卸载 Nuke 在 Linux 上，有几件事你需要做:

1。 导航到 /Usr/本地/ 并删除 Nuke 10.5v8 文件夹。
2. 删除、重命名或移动您的 。 Nuke 文件夹 (如果存在)。

的 。 Nuke 默认情况下，文件夹位于主目录中:

/Home/<登录名>/。nuke

3. 删除、重命名或移动默认情况下位于以下目录中的缓存文件:

/Var/tmp/nuke

注意: 如果使用 NUKE_TEMP_DIR 环境变量指定了备用目录，请清除这些文件以及默认位置。请参阅 Nuke 环境变量 欲了解更多信息。
