在 Windows 上安装

安装包安装 核武器 , NukeX ,和 核武器工作室 ,这些图标出现在您的安装文件夹中。

要安装 核武器 在 Windows 上，请参阅:

• 使用用户界面 (UI) 安装 ,或

• 在 Windows 上安装 .

使用用户界面 (UI) 安装

1。 下载正确的 。拉链 我们网站上的文件 Www.foundry.com .
2. 从解压缩安装程序 。拉链 文件，双击 。Exe 要安装的文件 核武器 .
3. 按照屏幕上的说明操作。默认情况下, 核武器 安装到 驱动器号 : \ 程序文件 \ 核武器 10.5v8 .
4. 我们网站上的 Nuke 插件页面打开，让您可以轻松访问大量可供使用的插件 核武器 .
5. 继续到 Windows 许可 .

注意: 在 Windows 上，如果您安装 核武器 要在多台计算机上运行网络驱动器，请确保在每台运行的计算机上安装了正确的 Microsoft 运行时库 核武器 。要做到这一点，运行 Vcredist_x64.exe 在每台机器上。可以在以下文件中找到相应的文件 VCRedist 文件夹中的子目录 核武器 已安装。

运行 核武器 如果不安装计算机上的库，可能会正常工作，尤其是许多系统 (如默认情况下的 Windows Vista) 已经有了这些库。如果图书馆不存在, 核武器 仍然可以正确运行，但是某些插件可能无法加载，并出现错误消息，例如 “由于应用程序配置不正确，此应用程序无法启动。重新安装应用程序可能会解决这个问题。"

请注意，这些库是在运行 核武器 安装程序，因此用户在本地机器上安装不需要担心这个问题。

从命令行安装

要安装 核武器 在命令行中，执行以下操作:

1。 下载正确的 。Exe 安装文件来自我们的网站 Www.foundry.com .
2. 要打开命令提示窗口，请选择 开始 > 所有程序 > 配件 > 命令提示 .
3. 使用 光盘 (更改目录) 命令移动到保存安装文件的目录。例如，如果将安装文件保存在 C: \ Temp ,使用以下命令并按 返回 :

Cd \ Temp

4. 要安装 核武器 ,执行以下操作之一:

• 要安装 核武器 到当前目录并显示安装对话框，键入不带文件扩展名的安装文件的名称，然后按 返回 :

核武器 10.5v8 Win-x86-release-64

• 要安装 核武器 到指定的目录并显示安装对话框，请使用 /Dir 安装选项:

核武器 10.5v8 -Win-x86-release-64/dir = “C: \ 核武器 ”

• 要安装 核武器 静默，以便安装程序不会提示您输入任何内容，而是显示进度条，请输入 /沉默 安装命令后:

核武器 10.5v8 -Win-x86-release-64/沉默

• 要安装 核武器 默默地，以便没有显示，输入 /Verysilent 安装命令后:

核武器 10.5v8 -Win-x86-release-64/verysilent

提示: 您可以添加 /MERGETASKS = "!desktopicon" 到 /Verysilent 标志，以避免为所有口味创建桌面图标 核武器 .

• 您还可以使用安装选项的组合:

核武器 10.5v8 -Win-x86-release-64/静音/dir = “C: \ 核武器 ”

注意: 通过使用 沉默 或 Verysilent 安装选项，您同意的条款 核武器 最终用户许可协议。要查看本协议，请参阅 附录 E: EULA 或者在标准的非静默模式下运行安装程序。
