在 Mac OS X 上安装

安装包安装 Nuke , NukeX ,和 Nuke 工作室 ,这些图标出现在您的安装文件夹中。

要安装 Nuke 在 Mac OS X 上，请参阅:

• 使用用户界面 (UI) 安装 ,或

• 从终端安装 .

使用用户界面 (UI) 安装

1。 下载正确的 。Dmg 我们网站上的安装文件 在 Www.foundry.com .
2. 双击上的 Nuke 10.5v8 -Mac-x86-release-64.dmg 档案，以提取 。Pkg 安装程序:

一个 。Pkg 文件已创建。

3. 双击上的 。Pkg 文件。
4. 按照屏幕上的说明安装 Nuke 。默认情况下, Nuke 安装到

/应用程序/ Nuke 10.5v8 .

5. 我们网站上的 Nuke 插件页面打开，让您可以轻松访问大量可供使用的插件 Nuke .
6. 继续到 在 Mac OS X 上启动 .

从终端安装

1。 下载正确的 。Dmg 安装文件来自我们的网站 Www.foundry.com .
2. 启动终端窗口。
3. 安装的 。Dmg 安装文件，使用 Hdiutil 附加 命令，包含保存安装文件的目录。例如，如果将安装文件保存在 构建/ Nuke ,使用以下命令:

Hdiutil 附加/构建/ Nuke / Nuke 10.5v8 Mac-x86-release-64.dmg

4. 输入以下命令:

Pushd/卷/ Nuke 10.5v8 /

这将目录路径存储在内存中，以便以后可以返回。

5. 要安装 Nuke ,使用以下命令:

Sudo 安装程序-pkg Nuke 10.5v8 -Mac-x86-release-64.pkg-目标 "/"

系统会提示您输入密码。

6. 输入以下命令:

Popd

这将更改 pushd 命令存储的目录。

7. 最后，使用以下命令弹出装入的磁盘映像:

Hdiutil 分离/卷/ Nuke 10.5v8

注意: 通过运行的静默安装 Nuke ,您同意最终用户许可协议的条款。要查看本协议，请参阅 附录 E: EULA 或者在标准的非静默模式下运行安装程序。
