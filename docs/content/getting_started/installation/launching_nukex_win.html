Launching NukeX on Windows

To run NukeX on Windows, do one of the following:

• Double-click the NukeX icon on the Desktop.

• Select Start > All Programs > The Foundry > Nuke8.0v6 > NukeX8.0v6.

• Using a command prompt, navigate to the Nuke application directory (by default, \Program Files\Nuke8.0v6), and enter Nuke8.0v6 --nukex.
