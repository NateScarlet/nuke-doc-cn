Launching NukeX on Linux

To launch NukeX on Linux, do one of the following:

• Double-click the NukeX icon on the Desktop.

• Open the Nuke application directory (by default, /usr/local/Nuke8.0v6) and double-click the NukeX icon (or list item).

• Using a terminal, navigate to the Nuke application directory (by default, /usr/local/Nuke), and enter ./Nuke8.0v6 --nukex.
