Getting Started

This chapter covers installation and licensing, the default Nuke environments and workspaces, and the Preferences dialog.

Organization of the Section

These are the topics covered by this section:

• Meet the Nuke Product Family describes the different products and modes of Nuke and explains some key concepts.

• Installation and Licensing explains how to install and license Nuke on Windows, Mac OS X, and Linux.

• Nuke Studio Environments runs through the differences between Nuke Studio's compositing and timeline environments. Timeline functionality is only available in Nuke Studio.

• Workspaces outlines the default panel layouts that ship with Nuke's different modes.

• Setting Preferences explains how set your interface preferences, but the preferences themselves are listed in Appendix A: Preferences because there are quite a few.

• Using the Compositing Environment explains the basics of Nuke's compositing environment.

• Using Nuke Studio's Timeline Environment explains the basics of Nuke Studio's timeline environment.
