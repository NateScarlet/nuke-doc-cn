在节点图内导航

随着脚本复杂性的增加，您需要能够快速平移到特定的节点集群。节点图提供了几种方法。

平移

可以使用鼠标或显示在节点图右下角的导航器图进行平移。

用鼠标平移

要使用鼠标平移，请按下鼠标中键并将鼠标指针拖动到工作区上 (也可以使用 Alt 拖动)。脚本随指针移动。

注意: 在许多 Linux windows 管理器中, Alt 默认情况下，key 用作鼠标修饰键。这可能会在 3D 应用程序中导致问题 Alt 用于 3D 环境中的相机导航。
可以使用键映射将鼠标修改器指定给另一个键，例如 (超级或 Windows 徽标) 键，但是方法会根据您使用的 Linux 风味而改变。有关更多信息，请参考特定 Linux 发行版的密钥映射文档。

用地图平移

如果您的脚本大于可见工作区，导航器地图会自动出现在右下角。

地图显示了整个脚本的微型视图，粉红色矩形显示了您在工作区边界内看到的脚本部分。

要使用地图进行平移，请拖动粉红色矩形以平移到脚本的不同视图。

当整个脚本包含在窗口边框中时，地图会自动消失。

提示: 导航地图可调整大小。拖动它的左上角，使其尽可能大或小。

放大

要放大，您可以:

• 将光标移动到要放大的区域上，并重复按加法键 ()，直到工作区以所需的比例显示脚本。

或

• 新闻 Alt 按住鼠标中键的同时向右拖动。

缩小

您可以通过执行以下操作之一缩小:

• 将鼠标指针移动到要缩小的区域上，并重复按减法键 (-)，直到工作区以所需的比例显示脚本。

或

• 新闻 Alt 按住鼠标中键并向左拖动。

注意: 在许多 Linux windows 管理器中, Alt 默认情况下，key 用作鼠标修饰键。这可能会在 3D 应用程序中导致问题 Alt 用于 3D 环境中的相机导航。

在可见工作区中拟合选定节点

要在节点图中适合选定的节点，请单击鼠标中键或按 F .

在可见工作区中拟合节点树

要使整个节点树适合节点图，请单击节点图以确保没有选择节点，然后单击鼠标中键或按 F .

为平移和缩放级别添加书签

要为平移和缩放级别添加书签:

1。 根据需要平移和/或缩放脚本。
2. 要保存当前的平移和缩放级别，请导航到 编辑 > 书签 > 保存位置 1 (或按 Ctrl / Cmd + F7 )。

您可以以类似的方式保存三个位置，使用 保存位置 2 , 保存位置 3 ,和 保存位置 4 (或按 Ctrl / Cmd F8/F9/F10 )。

保存是临时的，不会保存到脚本中。

3. 要稍后恢复保存的位置，请选择 编辑 > 书签 > 恢复位置 1 , 恢复位置 2 , 恢复位置 3 ,或 恢复位置 4 (或按 转变 + F7 / F8 / F9 / F10 )。

注意: 恢复保存的位置在不同的节点图中不起作用 -- 你不能在一个节点图中保存一个位置，然后在另一个节点图中恢复它。

书签节点

一个非常有用的功能 Nuke 节点，是它们在整个项目中作为跳转点的能力。

1。 双击要添加书签的节点以打开其属性。
2. 转到 节点 选项卡并选择 书签 复选框 (或按 Ctrl / Cmd + 转变 + B )。

这会将节点添加到书签列表中。

默认情况下，背景节点具有 书签 复选框启用。所有其他节点都禁用了它。

3. 选择 编辑 > 书签 > 跳转到书签节点 (或按 J 在节点图上) 以显示书签跳转到菜单。
4. 开始键入要导航到的节点的名称。

Nuke 显示所有匹配 启用书签标志的节点。它们根据节点名称列出，或者，如果 标签 存在时，标签后跟方括号中的节点名称。

5. 要选择要导航到的节点，您可以单击该节点，或者使用 向上 和 下来 箭头键，然后按 返回 .
搜索节点

Nuke 的 搜索 功能允许您在脚本中搜索节点并选择找到的任何匹配项。作为搜索字符串，您可以输入全部或部分节点名称。例如，您可以通过输入来搜索脚本中的所有模糊节点 Bl 作为搜索字符串。

您还可以使用正则表达式进行更复杂的搜索，例如搜索脚本中的所有读写节点。

搜索节点的步骤

1。 选择 编辑 > 搜索 (或按 / ) 打开搜索对话框。
2. 在搜索字段中，输入要搜索的字符串。

如果要搜索脚本中的所有节点，请输入 * (星号)。

如果要搜索脚本中的所有读取节点，请输入 阅读 .

如果要搜索所有读写节点，请输入以下表达式:

( * 阅读 * | * 写 * )

3. 单击 好 .

Nuke 在脚本中搜索节点，选择它找到的所有匹配项，并将节点图的焦点放在第一个节点上。如果再次执行相同的搜索，节点图视图将按顺序聚焦到下一个节点。例如，如果您的节点图中有一个 ColorBars 节点、一个 ColorWheel 节点和一个 ColorCorrect 节点, 连续三次搜索 “颜色” 会选择所有这些节点，并按字母顺序关注每个节点。

注意: 当您在 “搜索” 对话框中输入表达式时，请记住搜索字段仅使用正则表达式。在正则表达式中有特定含义的任何字符，如方括号 ( [ 和 ] ,需要前面的 反斜杠 字符 ( \ )。

清理节点图

有时，当您创建和删除节点时，您可能会发现节点树变得有点混乱。有一个快速的解决方案可以用来整理你的树:

1。 选择要在节点图中重新排列的所有节点。
2. 单击 编辑 > 节点 > Autoplace 。或者，按 L .

Autoplace 函数会自动以整齐的树形排列选定的节点。它通过确保节点不与其他节点重叠并保持输入连接水平来确定节点的合理位置。
