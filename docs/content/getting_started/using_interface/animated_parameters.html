摄影表和曲线编辑器中的动画参数

将关键帧添加到参数时, Nuke 工作室 两者都会自动向它们的摄影表添加标记，并在其 “曲线编辑器” 面板上绘制曲线。

摄影表提供了一种简单的方法来编辑关键帧以及重新定位、修剪和滑动剪辑。

曲线编辑器允许您调整关键帧及其之间的插值。

您可以添加关键帧、删除关键帧，甚至调整关键帧之间的插值，而无需查看摄影表或曲线编辑器。然而，随着动画变得越来越复杂，您可能会发现直接操纵其关键帧更容易编辑动画。有关如何执行此操作的更多信息，请参见 涂料表允许您访问您创建的所有关键帧，并提供编辑它们的简单方法。您还可以使用它来重新定位、修剪和滑动读取节点和某些时间节点。 和 曲线编辑器使您可以编辑曲线，而无需在 “属性” 窗格中实际输入信息。 .
