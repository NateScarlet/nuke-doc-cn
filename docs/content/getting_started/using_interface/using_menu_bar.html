使用菜单栏

的 核武器工作室 菜单栏包括以下功能:

菜单

函数

文件

用于磁盘操作的命令，包括加载、保存和导入脚本。

编辑

编辑功能、首选项和项目设置。

布局

恢复和保存布局。

观众

添加和连接查看器。

渲染

渲染输出。

缓存

缓存文件、清除磁盘缓存和清除缓冲区。

帮助

访问热键列表、用户文档、发行说明、培训资源、教程文件、 Nukepedia 、 Foundry 网站上的 Nuke 插件页面以及 核武器工作室 -相关电子邮件列表。

要快速浏览可用菜单并查看它们包含的内容，请单击菜单并将鼠标指针移动到其他菜单上。 核武器工作室 打开和关闭菜单。
