感兴趣区域 (ROI)

的 ROI 按钮允许您仅通过感兴趣区域 (您明确选择的图像的一部分) 启用渲染。这对于在重进程脚本中快速查看渲染结果非常有用。

要定义感兴趣的区域:

1。 点击 ROI 查看器控件中的按钮。ROI 覆盖出现。

2. 拖动可根据需要调整和移动 ROI 覆盖。

自由-绘制感兴趣的区域:

1。 在观众，按 Alt + W 一次 (不要按住钥匙)。ROI 按钮变为红色，但 ROI 覆盖不显示。这允许您自由绘制自己的 ROI，而不是调整默认覆盖。
2. 拖动字幕，在需要的地方绘制感兴趣的区域。

要清除感兴趣的区域:

1。 设置了感兴趣的区域后，您可以通过按 Alt + W 在观众。然后可以拖动新的选框来定义新的感兴趣区域。
2. 要关闭该功能并使用最近的更改更新整个查看器，请单击 ROI 再次按下按钮 (或按 转变 + W )。
