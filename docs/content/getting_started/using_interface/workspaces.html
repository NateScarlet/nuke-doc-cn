自定义工作空间

自定义窗格

您可以调整大小和分割窗格，为屏幕上的不同元素腾出更多空间。要调整窗格的大小，请将窗格的分隔线拖到新位置。

您可以通过单击内容菜单按钮来拆分窗格 在窗格的左上角，然后选择 垂直分割 或 水平分割 从打开的菜单中。

注意: 按下并按住 空格键 显示该窗格的右击菜单 (如果可用)。

移动工具栏

通过为工具栏添加新面板、隐藏面板名称和控件以及调整面板大小，可以将其移动到新位置。有关如何执行此操作的更多信息，请参见 添加选项卡 ,和 隐藏选项卡名称和控件 .

添加选项卡

当您无法将更多元素放入显示器时，您可以使用标签来节省空间。还可以使用选项卡将工具栏移动到新位置。

您可以通过单击内容菜单按钮来添加选项卡 在窗格的左上角，然后选择要添加的选项卡类型。例如，您可以添加 节点工具栏 , 节点图 , 新观众 ,或 脚本编辑器 。新选项卡将添加到现有选项卡的顶部。

要移动选项卡，请单击选项卡的名称，并将其拖到同一窗格或另一个窗格中的新位置。

您可以通过单击 X 在要关闭的选项卡的右上角。

注意: 关闭链接选项卡将关闭所有关联的选项卡。如果你坚持 Alt 关闭链接选项卡时，它只关闭该选项卡。

独奏标签

您可以通过右键单击选项卡名称或单击内容菜单，然后选择 个人标签 。这将自动关闭同一窗格中任何其他打开的选项卡，除了您选择单独打开的选项卡。

浮动窗户

您可以将选项卡和窗格变成浮动窗口，反之亦然。要将选项卡或窗格转换为浮动窗口，请单击 “内容” 菜单按钮 在要浮动的选项卡或窗格的左上角，然后选择 浮动选项卡 或 浮动窗格 。您也可以通过单击来浮动标签 Ctrl / Cmd 单击选项卡名称，或右键单击选项卡名称并选择 浮动选项卡 .

要将浮动窗口更改为选项卡或窗格，请单击浮动窗口中的选项卡或窗格名称，并将其拖动到希望停靠的位置。您可以通过单击 X 选项卡或窗格右上角的按钮。

最大化窗口

要制作窗口全屏，首先确保要制作全屏的窗口处于活动状态，然后按 Alt S 。这可能是主应用程序窗口或浮动查看器。使其全屏删除窗口边框。您还可以通过按空格键最大化选项卡和面板。

隐藏选项卡名称和控件

您可以隐藏选项卡的名称和控制按钮，因为在所有面板 (如工具栏面板) 中可能不需要它们。要隐藏选项卡上的名称和控件，请单击 “内容” 菜单按钮 在选项卡的左上角，并禁用 显示选项卡 .

通过将光标移动到窗格区域的顶部，直到窗格的顶部边缘突出显示，右键单击以打开 “内容” 菜单，可以再次显示选项卡上的名称和控件。并选择 显示选项卡 .

保存和加载工作区

自定义工作区并对其满意后，您可以通过选择 工作空间 > 保存工作空间… 然后要求你说出它的名字。保存后，您的自定义工作区将出现在 工作空间 现有默认工作区下的下拉列表。从下拉列表中选择它以加载它。

设置启动工作区

当你启动 Nuke ,它将在首选项中设置的工作区中打开。通过执行以下操作，可以将启动工作区更改为任何其他默认工作区或自定义工作区:

1。 打开 偏好 按对话框 转变 + S .
2. 在 行为 部分，选择 启动 .
3. 使用 启动工作区 下拉菜单选择所需的工作区 Nuke 启动时加载。
