书签节点

一个非常有用的功能 核武器工作室 节点是它们在整个项目中作为跳转点的能力。

1。 双击要添加书签的节点以打开其属性。
2. 转到 节点 标签和检查 书签 (或按 Ctrl / Cmd + 转变 + B )。

这会将节点添加到书签列表中。

默认情况下，背景节点具有 书签 复选框启用。所有其他节点都禁用了它。

3. 选择 编辑 > 书签 > 跳转到书签节点 (或按 J 在节点图上) 来提出 书签跳转到菜单。
4. 开始键入要导航到的节点的名称。

核武器工作室 显示所有匹配 启用书签标志的节点。它们根据节点名称列出，或者，如果 标签 存在时，标签后跟方括号中的节点名称。

5. 要选择要导航到的节点，请执行以下操作:

• 点击它，或者

• 滚动到它与 向上 和 下来 箭头键并按 返回 .
