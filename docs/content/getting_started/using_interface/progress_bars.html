Progress Bars

Nuke displays a progress bar for each active task it performs. By default, progress bars appear in a pop-up dialog, but you can also display them in the Progress panel. To do so, click on a content menu button and select Windows > Progress. This opens a Progress panel. The next time you get a progress bar, it appears in the panel. If you delete the panel, progress bars appear in a pop-up dialog again.

If you want to have the Progress panel appear in the same position in the future, you can save it as part of a workspace. For more information, see Workspaces.
