使用色样

通过执行以下操作，您可以使用色样从查看器中采样颜色:

1。 单击 色样 激活滴眼器工具。
2. 将光标移动到查看器上，移动到要采样的颜色上。

您可以根据需要放大和平移，直到要取样的区域清晰为止。

3. Ctrl / Cmd + 单击以采样颜色值，或 Ctrl / Cmd + Alt + 单击从节点的输入中采样颜色，同时查看其输出。

您可以通过按下来采样一个区域，而不仅仅是一个像素 转变 在采样颜色时拖动。当取样一个区域时, 核武器 计算区域的平均颜色。

如果您对采样的像素或区域不满意，您可以简单地重复采样过程，直到您对选定的像素或区域满意。

提示: 您可以通过 Ctrl / Cmd 在查看器中单击鼠标右键。

4. 当您对颜色样本满意时，再次单击色样。这将关闭滴管工具，现在显示您选择的样本。
使用面板颜色选择器

如果你 偏好 > 面板 > 控制面板 > 彩色面板 > 颜色选择器按钮打开 下拉设置为 面板颜色选择器 ,你可以显示的颜色内滑轮子 属性 面板，单击 颜色选择器 按钮 。您可以同时打开多个窗格内颜色选择器。

提示: 持有 Ctrl / Cmd 单击颜色选择器按钮，将交替颜色选择器打开到 偏好 对话框。

使用面板中的色轮

可以通过将色轮边缘的标记 (或色轮内部的标记) 拖动到所需的色调来调整色调。可以通过将滚轮内的标记拖动到所需的饱和度来调整饱和度。

提示: 你也可以 Ctrl / Cmd 点击色轮只影响色调，然后 转变 单击仅影响饱和度。

要调整颜色的值 (亮度)，请按住 Ctrl / Cmd 转变 然后向右或向左拖动。

默认情况下，色轮使用相对光标定位。这意味着光标位置可以从标记位置偏移，这允许非常精细的调整。您可以禁用此行为，并通过保持使用绝对光标定位 Alt 移动标记时。

注意: 如果任何红色、绿色或蓝色值高于 1，则滚轮内的标记变为红色。如果红色、绿色和蓝色通道的值都小于 0.1，标记将变为蓝色。如果至少有一个通道为负，至少有一个通道为正，则色调指示器变为红色。

使用浮动颜色选择器窗口

如果你 偏好 > 面板 > 控制面板 > 彩色面板 > 颜色选择器按钮打开 下拉设置为 浮动颜色选择器 ,你可以显示颜色滑块和车轮的浮动窗口中按一下 颜色选择器 按钮 .

TIP: Holding Ctrl / Cmd and clicking the color picker button opens the alternate color picker to the one specified in the Preferences dialog.

通过拖动窗口的一个角来调整窗口的大小，可以将浮动窗口更改为水平窗口。当它足够宽时，滑块会自动变成水平的。

• 从 TMI , HSV ,和 RGB 按钮，您可以选择要显示的滑块。

• 色轮有三种状态: 隐藏色轮、显示色轮和显示方形色轮。您可以通过简单地点击这些状态循环 色轮 按钮多次。

• 您可以通过点击选择显示的颜色色板 色样 按钮。

• 如果您希望滑块的背景显示滑块设置为当前位置时颜色的值，请单击 Dyn 按钮。

在浮动窗口中使用色轮

在浮动窗口中使用色轮与使用面板内色轮完全相同; 拖动色轮边缘上的标记来调整色调, 并在车轮内拖动标记以调整饱和度。

但是，您也可以在浮动窗口中平移和缩放色盘:

• 要平移，按 Alt 并将光标拖动到色轮上。

• 要缩放，请按 Alt 并用鼠标中键向左或向右拖动。

• 只需在色轮上单击鼠标中键，即可重置缩放和/或平移。

在浮动窗口中使用颜色滑块

在浮动颜色窗口中使用滑块与使用面板内滑块完全相同。

在浮动窗口中使用色样

当你对一种颜色感到满意时，你可以通过在你想保存它的样本上点击鼠标右键来保存它。这取代了原来的样本。您还可以拖放要保存在任何色板上的颜色来替换它。

滑块上方的矩形在右侧显示原始颜色，在左侧显示当前选定的颜色。当您将光标悬停在此上方时，会出现一个箭头，允许您通过单击矩形将一个复制到另一个。

打开另一个浮动颜色选择器窗口

您可以打开多个颜色选择器浮动窗口，按 Ctrl / Cmd Alt 单击另一个参数的颜色选择器按钮。

使用通道选择器

默认情况下，许多参数 核武器 自动为您分组频道。例如，如果您拖动 增益 在 ColorCorrect 节点的滑块中，您同时影响 R 、 G 和 B 通道 (假设您正在处理 RGB 层)。))您可以使用 通道选择器 按钮 显示和编辑单个通道值。频道选择器按钮上显示的数字表示可供编辑的频道数。

可以通过执行以下操作编辑单个通道的值:

1。 单击 通道选择器 按钮。

可用通道将显示每个通道的值字段。例如，当你按下 增益 - 通道选择器 按钮，出现四个通道值字段 (R 、 G 、 B 和 A)，因为通道选择器显示有四个可用的通道进行编辑。

2. 在通道值字段中输入所需的值。

当您对这些值满意后，只需再次单击通道选择器按钮来关闭并保存您的设置。

颜色滑块函数

RGBA 滑块

滑块

函数

红色滑块 ( R )

这允许您控制红色通道的值 (或者如果您正在处理除了 RGBA 之外的另一层，则控制层中的第一个通道)。

绿色滑块 ( G )

这允许您控制绿色通道的值 (或者如果您正在处理除了 RGBA 之外的另一层，则控制层中的第二个通道)。

蓝色滑块 ( B )

这允许您控制蓝色通道的值 (或者如果您正在处理除了 RGBA 之外的另一层，则控制层中的第三个通道)。

Alpha 滑块 ( 一个 )

这允许您控制 alpha 通道的值 (或者如果您正在处理除了 RGBA 之外的另一层，则控制层中的第四个通道)。

TMI 滑块

Slider

Function

温度滑块 ( T )

这允许您通过反向影响红色和蓝色值来控制表观色温 (假设您正在处理 RGBA 层)。

洋红色/绿色滑块 ( M )

这允许你控制绿色和品红色的混合。要添加更多的洋红色 (增加红色和蓝色通道的值，同时减少绿色通道的值)，请向上拖动。要添加更多绿色 (增加绿色通道的值，同时减少红色和蓝色通道)，请向下拖动。

强度滑块 ( 我 )

这允许您同时控制红色、绿色和蓝色通道值。要将所有通道的值增加相同的量，请向上拖动。要将所有通道的值减少相同的量，请向下拖动。

HSV 滑块

滑块

函数

色调滑块 ( H )

这允许您控制颜色在传统色轮上的位置 (例如，颜色是红色、黄色还是紫色)。

饱和滑块 ( S )

这允许您控制颜色的强度或纯度。

值滑块 ( V )

这允许您控制颜色的亮度 (红色、绿色和蓝色的最大值)。

注意: HSV 滑块仅在浮动颜色选择器窗口中可用。在面板颜色选择器中，可以使用色轮调整色调、饱和度和值。
