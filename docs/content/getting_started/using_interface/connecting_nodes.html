连接节点

在脚本中添加或粘贴节点时, Nuke 自动生成当前选定节点和新节点之间的管道。在构建脚本时，需要移动这些管道，或者在节点之间运行新管道。在 Nuke ,您可以通过在管道的后端 (没有箭头的末端) 拖动来进行此类修改。

断开节点

可以通过将连接箭头的头部或尾部拖动到工作区的空白区域，或者选择树中的较低节点并按 Ctrl / Cmd D .

重新连接节点

可以通过拖动连接箭头的头部或尾部并将其放在要连接的节点的中心来重新连接节点。

重新连接管道。

注意: Nuke 使用标签区分可能遇到合并节点的双输入 一个 和 B . 一个 引用前景元素，并且 B 到背景元素。 Nuke 总是从 一个 输入到 B 。这意味着，如果您以后决定禁用与 一个 输入，数据流保持流动，因为默认情况下，它使用 B 输入。

复制连接箭头

要复制连接箭头，请按住 转变 并将连接箭头拖动到要创建连接的节点的顶部。 Nuke 复制连接箭头，使原始连接保持不变。

在节点之间添加节点

要在两个连接的节点之间添加节点，请将该节点拖到已经连接的节点之间的空间中。当您这样做时，您会看到这两个节点之间的链接变得活跃。当这种情况发生时，只需释放您正在拖动的节点，它就会自动放置并连接在两个节点之间。

弯曲连接箭头

要弯曲连接箭头，请执行以下操作:

1。 选择要弯曲的连接器之前的节点。
2. 从工具栏中选择 其他 > 点 .

选定节点后会出现一个点，导致连接器发生弯曲。

3. 根据需要拖动点以重新定位折弯。

提示: 也可以通过按向现有连接添加点 Ctrl / Cmd 点击连接箭头上出现的黄色点。
