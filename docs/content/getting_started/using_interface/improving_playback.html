提高播放性能

中的逐帧更新 Nuke 工作室 的界面可能很昂贵，因此实时回放可能会受到影响。曲线编辑器、涂料表和 RotoPaint 或跟踪器节点是界面中更新较多区域的好例子。

的 播放期间优化查看器 在 RAM 缓存回放期间，查看器工具中的按钮会停止界面上其他地方的更新，有助于优化回放帧速率。

注意: 此按钮与查看器设置中的同名复选框联合使用 (按 S 在查看器中)，并适用于脚本中的所有查看器。

默认情况下，可以为中的所有查看器启用此功能 新脚本 使用 偏好 > 观众 > 播放期间优化查看器 复选框。

禁用时，默认模式下，所有接口元素都将根据需要更新。启用 播放期间优化查看器 影响以下接口区域，直到播放停止，此时整个接口都将更新:

• 属性面板控件 -在播放过程中，控件不会更新，它们会显示向前或向后按下播放时您所在帧的值。停止播放时，控件将更新以显示当前帧的值。

• 节点图 -汽车标签和邮票不更新。

• 曲线编辑器/摄影表 -黄色时间线指示器被隐藏，直到播放停止，此时它显示在当前帧。

您仍然可以正常操作和启用控制、曲线和关键帧 播放期间优化查看器 不会影响查看器中的句柄。
