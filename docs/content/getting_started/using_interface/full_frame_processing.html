全帧处理

默认情况下，当您在查看器中显示或播放序列时, 核武器工作室 只计算和缓存可见区域的扫描线。它不是缓存整个帧，而是您正在查看的扫描线的子集。例如，如果您已将缩放级别设置为 ÷ 4 , 核武器工作室 仅缓存组成帧的扫描线的 1/4。在很多情况下，这允许更快的回放，可能是你想要的。

但是，如果您播放一个序列，然后平移或更改缩放级别, 核武器工作室 必须计算它以前没有计算的扫描线。这可能不是你想要的，如果你正在执行的任务需要你不断平移或放大和缩小盘子 (如油漆和旋转)。如果是这样，你可以点击 全帧处理 查看器顶部的控件中的按钮强制 核武器工作室 处理图像中的每条扫描线。与默认模式相比，最初这可能需要稍长的时间，并且需要更多的查看器缓存空间，但一次 核武器工作室 缓存了所需的帧后，您可以在不破坏缓存或影响回放的情况下平移并更改缩放级别。
