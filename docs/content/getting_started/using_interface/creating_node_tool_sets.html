创建节点工具集

如果您发现自己重复创建同一组节点，则可以在 Nuke 工具栏。这允许您快速轻松地从 工具集 菜单。

工具集可以在艺术家之间共享，如果他们使用集中。核弹文件夹。这需要通过可以设置的 NUKE_PATH 环境变量来访问 更多信息 .

要创建工具集，请执行以下操作:

1。 在节点图中，选择任意数量的节点 (或仅选择一个节点)。该节点不都是联系在一起的。
2. 单击 ( 工具集 ) 在工具栏上，然后选择 创建 。也可以在节点图上单击鼠标右键，然后选择 工具集 > 创建 。的 创建 工具集 出现对话框。您也可以通过按来访问工具集 标签 并在搜索栏中搜索它们。
3. 在 工具集 菜单 下拉菜单，选择要放置新工具集的菜单。然后给它一个名字 菜单 项目 字段并单击 创建 .

默认情况下，您的新工具集位于 工具集 菜单，但是如果你想为工具集创建一个子文件夹，你可以通过在工具集名称之前指定文件夹名称，在 菜单 项目 字段。例如，输入 Roto / Basmicroto 将创建一个名为 Roto 的子文件夹 工具集 菜单，并在其中放置一个名为 basmicroto 的新工具集。

例如，有一些用于创建粒子的工具集内置于 Nuke 。这些也可以在 工具集 菜单。为了防止内置工具集和您创建的工具集之间的混淆，如果您碰巧创建了与内置工具集同名的工具集文件夹, Nuke 添加 [用户] 为了清楚起见，请在文件夹名称前面。

4. 要删除已创建的工具集，可以单击 工具集 > 删除 在工具栏中，选择要删除的工具集。
