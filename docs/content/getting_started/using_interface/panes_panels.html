窗格和面板

Nuke 的主窗口分为三个窗格: 节点图形/曲线编辑器/摄影表窗格、属性/脚本编辑器窗格和查看器窗格。

您可以将面板添加到这些窗格上，例如:

• 用于选择节点的工具栏

• 用于构建节点树的节点图 (也称为 dag)

• 用于编辑动画曲线的曲线编辑器

• 用于调整节点控件的属性箱

• 预览输出的查看器

• 显示进度条的进度面板

• 用于执行 Python 命令的脚本编辑器。

默认情况下，左下角有一个节点图形面板，左上角有一个查看器面板，右边有一个属性面板。

节点图是添加节点和构建节点树的位置。将节点添加到面板时，其 “属性” 面板将显示在右侧的 “属性” 框中。这是您可以调整节点以产生您所追求的效果的地方。要检查结果，可以在查看器中查看输出。

可以使用中描述的内容菜单打开更多面板 工具栏、菜单栏和内容菜单 。您可以在同一窗格上添加几个面板，并使用窗格顶部的标签页在它们之间切换。

标签面板

窗格分为窗格顶部的选项卡式面板。

要转到不同的选项卡，只需单击选项卡名称或使用循环浏览可用的选项卡 Ctrl/Cmd T .
