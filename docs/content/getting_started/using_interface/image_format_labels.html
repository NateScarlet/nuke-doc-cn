图像格式标签

像素值指示器显示指针下面的像素或采样像素或像素区域的信息。(您可以通过按从查看器中采样单个像素 Ctrl/Cmd 单击时，通过按从查看器中选择一个区域 Ctrl/Cmd 转变 拖动时，按节点输入的单个像素 Ctrl / Cmd Alt 单击时，按节点输入的区域 Ctrl / Cmd Alt 转变 拖动时。)从左到右，指标显示关于当前像素或样本的以下内容: 它的 x 和 y 位置; 它的红色、绿色、蓝色和 Alpha 值; 和其他值，具体取决于您从右侧的 “颜色类型” 菜单中选择的颜色类型。

格式指示器显示图像分辨率和边界框的大小。
