分离通道

默认情况下，许多参数 核武器工作室 自动为您提供帮派频道。例如，如果在 ColorCorrect 节点的增益滑块上拖动，则会同时影响 R 、 G 和 B 通道 (假设您正在处理 RGB 层)。但是，可以使用参数的 “通道选择器” 按钮来显示和编辑分配给各个通道的值。

编辑单个通道的值

1。 单击参数的通道选择器按钮。其表面上的数字可以让您知道有多少个频道可供编辑。将出现一系列输入字段 (每个通道各一个)。

2. 根据需要编辑任何显示输入字段中的值。

您也可以使用颜色滑块来编辑单个频道。请参阅 使用颜色控件 .
