显示单个通道

你可以按下 R , G , B ,和 一个 键盘上的键分别显示红色、绿色、蓝色和 alpha 通道。或者，您也可以从 RGB 左上角的下拉菜单。

再次按下其中一个通道键，切换并显示所有通道。

提示: 如果你按 转变 选择频道时，您的选择仅影响查看器节点的当前活动输入。这样，您可以从查看器的不同输入中显示不同的频道。例如，当设置键控时，可以查看一个输入的 RBG 通道和另一个输入的 alpha 通道，并在两者之间切换。要实现这一点，请执行以下操作:
1。创建一个带有几个输入的查看器。请参阅 连接查看器节点 .
2。通过在查看器上按下其数字 (例如 1) 来激活其中一个输入。
3.新闻 转变 并选择 RGB 从频道下拉菜单。
4。通过在查看器上按其数字来激活另一个输入 (例如，输入 2)。
5.新闻 转变 并选择 一个 从频道下拉菜单。
6。通过按数字或上下箭头键在输入之间切换。
