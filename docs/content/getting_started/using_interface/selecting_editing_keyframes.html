在摄影表中选择和编辑关键帧

核武器工作室 提供了在摄影表中选择和编辑关键帧的几种方法。

选择关键帧

• 要选择多个关键帧，请在关键帧视图中的关键帧上拖动选框。

• 要将关键帧添加到选择框或从选择框中删除它们，请按住 转变 并在一些选定的关键帧周围拖动另一个字幕。

• 要选择单个关键帧，请在关键帧视图中单击它。它的父关键帧和所有其他取决于它的关键帧也被选中。关键帧的帧编号显示在其旁边。

• 要选择一个节点中的所有关键帧，请单击层次视图中的节点名称。将选择该节点下的所有子关键帧。

• 如果从动画节点到当前活动的查看器有多个路径，则动画节点的关键帧将多次出现在摄影表中。在一个位置选择它们时，它们会变成橙色，而在其他位置相应的关键点会变成白色。


注意: 选择多次出现在摄影表中的关键点 (通过多个路径) 时，选框选择将只限制为一个路径。

移动和重新缩放关键帧

选择关键帧后，可以移动和重新缩放它们:

• 要移动关键帧，请在关键帧视图中拖动选区。

• 要向右或向左轻推选定的关键帧，请使用数字键盘上的箭头键。

• 要直接编辑关键帧的帧号，请双击关键帧旁边的帧号，并在 x 弹出的字段。

• 也可以使用摄影表底部的移动控件移动关键帧: 输入要移动选定关键帧的帧数，然后单击 移动 。您可以输入负值来向后移动关键帧。如果你不输入一个数字, 移动 字段显示上次移动的数量。

• 要重新缩放，请抓住字幕选择框的右侧或左侧并拖动它。盒子的另一边保持放置，关键帧根据你如何调整盒子比例重新调整。

• 如果从动画节点到当前活动的查看器有多个路径，则动画节点的关键帧将多次出现在摄影表中。调整一个位置的关键点也会更新另一个位置的关键点。


注意: 您不能将关键帧移动过其他关键帧。

复制和粘贴摄影表中的关键帧

可以在摄影表中复制和粘贴关键帧。使用 Ctrl/Cmd C 复制关键帧或选择关键帧的快捷方式，以及 Ctrl/Cmd V 将它们粘贴到涂料表的另一个位置。

当使用 Ctrl / Cmd V 热键，粘贴关键帧相对于当前帧指示器的位置。换句话说，如果您在 1 、 10 和 20 有关键帧，并且在将选择粘贴到 21 帧之前复制了此选择，则新的关键帧将设置为 21 、 30 帧, 和 40。

要将关键帧粘贴到与源关键帧相同的帧上，请选择要接收关键帧的控件，在摄影表上单击鼠标右键，然后选择 编辑 > 粘贴 > 粘贴绝对 .

删除和添加摄影表中的关键帧

可以删除摄影表中的关键帧。要删除关键帧，只需选择它并按 删除 / 退格 按钮或右键单击关键帧并选择 编辑 > 擦除 .

您还可以在摄影表中添加关键帧，以添加当前没有的动画。按添加关键帧 Ctrl / Cmd Alt 点击涂料表中的空白位置。将创建一个新的关键帧，您可以根据需要对其进行调整。
