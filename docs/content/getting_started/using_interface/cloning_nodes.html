克隆节点

可以将节点直接克隆到节点图中，也可以复制克隆节点，以便在脚本中将它们粘贴到其他位置。克隆的节点继承其父节点的值，但与复制的节点不同，它们还维护与其父节点值的活动链接。如果更改已克隆的节点的值，克隆将自动继承这些更改。

克隆有助于在多个元素之间保持一致的设置。例如，您可以使用克隆将相同的胶片纹理设置应用于在同一股票上拍摄的一系列元素。如果您需要更改设置，这些更改将自动在整个脚本中产生涟漪。

注意: 您不能克隆小发明。这适用于在您的设施中创建的 gizmos 和 Nuke 工作室 默认工具栏，实际上是小控件，如 LightWrap 节点、 Grain 节点和 IBK 节点。 有关 gizmos 的更多信息，请参见 配置 Nuke 第一章。

克隆节点

要克隆节点，请执行以下操作:

1。 选择要克隆的一个或多个节点。
2. 单击 编辑 > 克隆 .

Nuke 工作室 克隆节点，同时保持与父节点的活动链接。克隆与原始节点具有相同的名称，并用将克隆连接到其父节点的橙色行表示。

通过执行以下操作，可以将节点复制为克隆，并将其粘贴到不同的位置:

1. Select the node or nodes you want to clone.
2. 单击 编辑 > 复制为克隆 (或按 Ctrl + K )。

取消链接节点

对于 declone 节点，请执行以下操作:

1。 选择要删除的一个或多个节点。
2. 单击 编辑 > Declone (或按 Alt + 转变 + K )。

Nuke 工作室 删除选定节点的克隆状态。
