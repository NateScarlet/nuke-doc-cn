自定义节点显示

您可以修改特定节点显示的颜色、名称和注释。这样做可以让其他艺术家更容易解读你剧本的意图。例如，您可以将与键控相关的所有节点涂成绿色。

修改节点显示特征

要修改节点的显示特征，请执行以下操作:

1。 双击节点图中的节点，以在 属性 窗格。
2. 单击 节点 中的对话框顶部的选项卡 属性 窗格。显示其属性:

在节点的属性中，您还可以执行以下操作:

• 选择 隐藏输入 复选框以隐藏节点的传入管道。这可以增强大型脚本的可读性。

• 选择 邮票 复选框在其表面上显示节点输出的缩略图渲染。你也可以按 Alt P 在 属性 窗格或节点上，切换邮票。

如果希望邮票显示固定框架 (而不是更新以匹配当前框架)，请在 静态框架 字段。要做到这一点，你还需要按 转变 S 要打开 偏好 对话框，转到 面板 > 节点图 ,并确保 邮票模式 下拉设置为 静态框架 。请注意，如果您使用的帧号超出了节点的帧范围，它将被夹到该范围中的第一个或最后一个帧。

提示: 你也可以 Nuke 工作室 根据节点的功能自动为节点设置颜色代码。请参阅 可用的首选项设置 .
