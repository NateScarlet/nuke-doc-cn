搜索节点

Nuke 工作室 的搜索功能允许您在脚本中搜索节点，并选择找到的任何匹配项。作为搜索字符串，您可以输入全部或部分节点名称。例如，您可以通过输入来搜索脚本中的所有模糊节点 Bl 作为搜索字符串。

使用正则表达式，您还可以执行更复杂的搜索，例如搜索脚本中的所有读写节点。

搜索节点的步骤

1。 选择 编辑 > 搜索 (或按 / ) 打开搜索对话框。
2. 在搜索字段中，输入要搜索的字符串。

如果要搜索脚本中的所有节点，请输入 * (星号)。

如果要搜索脚本中的所有读取节点，请输入 阅读 .

如果要搜索所有读写节点，请输入以下表达式:

(* 读 * | * 写 *)

3. 单击 好 .

Nuke 工作室 在脚本中搜索节点，选择它找到的所有匹配项，并将节点图的焦点放在第一个节点上。如果再次执行相同的搜索，节点图视图将按顺序聚焦到下一个节点。例如，如果您的节点图中有一个 ColorBars 节点、一个 ColorWheel 节点和一个 ColorCorrect 节点, 连续三次搜索 “颜色” 会选择所有这些节点，并按字母顺序关注每个节点。

注意: 当您在 “搜索” 对话框中输入表达式时，请记住搜索字段仅使用正则表达式。在正则表达式中有特定含义的任何字符，如 [ 和 ] ,需要前面的 \ 性格。
