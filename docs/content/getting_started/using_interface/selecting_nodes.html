选择节点

核武器工作室 提供了许多选择节点的选项。选择的节点是突出的颜色是确定的 偏好 对话框。请参阅 可用的首选项设置 欲了解更多信息。

选择的类型

如何进行选择

选择单个节点

要选择一个节点，只需点击它。

选择多个节点

要选择多个节点，可以按 转变 单击要选择的每个节点，或在工作区中单击并拖动以在要选择的节点周围绘制选框。

选择所有上游节点

新闻 Ctrl / Cmd 拖动节点时。 核武器工作室 选择向选定节点提供数据的所有节点。你也可以 Ctrl / Cmd 转变 单击以选择更多节点，而不清除当前选择。

选择节点树中的所有节点

单击节点图中的节点并选择 编辑 > 选择连接的节点 (或按 Ctrl / Cmd Alt 一个 )。这将选择节点树中的所有节点，无论它们是来自当前节点的上游还是下游。未选择任何其他节点树中的节点。

选择脚本中的所有节点

选择 编辑 > 选择全部 (或按 Ctrl / Cmd 一个 )。

按名称选择节点

1。 选择 编辑 > 搜索 ,或按正斜杠 ( / )。

出现一个对话框。

2. 键入包含在要选择的节点名称中的字母数字字符串。单击 好 .

提示: 键入上述字母数字搜索字符串时，可以使用星号 ( * ) 和问号 ( ? ) 作为通配符。星号代表多个字母数字字符。问号只代表一个字符。

反转选择

选择 编辑 > 反转选择 .
