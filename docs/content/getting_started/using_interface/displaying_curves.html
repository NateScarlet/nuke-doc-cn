显示曲线

有几种方法可以在曲线编辑器中显示动画曲线并放大和缩小它们。

显示动画曲线的步骤

1。 单击动画按钮 在要查看其曲线的参数旁边。
2. 选择 曲线编辑器 。“曲线编辑器” 面板将显示，焦点位于选定参数的曲线上。

垂直或 y 轴表示参数的值。

水平或 x 轴表示时间 (以帧为单位)。

Nuke 工作室 的曲线编辑器面板。

在编辑器中显示曲线

1。 在左侧的参数树中，单击 + 和 - 必要时，标志着层次结构的扩展和折叠。
2. 单击参数的名称，使其曲线成为编辑器的焦点。要同时聚焦多条曲线，请执行以下操作:

• 保持 转变 点击曲线创建一个连续的列表,

OR

• 保持 Ctrl/Cmd 然后单击单个曲线，将它们添加到选择中。

3. 要为每个通道显示单独的曲线，请在节点的 “属性” 面板中分隔相关控件的通道。

左侧的参数树允许您聚焦脚本中的任何曲线。

提示: 如果要查看动画曲线下的 AudioRead 节点，可以通过右击并选择 视图 > 音频 > 绘制 风格 > 下面 . 有关音频选项的更多信息，请参见 Nuke 中的音频 第二章 Nuke 工作室 用户指南。

在编辑器中启用或禁用曲线

在左侧的参数树中，单击 和 - 必要时，标志着层次结构的扩展和折叠。

在编辑器中放大或缩小

1。 单击要放大或缩小的区域。
2. 按 + 按钮放大，或按-按钮缩小,

OR

使用鼠标滚轮向上滚动以放大，或向下滚动以缩小。

提示: 根据您的鼠标首选项，您可以缩放到曲线编辑器中的自定义区域，在编辑器上单击鼠标中键，然后拖动以选择带有选框的区域。释放鼠标按钮后，编辑器将缩放以适合编辑器中的选定区域。

在编辑器中平移

按住鼠标中键并在编辑器上拖动。您也可以使用 Alt 拖动。

重置缩放和平移

1。 在曲线编辑器上单击鼠标右键。
2. 从打开的菜单中，选择 视图 > 框架全部 .

或

新闻 一个 在编辑上。

OR

单击鼠标中键 (取决于您的鼠标首选项)。

Nuke 工作室 在编辑器中使曲线居中，重置缩放。

在编辑器中使曲线的一部分居中

1。 在编辑器中选择要居中的点。
2. 在编辑器上单击鼠标右键，然后选择 视图 > 选定帧 (或按 F 在编辑上)。

Nuke 工作室 在编辑器中使曲线的选定部分居中。如果没有选择点, Nuke 工作室 使选定曲线或所有曲线居中。
