使用动画参数

“动画” 菜单允许您设置关键帧、删除关键帧以及对动画参数的曲线执行其他编辑操作。

设置关键帧

要设置关键帧，请执行以下操作:

1。 使用查看器导航到要放置关键点的帧。
2. 单击动画按钮 在要设置动画的参数旁边。
3. 选择 设置密钥 从下拉菜单。参数的输入字段变为蓝色，表示已插入关键帧。 Nuke 工作室 切换到自动键模式: 当您在另一个帧更改参数值时，它会自动为您插入另一个关键帧。

还可以为节点中的所有控件设置键。为此，请在节点的 “属性” 面板上单击鼠标右键，然后选择 在所有旋钮上设置键 .

4. 导航到要放置关键点的下一帧。
5. 使用输入字段、常规滑块或颜色滑块编辑参数值。你改变价值的那一刻, Nuke 工作室 创建关键帧。
6. 根据需要继续添加关键帧。
7. 使用查看器的洗涤器预览结果。

删除关键帧

要删除单个关键帧:

1。 使用查看器的下一个编辑/关键帧和上一个编辑/关键帧按钮 导航到要删除的关键帧。请注意，擦洗栏指示带有蓝色标记的关键帧。
2. 单击动画按钮 .
3. 选择 删除密钥 从下拉菜单。

Nuke 工作室 删除关键帧。

要从参数中删除所有关键帧:

1。 单击动画按钮。
2. 选择 没有动画 从下拉菜单。出现确认对话框。单击 是 .

Nuke 工作室 从参数中删除所有关键帧，并将静态值设置为与当前帧的值匹配。
