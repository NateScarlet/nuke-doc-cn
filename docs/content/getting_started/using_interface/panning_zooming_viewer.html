平移和缩放查看器窗口

Nuke 工作室 提供了几种方法，包括一些有用的键盘快捷键，来关注您需要查看的查看器区域。

平移框架

要平移框架，请按住鼠标中键并在显示屏上拖动 (也可以使用 Alt 拖动)。框架跟随鼠标指针。

注意: 在 Linux 上, Alt 拖动可能无法按预期工作。这是由于 Gnome 上的默认窗口功能。要绕过它，你可以使用 ( 窗户 ) 键代替 Alt 当平移。
或者，您可以更改 Gnome 的窗口首选项来解决问题:
1.选择 应用 > 偏好 > 窗户 要打开 窗口首选项 对话框。
2.下 移动键 ,选择 超级 (“或 Windows 徽标”) .
你现在应该可以用 Alt 拖动。

重新居中框架

您可以通过单击鼠标中键或按 F .

放大和缩小

要放大帧:

1。 将指针移动到要缩放的显示区域上。
2. 按住鼠标中键和鼠标左键的同时拖动鼠标。

OR

按下加号按钮 ( ) 重复，直到框架达到所需的比例。

OR

选择 放大 从缩放下拉菜单 在右上角。

要从框架中缩小:

1。 将指针移动到要缩放的显示区域上。
2. Drag the mouse while pressing the middle mouse button and the left mouse button.

OR

按减号按钮 ( - ) 重复，直到帧以所需的比例显示。

OR

选择 缩小 从缩放下拉菜单 在右上角。

还原缩放到 100%

您可以通过按将缩放恢复到 100% Ctrl / Cmd + 1 .
