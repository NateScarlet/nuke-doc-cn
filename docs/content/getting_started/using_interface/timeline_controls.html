时间轴控制

沿时间线拖动橙色标记，以快速提示到特定帧。当前帧的编号出现在 当前 时间线上方的字段。您还可以通过直接在此字段中键入帧的编号来提示帧。

默认情况下, Nuke 工作室 自动调整每个查看器窗口的时间轴，以显示在项目设置中定义的帧范围。如果没有定义帧范围，则读取的第一个图像的帧范围将用作全局帧范围。

Viewer timeline 控件也有一个帧范围源下拉菜单，您可以使用该菜单定义时间线获取其帧范围的位置。您可以将此菜单设置为 全球 , 输入 ,或 自定义 . 全球 是上述默认设置。

要让观众调整时间线以显示当前输入剪辑的 “输入” 和 “输出” 帧，请选择 输入 从框架范围源下拉菜单。剪辑中第一帧的编号显示在时间线的左端，最后一帧的编号显示在右端。如果更改查看器的输入，时间轴上的帧范围将相应调整。

要手动调整当前查看器窗口的帧范围，请在时间轴上平移和缩放，直到看到所需的帧范围和 自定义 在 “帧范围源” 下拉菜单中处于选中状态。 Alt 拖动到平移，然后 MMB 拖动以放大。也可以使用鼠标滚轮放大或缩小时间线。要重置缩放，请在时间轴上按下鼠标中键。

要调整当前查看器窗口的播放范围, Ctrl 拖动 (Mac 用户) Cmd 将时间线上的红色播放范围标记拖到新的 “输入” 和 “输出” 帧，如下所示, 或单击 “帧范围锁定” 按钮，并在 “播放范围” 字段中输入新的播放范围。
