Disabling and Deleting Nodes

Nuke Studio allows you you to disable, re-enabe, and delete nodes in your Node Graph.

Action Instructions
Disable node

To disable nodes:

1.   Select the node or nodes you want to disable.
2.   Select Edit > Node > Disable/Enable (or press D).

Nuke Studio disbales the nodeâ€™s effect on the data stream.
Re-enable node

To re-enable nodes:

1.   Select the node or nodes you want to re-enable.
2.   Select Edit > Node > Disable/Enable (or press D).
Delete nodes

To delete nodes:

1.   Select the node or nodes you want to delete.
2.   Select Edit > Delete (or press Delete).

Nuke Studio removes the node(s) from the script.
