工具栏、菜单栏和内容菜单

工具栏位于左边的浏览器中, 合成环境 。它由许多菜单图标组成。不同的节点根据它们的功能在这些图标下分组。使用工具栏将节点添加到节点图中。

在菜单栏中位于顶部 Nuke 窗口 (在所有工作区中)。它的菜单，如 文件 或 编辑 菜单中，您可以执行与项目或脚本、查看器或编辑相关的更一般的操作，而不是执行某些单个节点。

除了工具栏和菜单栏之外，您还应该熟悉内容菜单。它们是每个窗格左上角的灰色方格框。如果单击该框，将打开一个菜单，如下图所示。您可以使用菜单中的选项自定义工作区。

最后，为了更快地工作，您可以在不同的面板上单击鼠标右键，以显示一个菜单，其中包含与该特定面板相关的选项。
