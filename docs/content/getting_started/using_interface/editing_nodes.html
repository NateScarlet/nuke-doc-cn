编辑节点

要在节点图中复制、粘贴和执行其他编辑功能，可以使用标准编辑键 (例如, Ctrl / Cmd C 要复制，并 Ctrl / Cmd V 粘贴)。可以将节点复制到文件或内存。复制的节点继承其父节点的值，但这些值与克隆节点中的值不同，不会主动链接。复制的节点允许您为原始节点和副本指定不同的值。

粘贴节点时, 核武器工作室 自动将它们连接到粘贴操作之前选定的节点。如果不想连接任何东西，请单击工作区的空白区域，在粘贴之前取消选择任何选定的节点。

编辑类型

如何预制它

将节点复制到剪贴板

要将节点复制到剪贴板:

1。 选择要复制的一个或多个节点。
2. 选择 编辑 > 复制 (或按 Ctrl / Cmd + C )。

将节点复制到文件

要将节点复制到文件:

1。 选择要复制的一个或多个节点。
2. 选择 文件 > 导出 Comp 节点.
3. 导航到要将节点存储为脚本的目录。
4. 在路径的末尾键入脚本的名称，然后键入扩展 。Nk .

切割节点

要切割节点:

1。 选择要剪切的一个或多个节点。
2. 选择 编辑 > 切 (或按 Ctrl / Cmd + X )。

核武器工作室 从脚本中删除节点，并将节点写入剪贴板。

从剪贴板粘贴节点

要从剪贴板粘贴节点:

1。 选择希望粘贴节点跟随的节点。
2. 选择 编辑 > 粘贴 (或按 Ctrl / Cmd + V )。

核武器工作室 将节点添加到脚本中，将它们连接到选定的节点。

从文件加载节点

要从文件加载节点:

1。 选择希望加载的节点跟随的节点。
2. 选择 文件 > 导入脚本 .
3. 导航到存储脚本的目录。
4. 选择脚本，然后单击 导入 .

核武器工作室 将文件描述的节点添加到选定节点。
