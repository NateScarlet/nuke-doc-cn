使用摄影表界面

要显示摄影表，请单击 涂料表 标签。如果看不到选项卡，请右键单击节点图标题栏并选择 涂料表 .

在涂料表中，您可以看到:

• 层次视图 在层次视图中单击鼠标右键，查看节点和动画控件的层次列表。如果在列表中看不到节点，请尝试打开其属性面板。

提示: 通过在层次视图中单击节点，可以打开节点的 “属性” 面板。

• 读取节点 -任何打开属性面板的读取节点在涂料表中显示为灰色条。

• 时间节点 -任何 AppendClip 、 FrameRange 、重定时、时间偏移、时间扭曲和时间片节点都打开了它们的属性面板，显示为绿色条。

• 关键帧 -您的关键帧在关键帧网格中显示为灰色标记。如果进入最低层次，还可以查看在立体或多视图项目中为不同视图设置的关键帧。

• 当前帧指示器 -当前帧指示器显示为橙色线。

• 项目中的第一帧和最后一帧 -项目帧范围中的第一帧和最后一帧 (如项目设置中定义的) 显示为灰线。
