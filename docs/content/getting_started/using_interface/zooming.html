缩放

您可以通过几种方式放大或缩小脚本。

要放大

将鼠标指针移动到要放大的区域上，并重复按加号键 ()，直到工作区以所需的比例显示脚本。

OR

新闻 Alt 按住鼠标中键的同时向右拖动。

要缩小

将鼠标指针移动到要缩小的区域上，然后反复按减号键 (-)，直到工作区以所需的比例显示脚本。

OR

新闻 Alt 按住鼠标中键向左拖动。

注意: 在 Linux 上, Alt 中间拖动可以缩放整个 Nuke 工作室 窗口而不是节点图。这是 Gnome 的默认功能。要绕过它，你可以使用 ( 窗户 ) 键代替 Alt 放大时。
或者，您可以更改 Gnome 的窗口首选项来解决问题:
1.选择 应用 > 偏好 > 窗户 打开窗口 偏好 对话框。
2.下 移动键 ,选择 超级 (“或 Windows 徽标”) .
您现在应该能够放大和缩小节点图 Alt 中间阻力。

在节点图中适合选定节点

要在节点图中适合选定的节点，请单击鼠标中键或按 F .

在节点图中适合节点树

要使整个节点树适合节点图，请单击节点图以确保没有选择节点，然后单击鼠标中键或按 F .
