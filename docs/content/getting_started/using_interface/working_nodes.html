使用节点

节点是任何复合材料的基本构建块。您可以通过插入和连接节点来创建新的合成脚本，以形成操作网络。这些操作连接并允许您操纵图像。

注意: 并非所有产品版本都支持所有节点。有关支持的节点的运行情况，请参见 Nuke 产品 .

添加节点

可以使用工具栏、选项卡菜单或右击菜单添加节点。添加节点时, Nuke 自动将其连接到当前选定的节点。

使用工具栏

1。 通过单击新节点来选择要跟随的现有节点。
2. 单击工具栏上的图标，然后从显示的菜单中选择节点。例如，如果要添加模糊节点，请单击 过滤器 图标并选择 模糊 .

注意: 您可以按菜单图标上的鼠标中键来重复该菜单中使用的最后一项。例如，如果您首先从 过滤器 菜单中，您可以添加另一个模糊节点，只需按下鼠标中键 过滤器 图标。

使用标签菜单

1. Select the existing node that you want the new node to follow by clicking on it.
2. 按下 标签 键入并开始键入要创建的节点的名称。

这将打开一个提示，显示匹配列表。

3. 要从列表中选择要添加的节点，您可以单击该节点，或者使用 向上 和 下来 箭头键并按 返回 .

注意: 要添加使用此方法创建的最后一个节点，只需按 标签 然后 返回 .

使用右击菜单

要使用右击菜单添加节点，请执行以下操作:

1。 在您希望新节点跟随的现有节点上单击鼠标右键。
2. 从打开的菜单中，选择要添加的节点。

注意: 还可以使用键盘快捷键添加节点。工具栏中的大多数菜单都在相关项目旁边包含相关键盘快捷键的注释。

添加节点树的新分支

要在节点树的新分支中添加节点，请执行以下操作:

1. Select the existing node that you want the new node to follow by clicking on it.

2. 按住 转变 并使用工具栏、选项卡菜单或右键单击菜单创建节点。要使用选项卡菜单在新分支中添加节点，请按 标签 先键，然后按住 转变 选择新节点时。

节点将添加到节点树的新分支中的选定节点之后。

编辑节点

要在节点图中复制、粘贴和执行其他编辑功能，可以使用标准编辑键 (例如, Ctrl / Cmd C 要复制，并 Ctrl / Cmd V 粘贴)。可以将节点复制到文件或内存。复制的节点继承其父节点的值，但这些值与克隆节点中的值不同，不会主动链接。复制的节点允许您为原始节点和副本指定不同的值。

粘贴节点时, Nuke 自动将它们连接到粘贴操作之前选定的节点。如果不想连接任何东西，请单击工作区的空白区域，在粘贴之前取消选择任何选定的节点。

克隆节点

可以将节点直接克隆到节点图中，也可以复制克隆节点，以便在脚本中将它们粘贴到其他位置。克隆的节点继承其父节点的值，但与复制的节点不同，它们还维护与其父节点值的活动链接。如果更改已克隆的节点的值，克隆将自动继承这些更改。

克隆有助于在多个元素之间保持一致的设置。例如，您可以使用克隆将相同的胶片纹理设置应用于在同一股票上拍摄的一系列元素。如果您需要更改设置，这些更改将自动在整个脚本中产生涟漪。

注意: 您不能克隆小发明。这适用于在您的设施中创建的 gizmos 和 Nuke 默认工具栏，实际上是小控件，如 LightWrap 节点、 Grain 节点和 IBK 节点。有关 gizmos 的更多信息，请参见 配置 Nuke 第一章。

克隆节点

要克隆节点，请执行以下操作:

1。 选择要克隆的一个或多个节点。
2. 单击 编辑 > 克隆 .

Nuke 克隆节点，同时保持与父节点的活动链接。克隆与原始节点具有相同的名称，并用将克隆连接到其父节点的橙色行表示。

复制节点作为克隆

通过执行以下操作，可以将节点复制为克隆，并将其粘贴到不同的位置:

1. Select the node or nodes you want to clone.
2. 单击 编辑 > 复制为克隆 (或按 Ctrl + K )。

对于 declone 节点，请执行以下操作:

Nuke 删除选定节点的克隆状态。

创建节点工具集

如果您发现自己重复创建同一组节点，则可以在 Nuke 工具栏。这允许您快速轻松地从 工具集 菜单。

工具集可以在艺术家之间共享，如果他们使用集中。核弹文件夹。这需要通过可以设置的 NUKE_PATH 环境变量来访问 (请参见 配置 Nuke 更多信息)。

要创建工具集，请执行以下操作:

1。 在节点图中，选择任意数量的节点 (或仅选择一个节点)。该节点不都是联系在一起的。
2. 单击 ( 工具集 ) 在工具栏上，然后选择 创建 。也可以在节点图上单击鼠标右键，然后选择 工具集 > 创建 。的 创建 工具集 出现对话框。您也可以通过按来访问工具集 标签 并在搜索栏中搜索它们。
3. 在 工具集 菜单 下拉菜单，选择要放置新工具集的菜单。然后给它一个名字 菜单 项目 字段并单击 创建 .

默认情况下，您的新工具集位于 工具集 菜单，但是如果你想为工具集创建一个子文件夹，你可以通过在工具集名称之前指定文件夹名称，在 菜单 项目 字段。例如，输入 Roto / Basmicroto 将创建一个名为 Roto 的子文件夹 工具集 菜单，并在其中放置一个名为 basmicroto 的新工具集。

例如，有一些用于创建粒子的工具集内置于 Nuke 。这些也可以在 工具集 菜单。为了防止内置工具集和您创建的工具集之间的混淆，如果您碰巧创建了与内置工具集同名的工具集文件夹, Nuke 添加 [用户] 为了清楚起见，请在文件夹名称前面。

4. 要删除已创建的工具集，可以单击 工具集 > 删除 在工具栏中，选择要删除的工具集。
