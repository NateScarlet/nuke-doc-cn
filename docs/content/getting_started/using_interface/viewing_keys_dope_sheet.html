查看摄影表中的关键点

要查看摄影表中的关键点，需要先创建一些读取节点、时间节点或关键帧。摄影表显示:

• 所有打开属性面板的读取节点,

• 任何 AppendClip 、 FrameRange 、重定时、时间偏移、时间扭曲和时间片节点都打开了它们的属性面板，以及

• 在其上设置关键帧并打开其属性面板的任何节点。

提示: 要强制上述节点在摄影表中显示，即使它们的属性没有打开，请检查 总是在涂料表中 上的框 节点 节点属性面板的选项卡或在节点图中选择节点并按 Alt D .

提示: 如果要在关键帧下查看 AudioRead 节点，可以通过右击并选择 视图 > 音频 > 绘制 风格 > 下面 . 有关音频选项的更多信息，请参见 Nuke 中的音频 第二章 Nuke 工作室 用户指南。

摄影表和时间节点

如果在正在设置动画的节点下游有一个 AppendClip 、帧范围、重定时、时间偏移、时间扭曲或时间片节点，则摄影表将在当前活动查看器的上下文中显示您的关键帧。每当您更改查看器输入时，摄影表中的键都会相应移动，以反映它们在查看器中的实际效果:

• 如果在及时移动关键点的节点之前查看关键点，您会在它们的原始时间看到它们。

• 如果在时间操作后查看关键点，您会看到它们被时间操作移动。

摄影表中的多个节点分支

如果从动画节点到当前活动的查看器有多个路径，则动画节点的关键帧将多次出现在摄影表中。例如，在下面的脚本中，从动画模糊节点到查看器有两条路径:

1。 模糊> 合并> 查看器和
2. 模糊> 时间偏移> 合并> 查看器。

因此，模糊节点的关键帧在摄影表中显示两次。调整一个位置的关键点也会更新另一个位置的关键点。
