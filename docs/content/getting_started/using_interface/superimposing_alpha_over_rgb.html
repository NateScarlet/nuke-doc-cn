将图像的 Alpha 通道叠加到其 RGB 通道上

当您阅读了具有 alpha 通道的图像时，您可以将 alpha 通道显示为图像的红色、绿色和蓝色通道顶部的红色覆盖。

要在图像的 RGB 通道上显示图像的 Alpha 通道:

1。 选择 图像 > 阅读 在图像中阅读。
2. 将查看器节点连接到读取节点。

默认情况下, 核武器工作室 在查看器中显示红色、绿色和蓝色通道。

3. 单击查看器以确保它是当前活动的面板。
4. 新闻 M .

核武器工作室 将图像的 alpha 通道显示为 RGB 通道顶部的红色覆盖。

5. 要返回到 RGB 显示屏，请按 M 再次。
