合成观众

查看器节点在查看器面板中显示任何连接的进程节点的渲染输出，并且不编辑任何数据。查看器节点允许您快速为参数分配正确的值，因为它们允许您在上下文中编辑，也就是说, 在脚本中上游编辑给定节点的参数，同时查看下游更改的效果。

您可以在脚本中放置任意多个查看器节点，这允许您同时查看多个输出。您还可以将多达十个进程节点的输出管道传输到单个查看器节点，然后循环显示各种显示。这允许您通过给定的效果轻松比较处理前后的图像。

注意: 查看器可以显示的最大图像大小为 64 k x 4 k (或其他分辨率下的等效总像素数)。不过，如果你想使用最大图像大小，请确保你有足够的 RAM 内存。

添加查看器节点

查看器具有显示在节点图中的相应节点。这些节点不为渲染生成输出; 它们只生成显示数据。您可以连接查看器节点，如中所述 使用节点 。实际上，使用下面描述的查看器键盘快捷键会更快。

要添加查看器节点:

1。 选择要查看其输出的节点。
2. 执行以下操作之一:

• 使用菜单栏，选择 观众 > 创建新查看器 .

• 使用工具栏，选择 图像 > 观众 .

• 使用键盘快捷键，按 Ctrl / Cmd 我 .

Nuke 将查看器节点连接到您在步骤 1 中选择的节点，并在 “查看器” 面板中显示该节点的输出。您还可以插入一个查看器节点，并通过简单地按一下来设置其第一个连接 1 在节点图上。

连接查看器节点

将查看器节点添加到脚本后，只需选择进程节点，然后按任意数字键，就可以快速将任何进程节点的输出传递给它。这样，将输出管道连接到每个查看器节点上可用的十个输入端口之一 ( 0 键代表第十个插槽)。

切换视图

如果一个查看器节点有多个输入，如上所示, 您可以按向上或向下箭头键快速循环浏览视图 (您的光标需要在查看器窗口中)。要查看特定节点，请按数字键 ( 1 、 2 、 3.0 ) 对应于要查看其内容的管道编号。

平移和缩放查看器窗口

Nuke 提供了几种方法，包括一些有用的键盘快捷键，来关注您需要查看的查看器区域。

平移框架

要平移框架，请按住鼠标中键并在显示屏上拖动 (也可以使用 Alt 拖动)。框架跟随鼠标指针。

注意: 在许多 Linux windows 管理器中, Alt 默认情况下，key 用作鼠标修饰键。这可能会在 3D 应用程序中导致问题 Alt 用于 3D 环境中的相机导航。
可以使用键映射将鼠标修改器指定给另一个键，例如 (超级或 Windows 徽标) 键，但是方法会根据您使用的 Linux 风味而改变。有关更多信息，请参考特定 Linux 发行版的密钥映射文档。

重新居中框架

您可以通过单击鼠标中键或按 F .

放大和缩小

要放大帧:

1。 将指针移动到要缩放的显示区域上。
2. 按住鼠标中键和鼠标左键的同时拖动鼠标。

OR

按下加号按钮 ( ) 重复，直到框架达到所需的比例。

OR

选择 放大 从右上角的缩放下拉菜单。

要从框架中缩小:

1。 将指针移动到要缩放的显示区域上。
2. Drag the mouse while pressing the middle mouse button and the left mouse button.

OR

按减号按钮 ( - ) 重复，直到帧以所需的比例显示。

OR

选择 缩小 从右上角的缩放下拉菜单。

还原缩放到 100%

您可以通过按将缩放恢复到 100% Ctrl / Cmd + 1 .

显示/隐藏浮动查看器

您可以使用方便的键盘快捷键隐藏和显示浮动查看器。

要隐藏或显示浮动查看器，请按 ' (反向键)。

显示/隐藏查看器工具栏

要隐藏或显示顶部工具栏，请按 { .

要隐藏或显示底部工具栏，请按 } .

锁定查看器缩放级别

您可以选择锁定查看器的缩放级别，这样当您在不同大小的输入之间切换时，它不会改变。在查看器上单击鼠标右键，然后选择 防止自动缩放 在保持所有输入的相同缩放级别 (打开) 和根据屏幕上的图像尺寸 (关闭) 进行更改之间切换。或者，您也可以按 Alt Z 在查看器上，或切换 防止自动缩放 在查看器设置中。

在查看器中查看过扫描

如果您希望查看器显示过扫描，即超出帧的左/右或顶部/底部的任何像素，您可以检查 显示 过扫描 在查看器设置中框或选中查看器右击菜单中的选项。如果您的一些节点需要访问帧外，查看帧边缘之外的像素可能会很有用。你还可以控制过扫描显示指定的像素数量的 过扫描 字段。
