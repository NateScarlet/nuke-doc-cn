使用查看器复合显示模式

Wipe 控件提供了一个用于显示两个图像的分割屏幕的选项，这可以帮助您比较颜色校正、过滤和其他图像处理的前后版本。此控件还包括覆盖不同图像的显示合成选项。

显示比较擦除

1。 在脚本中选择一个节点，然后按 1 在查看器中显示其输出。
2. 选择要比较的节点，然后按 2 .

5. 拖动十字准星的手柄以调整擦拭:

• 拖动十字线中心以更改其位置。

• 拖动长手柄 (右侧) 旋转擦拭。

• 拖动 “arc” 手柄以交叉溶解第二幅图像。

6. 完成拆分屏幕后，选择 “无” ( - ) 从查看器复合下拉菜单中。
