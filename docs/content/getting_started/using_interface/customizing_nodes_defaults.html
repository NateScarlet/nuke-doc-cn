Customizing a Node’s Defaults with Node Presets

Sometimes the default control values on a node aren’t the ones you use most often. If you find yourself frequently adjusting many control defaults, you may want to save your customized default values as a node preset.

To Create a Node Preset:

1.   Set the controls in a node’s control panel to the values you want.
2.   Click the load node presets button on the control panel title bar and select Save as Preset.
3.   Give your preset a name and click Create.

Your new preset now appears as an option when you click the load node presets button and select Apply Preset.

Loading and Deleting a Preset

1.   To load a preset on a node, click the load node presets button and select your preset under Apply Preset. User presets are your own personal presets, and shared presets can also be used by other users.
2.   To delete a preset, click the load node presets button and select Delete Preset. In the dialog that displays, select the preset you want to delete from the dropdown menu, and click Delete.
