使用节点预设自定义节点的默认值

有时，节点上的默认控制值不是您最常使用的值。如果您发现自己经常调整许多控件默认值，您可能需要将自定义的默认值保存为节点预设。

要创建节点预设:

1。 将节点控制面板中的控件设置为所需的值。
2. 单击 加载节点预设 控制面板标题栏上的按钮，然后选择 另存为预设 .
3. 给你的预设一个名字，然后点击 创建 .

你的现在看起来像是一个选项，当你点击的预设 加载节点预设 按钮并选择 应用预设 .

加载和删除预设

1。 要在节点上加载预设，请单击 加载节点预设 按钮并在下选择预设 应用预设 。用户预设是您自己的个人预设，其他用户也可以使用共享预设。
2. 要删除预设，请单击 加载节点预设 按钮并选择 删除预设 。在显示的对话框中，从下拉菜单中选择要删除的预设，然后单击 删除 .
