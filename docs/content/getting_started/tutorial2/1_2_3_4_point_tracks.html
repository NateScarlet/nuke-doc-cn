一点，两点，三点，四

在我们进入第一个例子之前，让我们回顾几个跟踪概念。您可以使用中的跟踪器节点跟踪所需的任意多个特征或模式 Nuke 。你如何决定是跟踪一个、两个还是更多的功能？这取决于你想用数据做什么，以及你在结果中需要的准确性水平。以下是一些一般准则:

一个轨道:
仅 X 和 Y 位置。 		两条轨道:
X 、 Y 和 Z 旋转。

三条轨道:
X 、 Y 、 Z-旋转和缩放。

• 一点跟踪 -跟踪一个特征的水平 (x轴) 和垂直 (y轴) 位置，图像上的透视变化很小或没有变化。您可以应用此信息来移动组合中的其他元素，或者应用反转来稳定图像。

• 两点跟踪 跟踪两个特征的水平和垂直位置。相对于彼此的特征位置指示图像是顺时针旋转还是逆时针旋转 (z轴旋转)。在某些情况下，两个跟踪点也足以计算特征的缩放。

• 三点跟踪 跟踪三个特征的水平和垂直位置。通过一组额外的跟踪数据提供两点跟踪的所有好处，从而提高 z 旋转和缩放的准确性。

• 多点跟踪 + 同样，使用额外的跟踪数据集减少跟踪的所有好处。三点通常足以满足大多数 2D 跟踪需求，但是多点可以扭曲和匹配-将另一个元素移动到您跟踪的特征点，例如, 使用四个轨道和一个 CornerPin2D 节点。
