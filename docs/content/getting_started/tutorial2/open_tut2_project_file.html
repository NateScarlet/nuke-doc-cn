打开教程项目文件

在本教程中，您将从已经包含节点树的项目文件中工作。每个树都是为下面的例子设置的。

打开项目文件

1。 启动 Nuke 应用和选择 文件> 打开 Comp 从菜单栏。
2. 在文件浏览器中，导航到 Nuke _ 教程/跟踪/ 文件夹中，选择 Tracking_tutor.nk 项目文件并单击 打开 .
3. 它应该显示一些错误的节点。别担心!它找不到教程文件。因此，在做其他事情之前，你必须告诉这个脚本在哪里可以找到这些教程图像。双击 NoOp 节点在左上角的节点的图表。它被称为 Tutorial_Path。双击会在右侧显示属性面板。在教程项目目录中输入教程文件的路径。使用文件浏览器，因为这通常比输入文件更容易。然后，您应该看到教程图像出现。
4. 将鼠标指针移动到节点图形上，然后按 F 构建项目文件的全部内容。

此项目文件中的示例使用彩色框进行分组，称为 背景 ,并且每个包含一个节点树，用于后面的教程示例。

提示: 背景使您可以组织节点组，如此项目文件中显示的节点组。选择 其他 > 背景 从工具栏中拖动背景标题栏以移动它。拖动背景角以调整其大小。当您拖动背景的标题栏时，任何被背景边框包围的节点都会随背景移动。
