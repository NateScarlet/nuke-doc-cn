跟踪单个特征

在第一个示例中，您将学习如何设置跟踪锚点，然后跟踪单个功能，这是最基本的 2D 跟踪操作。在为一个功能实现坚实的跟踪后，您可以在此基础上构建并根据需要跟踪其他功能。

设置跟踪锚

1。 在项目工作区中 Tracking_tutor.nk 文件中，找到标记为 跟踪图像 .
2. 点击 伦敦眼 读取节点以选择它。
3. 使用 RAM 缓存或你选择的 Flipbook 播放几次，以查看镜头。
4. 查看图像中的特征，并在剪辑播放时注意移动的数量和方向。
5. 选择 变换 > 跟踪器 从工具栏将新的跟踪器节点附加到 伦敦眼 读取节点。

6. 将查看器节点连接到跟踪器节点，如图所示。
7. 在查看器中，将时间滑块摩擦到帧 1，以确保您处于拍摄的开始。
8。 双击跟踪器节点以显示其 “属性” 面板。
9. 单击 添加轨道 在查看器中创建跟踪锚点。
10. 如图所示，将跟踪锚拖到塔尖上。使用查看器中的缩放窗口来帮助您定位。

11。 单击跟踪锚点的图案框 (内框)，并调整其大小以包含特征。
12. 单击跟踪锚点的搜索区域 (外框)，并调整其大小以包含您认为该功能可能在帧之间移动的空间量。

大的搜索区域需要更多的计算时间，所以保持尽可能小。然而，当搜索区域为 太 小，功能可能会移动到盒子外面，你会失去轨道。如果你不确定要创建多大的搜索区域，回去查看你的图像的 flipbook。

自动跟踪与关键帧跟踪

放置跟踪锚后，你就可以计算你的轨迹了。 Nuke 的 Tracker 提供了两种计算方法:

• 自动跟踪 -非常适合简单的轨道，一旦你设置了跟踪锚，就没有额外的准备步骤。

• 关键帧跟踪 -一个更复杂的方法，要求您在序列上设置关键帧，以便计算轨迹。关键帧跟踪可能是更复杂的模式和运动的更好的选择。

使用自动跟踪

1。 启用 在轨道路径上显示错误 通过单击查看器工具中的交通灯图标。

此颜色代码跟踪显示其模式匹配错误值，绿色表示匹配良好，红色表示匹配不良。

2. 在 “跟踪器属性” 面板中，选择要在 轨道 列表。
3. 选择轨迹要输出的运动类型: 平移、旋转或缩放。在这个简单的例子中，你只需要选择 T Ranslation。
4. 在查看器的顶部，单击 轨道向前 用于生成轨迹的按钮。

如果您只需要某个帧范围，请使用 按钮并输入所需范围。

跟踪器完成后，您将看到沿曲线带有颜色编码点的轨迹曲线。

注意: 如果跟踪失败，请尝试调整模式或搜索框的大小，如中所述 设置跟踪锚 并重新追踪。

5. 使用 下一帧 和 上一帧 时间轴上的按钮可逐步通过轨迹以验证其准确性。

在这个例子中，轨道相当牢固。然而，一些图像不像这个图像那么容易跟踪。你有什么选择？你可以重新追踪 使用关键帧跟踪 如下所述，或者您可以在曲线编辑器中编辑现有轨迹，请参见 编辑轨迹数据 .

注意: 请记住，红色关键帧并不一定意味着跟踪结果不佳，只是跟踪器无法可靠地将模式从一个关键帧匹配到下一个关键帧。

使用关键帧跟踪

1。 在 “跟踪器属性” 面板中，选择要在 轨道 列表。
2. 选择轨迹要输出的运动类型: 平移、旋转或缩放。在这个简单的例子中，你只需要选择 T Ranslation。
3. 将该序列擦洗几帧，并通过将锚点拖动到图案的位置来调整跟踪锚点的位置。您可以使用 “缩放” 窗口微调定位。根据需要继续执行序列。

在每一帧，一个新的关键帧窗口被添加到缩放窗口的右侧。最接近当前播放头帧的关键帧以橙色突出显示。

在复杂或更大的运动区域周围放置更多的关键帧是一个好主意，在直接翻译上放置更少的关键帧。一般来说，更多的关键帧会产生更好的跟踪，但会牺牲处理时间。

4. 当你对关键帧满意时，确保在 轨道 列表，然后单击 跟踪所有关键帧。

您还可以通过单击来强制选定的轨迹在最近的两个关键帧之间重新计算 在查看器工具栏中。

5. 使用 下一帧 和 上一帧 时间轴上的按钮可逐步通过轨迹以验证其准确性。

NOTE: Bear in mind that a red keyframe doesn’t necessarily mean that the tracking result is poor, only that Tracker couldn’t reliably match the pattern from one keyframe to the next.

6. 继续到 编辑轨迹数据 如果赛道仍然不令人满意。

编辑轨迹数据

1。 在 “跟踪器属性” 面板中，选择要查看的轨迹。
2. 单击动画旁边的动画按钮 轨道 列表，然后选择 曲线编辑器 .

3. 单击 Track_x 和 Track_y 曲线编辑器树中的项目，您将看到跟踪过程中每个参数的记录值。

4. 同时选择压低的曲线 转变 键并单击 Track_x 和 Track_y 曲线，在 轨迹 1 .
5. 按下 F 在曲线编辑器中 “框” 曲线的关键帧。

• 要调整值，请选择点并向上或向下拖动。

• 要更改特定点的帧，请选择该帧，然后按住 Ctrl 键入并向左或向右拖动点。

6. 让我们假设你想通过对值应用过滤器来平滑曲线。在曲线的一段周围按鼠标左键的同时绘制一个选框，以选择多个点。

7. 单击鼠标右键以显示曲线编辑器选项的菜单。选择 编辑 > 过滤器 并进入 2 作为过滤关键帧的次数。

Nuke 基于周围点的值平均每个点的位置，平滑曲线。

8。 关闭 “曲线编辑器” 窗口，然后在查看器中播放结果。

这些是跟踪和编辑单个功能的结果的基础。在下一个例子中，我们将使它变得更加困难 -- 跟踪一个移出视野的特征。
