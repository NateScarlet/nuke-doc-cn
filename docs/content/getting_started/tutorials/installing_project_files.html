安装项目文件

在继续之前，从我们的网站下载教程项目文件，并将它们移动到您将要创建的目录，名为“ Nuke _ 教程 ”。这取决于你把你的教程文件放在哪里，但是根据你的操作系统，下面是我们的建议。无论你做什么，你都需要记住你把这些文件放在哪里。

提示: 如果您使用的是 Mac 或 Linux 系统，请在管理员权限下登录，以避免安装文件时出现权限问题。

创建教程目录 (Windows)

1。 在 Windows 桌面上，双击 我的电脑 图标打开文件浏览器。
2. 双击上 本地磁盘 (C :) 然后打开 C: \ 文档和设置 \ 所有用户 \ 应用程序数据 文件夹。
3. 在显示的目录上单击鼠标右键，然后选择 新建> 文件夹 .
4. 命名文件夹: Nuke _ 教程 .

创建教程目录 (Mac OS 或 Linux)

1。 打开外壳或终端窗口。
2. 在命令行中，输入 Mkdir ~/ Nuke _ 教程 在您的用户或 “主目录” 下创建教程目录。

下载并安装项目文件

1。 打开互联网浏览器，然后转到: Http://www.thefoundry.co.uk/support/user-guides .
2. 单击 NUKE 教程资产 将项目文件下载到本地计算机的链接。
3. 提取下载的文件并将其移动到 (或复制) Nuke 您之前创建的 _ 教程目录。

您现在可以开始第一个教程了 Nuke .
