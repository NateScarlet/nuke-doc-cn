全尺寸格式

当您在中启动新脚本时 Nuke ,您需要设置一个完整大小的格式。全尺寸格式确定从任何断开的节点输入中获得的图像大小。它还设置任何脚本生成元素的默认大小，如常量和颜色条。

注意: 完整大小的格式不会影响读取到脚本中的元素的格式。 Nuke 是独立于分辨率的，这意味着它尊重并保持图像的分辨率。它不会自动裁剪或填充元素以匹配项目设置。
如果希望读入的元素符合项目设置，可以使用 “重新格式化” 节点手动执行此操作。有关更多信息，请参见 重新格式化图像序列 .
如果使用代理格式，完整大小的格式也用于计算代理缩放。有关代理模式和代理格式的更多信息，请参见 代理模式 .

设置全尺寸格式的步骤

1。 如果尚未打开，请选择 编辑 > 项目 设置 (或按 S ) 以显示 项目设置 面板。
2. 从 全尺寸格式 下拉菜单中，选择渲染图像的最终输出分辨率。如果要使用的格式不在菜单中，请选择 新 。的 新格式 对话框显示。

在 名称 字段中，输入新格式的名称。

在 文件大小 字段，定义格式的宽度和高度。

如果愿意，还可以定义其他信息，如偏移和像素纵横比。

单击 好 保存格式。它现在出现在下拉菜单中，您可以在其中选择它。
