恢复备份

遇到系统或电源故障后，您可能希望恢复由 Nuke 的自动保存功能。

恢复备份的步骤

1。 重新启动 Nuke .

将打开一个对话框，询问您是否要恢复自动保存文件。

2. 单击 好 .

Nuke 打开备份文件供您使用。

有时您可能不想加载自动保存文件，而是需要加载最后保存的版本。例如，考虑这样一种情况: 您修改了脚本，但决定不提交更改，因此退出 Nuke 没有储蓄。在所有的可能性 Nuke 自动保存您的部分或全部更改，在这种情况下，如果您打开自动保存的文件，您不会按预期处理原始脚本。如果您不小心打开了一个自动保存的脚本，那么只需关闭它并重新加载上次保存的版本。

注意: Breakpad 崩溃报告允许您在不太可能发生崩溃的情况下向 Foundry 提交崩溃转储。

默认情况下，在 GUI 模式下启用崩溃报告，在终端模式下禁用崩溃报告。您可以切换报告和使用, -- 碰撞处理 1 或 0 命令行上的选项或通过将环境变量 NUKE_CRASH_HANDLING 设置为 1 或 0。

当在 GUI 模式下启用崩溃处理时，您可以使用 -- Nocrashprompt 命令行选项或通过将环境变量 NO_CRASH_PROMPT 设置为 0。

当启用崩溃处理时，终端模式中的崩溃会自动提交。
