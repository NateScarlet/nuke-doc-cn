定义帧范围

几个对话 Nuke ,如 要渲染的帧 和 框架到 flipbook 对话框，提示您输入帧范围。要定义一个，您需要输入一个开始帧和一个结束帧，用破折号分隔。例如，要将操作限制为帧 1 、 2 、 3 、 4 和 5，可以使用 1-5 作为帧范围。

下表为您提供了可以定义的帧范围的更多示例。

您可以使用上述方法来定义帧范围 Nuke 。除了对话框，它们还可以在命令行 (任何帧范围都应该在-F 开关之前) 和 Python 语句中使用。有关更多信息，请参见 命令行操作 以及 Nuke Python 文档 ( 帮助 > 文档 )。
