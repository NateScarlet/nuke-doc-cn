关闭脚本

要关闭脚本:

1。 选择 文件 > 关闭 Comp (或按 Ctrl / Cmd + W )。
2. 如果您对脚本进行了任何未保存的更改, Nuke 提示您选择是否保存它们。单击 是 保存您的更改或 没有 忽略它们。

Nuke 退出并重新启动，就像你再次运行它一样。它在启动时做的一切，除了显示闪屏。因此，您可以使用 Ctrl / Cmd W 作为清除内存的快速方法，请重新读取插件路径，并重新加载 Init.py 和 Menu.py 文件。(该 Init.py 和 Menu.py 文件是文件 Nuke 在启动时运行，可用于配置 Nuke 。如果您对这些文件进行了任何更改，您可能需要重新加载它们。)
