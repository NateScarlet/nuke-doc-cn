使用结果

一旦你对 DepthGenerator 的结果感到满意，有几种方法可以在 Nuke 和 NukeX :

• 如果你设置 深度输出 到 深度 1/Z , 您可以单击 创建卡片 在 DepthGenerator 中，控制创建一个组节点，该卡位于 3D 空间中，并根据深度通道置换。这允许您将深度可视化为 3D 空间中的曲面，从而更容易在实景拍摄中将元素 (如卡片) 放置在正确的深度。

深度通道。

• 如果将 DepthGenerator 设置为创建位置传递，则可以将位置传递提供给 PositionToPoints 节点 ( 3D > 几何 > PositionToPoints ) 创建三维点云。这是在 3D 空间中可视化深度的另一个好方法。

有关更多信息，请参见 使用 PositionToPoints 节点创建密集点云 .

• 如果使用深度生成器创建位置传递和法线传递，则可以将这些传递提供给重新点亮节点 ( 3D > 灯光 > 重新点亮 )。该节点根据位置和法线通道将着色器应用于其输入图像，从而创建逼真的照明。

有关更多信息，请参见 使用 3D 灯光重新显示 2D 图像 .

• 如果你想在镜头中引入雾和景深效果，你也可以使用深度贴图。在 Nuke 因此，ZDefocus 节点在其输入中需要深度贴图。

有关更多信息，请参见 仿真景深模糊 .
