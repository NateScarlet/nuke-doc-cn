提炼结果

要细化结果:

1。 调整 深度 细节 改变用于计算深度图的图像的分辨率。默认值 0.5 等于图像分辨率的一半。较低的值加快处理速度，并提供更平滑的结果。更高的值会获得更精细的细节，但也会增加处理时间。

深度细节 = 0.15 深度细节 = 0.85
2. 如果您发现深度图在平面图像区域中很嘈杂，请尝试增加 噪音 中的值 深度生成 参数组。这将设置计算深度贴图时输入素材中噪声深度生成器应忽略的量。值越高，深度贴图越平滑。

噪音 = 0 噪音 = 0.15
3. 如果你认为深度图应该仍然更平滑，你也可以增加 平滑度 。这不会影响深度计算，而是对结果应用智能模糊。

高平滑度可能会错过许多局部细节，但不太可能产生嘈杂的深度值。低平滑度集中在细节匹配上，即使生成的深度图参差不齐。

4. 的 强度 参数定义帧之间像素匹配的强度。通常可以将其保留为默认值。但是，如果深度图看起来不适合图像的某些边缘，则可以增加 强度 强制匹配细节被遗漏的地方。你也可以减少 强度 为了平滑深度图，但通常最好将其保持在相对较高的值，以确保算法匹配图像并正确计算深度。

强度 = 0.5 强度 = 1.5
5. 如果您对深度图中的对象边界不满意，请调整 锐度 。使用高值在对象之间产生不同的边界，或使用低值平滑边界。

锐度 = 0.05 锐度 = 0.95
6. 如有必要，使用 近剪辑平面 和 远剪辑平面 设置深度贴图中允许的最小值和最大值。超出此范围的所有深度值都被剪裁为这些值。
7. 要查看是否有深度计算不明确的区域，请检查 标记不良区域 .

任何不明确的区域都将在深度图中标记为非常大的值。如果 CameraTracker 无法使用相机数据计算所有像素的深度，例如，由于某些相机移动，这种区域可能会出现。

标记不良区域 已启用。
8。 一旦你设置了参数，这样你就对深度图满意了，查看法线通过 (如果你决定提前生成一个)。法线根据深度计算，法线方向由像素之间的深度变化定义。法线通道可能会很嘈杂，因为深度的微小变化会导致法线的巨大变化。要通过使用较低分辨率的深度贴图计算法线来平滑法线贴图，请减少 正常细节 。这集成了更广泛像素范围内的深度变化，减少了法线传递中的噪音。

默认值 0.25 会导致以四分分辨率计算法线传递。

正常细节 = 0.2 正常细节 = 0.8
9. 继续到 使用结果 .
