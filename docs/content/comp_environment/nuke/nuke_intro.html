该科的组织

这些是本节涵盖的主题:

• 管理脚本 描述创建、保存和加载脚本或 comps 的基础知识。

• 重新格式化元素 描述如何通过缩放、裁剪和像素方面调整来重新格式化图像。本章还介绍了边界框的使用。

• 渠道 显示如何使用管理图像数据 Nuke 1023 渠道的工作流程。

• 合并图像 教你如何将背景和前景元素分层，创建联系表，并将矩形从一个图像复制到另一个图像。

• 去噪 教你使用去噪节点从你的镜头中去除噪音。它使用空间过滤来去除噪声，而不会降低图像质量。

• 用灵长类动物键控 教你用蓝/绿幕抠像 Primatte 中 Nuke .

• 键控与键光 教你使用键控工具键控灯 Nuke .

• 使用 Ultimatte 键控 显示您在中使用最终键盘 Nuke .

• 使用 RotoPaint 显示如何使用 Nuke 这是 RotoPaint 节点。

• 跟踪和稳定 演示如何生成和编辑 2D 跟踪数据，以删除不需要的运动或将其应用于其他元素。

• 转换元素 介绍用于更改图像大小、位置和方向的工具，包括如何在 2D 和 3D 空间中平移、缩放、旋转和倾斜元素。本章还介绍了添加运动模糊。

• 扭曲图像 教你使用 GridWarp 和 SplineWarp 节点来扭曲和变形图像。

• 时间操作 解释如何应用基于时间的效果，如剪辑重新计时和运动模糊。本章还解释了如何执行编辑任务，如修剪和滑动。

• 使用颜色 解释了广泛的采样 Nuke 有许多颜色校正工具。

• 过滤和空间效应 处理应用过滤器，如卷积和模糊。

• 创建效果 描述如何在图像上创建效果，如星形滤镜效果。

• 分析和匹配剪辑 解释如何使用 CurveTool 节点分析和匹配图像序列。

• 3D 合成 教你如何创建和操作由对象、材料、灯光和相机组成的 3D 场景。

• 立体脚本 描述如何在复合材料立体的 Nuke .

• 深层合成 使用中设置的深层合成节点 Nuke .

• 使用文件元数据 描述如何使用 Nuke 用于处理图像中嵌入的信息的元数据节点。

• Nuke 中的音频 封面使用音频剪辑 Nuke .

• 预览和渲染 教你如何从脚本中写出图像序列，以便预览结果或创建最终元素。

• 组织脚本 旨在帮助您组织 Nuke 以清晰而有意义的方式编写脚本。

• 配置 Nuke 解释如何设置 Nuke 多个艺术家在同一个项目中工作。

• 表情 解释如何将表达式或脚本命令应用于 Nuke 参数。

• 脚本编辑器和 Python 带你通过使用 Nuke 用于执行 Python 命令的脚本编辑器。
