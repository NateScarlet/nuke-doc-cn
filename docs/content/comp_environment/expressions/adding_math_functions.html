向表达式添加数学函数

您可以将数学函数合并到参数中。例如，您可能会否定某个表达式，以便反转希望用于稳定元素的跟踪曲线 (此类表达式可能类似于以下内容: -(Transform1.translate.x) )。

您也可以依靠函数向表达式添加更复杂的数学运算。下表列出了您可以纳入的所有功能 核武器 表情。

Abs (x)

返回浮点数 x 的绝对值。

x

另见: 晶圆厂。

Acos (x)

计算 x 的弧余弦; 这是余弦为 x 的值。

如果 x 小于-1 或大于 1，acos 返回 nan (不是数字)。

另见: cos，cosh，asin，atan。

Asin (x)

计算 x 的弧正弦值; 这是其正弦值为 x 的值。

如果 x 小于-1 或大于 1，asin 返回 nan (不是数字)。

另见: sin，sinh，acos，atan。

阿坦 (x)

计算 x 的反正切，即切线为 x 的值。返回值介于-PI/2 和 PI/2 之间。

x

另见: tan，tanh，acos，asin，atan2。

Atan2 (x，y)

计算两个变量 x 和 y 的反正切。此函数用于计算两个向量之间的角度。

X，y

另见: sin，cos，tan，asin，acos，atan，hypot。

x

另见: 地板，trunc，rint。

x

x

FBm (x，y，z，八度，空白，增益)

分数布朗运动。这是对 noise () 的八度音阶调用的总和。对于它们中的每一个，输入点乘以 pow (空白，i)，结果乘以 pow (增益，i)。对于正常使用，缺口应大于 1，增益应小于 1。

X，y，z，八度，空白，增益

X，y

噪声 (x，y，z)

创建 3D Perlin 噪波值。这将在零上产生一个有符号范围中心。绝对最大范围为-1.0 到 1.0。这在所有整数上都产生零，所以你应该稍微旋转坐标 (将 y 和 z 的一小部分添加到 x 等)。) 如果你想用它来产生随机数。
