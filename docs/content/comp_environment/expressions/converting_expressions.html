在脚本语言之间转换表达式

根据您需要使用表达式的位置，您可能会发现您想要转换表达式。 核武器 表达式到 Tcl 表达式或将 Python 函数嵌入到 核武器 表达。不同的语言在不同的地方使用 核武器 :

• Python 可以在脚本编辑器中使用，在脚本命令中 ( 文件 > Comp 脚本命令 ) 和在脚本运行时 核武器 开始 (如 Init.py 和 Menu.py )。有关更多信息，请参见 核武器 Python 文档 ( 帮助 > 文档 )。

• Tcl 可以在大多数字符串旋钮中使用 (其中可以输入不仅仅是数字的文本)，在脚本命令对话框中 ( 文件 > Comp 脚本命令 )，打开一些兼容性启动脚本 (如 Init.tcl 和 格式.tcl )。

• 核武器 表达式可以在 “添加表达式” 对话框中使用，其中大多数旋钮位于 核武器 和表达式节点中的表达式输入字段。

可以使用以下函数将不同类型的表达式一起使用:

• Nuke.expression () 要使用 核武器 Python 代码中的表达式。

• 表达 要使用 核武器 Tcl 中的表达。

• Nuke.tcl () 在 Python 中运行 Tcl 代码。

• 巨蟒 在 Tcl 中运行 Python 代码。

• [] (方括号) 将 Tcl 嵌入到 核武器 表达式 (或字符串旋钮)。

• [蟒蛇 {.}] 将 Python 嵌入到 核武器 表达。

提示: 请注意，放置支架 ( {} ) 在将 Python 代码嵌入 Tcl 时，可能会让这个过程变得更容易一些，因为这阻止了 Tcl 在将其传递给 Python 解释器之前对 Python 代码执行自己的评估。例如: [python {“hello” “world”}]

提示: 请注意，默认情况下，“python” Tcl 命令计算一行代码并返回结果。如果要运行多行，请使用 “-exec” 选项 (例如，“python-exec”)。请参考 核武器 Tcl 脚本文档 ( 帮助 > 文档 > TCL 脚本 ) 了解更多信息。
