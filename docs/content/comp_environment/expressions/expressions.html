表情

本章旨在作为如何将表达式 (编程命令) 应用于 Nuke 参数。它解释了如何用表达式执行一些常见的任务 (例如，如何将一个参数的值链接到另一个参数), 并以一个表结束，该表包含了作为表达式一部分的所有函数。

快速启动

以下是工作流的快速概述:

1。 你进入 Nuke 表达式对话框中的表达式，您可以通过按等号 ( = ) 在参数上或通过右击参数并选择 添加表达式 .
2. 在 “表达式” 对话框中，输入引用其他参数值的文本 (创建链接表达式-请参见 链接表达式 ) 或将某种数学函数应用于当前值 (请参见 向表达式添加数学函数 )。前者的一个例子是 Parent.Transform1.rotate ,这表明该控件从父控件获取其值，转换节点的 旋转 控制。
3. 如有必要，您还可以在脚本语言 (即在 Nuke 表达式、 Tcl 和 Python)。请参阅 在脚本语言之间转换表达式 .
