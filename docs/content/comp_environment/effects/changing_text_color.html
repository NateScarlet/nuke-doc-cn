更改文本颜色

要更改文本颜色:

1。 在文本节点属性中，单击 颜色 标签。
2. 调整 颜色 参数或单击颜色选择器按钮 为文本选择颜色。
3. 如果要在文本中创建颜色渐变，请选择除 没有 从 坡道 :

• 线性 -渐变从一种颜色线性变化到另一种颜色。

• Smooth0 -渐变颜色逐渐缓和到点 0 端。这意味着点 0 端的颜色比点 1 端的颜色分布宽。

• Smooth1 -渐变颜色简化为点 1 末端。这意味着点 1 端的颜色比点 0 端的颜色分布宽。

• 平滑 -渐变颜色逐渐简化为两端。这意味着点 0 和点 1 末端的颜色分布比坡道中心的颜色宽。

线性 坡道。 平滑 坡道。
Smooth0 坡道。 Smooth1 坡道。
4. 使用 颜色 控制以选择点 1 末端 (默认情况下为顶端) 的坡道的颜色。然后，使用 颜色 0 选择点 0 端 (默认情况下为底端) 的坡道颜色。
5. 要调整坡道的展开和角度，请拖动 P0 和 P1 查看器句柄指向新位置。你可能需要按 O 两次在查看器上显示这些控件。

或者，您可以输入新 x 和 y 的值 点 1 和 点 0 文本节点属性中的控件。
