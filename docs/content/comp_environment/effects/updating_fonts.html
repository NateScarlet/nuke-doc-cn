更新字体缓存

一旦生成了字体缓存, 核武器 接受缓存的内容，不再在每次字体控件请求时从头开始生成。如果添加或删除字体，缓存不会自动更新，您需要手动更新。有几种方法可以这样做:

• 导航到 项目设置 > 字体 并单击 重新扫描字体路径 .

• 手动删除 Fontmapping.fcache.xml 从 NUKE_TEMP_DIR 字体缓存文件并重新启动 核武器 .

• 打开 核武器 的脚本编辑器和执行 Nuke.rescanFontFolders ()

转到 帮助 > 文档 > Python 开发人员指南 有关与字体缓存相关的 Python 命令的更多信息。
