将矩形从一个图像复制到另一个图像

1。 选择 合并 > 复制矩形 在包含要复制的区域 (输入 a) 和要将该区域复制到的图像 (输入 B) 的图像之后插入 CopyRectangle 节点。创建以下设置:

2. 在 CopyRectangle 控件中，使用 渠道 下拉菜单从输入 A 中选择要复制的频道。
3. 要定义要复制的矩形，请在查看器中调整 CopyRectangle 覆盖的大小并重新定位。拖动覆盖的中心以重新定位，并拖动边缘以调整大小。如果在查看器中看不到覆盖，请打开 CopyRectangle 属性面板，然后双击节点图中的查看器节点。
重新定位矩形。 调整矩形的大小。
4. 要控制矩形的边缘看起来有多软，请调整 柔软 滑块。值越高，边缘越柔和。
一个低 柔软 价值。 一个高 柔软 价值。
5. 要在完全复制矩形效果和输入 B 之间溶解，请调整 混合 滑块。
