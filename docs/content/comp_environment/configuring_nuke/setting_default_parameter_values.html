设置控件的默认值

您可以通过向节点控件添加简单的 Python 行来设置节点控件的默认值 Init.py 文件。设置默认值后，所有具有匹配名称的控件都将默认为该值。例如，您可以在读取、写入或其他与文件格式相关的节点中为特定于文件格式的控件设置默认值。要设置默认值，请使用以下语句:

Nuke.knobDefault ()

若要指定文件格式特定的默认值，请使用类名，后跟文件格式扩展名和控制名，所有这些都以句点分隔。例如:

Nuke.knobDefault (“读取.exr.压缩”，“2”)

也许你想要帧范围控件的最后一个帧值 项目设置 默认为帧 200。为此，请使用以下语句:

Nuke.knobDefault (“Root.last_frame”，“200”)

或者您可能希望在文本节点中设置不同的默认字体样式:

Nuke.knobDefault (“Text2.font”，“{Arial: Regular: arial.ttf: 0}”)
