自定义插件

的 Nuke 开发人员工具包 (NDK) 允许开发人员创建和编译自己的二进制插件。

源自定义插件

1。 将插件文件放在插件路径目录中。它的名字应该包括一个 。Dll (在 Windows 上), 。所以 (在 Linux 上) 或 。Dylib (在 Mac 上) 扩展。

有关插件路径目录的更多信息，请参见 加载 Gizmos 、 NDK 插件以及 Python 和 Tcl 脚本 .

2. 创建引用插件文件的菜单选项 (请参见 定义自定义菜单和工具栏 )。

或者指示艺术家通过打开脚本编辑器并输入来调用插件 Nuke.load (" 插件名称 ”) 在哪里 插件名称 表示没有扩展的插件的名称。
