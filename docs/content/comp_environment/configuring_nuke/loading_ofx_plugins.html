你在这里:

正在加载 OFX 插件

在启动时, Nuke 扫描不同的目录，寻找能够给用户带来额外功能的 OFX 插件 Nuke 。这些目录的路径因操作系统而异，但以下是您可以找到它们的示例:

• Linux:

/Usr/OFX/ Nuke

/Usr/OFX/插件

• 视窗:

C: \ 程序文件 \ 常见文件 \ OFX \ Nuke (或者，当使用 64 位 Nuke 在 64 位 Windows 上，\ Program Files (x86) \ Common Files \ OFX \ Nuke )

C: \ Program Files \ Common Files \ OFX \ Plugins (或者，使用 64 位时 Nuke 在 64 位 Windows 上，\ Program Files (x86) \ Common Files \ OFX \ Plugins)

• Mac OS X:

/图书馆/OFX/ Nuke

/Library/OFX/插件

如果你想 Nuke 在其他地方寻找 OFX 插件。只需定义指向新目录的环境变量 OFX_PLUGIN_PATH。

例如，在使用 csh 或 tcsh shell 的 Mac OS X 上:

Setenv OFX_PLUGIN_PATH/SharedDisk/OFX

或者，如果您使用的是 bash 或 ksh shell:

导出 OFX_PLUGIN_PATH =/SharedDisk/OFX
