示例案例

下面是一些可能需要更改默认 LUTs 的情况。

在视频色彩空间中工作

不建议模拟在视频色彩空间中工作的合成器软件。但是，如果您确实需要这样做，请执行以下操作:

1。 选择 编辑 > 项目设置 然后去 颜色 标签。
2. 下 默认 LUT 设置 ,更改 监视器 , 8 位文件 ,和 16 位文件 值为 线性 .

这可以防止 Nuke 从 sRGB 转换为线性。 Nuke 的节点仍然假设线性数据，但是图像处理应用于非线性视频彩色空间图像。

16 位文件中的线性数据

一些设施在 16 位文件中使用线性数据。如果您的工厂是这种情况，请执行以下操作:

1. Select Edit > Project Settings and go to the Color tab.
2. 下 默认 LUT 设置 ,更改 16 位文件 价值到 线性 .

Cineon 显示器

一些设施已经调整了监控电子设备，以正确显示 Cineon 数据。如果您的工厂是这种情况，请执行以下操作:

1. Select Edit > Project Settings and go to the Color tab.
2. 下 默认 LUT 设置 ,更改 监视器 价值到 Cineon .

色彩管理

但真实的色彩管理需要使用 Truelight 或其它节点，不妨近似与一甩即用 监视器 设置。这样，纹理贴图和邮票更准确地类似于最终显示。

如果您的颜色管理正在创建显示器校正的图像，您将需要设置 监视器 到 SRGB 因此，您可以在非查看器图像上获得合理的监视正确输出。
