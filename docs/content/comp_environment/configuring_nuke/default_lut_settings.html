默认 LUT 设置

默认情况下, Nuke 在以下情况下使用以下 lut:

LUT

文件类型/设备

默认 LUT

工作空间

这决定了什么颜色空间文件应该在读取和写入时转换为 Nuke 引擎盖下。在早期版本中 Nuke ,这个色彩空间是隐藏的，因为 线性 总是被选为 工作空间 。你可能会发现一些操作在色彩空间中工作得更好 线性 。例如，一些转换在 堵塞 色彩空间。

注意: 你只能改变 工作空间 如果你使用 OCIO 色彩管理 .

线性

监视器

这用于邮票、 OpenGL 纹理、颜色选择器显示和所有其他非查看器图像显示。当需要与监视器颜色空间进行转换时，Truelight 节点也可以使用。

SRGB

8 位文件

这在读取或写入包含 8 位数据的图像文件时使用。也由合并节点的 sRGB 开关使用，并将 Primatte 和 Truelight 节点的输入转换为 sRGB 及其来自 sRGB 的输出。Cineon LUT 也用于控制 Truelight 的输入。

SRGB

16 位文件

这在读取或写入包含 16 位整数数据 (而不是半浮点数) 的图像文件时使用。

SRGB

日志文件

这是在阅读或写作时使用的 。Cin 或 。Dpx 文件。当需要与日志颜色空间进行转换时，Truelight 节点也可以使用。

Cineon

浮动文件

这在读取或写入包含浮点数据的图像文件时使用。

线性

更改默认 LUT 设置的步骤

1。 选择 编辑 > 项目设置 打开脚本的设置。
2. 转到 颜色 标签。
3. 从下的下拉菜单 默认 LUT 设置 ,选择默认情况下要为每个文件类型或设备使用的 lut。

新的默认值现在用于您没有选择特定 LUT 的任何 LUT 设置。您设置为特定 LUT 的任何控件 (即，未设置为 默认 ) 继续使用选定的 LUT，只有那些设置为 默认 受到影响。
