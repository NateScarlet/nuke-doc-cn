加载 Gizmos 、 NDK 插件以及 Python 和 Tcl 脚本

在启动时, Nuke 扫描各种目录中的自定义行为的文件 Nuke 。它查找有关最喜欢的目录、菜单选项、图像格式、 gizmos 、 NDK 插件、 Python 脚本、通用 Tcl 脚本和首选项的信息。

警告: 值得说的是，你应该小心编辑 Python 文件，因为错误可能会停止 Nuke 从运行。
有关 Python 的更多信息 Nuke ,请参见 脚本编辑器和 Python 或 Python 开发人员指南 ( 帮助 > 文档 )。

要使您的自定义可用于特定版本的所有版本，请将它们放在以下目录中:

• Linux:/usr/local/Nuke/<version>/plugins/

• Mac OS X:/库/应用程序支持/Nuke/<版本>/插件/

• Windows: C: \ Program Files \ Common Files \ Nuke \ <version> \ plugins \

如果你想让 Nuke 在其他地方寻找插件，而不是在这些默认位置，你也可以自己定义一个通用的插件路径。因此，通过定义 Nuke 插件路径，您可以为自己分配一个公共共享目录来控制 Nuke 对于多个艺术家。请参阅 定义 Nuke 插件路径 欲了解更多信息。
