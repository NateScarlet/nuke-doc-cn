创建和使用模板脚本

要创建和使用模板脚本:

1。 创建要用作模板的脚本。
2. 选择 文件 > 将 Comp 另存为 。导航到 ~/。 Nuke 。波浪 ( ~ ) 代表您的主目录和句号 ( . ) 用于隐藏文件夹。
3. 命名您的脚本 模板.nk 并单击 保存 .

下次你发射的时候 Nuke 或选择 文件 > 新 Comp 或 文件 > 关闭 Comp , Nuke 从加载模板 ~/。 Nuke /模板。nk .

提示: 如果您不确定主目录的位置，请在 Linux 和 Mac 上打开终端窗口并键入 回声 $ 家庭 。终端将路径名返回到您的主目录。

在 Windows 上，您可以找到 。 Nuke HOME 环境变量指向的目录下的目录。如果未设置此变量 (这是常见的)，则为。Nuke 目录将位于 USERPROFILE 环境变量指定的文件夹下。要确定是否设置了 HOME 和 USERPROFILE 环境变量以及它们指向的位置，请在 Windows 资源管理器的地址栏中输入 % HOME % 或 % USERPROFILE %。如果设置了环境变量，它指向的文件夹将被打开。如果没有设置，你会得到一个错误。
