什么是终端？如何使用终端？

本章中的许多任务告诉您从终端或 shell 输入命令。这是指一个可以直接输入命令的窗口 比 通过用户间进行选择 脸。

下面描述如何为操作系统打开这样的窗口。

• Linux: 单击桌面上的鼠标右键，然后选择 新终端 (或 开放式终端 ) 从右击菜单。

• 视窗: 从 开始 菜单，选择 所有程序> 附件> 命令提示 .

• Mac OS X: 点击 终端 Dock 图标。

OR

浏览到 应用程序> 实用程序 系统硬盘上的文件夹，然后双击 终端 图标。

在终端或 shell 内部，您将看到一个命令提示符，它类似于:

Linux 和 Mac OS X 上的终端提示 Windows 的命令提示符

看到命令提示符后，可以输入命令来执行各种任务，如在目录中列出文件或运行程序。以下是一些具体的例子:

• 在 Linux 或 Mac OS X 上，键入 Pwd 和新闻 输入 查看当前目录的路径。在 Windows 上，等效的命令是 光盘 .

• 在 Linux 或 Mac OS X 上，键入 Ls 和新闻 输入 查看当前目录中的文件列表。在 Windows 上，等效的命令是 Dir .

• 在 Linux 、 Mac OS X 和 Windows 上，键入 光盘 后跟完整的路径名，然后按 输入 更改目录。
