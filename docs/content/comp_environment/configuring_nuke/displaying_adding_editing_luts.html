管理 Nuke 的本地 LUTs

的 颜色 选项卡中的 Nuke 项目设置 允许您查看、添加、编辑和删除 Nuke 他是土生土长的 LUTs。

注意: 你不能调整出现在 OCIO 下的 LUTs 色彩管理 计划。请参阅 OCIO 色彩管理 欲了解更多信息。

显示 LUT 曲线

1。 选择 编辑 > 项目设置 打开脚本的设置。
2. 转到 颜色 标签。
3. 从左侧的列表中，选择要在曲线编辑器中显示的 LUT。要选择几个 lut，请按 Ctrl (Mac 用户按 Cmd )，同时选择 LUTs。所有选定的 LUTs 同时显示在曲线编辑器中。

创建新的 LUT

1. Select Edit > Project Settings to open the settings for the script.
2. Go to the Color tab.
3. 单击加号按钮 ( + )。将打开一个对话框。
4. 输入新 LUT 的名称，然后单击 好 .
5. 调整查找曲线以满足您的需求。单击曲线以选择它。 Ctrl/Cmd + Alt + 单击 在曲线上添加点，并将点拖到新位置。要更改曲线的形状，请调整切线控制柄。

新的 LUT 现在在全局 LUT 设置中可用, 色彩空间 读写节点属性面板的下拉菜单，查看器控件。

编辑 LUTs

1. Select Edit > Project Settings to open the settings for the script.
2. Go to the Color tab.
3. 从左侧的列表中，选择要编辑的 LUT。
4. 调整查找曲线以满足您的需求。单击曲线以选择它。 Ctrl/Cmd + Alt + 单击 在曲线上添加点，并将点拖到新位置。要更改曲线的形状，请调整切线控制柄。

要使用通常的编辑命令，如复制和粘贴，请在曲线编辑器上单击鼠标右键，然后选择 编辑 。然后，选择要使用的编辑命令，就像在任何曲线编辑器上一样。

注意: 目前无法重命名现有 lut。如果你想重命名一个 LUT，你需要添加并命名一个新的 LUT，将信息从旧的 LUT 复制到新的 LUT 中，然后删除旧的 LUT。

将 LUT 曲线重置回其初始默认形状

1. Select Edit > Project Settings to open the settings for the script.
2. Go to the Color tab.
3. 从左侧的列表中，选择要重置的 LUT。要选择几个 lut，请按 Ctrl / Cmd 选择 LUTs 时。
4. 单击 重置 .

移除 LUTs

1. Select Edit > Project Settings to open the settings for the script.
2. Go to the Color tab.
3. 从左侧的列表中，选择要删除的 LUT。仅删除您意外创建的、未在脚本中使用的 LUTs。要删除 LUT，请单击减号按钮 ( - )。

LUT 将从 LUT 设置中删除, 色彩空间 读写节点属性面板的下拉菜单。

注意: 如果删除在节点中使用的 LUT，则节点将继续按名称引用 LUT 并引发错误。
