设置环境变量

本节介绍如何设置环境变量，检查特定的环境变量是否存在，以及显示设置的环境变量列表。

设置环境变量

在窗户上

1。 右击 on 我的电脑 并选择 属性 .
2. 转到 高级 标签。
3. 单击 环境变量 按钮。的 环境变量 对话框打开。
4. 单击 新 按钮下 用户变量 或 系统变量 ,这取决于您是要为当前用户还是所有用户设置变量。要为所有用户设置环境变量，您需要具有管理员权限。
5. 在 变量名称 字段中，输入要设置的环境变量的名称。有关环境变量的列表 Nuke 理解，见 Nuke 环境变量 .
6. 在 变量值 字段中，输入变量的值。例如，该值可以是目录路径。
7. 单击 好 .

注意: 编辑现有系统变量或添加或删除用户或系统变量时，您可能需要在对环境变量的更改生效之前注销并重新登录。

在 Mac 上

在 Mac OS X 上，您可以使用 Launchd.conf 文件以设置环境变量。您可能需要创建 Launchd.conf 文件 (如果文件中不存在) /Etc/ 目录。

注意: 如果您只需要为单个会话设置环境变量，或者您不想使用 Mac OS X Launchd.conf 文件，您还可以设置变量所使用的方法, 在 Linux 上 部分。

环境变量使用 Launchd.conf 文件被读取时 Nuke 是从 Nuke 图标，当它从终端启动时。

1。 打开终端窗口。
2. 创建 /Etc/launchd.conf 文件，如果它还不存在，然后使用以下格式将环境变量和值添加到文件中:

Setenv <变量> <值>

例如，要设置两个环境变量 NUKE_PATH 和 OFX_PLUGIN_PATH，以指向备用位置:

Setenv NUKE_PATH/SharedDisk/Nuke/

Setenv OFX_PLUGIN_PATH/SharedDisk/OFX

有关环境变量的列表 Nuke 理解，见 Nuke 环境变量 .

提示: 一个方便的命令行工具，用于在终端中创建和编辑文件 纳米 。要以正确的权限启动 nano，请输入:

Sudo nano/etc/launchd.conf

3. 强制操作系统读取 Launchd.conf 文件在启动时，输入:

Launchctl </etc/launchd.conf; sudo launchctl </etc/launchd.conf

4. 重新启动 Mac 以应用更改。

在 Linux 上

1。 设置环境变量的过程取决于默认的 shell。要获取正在使用的 shell 的名称，请启动 shell 并输入 Echo $ SHELL .
2. 根据上一步的输出，执行以下操作之一:

• 如果您的 shell 是 csh 或 tcsh shell，请将以下命令添加到。Cshrc 或。主目录中的 tcshrc 文件: Setenv 变量值 。替换 变量 使用环境变量的名称和 价值 例如，你想给它的值 Setenv NUKE_PATH/SharedDisk/Nuke .

• 如果您的 shell 是 bash 或 ksh shell，请将以下命令添加到。Bashrc 或。主目录中的 kshrc 文件: 导出变量 = 值 。替换 变量 使用环境变量的名称和 价值 例如，你想给它的值 导出 NUKE_PATH =/SharedDisk/Nuke .

For a list of the environment variables that Nuke understands, see Nuke Environment Variables .

检查环境变量是否存在

On Windows

1。 选择 开始 > 所有程序 > 配件 > 命令提示 .
2. 在打开的命令窗口中，输入 回声 % 变量 % 。替换 变量 使用环境变量的名称。例如，要检查是否设置了 NUKE_DISK_CACHE，请输入 Echo % NUKE_DISK_CACHE % .

如果设置了变量，其值将显示在命令窗口中。

在 Mac 或 Linux 上

1。 发射终端或外壳。
2. 输入 回声 $ 变量 。替换 变量 使用环境变量的名称。例如，要检查是否设置了 NUKE_DISK_CACHE，请输入 Echo $ NUKE_DISK_CACHE .

如果设置了变量，其值将显示在终端或 shell 窗口中。

显示设置的环境变量列表

On Windows

1. Select Start > All Programs > Accessories > Command Prompt .
2. 在打开的命令窗口中，输入 设置 .

设置的所有环境变量的列表将显示在命令窗口中。

On Mac or Linux

1. Launch Terminal or a shell.
2. 输入 Printenv .

设置的所有环境变量的列表将显示在终端或 shell 窗口中。
