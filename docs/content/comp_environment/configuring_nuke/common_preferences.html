的 Nuke 首选项对话框 ( 编辑 > 偏好 ) 允许任何用户对界面进行各种行为和显示调整。但是，您可能希望为艺术家指定某些默认首选项。

定义默认首选项

要定义默认首选项:

1。 选择 编辑 > 偏好 要显示 偏好 对话框。
2. 根据需要修改对话框中的控件。有关控件的操作的说明，请参见 附录 A: 偏好 .
3. 单击 好 . Nuke 将修改的首选项写入名为 偏好 10.5 。Nk ,它存储在您的 [主目录]/。nuke 目录。

提示: 如果您不确定主目录的位置，在 Linux 和 Mac 上，您可以转到终端窗口并键入 回声 $ 家庭 。终端将把路径名返回到您的主目录。

在 Windows 上，您可以找到。Nuke 目录位于 HOME 环境变量指向的目录下。如果未设置此变量 (这是常见的)，则为。Nuke 目录将位于 USERPROFILE 环境变量指定的文件夹下。要确定是否设置了 HOME 和 USERPROFILE 环境变量以及它们指向的位置，请在 Windows 资源管理器的地址栏中输入 % HOME % 或 % USERPROFILE %。如果设置了环境变量，它指向的文件夹将被打开。如果没有设置，你会得到一个错误。

以下是不同平台上路径名的示例:
Linux :/Home/登录名
Mac :/Users/登录名
窗户 : 盘符: \ 文档和设置 \ 登录名或盘符: \ Users \ 登录名

4. 移动结果 偏好 10.5 。Nk 文件到您的 Nuke 插件路径目录。

有关插件路径目录的更多信息，请参见 加载 Gizmos 、 NDK 插件以及 Python 和 Tcl 脚本 .

您的偏好现在作为您的艺术家的默认设置。然而，他们是否应该使用 偏好 对话框中，这些更改将覆盖您的默认值。

删除 (和重置) 首选项

到 删除 (并重置) 首选项:

1。 按照本章开头对操作系统的说明打开终端 (或 shell)。
2. 使用提示，转到。核弹目录，在你的主目录下。
3. 输入 Pwd 显示并验证路径。

你应该看到类似的东西

• /用户/ 登录名 /。Nuke (在 Linux 上),

• /用户/ 登录名 /。 Nuke (在 Mac 上) 或

• 驱动器号 : \ 文档和设置 \ 登录名 \.Nuke 或 驱动器号 : \ 用户 \ 登录名 \.Nuke

然而，情况并不总是这样，因为在 Windows 上。Nuke 文件夹可以在 HOME 环境变量指向的目录下找到，或者 (如果没有设置 HOME) USERPROFILE 环境变量。

要确定是否设置了 HOME 和 USERPROFILE 环境变量以及它们指向的位置，请在 Windows 资源管理器的地址栏中输入 % HOME % 或 % USERPROFILE %。如果设置了环境变量，它指向的文件夹将被打开。如果没有设置，你会得到一个错误。

4. 输入 Rm 首选项 10.5 。Nk 删除首选项文件。
5. 关闭终端或外壳。

下次你发射的时候 Nuke ,它使用默认首选项重建文件。
