向网格添加纹理

您可以使用 Project3D 节点快速将纹理添加到点云网格。执行以下操作:

1。 单击 3D > 着色器 > Project3D .
2. 将已解决的相机节点连接到 Project3D 凸轮 输入。
3. 将 Project3D 节点的另一个输入连接到序列中。
4. 连接 Group1_Mesh 节点 Img 项目 3D 节点的输入。

网格现在被原始 2D 序列纹理化。

提示: 您可以将相机、点云和网格导出为 Alembic ( 。Abc ) 其他应用程序的对象。有关导出的更多信息 。Abc 文件，请参见 导出几何图形、摄影机、灯光、轴或点云 .
