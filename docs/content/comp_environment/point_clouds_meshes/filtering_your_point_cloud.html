过滤您的点云

您可以通过调整包含的点的数量和质量来过滤点云。 NukeX 直接过滤跟踪的点云，而无需重新计算，从而简化您的工作流程。

1。 确保 显示拒绝点 启用并调整阈值控件以动态更新 3D 查看器。拒绝的曲目以红色突出显示。
2. 调整 角度 阈值 设置三角化 3D 点的最小可接受角度 (以度为单位)。三角测量角度较大的点往往更准确。

设置阈值 0 三角化所有点或增加阈值以突出显示最不准确的点。

提示: 根据经验，任何低于 5 度的东西都可能是不正确的。

3. 调整 密度 阈值 设置可接受的最小点密度。孤立点往往不太准确。

设置阈值 0 输出所有点或增加阈值以突出显示最孤立、不太准确的点。

4. 观众 擦拭 工具可以通过将 2D 源素材与点云进行比较来帮助您定位孤立点。

将输入 A 设置为源图像的 PointCloudGenerator 节点，将输入 B 设置为源图像的读取节点。

请参阅 使用查看器控件 欲了解更多信息。

5. 继续到 删除被拒绝的点 .
