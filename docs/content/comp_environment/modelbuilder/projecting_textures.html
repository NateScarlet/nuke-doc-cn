将纹理投影到形状上

如果 ModelBuilder 模型与原始 2D 素材非常匹配，则可以简单地将 2D 素材投影到几何图形上。要这样做:

1。 在 ModelBuilder 属性面板的底部，将烘焙菜单设置为 投影 并单击 烘焙 .

这将在当前纹理帧处创建一个投影: 一个项目 3d 节点，该节点的帧保持节点设置为将输入图像和相机锁定到纹理帧。Project3D 节点也连接到 垫子 ApplyMaterial 节点的输入。

2. 要将投影应用于几何图形，请连接 ModelBuilder 节点 (或烘焙创建的几何图形节点) 选定几何 ) 到 ApplyMaterial 节点的未命名输入并查看结果。

提示: 要在多个帧上纹理几何图形，可以烘焙 投影 在几个帧上，使用 Roto 节点在使用 MergeMat 节点组合 Project3D 节点之前屏蔽掉一些东西。然后，将 MergeMat 节点连接到 ApplyMaterial 节点的 垫子 输入和 ModelBuilder 到其未命名的输入。将查看器连接到 ApplyMaterial 的输出并查看结果。

提示: Bake 菜单中的选项是用 Python 实现的，您也可以使用 Python 将自己的条目添加到菜单中。要了解内置选项是如何实现的，请查看 Modelbuilder.py 模块中的 Nukescripts 安装文件夹 (有关此文件位置的详细信息，请参阅 查看更多示例 )。然后，要创建自己的菜单选项，请编辑 Modelbuilder.py 文件，并使用 populateBakeMenu 函数添加条目，上面写着 “# 如果需要，在这里添加您自己的条目”。
