连接 ModelBuilder 节点

要连接 ModelBuilder 节点:

1。 如果您还没有匹配您的镜头的跟踪相机，请使用 CameraTracker 节点创建一个 (请参见 相机跟踪 )。

ModelBuilder 需要一个变化的相机 (变化超过 5 度)，以便从 2D 镜头中计算 3D 点。

2. 选择 3D > 几何 > ModelBuilder 创建 ModelBuilder 节点。
3. 将相机节点连接到 ModelBuilder 凸轮 输入，和你的 2D 镜头 Src 输入。

请注意，您的对象 Src 要建模的素材不应该移动，因为 ModelBuilder 最适用于静态对象。

4. 如果您有 3D 几何图形 (如形状或点云)，希望在定位新形状时用作参考，请将其连接到 ModelBuilder 的 Geo 输入。

例如，可以使用 PointCloudGenerator 节点 (请参见 创建密集的点云 ) 为您的镜头创建三维点云，并将其连接到 Geo 输入。创建新形状时，可以选择点云上的顶点以自动将形状与点云对齐。这给你一个形状的大致初始位置。

5. 如果您有一个 2D 纹理图像，您希望在 UV 预览窗口和 3D 查看器中的模型上显示为背景，请将其连接到 ModelBuilder 的 Tex 输入。
6. 将查看器附加到 ModelBuilder 节点的输出。

7. 继续到 创建形状 下面。
