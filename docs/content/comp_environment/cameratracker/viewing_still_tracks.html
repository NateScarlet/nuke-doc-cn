查看参考帧和轨迹数据

跟踪完成后，您可以查看 CameraTracker 如何连接分析的帧:

1。 执行以下操作之一:

• 切换 缩略图 控件位于查看器上方 跟踪 。CameraTracker 在查看器底部的缩略图库中显示连接到当前帧的帧。这允许您简单地逐帧浏览序列来评估连接。

• 切换 缩略图 控件位于查看器上方 所有 让 CameraTracker 在缩略图库中显示跟踪范围内的所有帧。这允许您将光标悬停在缩略图上，以查看其与以橙色突出显示的相邻参考帧的连接。断开的帧以红色突出显示。请参阅 断开帧集 欲了解更多信息。

提示: 您可以单击并拖动以左右滚动画廊，或者单击缩略图将播放头移动到相应的参考框架。要调整缩略图库的大小，请上下拖动库的顶部。

2. 要显示轨迹信息，请将鼠标悬停在某个特征上或选择多个特征。

注意: 如果您进行多个选择，您将看不到帧中的轨道长度，只有轨道本身。

查看自动跟踪曲线

详细的跟踪信息显示在 “属性” 面板的曲线中 自动跟踪 标签。您可以一次选择所有轨迹曲线，或者获得一条曲线的更详细视图，例如 Num 轨道 .

提示: 按 F 聚焦在曲线上可以在可用空间中最大化选定曲线。

选择所有轨迹曲线。 构建一条曲线。

曲线可以指示跟踪遇到问题的序列区域。例如，右侧的 “轨迹数” 曲线在当前帧上显示了显著的倾斜，如播放头位置所示。

在这种情况下，您可以尝试调整 功能数量 或 最小长度 并重新追踪。请参阅 故障排除仍然跟踪
