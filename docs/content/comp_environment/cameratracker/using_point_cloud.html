Using the Point Cloud

If you elected to create a point cloud and scene linked to the result of the solve. You can use the point cloud to position scene elements.
Viewing the point cloud in 3D view.

TIP:  If your point cloud is very dense, you can reduce the size of it by only viewing the tracks visible in your key frames. That way you can view fewer, more accurate 3D points. To reduce the point cloud size, check the KeyTracks box on the CameraTracker tab.
