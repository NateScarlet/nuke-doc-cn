创建相机节点

使用跟踪和求解数据，CameraTracker 可以创建链接或烘焙的相机，在设置时模拟原始相机轨迹。您可以为多视图项目创建单个摄影机或摄影机装备，具体取决于您的脚本。

创建相机节点

1。 选择 相机 从 出口 下拉菜单。
2. 启用或禁用 链路输出 控件来确定相机节点是链接的还是烘焙的:

• 启用时，CameraTracker 将创建一个表达式链接的摄影机，以便在 “属性” 面板中进行任何调整 输出 > 相机 控制更新相机。

• 禁用时，相机将忽略所做的任何调整。

3. 单击 创建 .

相机节点将添加到节点图中。表达式链接的摄影机将使用绿色表达式箭头链接到 CameraTracker 节点。

摄像机节点数据可以用于各种 核武器 工作流，如 创建密集的点云 和 生成深度图 .

创建照相机装备

CameraTracker 创建的相机和脚本中的视图一样多，但是钻机最常见的用途是使用左右视图的立体脚本。

1。 选择 相机 钻机 从 出口 下拉菜单。
2. 启用或禁用 链路输出 控件来确定相机节点是链接的还是烘焙的:

• 启用后，CameraTracker 将创建表达式链接的摄影机，以便在 “属性” 面板中进行任何调整 输出 > 相机 控制更新相机。

• 禁用时，相机将忽略所做的任何调整。

3. Click Create .

注意: 如果尝试创建只有一个视图的装备，将显示错误。

对于脚本中出现的每个视图，相机节点将添加到节点图中。表达式链接的摄影机将使用绿色表达式箭头链接到 CameraTracker 节点。

Camera node data can be used in various Nuke workflows, such as Creating Dense Point Clouds and Generating Depth Maps .

创建相机集

CameraTracker 可以自动为每个解决的帧创建一个相机。

1。 选择 相机设置 从 出口 下拉菜单。
2. 启用或禁用 链路输出 控制以确定摄影机是链接的还是烘焙的:

• When enabled, CameraTracker creates expression linked cameras so that any adjustments made in the properties panel Output > Camera controls update the cameras.

• When disabled, any adjustments made are ignored by the cameras.

3. Click Create .

如果您正在处理大量帧，将显示确认对话框。

4. 单击 是 继续或 没有 取消出口。

CameraTracker 将一个组节点添加到包含与场景节点相连的每个帧的摄像机节点的节点图中。

5. 双击该组以打开 “属性” 面板，然后单击 S 在面板上方打开节点图中的组。

然后，您可以将相机数据写入 。Fbx 文件准备导入到 Modo 或 Maya 中进行投影绘画工作。
