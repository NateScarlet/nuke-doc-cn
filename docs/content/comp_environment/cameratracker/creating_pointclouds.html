创建点云

CameraTracker 可以创建烘焙点云，即在更改 CameraTracker 属性面板控件时不会更新的点。

1。 选择 点云 从 出口 下拉菜单。
2. 单击 创建 .

CameraTracker 将 CameraTrackerPointCloud 节点添加到节点图中。

3. 双击 “CameraTrackerPointCloud” 节点以打开 “属性” 面板。
4. 使用 “显示” 和 “渲染” 下拉菜单设置点在查看器中和渲染时的显示方式:

• 关闭 -隐藏 3D 点。

• 线框 -仅显示点。

• 固体 -使用所有点所代表的像素颜色显示所有点。

• 固体 线框 -使用它们所代表的像素的颜色和点本身显示所有点。

• 纹理 -仅显示表面纹理。

• 纹理 线框 -显示表面纹理和点。

5. 使用复选框将点云渲染为 2D 时，启用或禁用阴影投射和接收。

请参阅 投射阴影 有关照明和投射阴影的更多信息。

6. 使用设置渲染点的大小 点大小 控制。
