Attaching Objects to the Footage

Once your scene is set, you can start attaching 3D objects to the point locations.

1.   Select a track or a set of tracks where you want to attach your object.

Selected tracks are highlighted in yellow. You can attach another Viewer node to the CameraTracker node to see the highlighted 3D points alongside the 2D view. This way you can ensure the points are where you expect them to be.

2.   Right-click on your selection and select an object you want to attach to your footage:

• Select create > cube, cylinder, sphere or card to create a basic shape.

• Select create > card XY, card YZ or card ZX to create a card which is aligned with the x, y or z axis.

• Select create > axis to create a new axis.
