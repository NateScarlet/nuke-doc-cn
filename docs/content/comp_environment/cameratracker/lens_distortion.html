考虑镜头失真

默认情况下，添加到 3D 场景中的任何 CG 元素都不会应用镜头失真。因此，当你把它们和你的 2D 镜头结合起来时，它们看起来不像用同一台相机拍摄的。要解决这个问题，您可以:

• 使用 CameraTracker 计算 2D 镜头上的镜头失真，并生成一个将相同失真应用于 CG 元素的镜头失真节点。请参阅 扭曲你的 CG 元素以匹配你的 2D 镜头 .

• 使用 CameraTracker 计算 2D 镜头上的镜头失真，并生成一个镜头失真节点，该节点不会扭曲 2D 镜头以匹配您的 CG 元素。请参阅 不扭曲你的 2D 镜头，以匹配你的 CG 元素 (使用透镜失真) .

• 使用 CameraTracker 来计算 2D 镜头上的镜头失真，并且不失真镜头以匹配您的 CG 元素。请参阅 不扭曲你的 2D 镜头以匹配你的 CG 元素 (使用 CameraTracker) .

注意: 上述假设您设置 透镜失真 到 未知镜头 在创建解决方案之前。如果你设置 镜头失真 到 无镜头失真 在相机跟踪之前，使用一个单独的镜头失真节点来消除镜头失真，您可以使用相同的镜头失真节点将失真应用到您的 CG 元素或不失真您的 2D 镜头。请参阅 处理镜头失真 有关在中使用透镜失真节点的更多信息 Nuke .

扭曲你的 CG 元素以匹配你的 2D 镜头

1。 在 CameraTracker 属性中，设置 出口 菜单到 失真 并单击 创建 .

CameraTracker 创建一个镜头失真节点，该节点被设置为应用 2D 镜头中存在的镜头失真。

2. 将透镜失真节点连接到 ScanlineRender 节点的输出。

透镜失真节点扭曲 3D 场景中的 CG 元素，以匹配您的 2D 镜头。

3. 根据需要将 CG 元素和 2D 镜头组合在一起。

不扭曲你的 2D 镜头，以匹配你的 CG 元素 (使用透镜失真)

1。 在 CameraTracker 属性中，设置 出口 菜单到 不失真 并单击 创建 .

CameraTracker 创建一个镜头失真节点，该节点被设置为删除 2D 镜头中存在的镜头失真。

2. 将镜头失真节点连接到 2D 镜头的输出。

镜头失真节点移除 2D 镜头中的镜头失真，以匹配您的 CG 元素。

3. 将透镜失真节点连接到 Bg ScanlineRender 的输入。

未扭曲的 2D 镜头被用作 3D 场景的背景图像。

不扭曲你的 2D 镜头以匹配你的 CG 元素 (使用 CameraTracker)

1。 在 CameraTracker 属性中，启用 不失真输入 .

CameraTracker 移除其输入镜头中存在的镜头失真。

2. 将 CameraTracker 节点连接到 Bg ScanlineRender 的输入。

The undistorted 2D footage is used as a background image for the 3D scene.
