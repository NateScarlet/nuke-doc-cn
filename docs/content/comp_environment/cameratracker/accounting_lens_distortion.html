考虑镜头失真

您可以使用 CameraTracker 的镜头失真功能来解释镜头中的镜头失真。然后，您可以从原始镜头中跟踪和求解，并计算未扭曲镜头的相机。在 镜头 选项卡，您可以在镜头中定义失真类型，并调整其他镜头失真控制:

1。 在 镜头 失真 下拉列表中，选择预期的失真类型。

• 没有 镜头 失真 禁用所有镜头失真控制，并将镜头视为没有失真。这个选项给出了处理原始镜头的最佳解决方案。计算的焦距补偿镜头失真。

• 已知 镜头 允许您手动指定镜头失真，并在进行相机解算时使用该信息。当您有可用于定义镜头失真的网格时，请使用此选项。您可以先计算 lensdeximation 节点中的失真参数，然后将值粘贴到 CameraTracker 节点上。

• 精炼 已知 镜头 给你一个机会给一个近似失真使用，但试图在相机解决。当使用网格时，这是不必要的，但是如果使用替代方法或手动设置来计算镜头失真，这可以改进解决方案。

• 未知 镜头 从序列中自动计算镜头失真 (与 lensdistability 节点中的图像分析选项相同)，然后在相机解析度中细化失真。这样，您就不必使用透镜失真节点分别计算镜头失真。

2. 如果你选择了 已知 镜头 或 精炼 已知 镜头 ,输入的值 镜头 类型 , 径向 失真 , 失真 中心 控制，如有必要, 变形 挤压 , 不对称 失真 和 过滤器 控制。有关这些控件的更多信息，请参见 调整透镜失真参数 .
3. 直接移动到点击 解决 相机 和 创建 相机 查看结果或进一步调整的解算器控制 求解器 标签。

有了镜头失真值后，您也可以切换 不失真输入 框查看您的镜头扭曲或不扭曲。

在卡节点中使用镜头失真

CameraTracker 会自动重新计算镜头失真参数，以便您可以选择在卡片节点中应用镜头失真模型。执行以下操作:

1。 打开 卡 参数 上的文件夹 镜头 标签。
2. 按住 Ctrl / Cmd 并拖动 x , y 和 z 值从 规模 参数的值上的相应栏目, 卡 卡节点中的选项卡 ( 3D > 几何 > 卡 ) 属性面板。这为卡上的再现失真设置了正确的比例。
3. 按住 Ctrl/Cmd 并拖动 一个 , B, 和 c 值放在相应的字段上 镜头 失真 卡节点属性面板中的选项卡。这些值现在用于不扭曲附加到 Img 卡节点的输入。
