设置地平面和轴

解决后的相机不知道地平面在场景中的位置，这可能会产生一个意想不到的偏移和相对于你期望地面在 3D 空间中的角度。

设置地平面或轴组旨在为摄影师提供一个合理的参考系。虽然这样做不是绝对必要的，但它可以简化在 3D 环境中的工作。

在放置地平面之前，扫描你的序列，找到一个轨迹良好的框架，并在你知道是在地面的区域中解析数据。

1。 将鼠标悬停在可能的点上，以显示轨迹和求解数据，通常是 RMS 误差小于 1 的点。
2. 如果需要，按住 Shift 键并从多个帧中选择您选择的点。
3. 在 2D 和 3D 查看器之间切换以确认选定点的位置。

在 2D 查看器中选择的地平面点。 在 3D 查看器中查看的相同点。
4. 切换回 2D 查看器，并在选定的点上单击鼠标右键以显示 “轨迹选项” 菜单。
5. 选择 地平面 > 设置为选定 相对于选定的点定向相机。

指定为地平面的点以洋红色突出显示，相机相对于新的 “地面” 重新定向。

单独设置轴

在某些情况下，可能没有明确定义的地平面可供参考。使用 CameraTracker，可以分别设置 X 、 Y 和 Z 轴，而不是在地平面上对它们进行分组。

1。 使用三维点云来帮助您选择两个或更多已知位于所需轴上的点。
2. 右键单击一个点并选择 地平面 > 设置 X , Y ,或 Z 根据需要。

指定为轴的点以黄色突出显示，并且相机相对于新轴重新定向。

在场景中设置原点

定义层中的原点可以帮助您在组合中放置对象-如果您知道 0 、 0 的位置, 您可以轻松地将对象精确地放置在任何轴上所需的位置。

1。 从解析数据中选择单个轨迹作为原点。
2. 右键单击该点并选择 地平面 > 设置原点 .

指定为原点的点以黄色突出显示，并相对于新原点转换点云。
