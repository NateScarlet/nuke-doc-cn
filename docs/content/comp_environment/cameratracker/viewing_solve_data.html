查看解析数据

一旦解决数据被放置在查看器中，您可以放大以更清楚地显示这些点。你可以控制自己的浏览器中使用 显示 上的控件 设置 标签:

• 显示曲目 -显示或隐藏 2D 跟踪信息。

• 显示投影 3D 点 -显示或隐藏 3D 点的 2D 位置。

• 仅显示关键轨迹 -只显示用于计算求解的最长轨道。

• 显示 3D 标记 -在 3D 查看器中的每个点显示或隐藏 3D 标记。

交通灯方案适用于 2D 阈值的运动轨迹来帮助找到好排斥反应的抑制作用 自动跟踪 Tab-默认情况下，琥珀色，绿色代表好的曲目，红色代表糟糕的曲目。

圆圈和十字架被重新投影解决自动轨道。3D 点圆离特征点越近，解得越好。

将鼠标悬停在一个点上以显示其解析信息。

CameraTracker 还在 3D 查看器中创建一个点云，允许您交叉检查 2D 和已求解 3D 点的相对位置。在 2D 中选择一些点，然后按 标签 在查看器中，在 2D 和 3D 空间之间切换以检查点位置。

在 2D 序列中选择了几个点。 3D 查看器中显示的相同点。

查看自动跟踪曲线

详细的解析信息显示在 “属性” 面板的曲线中 自动跟踪 标签。您可以一次选择所有求解曲线，或者获得一条曲线的更详细视图，例如 错误-rms (均方根)。

提示: 按 F 聚焦在曲线上可以在可用空间中最大化选定曲线。

选择所有求解曲线。 构建一条曲线。

曲线可以指示求解遇到问题的序列区域。例如，右侧的均方根误差曲线显示了当前帧上的一个显著点，如播放头位置所示。在这种情况下，您可以尝试调整 关键帧间距 或 平滑度 并更新解决方案。请参阅 故障排除解决

图表中的曲线显示以下求解信息:

• 解决错误 -显示常数 解决 错误 参数。

• 错误最小 -每帧的最小重投影误差 (以像素为单位)。

• 错误 rms -每帧的根平均重投影误差 (以像素为单位)。

• 错误跟踪 -在每帧的轨道寿命上计算的最大根平均重投影误差 (以像素为单位)。

• 最大错误 -每帧的最大重投影误差 (以像素为单位)。

• 最大轨迹错误 -显示常数最大 RMS 错误参数。

• 最大错误 -显示最大错误阈值参数。

查看已解决的用户跟踪

当一个用户跟踪解决，您可以查看其错误的用户跟踪表 用户轨迹 标签。如果错误很高，请尝试查看 2D x，y 特征轨迹。或者，您可以取消选中用户跟踪的 e (启用) 列，以便在单击之前将其从场景中删除 解决 再次。

用户轨迹的 3D 位置 (X，Y，Z) 也显示在表格中，以及 3D 查看器中。

提示: 您可以在求解相机后创建用户轨迹，以便在镜头中提取特定的 3D 位置。要产生精确的 3D 点，用户轨迹必须在 3 个或更多具有良好视差的帧中定义。创建用户轨迹后，在 “用户轨迹” 表中选择它，然后单击 更新 XYZ 按钮从当前解析三角化 3D 位置。

提示: 自动跟踪后，您可能需要选择自动跟踪并将其提取为用户跟踪。这向解决者暗示这是一条重要的轨道，应该被锁定。如果场景的某个特定部分没有正确求解，尽管自动轨迹似乎在该区域跟踪良好，这尤其有用。

有关更多信息，请参见 提取用户轨迹 .

预览 Matchmove 质量

计算完解决方案后，您可能想通过预览添加到 3D 场景中的对象将如何坚持您的镜头来查看其质量。如果你设置 镜头失真 到 未知镜头 在创建解决方案之前，有两种方法可以做到这一点:

方法 1

1。 在 2D 查看器中，在轨迹点 (或多个选定点，理想情况下在同一平面上) 上单击鼠标右键，然后选择 创建 > 立方体 ,例如。

CameraTracker 创建一个立方体节点，并将其放置在选定点的平均位置。你将使用立方体来测试它与输入镜头的一致性。

提示: 如果需要，你可以调整大小，魔方用 均匀刻度 控制多维数据集属性。

2. 在 CameraTracker 属性中，设置 出口 菜单到 场景 + 并单击 创建 .

CameraTracker 创建一个 3D 场景，其中相机、 PointCloud 、 ScanlineRender 和一个被设置为不扭曲输入的透镜失真节点。

3. 将多维数据集节点连接到场景节点的一个输入。

4. 查看 ScanlineRender 的输出。

在步骤 1 中创建的立方体和生成的点云 CameraTracker 显示在 2D 查看器中未扭曲的素材的顶部。

5. 浏览时间线，看看立方体和点云是否粘在镜头上。如果需要，请删除不再需要的任何节点 (如多维数据集节点)，然后继续 调整场景 。如果他们不这样做，继续 故障排除解决 .

方法 2

1。 在 CameraTracker 属性中，设置 出口 菜单到 相机 并单击 创建 .

CameraTracker 创建一个相机节点，模拟设置中使用的相机。

2. 在 2D 查看器中，在轨迹点 (或多个选定点，理想情况下在同一平面上) 上单击鼠标右键，然后选择 创建 > 立方体 ,例如。

CameraTracker creates a Cube node and places it in the average position of the selected points. You are going to use the cube to test how well it sticks to the input footage.

TIP: If necessary, you can adjust the size of the cube using the uniform scale control in the Cube properties.

3. 启用 不失真输入 .

CameraTracker 从输入镜头中删除镜头失真。

4. 确保 CameraTracker 和 Camera properties 面板都打开。
5. 新闻 标签 在查看器上切换到 3D 视图。
6. 将查看器右上角的 “相机浏览” 菜单设置为步骤 1 中创建的相机，然后单击 按钮将 3D 视图锁定到该相机。
7. 在查看器的顶部，将查看器复合下拉菜单设置为 结束 ,以便您在同一 CameraTracker 节点上查看 CameraTracker 节点。

这将 CameraTracker 当前未扭曲的输入图像覆盖在 3D 查看器中的点云上。

8。 浏览时间线，查看您在步骤 2 中创建的立方体和生成的点云 CameraTracker 是否与素材保持一致。如果需要，请删除不再需要的任何节点 (如多维数据集节点)，然后继续 调整场景 。如果他们不这样做，继续 故障排除解决 .
