添加和删除镜头失真

如果您没有使用单独的镜头失真节点手动删除镜头失真，您可以使用 CameraTracker 的 输出 标签 镜头 控制以考虑失真，然后使用这些设置创建透镜失真节点以不失真或扭曲场景。请参阅 考虑镜头失真 欲了解更多信息。

1。 选择 失真 或 不失真 从 出口 下拉菜单，根据需要。
2. 启用或禁用 链路输出 控制以确定 lensdistrature 节点是链接表达式还是烘焙表达式:

• 启用时，CameraTracker 将创建一个表达式链接的摄影机，以便在 “属性” 面板中进行任何调整 输出 > 相机 控制更新相机。

• 禁用时，相机将忽略所做的任何调整。

3. 单击 创建 .

CameraTracker 在节点图中添加了一个 lensdistrature 节点。

4. 双击透镜失真节点以打开 “属性” 面板。

相关的失真控制是自动关键帧，但状态 不失真 复选框取决于您是否选择 失真 或 不失真 从 出口 下拉菜单。
