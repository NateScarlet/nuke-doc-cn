Adding and Removing Lens Distortion

If you didn't removed lens distortion manually using a separate LensDistortion node, you can use CameraTracker's Output tab Lens controls to account for distortion and then create LensDistortion nodes using those settings to undistort or distort your scene. See Accounting for Lens Distortion for more information.

1.   Select Distortion or Undistortion from the Export dropdown menu, as required.
2.   Enable or disable the Link output control to determine whether the LensDistortion node is expression linked or baked:

• When enabled, CameraTracker creates an expression linked camera so that any adjustments made in the properties panel Output > Camera controls update the camera.

• When disabled, any adjustments made are ignored by the camera.

3.   Click Create.

CameraTracker adds a LensDistortion node to the Node Graph.

4.   Double-click the LensDistortion node to open the properties panel.

The relevant distortion controls are automatically keyframed, but the state of the Undistort checkbox is dependent on whether you selected Distortion or Undistortion from the Export dropdown menu.
