使用多视图脚本

CameraTracker 可以像单视图项目一样跟踪和解决立体或多视图项目。

1。 按照中所述连接 CameraTracker 节点 连接 CameraTracker 节点 .
2. 使用 摄影师 或 设置 标签 主要观点 下拉菜单选择用于创建轨迹的视图。脚本中存在的任何其他视图中的轨迹都是从 主要观点 .
3. 遵循下所述的工作流程 相机跟踪 跟踪和解决相机。
4. 如果创建任何用户轨迹，请注意，它们目前最多只支持两个视图。要选择将哪些视图用作用户轨迹的左视图和右视图，请使用 用户跟踪视图 CameraTracker 属性的 UserTracks 选项卡上的控件。
5. 要为您的视图可视化两个相机，请通过选择为每个视图创建一个带有相机的多视图装备 相机钻机 从 出口 部分的 摄影师 标签。请参阅 创建相机节点 欲了解更多信息。
