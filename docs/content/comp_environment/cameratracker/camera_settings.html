设置相机参数

相机设置与设置中使用的相机的物理方面相关。精确的物理相机数据产生更好的相机跟踪和解决方案。

1。 从中选择 on set 相机的运动类型 摄影师 标签 相机运动 下拉菜单。这是链接到相同名称的控件 设置 标签。

• 仅旋转 -如果相机是静态和旋转的，例如，如果您使用三脚架安装的相机来安装 nodal pans，请选择此选项。

• 免费相机 -如果相机同时平移和旋转，请选择此选项。

• 线性运动 -如果相机具有直线路径，请选择此选项。

• 平面运动 -如果相机有平坦的路径，仅在二维平面中移动，请选择此选项。

2. 如果您已经使用了一个单独的透镜失真节点 (请参见 处理镜头失真 在第一页) 要消除镜头失真，你可以离开 镜头失真 控制设置为 无镜头失真 .

否则，设置 镜头失真 到 未知镜头 在您解决相机位置之前，强制 CameraTracker 计算失真。

3. 选择 焦距 从下拉菜单键入相机:

• 已知 -如果焦距可用，请选择此选项，并在 长度 控制。

• 近似 变化 -如果有一个近似焦距可用，请选择此选项，并在 长度 控制。

• 近似 常数 -如果近似焦距可用且没有缩放，请选择此选项，并在 长度 控制。

注意: CameraTracker 试图调整焦距的解决，如果你选择一个 近似 选项。

• 未知 变化 -如果焦距未知且正在更改，请选择此选项。

• 未知 常数 -这是默认选项。如果焦距未知且没有缩放，请使用此选项。

4. 要么选择一个 胶片背面预设 从下拉列表中填充 胶片背面尺寸 控件，或者如果您的相机不在列表中，请输入 胶片背面尺寸 手动。

提示: 您可以通过编辑添加自己的默认值 。。/NukeScripts/camerapresets.py Nuke 安装包中的文件。
