设置跟踪参数

首先，您需要调整一组跟踪控制，以获得最佳的跟踪结果。

1。 在 摄影师 选项卡，定义镜头摄影师应该跟踪的部分。

• 如果你想掩盖你的图像的部分，设置 面具 除了其他任何东西 没有 (默认)。有关更多信息，请参见 屏蔽图像的区域 .

• 对于立体或多视图镜头，设置 校长 视图 使用下拉列表。任何遮罩都应该应用于此视图，然后用于计算次摄影机。

• 与 分析 范围, 您可以设置要跟踪和求解的特定范围的帧。选择 源 剪辑 范围 跟踪和解决整个序列或 分析 范围 定义帧范围。输入范围的第一帧和最后一帧 分析 开始 和 分析 停止 分别为盒子。

2. 在 跟踪 选项卡您可以进一步定义特征跟踪的设置。

• 数量 的 特点 -定义要在每一帧中跟踪的特征数。默认为 150 个功能，理想情况下，每帧应该使用 100 多个轨道。在难以解决的序列中，考虑使用更多的特征。

• 检测 阈值 -在输入图像上设置特征的分布。如果输入低检测阈值，将在图像的所有部分均匀跟踪特征。使用 预览 特点 在跟踪之前预览您的功能。

• 特征 分离 -设置彼此之间特征的分布。要强制特征分离并以偶数距离在图像上均匀分布特征，请输入高特征分离值。使用 预览 特点 在跟踪之前预览特征分离。

• 细化特征位置 -检查此以将检测到的特征锁定到局部角。如果你激活它，CameraTracker 会在你的镜头中找到最近的角点，并锁定特征点。

• 预览 特点 -选中此框可预览要跟踪的特征。当您想在跟踪前进一步调整跟踪参数时，预览会派上用场。

预览您镜头上的特征。

• 最小长度 -为拒绝短轨道的最小轨道长度设置阈值。你可能会发现很多短轨道出现在长序列中，相机移动缓慢。

• 轨道 阈值 -将阈值设置在 0 和 1 之间。此阈值控制多个帧上相似特征的外观。您可以调整此值以测试轨道是否可靠。

• 轨道 平滑度 -设置平滑轨道生成的阈值。调整此值有助于防止复杂序列中的不良轨迹。增加 “平滑度” 值可删除随时间推移出现故障的轨迹。

• 轨道 一致性 -为一致的轨道生成设置阈值。增加此值以确保轨迹运动在本地一致。调整一致性以防止复杂序列中的不良轨迹。

• 轨道 验证 -告诉 CameraTracker 轨道应该适合哪种类型的相机运动。使用 免费 相机 和 旋转 相机 针对相机预期运动测试轨迹的选项。选择:

• 免费 相机 -期待一个自由移动的相机，旋转和翻译。

• 旋转 相机 -期待一个静止的相机，只旋转。

• 没有 -不要基于任何特定的相机运动来验证轨迹。当您需要更多的轨迹并希望将所有轨迹保留在图像中时，您可能需要使用此验证选项。

3. 当你快乐的时候 输入 设置，应用任何必要的蒙版，如下所述，然后单击 轨迹特征 在 摄影师 标签。
