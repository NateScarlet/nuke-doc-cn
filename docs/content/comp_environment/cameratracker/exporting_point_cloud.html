Exporting a Point Cloud

You can export a point cloud as a .fbx or .abc (Alembic) file, using the WriteGeo node, by following these steps:

1.   Create a WriteGeo node and attach it to your CameraTrackerPointCloud node.
2.   In the WriteGeo controls, select fbx or abc in the file type dropdown, and make sure you have the point clouds or pointClouds box checked.
3.   Browse to the destination you want to save the file in the file field and give your object a name.
4.   Click Execute.
5.   In the pop-up dialog, specify the frame range you want to include in the file and click OK.

A progress bar displays and the file is saved to the location specified.
