导出点云

您可以将点云导出为 。Fbx 或 。Abc (Alembic) 文件，使用 WriteGeo 节点，按照以下步骤操作:

1。 创建 WriteGeo 节点并将其附加到 CameraTrackerPointCloud 节点。
2. 在 WriteGeo 控件中，选择 Fbx 或 Abc 在 文件 类型 下拉列表，并确保您拥有 点 云 或 点云 复选框。
3. 浏览到要保存文件的目标 文件 字段并给你的对象一个名字。
4. 单击 执行 .
5. 在弹出对话框中，指定要包含在文件中的帧范围，然后单击 好 .

注意: 您应该只为一帧渲染导出的点云，因为默认情况下，使用 CameraTracker 节点或 PointCloudGenerator 节点生成的点云不会随着时间而改变。

显示进度条，文件将保存到指定的位置。
