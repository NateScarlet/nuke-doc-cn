查看轨迹和轨迹信息

您可以更改轨迹信息的显示方式，以帮助您评估轨迹并改进解决方案。在 摄影师 标签，旁边 显示 ,选择:

• 2D 点 -检查以在 2D 查看器中查看围绕重新投影点的圆。将鼠标悬停在一点上可以显示轨迹长度和错误信息。

跟踪 2D 点的信息。

• 3D 标记 -在 3D 模式下，查看您点的锥形 3D 标记。标记直接从查看器中的特征像素继承其颜色。您可以设置标记的尺寸使用 标记 规模 参数。

3D 查看器中的可扩展标记。

• 关键曲目 -选中仅查看在关键帧中可见的特征轨迹。如果要减小点云的大小并显示最可靠的点，这很有用。

查看所有曲目。 仅查看关键轨迹。
