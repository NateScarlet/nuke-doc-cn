缩放场景

可以使用定义场景的比例 制服 规模 和 比例限制 上的参数 场景 标签。请注意，您只能在 2D 视图中以这种方式缩放场景。

1。 使用设置场景比例 均匀刻度 滑块。

如果需要定义更具体的比例，可以添加 规模 约束 手动定义您的比例。执行以下操作:

2. 选择两个 (只有两个) 轨道使用 转变 + 点击浏览器覆盖。
3. 在其中一个轨道上单击鼠标右键，然后选择 场景 > 添加比例距离 .
4. 在 “属性” 面板中，将两个点之间的距离添加到第一个点 距离 行下 规模 约束 。场景的比例会根据您输入的距离进行调整。
5. 例如，如果您发现其中一些约束不如其他约束准确，您也可以删除这些约束。
