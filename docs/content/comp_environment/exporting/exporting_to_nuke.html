4. 选择 基本核弹射击 预设在 本地预设 自动完成快照模板或选择 项目快照模板 要使用 项目设置 模板。

注意: 您可以使用构建自定义快照模板 路径 代币, 内容 字段，以及文件夹和/-按钮。只需复制现有模板并根据需要进行编辑。

基本核弹射击 为每个剪辑或快照创建一个文件夹，其中包含 盘子 文件夹和 Nuke 文件夹包含 脚本 和 渲染 文件夹。

中的令牌 基本核弹射击 模板分解如下:

• {Filename} 只需提取出现在文件浏览器中的源文件名。

• {Shot} _ comp _ {version}.nk 提取每个剪辑的轨道项目名称和在 出口管制 。例如, Shot01_comp_v03.nk

• {Shot} _ comp _ {version}。#。{ext} 附加填充和指定的文件扩展名。例如，Shot01_comp_v03.0001.dpx

注意: 看到的 示例快照模板 有关如何使用快照模板在 Nuke 中创建多个格式写入节点的信息。

5. 单击 Nuke 项目文件 预设以显示 读取节点 和 写 节点 设置。

6. 检查从快照模板中的哪个路径应该用于 Nuke 读写节点。在这种情况下:

• {Shot}/plates/{filename} 在导出过程中解析到符号链接。

• {Shot}/nuke/render/{shot} _ comp _ {version}。#。{ext} 解析渲染路径的步骤 Nuke 工作室 希望找到核弹文件。

8。 单击 核写入节点 预设以显示 文件类型 设置。
9. 选择 文件类型 使用下拉列表进行渲染并完成相关字段，具体取决于 文件类型 已选择。

注意: 选择 Mov 从下拉列表中提供了额外的 QuickTime 特定控件，允许您选择 编解码器 和 YCbCr 矩阵 。矩阵控制使您能够使用新的 Rec 601 和 Rec 709 或的 遗产 编码方法，这是以前在 Nuke 和 Hiero 中使用的方法。

同样，选择 Exr 提供额外的元数据下拉列表，允许您导出或往返选定的元数据以及您的 。Exr 输出。

10. 设置以下所有文件类型通用的控件:

• 色彩空间 -使用下拉菜单设置要渲染的颜色空间，例如 线性 , REDLog ,或 Raw .

• 重新格式化 -选择所需的重新格式化选项以启用 格式 下拉列表。

• 格式 -设置在 Nuke 中渲染的格式，如 1920x1080 高清 1080 .

选择 定制... 创建未出现在预设列表中的格式。

• 调整大小 -设置要保留或覆盖原始纵横比的方法:

• 宽度 -缩放原始文件，直到其宽度与格式的宽度相匹配。然后以保持原始纵横比的方式缩放高度。

• 高度 -缩放原始文件，直到其高度与格式的高度相匹配。然后以保持原始纵横比的方式缩放宽度。

• 适合 -缩放原始图像，直到其最小边与格式的最小边相匹配。原始的较长的边然后以这样一种方式缩放，以保持原始的纵横比。

• 填充 -缩放原始图像，直到其最长边与格式的最长边相匹配。然后以保持原始纵横比的方式对输入的较短侧进行缩放。

• 扭曲 -缩放原始图像，直到其所有边与格式指定的长度匹配。此选项不保留原始纵横比，因此可能会发生扭曲。

• 通道 -使用下拉列表设置要导出的通道。

注意: 默认情况下, 所有 ,导出图像中的所有通道，因此创建的任何 Nuke Read 节点都具有 自动阿尔法 默认启用。

如果要导出非标准通道，请手动在字段中输入通道名称。

• 写入节点名称 + 如果你打算创建多个 Nuke 写节点，请在这里定义名称。默认情况下, Write _ {ext} ,将正在写入的文件扩展名附加到各个写入节点。但是，您可以使用任何令牌 Nuke 工作室 识别。

• 刻录小发明 -启用时，文本刻录将使用 Nuke gizmo 应用于媒体。单击 编辑 定义在老化过程中应用的信息。

请参阅 将刻录文本添加到导出中 欲了解更多信息。

11。 设置 手柄 ,如果需要:

• 夹子长度 -导出可用的完整剪辑长度，就像剪辑作为查看器打开一样。

• 切割长度 -仅导出时间表上包含的削减。

注意: 选择 切割长度 允许您向每个剪辑添加手柄，直到最大可用源剪辑长度。

12. 检查 应用 Retimes 导出时间轴上存在的任何 retimes。
13。 设置如何剪辑 开始帧 使用下拉菜单导出:

• 源 -使用源剪辑的开始帧。

• 自定义 -使用右侧的字段为所有剪辑指定开始帧。

14. 设置 版本 号码, 此出口的轨迹 ,和 标签过滤器 ,如果适用。

注意: 请参阅 使用版本 有关版本控制如何在中工作的更多信息 Nuke 工作室 .

15。 如果您使用的是渲染场，请从 渲染背景任务 下拉列表。

Nuke 工作室 需要一个 Python 脚本将导出传递到您选择的渲染场。脚本必须位于特定目录中，取决于平台，如中所示 启动期间运行 Python 。如果没有脚本，下拉列表将被禁用。

一些示例脚本包含在 Nuke 工作室 包在以下目录中:

• 在 Linux 上:/usr/local/ Nuke 工作室 9.0v1 /Plugins/site-packages/hiero/examples/

• 在 Windows 上: 驱动器号: \ 程序文件 \ 铸造 \ Hiero 9.0v1 \
插件 \ 站点包 \ hiero \ 示例

16。 单击 出口 .

的 导出队列 窗口显示每个导出组件预计需要多长时间的估计。

导出完成后，将创建快照模板中指定的文件结构。然后，当核弹剪辑准备就绪时，您可以在单独的轨道上导入它们。

当剪辑从 Nuke 工作室 此外，它们还标有核弹标签，标记哪些剪辑有出口历史。以这种方式标记的剪辑可以用来快速构建 VFX 轨道，如 building 中所述 从导出标签跟踪 .
