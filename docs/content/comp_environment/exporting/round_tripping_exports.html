Nuke 工作室 美国的出口套件可以通过 Nuke 自动往返，在两者之间来回发送剪辑 Nuke 工作室 和 Nuke 使用 Nuke 工作室 <> 核弹，转码，从时间线或 bin 中导出剪辑选择，写出 EDLs 和 XMLs，或者以所需的交付格式将整个时间线烘焙为一个剪辑。

通过添加 Python 绑定来执行相同的功能，这个基础设施提供了大量的灵活性，无论您的管道是 GUI 还是命令行导向的。

注意: Nuke 工作室 是无损的，如果你设置你的快照模板来镜像现有的文件结构，它可以插入你的管道。

Nuke 工作室 附带几个上下文相关和临时导出选项:

• 使用导出对话框进行往返 -准备出口项目的过程，并为 VFX 工作重新进入铺平道路 Nuke 工作室 .

• -使您能够在 Nuke 脚本中打开选择，从 bin 剪辑或跟踪项目，将剪辑和跟踪项目添加到现有脚本，并将渲染的写入节点从 Nuke 发送回 Nuke 工作室 .

• 高级发送到 Nuke -单个剪辑或 bin 导出 不需要导出对话框或 Hiero <> Nuke 的选项。

• 临时出口 -一个保护伞，涵盖了您可能无法在每个项目的基础上执行的导出，如 EDL 或 XML 导出。
