导出对话框简介

Nuke Studio 使用快照模板和处理器来执行一些导出操作，包括往返和 EDL/XML 创建。的 出口 对话框控制导出的内容和位置，以及是否期望版本控制剪辑作为从 Nuke 往返的一部分。

两种类型的 预设 可用于构建常用的导出任务:

• 本地预设 -使用 XML 文件格式保存在任务预设文件夹中。

• 项目预设 -你可以拖放 本地预设 在此面板中保存项目中的预设 。Hrox 文件。此选项专为协作工作而设计，允许您快速共享导出预设。

注意: 如果您选择 作为镜头处理 ,你会注意到有一个锁 项目快照模板 包括在 项目预设 。此预设只能从主窗口编辑 项目 > 编辑设置 菜单。请参阅 使用快照模板 欲了解更多信息。

Nuke Studio 还使用 内容预设 在所有快照模板中，允许您创建常用的导出条件，然后在所有项目中可用。某些预设仅适用于某些处理器，例如 EDL 出口商 预设不能与一起使用 作为镜头处理 .

可以过滤出口用 此出口的轨迹 和 标签过滤器 列表，仅导出特定标记的某些轨迹或轨迹项目。请参阅 使用标签 欲了解更多信息。

为了方便起见，Nuke Studio 附带了许多现成的预设，但您可以根据需要编辑这些预设:

• 转码图像 -定义转码参数，允许您保存最常用的文件类型转换。

• Nuke 项目文件 -定义 Nuke 在往返过程中读写节点使用的路径。

• 核写入节点 -定义 Nuke 写入节点的渲染格式。添加多个 Nuke 写入节点预设，以在 Nuke 中创建多个写入节点。

• 外部渲染 -定义第三方应用程序 (如 Adobe) 使用的导出路径®后影响®,在往返途中。

的 外部渲染 预设在模板中用作标记，以便其他任务可以找到外部渲染的位置，并可以编写自己的自定义导出器。你可以找到一个例子的标记功能 例子 目录，称为 FnExternalRender.py ,以及 Nukepedia 上的定制出口商示例 ( Www.nukepedia.com )。

请参阅 加载 Gizmos 、 NDK 插件以及 Python 和 TCL 脚本 有关在哪里可以找到的更多信息 例子 目录。

• EDL 出口商 -用于将序列导出为 EDL 格式。

• 符号链接导出器 -创建指向源文件位置的符号链接，而不是复制。

• XML 导出器 -用于将序列导出为 XML 格式。

• 复制导出器 -创建源文件到指定位置的副本，而不是符号链接。

• 音频导出 -将任何音轨复制到 。Wav 指定位置的文件。
