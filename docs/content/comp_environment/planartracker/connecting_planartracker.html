连接 PlanarTracker 节点

PlanarTracker 节点接受两个输入，其中一个 ( 源 输入) 是强制性的。Roto 节点自动创建与 PlanarTracker 节点，并连接到 源 输入。

连接 PlanarTracker 节点

要连接 PlanarTracker 节点:

1。 创建 PlanarTracker 节点 ( 变换 > PlanarTracker )。还创建了一个旋转节点，这样你就更容易开始创建你的飞机。
2. 将要跟踪的镜头连接到 Bg Roto 节点的输入。
3. 始终保持 PlanarTracker 连接到其关联的 Roto 节点。这是为了更容易绘制和分层各种平面，也允许您保留跟踪结果，但在完成跟踪后删除 PlanarTracker 节点。
4. 您可以使用 PlanarTracker 节点 面具 其他遮罩的输入，如色度键的结果。使用蒙版有助于进一步定义跟踪区域。
