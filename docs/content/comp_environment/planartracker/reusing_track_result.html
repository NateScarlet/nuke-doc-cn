重用跟踪结果

可以重复使用整个跟踪平面或单个跟踪。

重复使用跟踪飞机

您可以使用已跟踪的平面来分析位于同一平面上的较大平面。

1。 确保你和你以前画的形状在同一个框架里。
2. 在与以前跟踪的形状相同的平面上绘制形状。确保新形状是相同的 PlanarTrackLayer 作为笔触/形状列表中的第一个。
3. PlanarTracker 现在已经将您的新 roto 形状放置在与旧 roto 形状相同的平面上。如果你现在在时间线上擦洗，你会发现你的新形状与相机运动相吻合。

重用单个轨道

您还可以重复使用跟踪点，以验证整体结果是否与您的镜头一致。

1。 跟踪下所述的飞机 追踪一架飞机 ,并在必要时调整结果。
2. 当您对该飞机的跟踪结果感到满意时，您可以拖动跟踪形状的一个角，并将其放置在可能无法正确跟踪的特定细节上。请注意，您正在本地移动旋转形状，并且不会影响实际轨迹结果。
3. 擦洗框架，看看该点是否像你期望的那样停留在细节上。
4. 如果发生漂移，拖动该点以纠正它。
