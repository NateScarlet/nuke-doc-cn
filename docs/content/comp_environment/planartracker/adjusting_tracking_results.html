调整跟踪结果

您可以通过多种方式调整跟踪结果。

向平面添加平滑

要向平面添加平滑:

1。 检查 启用 抖动 平滑 允许 PlanarTracker 检测平面表面拐角周围的抖动并将其平滑。这有助于通过轻微平滑来改善结果，应用严重平滑可能会产生意想不到的结果。
2. 调整 平滑量 控制将抖动控制应用于平面表面拐角的程度。零值禁用抖动平滑，值 1 应用最大平滑。

调整跟踪控制

有时你的跟踪结果可能需要一些调整，然后你才会满意，你可以在 跟踪 PlanarTracker 属性面板中的选项卡。

1。 看看你的跟踪前后的轨迹。检查:

• 预览 特点 -检查以预览要跟踪的功能。当您想在跟踪前进一步调整跟踪参数时，预览会派上用场。

• 显示特征 轨道 -检查以显示跟踪后产生的轨迹。

2. 调整跟踪特征的放置方式。设置:

• 数量 的 特点 -定义要在每一帧中跟踪的特征数。默认为 100 功能。在难以解决的序列中，考虑使用更多的特征。

• 检测 阈值 -在输入图像上设置特征的分布。如果输入低检测阈值，将在图像的所有部分均匀跟踪特征，您可能会获得更准确的结果。例如，对于平坦的模糊区域，即使值为 0.005 也可能有用。使用 预览 特点 在跟踪之前预览您的功能。

• 特征 分离 -设置彼此之间特征的分布。要强制特征分离并以偶数距离在图像上均匀分布特征，请输入高特征分离值。如果你正在处理跟踪一个非常小的物体的镜头，或者一个结构很薄的物体，大约 3 或 5月4日的特征分离值是有用的。

• 精炼 特征 位置 -通过将检测到的特征锁定到局部角点来细化检测到的特征。

3. 调整跟踪过程。设置:

• 轨道 阈值 -将阈值设置在 0 和 1 之间。此阈值控制多个帧上相似特征的外观。您可以调整此值以测试轨道是否可靠。

• 轨道 平滑度 -设置平滑轨道生成的阈值。调整此值有助于防止复杂序列中的不良轨迹。增加 “平滑度” 值可删除随时间推移出现故障的轨迹。

• 轨道 一致性 -为一致的轨道生成设置阈值。增加此值以确保轨迹运动在本地一致。调整一致性以防止复杂序列中的不良轨迹。

4. 使用 校长 视图 下拉菜单选择在立体声项目中工作时要使用的视图。
