使用 UVProject 节点投影纹理

UVProject 节点设置对象的 uv 坐标，允许您将纹理图像投影到对象上。如果对象已经具有 uv 坐标，则此节点将替换它们。

使用 UVProject 节点

1。 选择 3D > 修改 > UVProject 要插入 UVProject 节点位于要修改的 3D 对象之后的任何位置。
2. 将查看器附加到节点以查看您的更改。
3. 在节点的控件中，使用 显示 下拉菜单，选择在对对象进行更改时在查看器中查看对象的方式。
4. 将轴或相机节点连接到 UVProject 节点的轴/凸轮输入。如果连接轴节点，请使用轴变换值 (即平移、旋转、比例等) 将纹理 UV 坐标投影到对象上。).如果连接相机节点，则执行与轴类似的投影，但也使用相机镜头信息，如光圈。
5. 调整以下参数:

• 从 投影 下拉菜单中，选择投影类型。通常，最好选择一个接近对象表面形状的类型。例如，如果您的对象是球体，如足球或行星，请选择 球形 .

• 从 飞机 下拉菜单中，选择投影方向: XY , YZ ,或 ZX 沿 z 、 x 或 y 轴投影纹理图像。此下拉菜单仅在您选择 平面 作为投影类型。

• 从 项目 上 下拉菜单，选择 两者都 , 前面 或 回来 取决于您是要将纹理投影到对象的正面、背面还是两者。物体的正面是面向相机的，同样，背面是离相机最远的。

• 检查 查看圆台剔除 如果希望 UVProject 节点仅影响摄影机视图视锥内的顶点。视锥外的任何顶点都不会受到影响，它们仍然保持其原始 UV 坐标。如果希望节点影响所有顶点，请取消选中。

• 要在水平方向镜像纹理 UV 坐标，请选中 反转 u 。要在垂直方向上镜像它们，请选中 反转 v .

• 要在水平方向缩放 (拉伸或挤压) 纹理 UV 坐标，请调整 U 刻度 滑块。要在垂直方向上缩放它们，请调整 V 刻度 滑块。值越高，纹理拉伸越多。

• 要更改用作顶点 UV 坐标以查找图像像素的属性的名称，请在 Attrib 名称 字段。
