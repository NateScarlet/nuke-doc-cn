使用图像方法 2 修改对象

像 DisplaceGeo 节点一样，置换着色器节点也执行置换映射，乍一看，节点看起来非常相似。然而，他们在执行位移映射的方法是不同的。

置换贴图是一种在渲染对象曲面时向它们添加几何细节的技术。与 “置换” 节点不同，“置换” 节点可以动态地这样做，只置换几何图形中在任何给定时刻可见的部分。置换考虑相机的视点，以确定置换的哪些部分需要渲染，从而节省渲染时间。也可以优化细分的级别，使其成为特定距离的对象所需的级别。

连接位移节点

1。 通过单击创建置换节点 3D > 着色器 > 位移 .
2. 将几何图形连接到置换节点的输出。如果需要，可以将纹理连接到置换节点的输入。
3. 连接要从中的创建置换的图像 位移 输入。
4. 或者，可以使用单独的贴图来计算法线。将此连接到 法线 输入。
5. 继续到 调整位移控制 下面。

调整位移控制

1。 使用 位移 通道 下拉菜单从您的 位移 要用作置换贴图的输入。
2. 如果你用的是 法线 输入，取消选中 构建 法线 和设置 正常扩张 至:

• 没有 要使用法线,

• XY 在 x 和 y 维度上乘以它们,

• XYZ 将它们乘以 x 、 y 和 z 尺寸。

3. 使用 规模 设置位移的总体比例。
4. 设置 过滤器 尺寸 采样输入图像时要使用的过滤器的大小。
5. 使用 过滤器 下拉菜单选择过滤算法。有关更多信息，请参见 选择过滤算法 .
6. 如果要在位移后自动计算法线，请选中 构建法线 。取消选中这个，如果你想从 法线 输入。
7. 继续到 调整渲染的位移控制 下面。

调整渲染的位移控制

在渲染使用置换的场景之前，请调整 镶嵌 选项卡以获得渲染过程的速度和质量:

1。 使用 最大细分 设置细分中发生的多边形细分的最大迭代次数。

如果您习惯于在其他应用程序中使用 DisplaceGeo 或 deplacement，值得注意的是，您可以使用 deplacement 节点通过较低的几何细分获得类似数量的细节。

例如，30x30 卡片是由 1800年三角形 (30x2) 的镶嵌构成的, 但是，您可以使用更少的卡细分的置换节点来实现类似的细分级别:

注意: 使用大量几何图形 和 位移细分可能会减慢渲染速度，到目前为止，它变得不可用 -- 10x10 卡 最大细分 设置为 4 生成 51200 个三角形!

2. 设置 模式 下拉菜单到多边形细分中使用的模式:

• 制服 -均匀多边形镶嵌。这是一个很好的选择，主要用于测试结果，只有在极少数情况下，才是实际用于置换渲染的最佳选择。

• 屏幕 -细分由屏幕大小决定。这是默认的，通常是最佳模式选项。镶嵌由屏幕上镶嵌多边形的大小决定。此模式可确保在达到特定多边形边缘屏幕长度后不会创建新多边形。

• 自适应 -细分是由位移的复杂性决定的。此选项尝试估计图像中不需要镶嵌的平面区域。计算基于阈值控制，只有在 自适应 模式被选中。

3. 如果你设置 模式 到 屏幕 或 自适应 ,设置 像素边缘长度 细分中使用的多边形的最大大小。
4. 如果你设置 模式 到 自适应 ,您也可以调整以下内容:

• 边缘阈值 -大于该阈值的边缘被分割，而小于该阈值的边缘根据正常阈值和置换阈值被细分。

• 正常阈值 -检测法线方向以确定曲面是否平坦。如果相邻法线之间的角度大于此阈值，将发生镶嵌。

• 置换阈值 -比较曲面上两个点的偏移程度，如果结果不匹配，将发生镶嵌。

注意: 请记住，将置换着色器应用于非常复杂的高分辨率几何图形可能非常慢。
