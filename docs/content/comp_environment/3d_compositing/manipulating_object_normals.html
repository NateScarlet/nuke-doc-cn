操纵对象法线

对象法线是垂直于曲面的向量。它们用于照明计算，以确定光线在任何特定点如何从表面反弹。通过操纵它们，您可以控制漫反射和高光的贡献。

操纵对象法线

1。 选择 3D > 修改 > 法线 在要调整其照明的 3D 对象之后的任何位置插入法线节点。
2. 将相机、轴或灯光节点连接到法线节点 看着 输入。
3. 在法线控件中，打开 行动 下拉菜单并选择:

• 不变 不要做任何改变。

• 设置 将法线值指定给法线 x 、 y 和 z 字段。

• 构建 根据周围的顶点重建每个法线。调整 阈值角度 滑块用于确定两个面不再构成光滑表面的折断角度。角度为 0 表示所有面都是平的，而 180 表示所有面都是平滑的。一个好的平均设置是 60。

• 看着 将所有法线指向法线节点的 看着 输入。

• 删除 从对象中删除命名属性。例如，如果删除 N 属性，则对象没有法线。
