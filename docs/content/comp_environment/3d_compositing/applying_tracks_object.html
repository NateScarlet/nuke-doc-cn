将轨迹应用于对象

Nuke 可以导入通道文件并将运动数据应用于任何相机或对象的变换参数。最常见的目的是模拟实际的相机沿着定义的路径移动或移动对象。

通道文件包含给定镜头中每个动画帧的一组笛卡尔坐标。此信息由 3D 跟踪软件 (如 3D 均衡器、 Maya 或 Boujou) 计算，然后导出为通道文件。

将通道文件应用于对象

1。 双击对象或相机节点以显示其参数。
2. 单击 导入 chan 文件 。将出现文件导航对话框。
3. 导航到通道文件，然后单击 好 .
4. Nuke 读取通道数据并显示有关导入的数据帧数量的状态消息。您还会注意到对象的平移参数变成绿色，表示这些参数现在由动画数据控制。擦洗查看器，您会注意到对象或相机现在会根据从通道文件导入的转换数据移动。

注意: 您可以使用 导出 chan 文件 按钮，将应用于给定对象的任何动画转换参数导出为 chan 文件。这是艺术家之间共享设置的有用方法。

提示: 您还可以使用通道文件将在其他应用程序中创建的相机导入 Nuke 。但是，由于 chan 文件格式不是标准文件格式，您可能需要一个文件格式转换器来从其他应用程序导出 chan 文件。
