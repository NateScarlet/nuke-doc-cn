使用 DepthToPoints 节点创建点云

DepthToPoints 是一个 gizmo，包含 DepthToPosition 和 PositionToPoints 节点。它可用于从深度通道和 3D 相机生成 3D 点云。DepthToPoints 采用图像文件中包含的深度数据和颜色信息，并将图像重新创建为三维点云。

使用 DepthToPoints 创建点云

1。 从 3D > 几何 菜单，选择 深度图 将 gizmo 添加到脚本中。
2. 读取带有深度通道的图像并将其连接到 图像 Gizmo 的输入。
3. 从 3D 菜单中，选择一个 相机 并将其连接到 相机 深度图输入。

4. 在 DepthToPoints 用户 选项卡上，从 深度 下拉菜单。如果您的频道包含普通频道，请从 表面正常 下拉菜单，或者将包含正常数据的图像连接到 规范 节点的输入。现在，您应该可以在 3D 查看器中看到点云。
5. 要更改点数，请调整 点细节 滑块。值为 0 表示不显示任何点。值为 1 将显示所有可用点。
6. 要更改点的大小，请调整 点大小 滑块。

提示: 如果您在查看图像和点云时遇到问题，请确保图像的 alpha 没有设置为黑色。
