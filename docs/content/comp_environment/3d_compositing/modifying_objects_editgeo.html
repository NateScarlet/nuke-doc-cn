使用 EditGeo 节点修改对象

EditGeo 允许您直接修改对象的顶点或面，具体取决于当前活动的查看器选择模式。如果 EditGeo 位于具有多个几何输入的 MergeGeo 节点的下游，也可以同时修改多个对象。

使用 EditGeo 修改对象

1。 在节点图中选择要修改的 3D 对象或 MergeGeo。
2. 导航到 3D > 修改 > EditGeo 插入 EditGeo 节点。
3. 在节点的控件中，使用 显示 下拉菜单，选择在对对象进行更改时在查看器中查看对象的方式。

提示: 如果您没有与几何图形关联的图像或纹理，您可能会发现启用 前照灯 Viewer 属性中的控件提高了可见性。

新闻 S 在查看器中显示查看器属性，然后启用 3D > 前照灯 复选框。

4. 使用查看器右上角的 3D 选择工具设置查看器选择模式:

• 顶点选择 -允许您选择几何图形上的单个顶点，在编辑时提供精细控制。您也可以选择多个顶点或通过按住 转变 .

• 面选择 -允许您选择几何图形上的各个面。您也可以选择多个面或通过按住 转变 .

注意: 你可以关掉 遮挡测试 通过点击 查看器中的禁用图标允许您选择顶点和
从视点隐藏的面。

例如，在禁用遮挡测试的立方体的 “正面” 上选择面也会选择立方体的相反隐藏侧的面。


请参阅 3D 选择工具 欲了解更多信息。

5. 设置轴控制柄如何与您的选择对齐 轴 对齐 下拉列表:

• 对象 -Xyz 轴的位置由选择中所有顶点的平均位置决定。轴的方向与对象的方向相同。

• 平均正常 -Xyz 轴的位置由选择中所有顶点的平均位置决定。轴的方向与当前选择的法线的平均值对齐。

在这种模式下，z 轴总是直接指向远离当前选择的位置。

对象对齐 平均正常对齐

提示: 持有 Ctrl / Cmd Alt 允许您通过几何图形的曲面拖动轴的中心，方向设置为最近的面法线。

6. 根据需要在查看器中进行顶点或面选择。保持 转变 添加到选择或 Alt + 转变 删除选择。
7. 当您对选择感到满意时，将与选择关联的轴拖到新位置以编辑几何图形。

动画编辑

1。 使用查看器右上角的 3D 选择工具设置查看器选择模式:

• Vertex selection - allows you to select individual vertices on geometry, giving you fine control while editing. You can also marquee select multiple vertices or use additive selection by holding down Shift .

• Face selection - allows you to select individual faces on geometry. You can also marquee select multiple faces or use additive selection by holding down Shift .

2. 在查看器中进行选择，然后单击 在属性面板中。
3. 将播放头擦洗到所需的帧，并使用查看器中的手柄变换几何图形。

关键帧会自动添加到新帧上。

4. 重复该过程以生成所需的动画。

提示: 单击 删除当前帧处的关键帧。

5. 单击 重置几何图形 将顶点返回到其原始位置。
