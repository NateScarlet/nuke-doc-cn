使用失真函数修改对象

1。 选择 3D > 修改 > 径向扭曲 在要修改的三维对象之后的任何位置插入 “径向扭曲” 节点。
2. 将查看器附加到节点以查看您的更改。
3. 在节点的控件中，使用 显示 下拉菜单，选择在对对象进行更改时在查看器中查看对象的方式。
4. 要选择失真是桶形失真还是销形失真，请调整 失真 滑块。低于 0 的值产生桶形失真，而高于 0 的值产生销垫形失真。如果将值设置为 0，3D 对象不会扭曲。
5. 要控制失真的大小，请调整 功率偏压 滑块。值越高，对象就会变得越扭曲。
6. 要移动扭曲的中心点，请在 旋转中心 字段。
7. 要控制每个 x 、 y 或 z 方向的失真量，请调整 规模 字段。
8。 要将对象的中心保持在 3D 空间中的原始位置，请选中 保存中心 .
