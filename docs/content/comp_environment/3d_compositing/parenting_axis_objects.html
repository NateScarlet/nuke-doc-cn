对轴对象的养育

轴对象通过添加一个新的转换轴作为空对象，其他对象可以作为父对象。即使对象已经有了自己的内部轴，有时在单独的轴中作为父轴也是有用的。

例如，轴节点已成为场景中其他对象的父对象 (两个图像平面和摄影机)。结果是一个轴对象 这在全球范围内控制了场景。例如，旋转它会旋转场景中的所有对象，如下图所示。

用轴对象旋转整个场景。

提示: 要将几个对象一起移动，还可以使用 MergeGeo 节点合并它们 (请参见 合并对象 )，然后使用 TransformGeo 节点控制它们 (请参见 使用 TransformGeo 节点 )。

添加轴对象的步骤

1。 单击 3D > 轴 要插入 轴节点。
2. 要创建父关系，请将轴节点连接到您希望使用轴转换控件控制的所有对象节点。

要创建嵌套的转换层次结构，请将附加轴节点链接到插入的第一个轴节点。例如，可以创建三轴节点的层次结构，以控制旋转、缩放和变换。

在上面的例子中, Axis3 (旋转) 连接到 Axis2 (比例)。 Axis2 连接到 Axis1 (变换)，和 Axis1 连接到所述 TransformGeo 节点 (s)，你想要的影响。以这种方式连接轴节点后，它们的变换数据会沿着链起伏，并添加到 TransformGeo 节点的设置中。

使用外观输入

您可以使用可选的 看 轴节点的输入，以便轴自动朝连接的输入旋转。可以将相机、灯光或轴节点附加到 看 输入。例如，您可以将相机节点连接到 看 输入，使轴旋转到相机，无论它被移动。请参阅 插入直接灯、点灯或聚光灯 或 使用相机 举个例子。
