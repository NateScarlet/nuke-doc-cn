使用 Project3D 节点投影纹理

Project3D 节点通过相机将输入图像投影到 3D 对象上。

使用 Project3D 节点

1。 选择 3D > 着色器 > Project3D 要插入 要投影的图像后的 Project3D 节点。将相机节点连接到 Project3D 节点的 cam 输入。
2. 在 Project3D 节点后插入 3D 几何节点 (例如球体)。
3. 将查看器附加到 3D 几何节点以查看更改。
4. 在节点的控件中，使用 显示 下拉菜单，选择在对对象进行更改时在查看器中查看对象的方式。
5. 从 项目 下拉菜单中，选择将图像投影到正面、背面或两个多边形上。
6. 要在输入图像的边缘用黑色扩展，请选中 作物 。要使用边缘颜色扩展图像，请取消选中 作物 .
7. 如果你想使用光线投射来测试投影并找出投影的哪些部分被遮挡，你可以使用遮挡模式下拉列表。选择:

• 没有 -禁用遮挡测试。

• 自我 -告诉 Project3D 只有连接到它的几何图形才能导致遮挡。

• 世界 -告诉 Project3D 场景中的其他对象可能会导致遮挡。
