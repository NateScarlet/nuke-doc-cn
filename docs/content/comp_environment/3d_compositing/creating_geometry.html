从头开始创建点云和几何图形

除了使用内置的基本体和导入的几何图形, Nuke 还允许您从头开始创建密集的点云和几何图形。

本节重点介绍如何使用中的 DepthToPosition 、 positiontoints 和 DepthToPoints 节点创建点云 Nuke 。然而，如果你有一个 NukeX 许可证中，您还可以使用 CameraTracker 和 PointCloudGenerator 节点创建点云，或使用 ModelBuilder 节点为 2D 快照构建 3D 模型。请参阅 创建密集的点云 和 使用 ModelBuilder 更多细节。
