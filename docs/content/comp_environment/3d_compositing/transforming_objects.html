变换几何图形、相机和灯光

变换操作包括移动、缩放、旋转 3D 场景中的对象。当对象节点处于活动状态时，可以在节点参数中输入特定的变换设置 (请参见 从节点属性面板转换 )，或者使用 3D 查看器中显示的变换手柄直接操作对象 (请参见 使用变换句柄 )。转换从对象的轴点位置发生 (请参见 变换和枢轴点 )。

Nuke 允许您将多个对象变换在一起，或者使一个对象始终面向另一个对象 (请参见 对轴对象的养育 和 使用 TransformGeo 节点 )。

还可以将变换参数链接到导入的轨迹或相机数据 (请参见 将轨迹应用于对象 )，或者使用动画曲线控制变换。

使用 FBX 或 Alembic 文件，可以从第三方 3D 应用程序 (如 Maya) 导入变换 (请参见 从 FBX 文件导入转换 和 从 Alembic 文件导入转换 )。
