使用 TransformGeo 节点

TransformGeo 节点允许您在与 MergeGeo 节点合并在一起的几个对象上移动、旋转、缩放和执行其他转换。它还允许将几何对象连接到轴节点。通过这样做，可以使用轴变换控件将所有连接的对象一起移动。您所需要做的就是在每个几何对象后插入一个 TransformGeo，将轴节点连接到 TransformGeo 节点 轴 输入，并调整轴节点的变换控件。有关更多信息，请参见 对轴对象的养育 .

TransformGeo 节点的另一种用途是使一个对象的旋转取决于另一个对象的位置，这样第一个对象总是旋转到面向或 “看” 第二个对象。例如，可以让球体对象始终面向圆柱体对象，而不管圆柱体的位置如何。当圆柱体移动到新位置时，球体会自动旋转以面对它。

让一个 3D 对象总是面对另一个

1。 选择要面向另一个对象的三维对象节点 (例如，球体)。
2. 选择 3D > 修改 > TransformGeo 插入 TransformGeo 节点。
3. 选择您希望第一个对象面对的对象 (例如，圆柱体)，并在此节点之后插入 TransformGeo。
4. 将第一个 TransformGeo 节点连接到 看 第二个 TransformGeo 节点的输入。

5. 打开第一个 TransformGeo 节点的控件，然后转到 看 标签。

6. 从 看轴 下拉菜单中，选择对象旋转的轴以面对另一个对象:

Z Y

X -Z

-Y -X
7. 使用 旋转 X , 旋转 Y ,和 旋转 Z 选中复选框以选择对象旋转的轴。要使第一个对象真正面向第二个对象，您需要选中所有三个复选框。
8。 调整 看起来力量 滑块来定义旋转的延伸。值越小，对象的旋转越少。将值设置为 0 不会产生旋转。
9. 如果要使用替代方案来计算旋转，请选中 使用四元数 。这可能对沿选定区域平滑不稳定的旋转很有用 看轴 .

如果现在调整第二个 TransformGeo 节点的变换控件，您将注意到第一个对象自动旋转，以面向第二个对象。有关如何调整变换控件的更多信息，请参见 使用变换句柄 和 从节点属性面板转换 .
