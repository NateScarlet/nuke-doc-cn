导出几何图形、摄影机、灯光、轴或点云

您可以将几何图形、相机、灯光、轴和点云导出到 FBX ( 。Fbx ) 或 Alembic ( 。Abc ) 使用 WriteGeo 节点的文件。

1。 创建 WriteGeo 节点并将其附加到场景节点。
2. 在 WriteGeo 控件中，选择 Fbx 或 Abc 在 文件 类型 下拉列表，并检查要导出的对象:

• 几何形状 -将场景几何图形写入文件。

• 相机 -将场景摄像机写入文件。

• 灯光 -将场景灯写入文件 ( 。Fbx 仅)。

• 轴 -将场景轴写入文件。

• 点云 -将场景点云写入文件。

3. 如果您正在编写 FBX 文件:

• 检查 Ascii 文件 格式 如果你不想写出一个二进制 。Fbx 文件。

• 检查 动画 网格 顶点 如果要对网格顶点进行动画处理，并在每一帧中创建关键帧。动画网格对数据使用顶点缓存。一个目录 _ Fpc 将创建附加到文件名以包含点缓存。默认情况下，“动画网格” 复选框处于关闭状态，因此不会创建点缓存目录。

4. 如果您正在编写 Alembic 文件，请选择 存储格式 要在写入文件时使用:

• HDF -保持向后兼容性的存储格式。

• 小川 -一种存储格式，提供更快的文件读取速度和更小的文件。

5. 浏览到要保存的目标 。Fbx 或 。Abc 文件中的 文件 字段并给它一个名字。
6. 检查 限制范围 如果要在指定帧范围之外执行时禁用节点，请选中此框。在 帧范围 字段中，输入要使用此写入节点进行可执行的帧范围。
7. 选择要导出的视图，如立体项目的左视图和右视图。
8。 单击 执行 .
9. 如有必要，可以使用弹出对话框更改要包含在文件中的帧范围，然后单击 好 .

将显示进度条，选定的文件将保存到指定的目录。
