用恒定颜色替换材料通道

FillMat 节点允许您用恒定颜色替换选定的材质通道。通常，您会使用此节点使一个对象容纳其他对象。当您将 FillMat 颜色设置为 0 时，它充当 “3D 饼干切割器”，并在材料原本所在的地方制造一个黑洞。

立方体前面的球体。 同样的场景
球体材料的 rgba
通道设置为黑色
使用 FillMat 节点。

之后的 alpha 通道
应用 FillMat 节点。

这类似于使用黑色常数节点作为输入纹理。然而，使用 FillMat 节点的优点是，除了 rgb 通道之外，您还可以轻松地将其应用于 alpha 通道。另一个优点是 FillMat 节点不会破坏着色序列，因此您可以将其插入到节点树中的其他材质节点之后。

用恒定颜色替换选定的材质通道

1。 选择 3D > 着色器 > FillMat 在用于曲面纹理的 2D 图像和创建曲面的 3D 对象节点之间插入 FillMat 节点。

2. 在 FillMat 控件中，使用 通道 用于选择要替换为固定颜色的通道的控件。
3. 使用 颜色 控件选择恒定颜色。默认情况下，此设置为黑色 (0)。
