使用 DepthToPosition 节点创建位置传递

DepthToPosition 获取包含在图像文件中的深度数据和相机数据，以创建 2D 位置 (xyz) 通道。此过程是通过通过相机投影深度并记录每个投影点的 xyz 位置来创建的。DepthToPosition 节点可以与 PositionToPoints 节点一起使用来创建点云，类似于 DepthToPoints 节点的效果。

使用深度定位创建位置通道

1。 从 3D > 几何 菜单，选择 深度定位 将节点添加到脚本中。
2. 读取带有深度通道的图像并将其连接到 图像 节点的输入。
3. 从 3D 菜单中，选择一个 相机 并将其连接到 相机 深度定位节点的输入。

4. 在 深度定位 “深度定位” 节点的选项卡上，从 深度 下拉菜单。现在，您应该可以在查看器中看到位置传递。

5. 如果你想改变的输出通道，选择您要，或创建自己的, 输出 下拉菜单。
6. 设置 远 控制值，以指定在与相机的距离处忽略深度值。这可以防止大深度值因浮动数学计算的精度而产生不必要的条带。

DepthToPosition 节点允许您从图像的深度通道创建位置通道，并将其馈送到 PositionToPoints 以最终创建点云。事实上，DepthToPosition 、 PositionToPoints 和 DepthToPoints 这三个节点可以一起使用，因为 DepthToPoints 是一个 gizmo，包含 DepthToPosition 和 PositionToPoints 节点。

要了解更多关于生成深度图的信息，请参考 生成深度图 .
