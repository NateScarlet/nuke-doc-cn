设置场景

每个 3D 场景包括以下对象: 场景节点、相机节点、一个或多个几何节点 (即卡片、球体、 obj) 和 ScanlineRender 节点。3D 场景的示例显示在 使用 2D 和 3D 操作员的脚本 和下图。在中所示的示例中 使用 2D 和 3D 操作员的脚本 ,场景节点接收来自两个几何节点的输出 ( Card1 和 Card2 ) 并将这些对象的组合发送到 ScanlineRender 节点，在该节点中，输出被转换回 2D。

3D 复合材料的核心节点。

脚本可能包含多个场景节点、摄影机和 3D 渲染节点。在 “特性” 框中加载的所有 3D 对象都将出现在 3D 查看器中，无论它们是否连接到同一场景节点。

场景节点

不管它在脚本中的位置如何，场景节点都是场景层次结构中的最高级别节点，因为它引用了 3D 工作区中的所有元素 -- 所有几何对象、相机、和材料。

添加场景节点的步骤

选择 3D > 场景 从工具栏中。

ScanlineRender 节点

脚本中的每个场景节点都应该连接到一个 ScanlineRender 节点，该节点告诉 Nuke 渲染场景的结果。ScanlineRender 节点还允许您在场景的 2D 和 3D 视图之间切换。

添加 ScanlineRender 节点

1。 选择场景 节点。
2. 选择 3D > 扫描线 渲染 从工具栏中。
3. 连接 Obj/scn 输入到场景或几何节点。
4. 连接 凸轮 输入到主摄像头。
5. 连接可选的 Bg 输入将背景图像合成到场景中。
6. 新闻 Ctrl + I ( Cmd + 我 在 Mac 上) 打开新查看器以显示 ScanlineRender 节点的输出。

当图像连接到 bg 输入时，其分辨率将成为 ScanlineRender 节点的输出分辨率。

摄像机节点

摄像机可以连接到场景节点或 ScanlineRender 节点。连接到 ScanlineRender 节点的摄影机是用于渲染的摄影机。

1。 选择 3D > 相机 插入相机节点。
2. 将输出连接器从摄影机节点拖到场景节点，或将摄影机节点连接到 ScanlineRender 节点的 cam 输入。
