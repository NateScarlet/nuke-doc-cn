3D 选择工具

使用查看器右上角的 3D 选择工具，可以选择 3D 对象上的节点、顶点和面。此外，您可以在 3D 视图中选择单个 3D 对象，当节点中有多个对象时，这很方便。

选择节点

可以使用节点选择工具选择节点。 这是默认的选择工具，对应于在查看器中选择 3D 对象节点的熟悉方式。

选择 3D 对象上的顶点

可以使用顶点选择工具选择三维对象上的顶点。要将选择保存到脚本中，需要使用 GeoSelect 节点。您可以在查看器的右上角找到 3D 选择工具。

注意: 选择模式由查看器设置-更改几何节点将您的选择转换为新对象。

也可以使用顶点选择工具使用 EditGeo 节点编辑几何图形。请参阅 使用 EditGeo 节点修改对象 .

在查看器中选择顶点

1。 将 3D 对象附加到查看器并激活查看器的 3D 模式。
2. 单击 顶点选择 工具 来激活它。默认情况下禁用它。
3. 通过在要选择的顶点上拖动选框来选择对象上的顶点。
4. 默认情况下，新选择将替换现有选择，但您也可以从现有选择中添加和删除:

• 要添加到现有选择中，请按住 转变 选择时。

• 要从现有选择中删除，请按住 Shift Alt 选择时。

5. 您还可以通过选择 遮挡测试 按钮 。如果不想在选择中包括被遮挡的顶点，请取消选择它。如果要同时选择可见顶点和遮挡顶点，请选择。

使用 GeoSelect 节点保存或恢复选定顶点的步骤

使用默认的 Viewer 3D 选择工具，您只能进行临时选择。要保存或恢复选定内容，可以在查看器中使用与 “地理选择” 节点相同的 3D 选择工具:

1。 创建一个地球选择节点 ( 3D > 修改 > 地球选择 ) 并将 3D 对象附加到其输入。需要打开 “地球选择属性” 面板，以便使用节点选择顶点。
2. 单击 顶点选择 工具 从 3D 选择面板。
3. Select vertices on the object by dragging a marquee over the vertices you want to select.
4. 默认情况下，新选择将替换现有选择，但您也可以从现有选择中添加和删除:

• 要添加到现有选择中，请按住 转变 选择时。

• To remove from existing selection, hold down Shift+Alt while selecting.

5. 在 “地球选择属性” 面板中，您也可以取消选中 可选 框来冻结您的选择，所以它不能被改变。再次选中该框将再次启用选择，您可以继续正常选择顶点。
6. 随着 显示 和 渲染 控件可以为对象选择所需的显示选项。有关这些控件的更多信息，请参见 对象显示属性 .
7. 当您对您的选择感到满意时，单击 保存 选择 将顶点存储在 GeoSelect 节点中。
8。 顶点由 GeoSelect 节点保存，可以通过单击来恢复 恢复 选择 .

选择三维对象上的面

你可以选择面的 3D 用途使用 面选择 工具。 通过线框线将每个三维对象划分为多边形面，使用 “面选择” 工具可以选择其中的一个或多个。

也可以使用面选择工具使用 EditGeo 节点编辑几何图形。请参阅 使用 EditGeo 节点修改对象 .

1。 单击 面选择 工具来激活它。默认情况下禁用它。
2. 通过在要选择的面上拖动选框来选择对象上的面。
3. 默认情况下，新选择将替换现有选择，但您也可以从现有选择中添加和删除:

• 要添加到现有选择中，请在选择时按住 Shift 键。

• 要从现有选择中删除，请按住 Shift Alt 选择时。

选择 3D 对象

当你在一个节点中有多个 3D 对象时 (比如在你在.具有多个对象的 fbx 文件)，您可以使用对象选择工具 来选择它们。要选择对象，请激活对象选择工具，然后单击要在三维视图中选择的对象。

将位置、方向和大小与 3D 选择相匹配

可以将三维对象 (如卡片节点) 的位置、方向和大小与另一个对象中的选定顶点相匹配。执行以下操作:

1。 单击 捕捉 下拉菜单 在要移动和选择的 3D 对象的 “属性” 面板上 匹配选择位置 。对象现在被定位为与选定顶点匹配。
2. 如果您选择 捕捉 > 匹配选择位置，方向 ,您的 3D 对象将根据 3D 顶点选择进行定位和对齐。
3. 如果您选择 捕捉 > 匹配选择位置、方向、大小 ,3D 对象将根据选定顶点进行定位、对齐和缩放。
