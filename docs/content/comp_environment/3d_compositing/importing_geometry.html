从其他应用程序导入几何图形和点云

有时，您可能需要导入在 3D 应用程序中创建的文件或对象，如 Maya 或 Boujou。根据您要导入的内容以及从哪里导入，有不同的导入方法:

• 从 OBJ 导入几何图形 ( 。Obj ) 文件，请参见 从 OBJ 文件导入几何图形 .

• 从 FBX 导入几何图形或点云 ( 。Fbx ) 文件，请参见 从 FBX 文件导入几何图形和点云 。FBX 是许多应用程序可以导出的标准文件格式, 。Fbx 文件包含 3D 场景，您可以从中导入相机、灯光、变换、网格和点云 Nuke .

• 要从 Alembic 导入几何图形或点云 ( 。Abc ) 文件，请参见 从 Alembic 文件导入几何图形和点云 。您可以从最流行的 3D 应用程序导出到 Alembic。
