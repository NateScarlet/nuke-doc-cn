概述

本章说明了如何建立一个 3D 场景中 核武器 ,以及如何在 3D 工作区中添加对象和相机。您还将看到如何纹理对象，变换对象和摄像机，以及渲染场景以用于脚本的其他区域。

尽管 3D 工作区有许多潜在的用途，但你最有可能使用它 -- 至少在最初 -- 来创建平铺场景。这些场景的 2D 图像平面排列成弯曲的形状，然后通过动画相机渲染出来，给人一种无缝环境的错觉。
