使用 PositionToPoints 节点创建密集点云

PositionToPoints 获取包含在图像文件中的位置数据 (从 3D 应用程序渲染)，并在 Nuke 。位置通道中的 x 、 y 和 z 顶点用于定义三维空间中的点位置，可以使用 点大小 和 点细节 控制。

使用 PositionToPoints 创建点云

1。 从 3D > 几何 菜单，选择 PositionToPoints 将节点添加到脚本中。
2. 读取位置传递并将其连接到节点。

如果您有多个包含单独位置和正常信息的图像，请将它们连接到 Pos 和 规范 分别输入。

请注意, Pos 和 规范 只有连接了未命名的输入后，输入才会出现。

3. 从中选择位置通道 表面点 下拉菜单。如果您的图像包含正常通道，请从 表面正常 下拉菜单。现在，您应该可以在 3D 查看器中看到点云。

如果你提供的节点的职位和一般信息 Pos 和 规范 输入, 表面点 和 表面正常 下拉列表已禁用。

4. 要更改点数，请调整 点细节 滑块。值为 0 表示不显示任何点。值为 1 将显示所有可用点。
5. 要更改点的大小，请调整 点大小 滑块。
