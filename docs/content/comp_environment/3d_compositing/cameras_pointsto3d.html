从动画相机定位 3D 点

在第三方应用程序 (如 Maya) 中创建的动画相机，或 Nuke 的 CameraTracker 包含足够的视差信息，可以使用 PointsTo3D 节点在 2D 场景中定位 3D 点。PointsTo3D 使用图像序列中的三个参考帧来推断 2D 点在 3D 空间中的位置。

连接 PointsTo3D 节点

1。 阅读你的图像序列。请参阅 导入图像序列 欲了解更多信息。
2. 按照下所述导入动画相机 相机 .

提示: NukeX 和 Nuke 工作室 包括 CameraTracker，它无需使用第三方应用程序即可创建动画相机。请参阅 相机跟踪 欲了解更多信息。

3. 在界面左侧的工具栏中，导航到 变换 > PointsTo3D 添加 PointsTo3D 节点。
4. 将图像序列和相机连接到 PointsTo3D 节点，然后连接查看器。

设置参考框架

1。 在 属性 面板，设置 相机类型 到 自由移动 .
2. 擦洗序列，挑出三个特征清晰可见的帧。
3. 擦洗到您的特征可见的第一个参考框架，然后拖动 PointA 查看器小部件从查看器的左下角到功能的位置。
4. 单击 设置框架 .

将二维坐标和参考帧编号插入到 A点 控制。

提示: 您可以调整 2D 点用 x 和 y 用于像素完美定位的属性中的控件。

5. 擦洗到您的特征可见的第二个参考框架，然后拖动 PointB 查看器小部件从查看器的左下角到功能的位置。
6. 单击 设置框架 .
7. 擦洗到您的特征可见的第三个参考框架，然后拖动 PointC 查看器小部件从查看器的左下角到功能的位置。
8。 单击 设置框架 .

现在可以在属性中查看所有三个参考帧的坐标和帧信息。

计算 3D 位置

1。 单击 计算 显示 “范围” 对话框。
2. 单击 帧范围 下拉菜单选择范围边界和要计算的帧 (如果需要)。
3. 要使用较低的分辨率计算图像序列的位置，请启用 使用代理 。这可以加快计算速度，但结果可能不准确。
4. 若要在过程中出现错误时停止计算，请禁用 继续出错 .
5. 单击 好 .

Nuke 穿过序列，对照三个参考帧跟踪 2D 位置，以确定相应的 3D 点。

一旦计算完成, Point2d Viewer 小部件放置在 2D 空间中的跟踪点上。

6. 新闻 标签 在查看器中切换到 3D 透视视图。计算的 3D 点在 3D 空间中高亮显示。

用于计算 3D 位置的 2D 点。 3D 查看器中的结果位置。

您可以使用计算出的 3D 点帮助您在场景中放置元素，或单击 生成轴 添加轴节点以控制场景中的所有元素。请参阅 对轴对象的养育 有关 axes 的更多信息。
