使用变换句柄

将具有变换功能的 3D 对象加载到属性 Bin 中时，将显示变换控制柄。控制柄的颜色对应于 3D 空间中可用的轴: 红色变换 x 轴，绿色变换 y 轴，蓝色变换 z 轴。

使用变换控制柄移动对象

• 拖动对象以在任何轴上移动它。

• 转变 拖动以将运动约束到一个轴。

使用变换控制柄旋转对象

• Ctrl 点击 (Mac 用户) Cmd 在旋转环之间单击) 并拖动以在任何轴上旋转对象。

• 单击特定的轴环并拖动以将旋转约束到一个轴。

使用变换控制柄缩放对象

• Ctrl 转变 点击 (Mac 用户) Cmd 转变 单击) 并拖动中心黄色正方形，以在所有三个轴上均匀缩放。

• Ctrl 转变 点击 (Mac 用户) Cmd 转变 单击) 并在单个方形轴上拖动以在该轴上缩放。
