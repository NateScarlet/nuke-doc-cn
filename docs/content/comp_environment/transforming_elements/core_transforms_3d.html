在 2.5D 中应用核心转换

Nuke 的 Card3D 节点允许您应用与变换节点相同的几何变换，但提供了一个额外的操作轴 z。

要清楚的是，Card3D 节点的转换不是真正的 3D，而是有时被称为 “2” 的转换。 5D ”-- 意思是你可以将一个元素移回到 z轴上，但是这样做并不能传达它在另一个元素后面或前面的感觉。2.5D 转换对于 “欺骗” 元素的视角或 “伪造” 相机变焦等任务非常有用。

然而，请记住 Nuke 不限制你 2.5 的尺寸。如果你需要真正的 3D 功能，你可以构建一个 3D 场景。请参阅 3D 合成 欲了解更多信息。
