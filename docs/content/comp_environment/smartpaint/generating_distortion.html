将运动向量应用于源

一旦你生成了你的运动向量并将你的油漆添加到源序列中, 向量扭曲节点从参考帧中获取绘制，并使用来自 SmartVector 节点的运动向量将其传播到序列的其余部分。

扭曲向量

在生成向量、应用更正并掩盖您不想处理的序列区域后，使用向量扭曲将您的工作推进到序列的其余部分。

1。 将向量扭曲节点添加到节点图中，并连接 Src 输入到您要传播的油漆或图像的下游序列。
2. 连接 SmartVector 输入到 SmartVector 节点或引用 。Exr 包含运动矢量数据的文件。

3. 双击 “矢量扭曲” 节点以打开其属性 (如果尚未打开)。
4. 擦洗到包含油漆或图像的框架，然后单击 设置为当前帧 设置矢量扭曲参考框架。

提示: 如果你知道你的油漆或图像被应用到什么框架，你可以在 参考框架 手动控制。

在这个阶段，观众中的擦洗会引入不必要的扭曲，因为整个源帧都是扭曲的。当您从参考系移动得更远时，扭曲会增加。

扭曲接近参考框架。

远离参考系的扭曲。

5. 设置 面具频道 到下指定的通道 遮罩扭曲区域 。这限制了扭曲的范围，并加快了处理时间。
6. 要将扭曲应用于原始序列，请预乘油漆或图像，然后将结果合并到源上。

对于短序列，或具有最小运动和细节的序列，您的更正应该很好地传播，如图所示。

如果您发现更正随着时间的推移不正确地扭曲，您可以使用向量扭曲的属性来按摩结果 (请参见 改善扭曲 ) 或使用多个矢量扭曲节点为几个参考帧应用扭曲 (请参见 扭曲多个参考帧 )。

遮罩扭曲区域

默认情况下，矢量扭曲会扭曲整个帧，因此可以通过屏蔽不想扭曲的帧区域来提高性能。具有非零的任何像素 面具频道 是扭曲的，也就是说，面具内的任何东西。与大多数遮罩通道不同，矢量扭曲的遮罩应用从 0 到 1 的逐渐扭曲，其中 0 是无扭曲，1 是完全扭曲。

要将蒙版应用于序列:

1。 将更正下游的 Roto 节点添加到源。
2. 画一个贝塞尔 参考框架 在你想弯曲的区域周围。

提示: 您可能会发现，简单的面膜，如果你能够 垫子 显示样式。新闻 M 在浏览器上显示您的更正。

3. 现在，当您向脚本中添加一个向量扭曲节点时，请设置 面具频道 到 Roto 节点中指定的通道。请参阅 扭曲向量 更多细节。

注意: 如果使用自定义遮罩通道，请设置 面具频道 控制至 所有 以确保面具的应用。
