修改粒子的运动

从物体上反弹粒子

使用 ParticleBounce 节点，您可以使粒子从 3D 对象的形状反弹，而不是穿过它。将此节点连接到其他粒子节点，并调整 ParticleBounce 控件:

1。 设置粒子从反弹对象的外部反弹 (由 对象 控件)，在 外部反弹模式 下拉列表: 没有 要应用无外部反弹效果, 反弹 反弹粒子, 杀死 结束粒子反弹时的生命。
2. 选择一个通道，一个粒子将分为反弹时被发现的 新 通道 下拉列表。将此设置为 没有 不执行任何频道分配。
3. 在 反弹 场，指定外部反弹效果的强度，并调整 摩擦 为了控制外部反弹效果的摩擦力。
4. 设置粒子从反弹对象的内部反弹 (由 对象 控制)，使用 内部反弹模式 下拉列表:

• 没有 -应用没有内部反弹效果。

• 反弹 -反弹粒子。

• 杀死 -结束粒子反弹时的生命。

5. 选择一个通道，一个粒子将分为反弹时被发现的 新 通道 下拉列表。将此设置为 没有 不执行任何频道分配。
6. 在 反弹 场，指定内部反弹效果的强度，并调整 摩擦 为了控制内部反弹效果的摩擦力。
7. 在中选择要用于反弹效果的对象 对象 下拉列表:

• 飞机 , 球体 或 气缸 -添加一个 Nuke 标准基元，用作粒子系统的反弹曲面。您可以使用这些形状来快速测试粒子效果。

• 输入 -添加附加到的自定义几何图形 几何 输入。

8。 使用变换控件微调几何图形的位置，以获得所需的反弹结果。

向粒子添加阻力

使用 ParticleDrag 节点，您可以对粒子应用阻力。随着时间的推移，这逐渐减慢了他们的速度。

1。 将 ParticleDrag 节点连接到粒子流。
2. 调整 ParticleDrag 控件:

• 拖动 -增加以向粒子添加拖动效果，在粒子与粒子中心保持距离时减缓和停止它们的运动。

• 旋转 拖动 -增加添加到您的粒子的旋转拖动效果，减缓和停止它们的旋转。

提示: 还可以输入负值以应用反向拖动效果并加快粒子速度。

在粒子上添加湍流运动

粒子湍流节点将柏林噪声应用于粒子运动 x , y 和/或 z 方向。

1。 将粒子湍流节点连接到粒子流。
2. 调整粒子湍流控制:

• 强度 -设置湍流效果的强度 x , y 和 z 斧头。

• 规模 -设置的规模效应，或影响的区域的大小, x , y 和 z 斧头。

• 偏移 -在上设置应用于效果的偏移或受影响区域的偏移 x , y 和 z 斧头。

向粒子添加螺旋运动

ParticleVortex 节点对粒子施加圆形力，并将它们吸引到想象的线，从而产生粒子漩涡。

1。 将 ParticleVortex 节点连接到粒子流。
2. 查看器中会出现一个箭头，您可以拖动该箭头来确定涡旋效果的方向和速度。箭头越大、越长，效果越强。或者，您可以使用 从 和 到 属性面板中的控件。
3. 在 ParticleVortex 属性面板中，使用 平行 控制。这将加速粒子在假想涡旋中心线的方向。如果将其设置为 0，则不会应用平行力，并且正值和负值确定力的方向。调整 平行衰减 要通过选择衰减类型来选择平行力相对于距离的衰减速度, 没有 没有衰减, 逆 对于反向衰减，或 逆 广场 对于反向衰减平方。
4. 随着 切向 滑块可以强制粒子循环漩涡中心线。调整 切向 衰减 要通过选择衰减类型来选择切向力相对于距离的衰减速度, 没有 没有衰减, 逆 对于反向衰减，或 逆 广场 对于反向衰减平方。
5. 使用 径向 滑块将吸引粒子的力从中心线调整为 (正值) 或排斥 (负值)。调整 径向 衰减 要通过选择衰减类型来选择径向力相对于距离的衰减速度, 没有 没有衰减, 逆 对于反向衰减，或 逆 广场 对于反向衰减平方。

提示: 如果你想创建一个粒子螺旋，你可以打开两个 平行 和 切向 值。这将创建一个螺旋状的粒子涡旋。

向粒子添加风效果

使用 ParticleWind，您可以模拟粒子上的风。

1。 将 ParticleWind 节点连接到粒子流。一个箭头出现在观察者中，然后您可以使用它来确定风的方向和速度。箭头越大、越长，风的效果就越强。或者，您可以使用 从 和 到 属性面板中的控件。
2. 在 ParticleWind 控件中，选中 空气 电阻 启用风的阻力效果。
3. 调整 拖动 滑块增加或减少模拟空气阻力。
