缓存播放缓存中的帧

Nuke 工作室 的播放缓存将帧放在 RAM 中，以便在播放期间快速检索，而不是像使用 本地缓存媒体 .

查看器下的绿色条表示播放缓存的内容，一个完整的条表示当前可以从 RAM 访问整个剪辑或时间线，从而优化播放。您可以:

• 使用查看器上方的暂停按钮暂时禁用缓存，或者使用 P 热键。

再次单击暂停，从播放头位置恢复缓存。

• 通过导航到完全刷新缓存 缓存 > 清除播放缓存 。刷新后会自动暂停缓存，但单击 “暂停” 按钮会从播放头位置恢复缓存。

也有一些 偏好 这将影响有多少内存可用以及何时应该进行缓存。要设置缓存行为:

1。 导航到 Nuke 工作室 > 偏好 (OS X) 或 编辑 > 偏好 (Linux 和 Windows),

OR

使用首选项热键 转变 S .

2. 单击 性能 > 缓存 子菜单。
3. 使用设置可用的总 RAM 缓存 播放缓存大小 字段。

注意: 您不能将其设置为大于可用内存 80% 的值 (四舍五入到最接近的半 GB)。例如，如果您有 6 GB 可用内存，最大可用缓存大小为 4.5 GB。

4. 启用 当应用程序进入后台时暂停缓存 当应用程序失去焦点时暂停播放缓存。

当你点击返回 Nuke 工作室 ,缓存从它停止的地方开始。

5. 启用 当应用程序进入后台时清除缓存 当应用程序失去焦点时刷新播放缓存。

当你点击返回 Nuke 工作室 ,缓存再次从播放头的位置开始。

Mac OS X 特定控件

Mac OS X 处理内存的方式与 Linux 或 Windows 不同，因此，这些控件可以提高性能:

1。 启用 将缓存锁定到内存中 这样操作系统就不会交换分配给播放缓存的内存。

此设置可以提高播放性能，但会减少运行其他应用程序的可用内存量。

注意: 默认情况下启用缓存锁定。

2. 启用 当应用程序在后台时解锁缓存 当应用程序在后台运行时，允许其他应用程序使用保留的系统 RAM，从而提高系统性能。

请记住，当应用程序再次变为活动状态时，可能会出现播放延迟，而缓存内存被重新交换。

注意: 默认情况下启用缓存释放。
