显示三种颜色:

• 黑色像素在最终合成中显示纯背景区域。

• 白色像素显示纯前景区域。

• 灰色像素是最终合成中前景和背景像素的混合。你需要 前景边缘周围的灰色像素，以便在前景边缘获得一个好的键。前景和背景混合的像素只以一种灰色显示。这样做是为了突出关键的潜在问题。这些灰色像素可能代表前景/背景混合 50/50 或 99/1。这一比率没有区别。
