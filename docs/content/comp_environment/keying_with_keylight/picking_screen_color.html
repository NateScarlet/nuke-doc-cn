挑选屏幕颜色

屏幕颜色可能是最重要的参数，在做其他事情之前，你应该总是选择屏幕颜色。它应该设置为前景对象后面的绿色或蓝色窗帘的颜色。直接从图像中选择屏幕颜色 Ctrl/Cmd Shift Alt 在蓝色像素上拖动矩形。使用选定像素的平均值。

提示: 如果你按 Alt 采样颜色时, 核武器 无论你在看什么，都要对源图像进行采样。这意味着即使您正在查看哑光、状态或合成，您也可以选择蓝屏颜色。

提示: 您可以通过 Ctrl / Cmd 在查看器中单击鼠标右键。

提示: 从屏幕颜色中挑选不同深浅的蓝色可以得到完全不同的结果。从屏幕的不同部分挑选来获得最佳效果是值得的。

选择的 屏幕 颜色 创建用于在背景上合成前景的屏幕遮罩。它还设置了 屏幕 余额 (如果这还没有手动设置)，并放弃前景。
