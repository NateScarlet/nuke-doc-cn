使用 ModifyRIB 节点

您可以使用 ModifyRIB 节点将 RIB (RenderMan Interface Bytestream) 语句插入脚本，以便在将 RIB 流传递给 PrmanRender 节点之前修改它。这在您可能希望调整对象表面的着色、替换对象或对场景中的多个对象执行各种变换和调整的情况下非常有用。例如，用 RenderMan 的基本几何图形替换脚本中的对象:

1。 将场景节点添加到您在本章开头创建的基本脚本中，并连接两个灯光节点和一个摄像机。
2. 单击 3D > 修改 > RenderMan > ModifyRIB 并在立方体和场景之间插入节点。
3. 旁边的 存档 字段，取消选中 使用 。这激活了 声明 字段。

注意: 您可以通过离开来加载肋骨档案 使用 选中并单击 选择文件 按钮来定位它。

4. 在 ModifyRIB 控制面板中选择 替换 从 操作 下拉菜单，然后在 声明 字段:

球体 0.25 360

现在，您应该可以在查看器中看到立方体所在的基本球体。

5. 要更改球体的颜色并向球体添加基本表面着色器，请在下面输入以下语句:

颜色 0.4 0.6 0.1

表面 “木材”

这将球体的颜色更改为绿色，并将类似木材的光洁度应用于曲面。

注意: 如果您直接从 RIB 文件中复制语句，则只从两者之间复制语句 世界开始 和 世界末日 因为复制文件的全部内容可能会导致不稳定问题。

有关 RIB 语句的更多信息，请参考 RenderMan 提供的文档。
