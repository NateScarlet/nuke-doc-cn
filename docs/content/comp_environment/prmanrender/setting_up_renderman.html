设置 RenderMan Pro 服务器和 PrmanRender

为了使用 PrmanRender 节点，你需要有皮克斯的照片级 RenderMan Pro 服务器 20 岁或更早 ,安装在您的机器上并获得许可 (为了简洁起见，我们从现在开始称之为 RenderMan)。要做到这一点:

1。 如果您使用的是 Mac OS X 10.8 (Mountain Lion) 或更高版本，请确保系统上安装了 X11。与 OS X 的早期版本不同，10.8 及以上版本默认情况下没有安装 X11。有关更多信息，请参见 Http://support.apple.com/kb/HT5293 .
2. 按照 RenderMan 安装指南中的说明，让 RenderMan 在您的机器上工作。在大多数情况下，这足以让你同时使用 RenderMan 和 PrmanRender 节点 Nuke 。请注意，RenderMan 特别需要设置两个环境变量才能使用 Nuke :

• RMANTREE-这需要指向 RenderMan 分布的位置。

• 根据您使用的操作系统 (在 Mac 上) 、 LD_LIBRARY_PATH (在 Linux 上) 或 PATH (在 Windows 上)。此环境变量需要指向 “% RMANTREE %/lib”。有关设置环境变量的更多信息，请参见 设置环境变量 .

3. 以确保您的 PrmanRender 节点在中工作 Nuke ,从终端启动应用程序 (有关更多信息，请参见 安装和许可 章)。
4. 在中创建 PrmanRender 节点 Nuke ( 3D > RenderMan > PrmanRender ) 并将其连接到您的节点。您可以尝试以下节点组合作为测试: 棋盘 > 立方体 > PrmanRender > 观众 .
