使用 RIB (RenderMan 接口 Bytestream) 参数，您可以选择过滤信息 核武器 为 RenderMan 生成，将参数设置为它，并输出自己的 RIB 文件。在 肋骨 标签:

• 过滤器-检查这个来过滤场景上的信息，由 核武器 给 RenderMan 的为了做到这一点, 核武器 调用一个名为 nukescripts.renderman.filterRIB 的 Python 函数。随着为每个渲染创建临时 RIB 文件，过滤可以使渲染启动稍微慢一些。

• 参数-指定用于筛选的参数。这个字符串被传递 核武器 的 Python 过滤器功能作为 RenderMan 的额外参数。如果你想使用你自己的过滤器，你也可以替换 核武器 的 Python 函数，并将您的参数直接传递给您自己的 Python 函数。例如，您可以将过滤器参数设置为 “-rif myfilter.so” 以加载您自己的 RenderMan 接口过滤器。

提示: 有关过滤的更多细节，请查看您的 RenderMan 文档。

RIB 文件是一个与 RenderMan 兼容的 ASCII 文件，其中包含 核武器 渲染素材时生成。要输出 RIB 文件:

1。 在 文件 字段下 输出 ,指定 RIB 文件的文件名和位置。
2. 单击 执行 .
