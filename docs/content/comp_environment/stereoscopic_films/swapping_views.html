交换视图

您可以使用 ShuffleViews 节点重新排列脚本中的视图。例如，您可以在管道中交换左右视图，以便 Nuke 右眼使用左侧输入，反之亦然。

重新排列视图

1。 选择 视图 > ShuffleViews 将 ShuffleViews 节点插入到脚本中的适当位置。
2. 在 ShuffleViews 控件中，单击 添加 如有必要。
3. 使用按钮或下拉菜单选择要替换哪个视图。例如，要交换左右视图，您需要进行以下选择:

• 在一行中，选择 左 下 得到 ,和 右 下 从 (“从右往左”)。左视图现在被右视图替换。

• 在另一行上，选择 右 下 得到 ,和 左 下 从 (“从左向右”)。

右视图被左视图替换。

如果 ShuffleViews 节点的 “属性” 面板上没有足够的按钮行或下拉菜单，请单击 添加 添加行的按钮。

要删除 ShuffleViews 节点控件中不必要的行，请单击 删除 要删除的行旁边的按钮。
