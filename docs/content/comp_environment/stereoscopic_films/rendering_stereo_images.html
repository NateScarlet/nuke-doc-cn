渲染立体图像

可以使用单个写入节点渲染多个视图。当使用立体声扩展的 。Exr 文件格式, 核武器 将两个视图的输出写入一个文件。对于任何其他文件类型，视图被写入各自的文件中。

渲染。Exr 文件

渲染。Exr 文件:

1。 选择 图像 > 写 在脚本中的适当位置插入写节点。
2. 在写入节点的控件中，选择 Exr 从 文件类型 下拉菜单。
3. 从 视图 下拉菜单中，选择要渲染的视图，例如 左，右 .
4. 根据需要调整任何其他写入控件，然后单击 渲染 . 核武器 提示您输入要渲染的帧。

核武器 将选定的视图写入单个文件。

渲染其他文件格式

渲染不在中的文件。Exr 文件格式:

1. Select Image > Write to insert a Write node in an appropriate place in your script.
2. 在写入节点的控件中，从 文件类型 下拉菜单。
3. 输入渲染图像序列的名称时，可以使用变量 % V (用大写字母 V) 来代表单词 左 和 右 (或任何其他完整视图名称) 在文件名中，例如 文件名。% V.#.exr 。代表字母 l 和 r (或任何视图的第一个字母)，使用变量 % V (用小写 v) 代替。渲染时, 核武器 然后用 left 、 right 、 l 或 r 填充，并呈现您在下一步中指定的所有视图。
4. 根据需要调整任何其他写入控件，然后单击 渲染 . 核武器 提示您输入要渲染的帧以及要执行的视图 (假设您已经在项目设置中设置了多个视图)。

核武器 呈现几个视图，但将它们写入单独的文件。如果没有在文件名中指定视图 (使用视图的名称、第一个字母或变量)，则只能呈现一个视图。

注意: 对于命令行渲染，您可以传递- 视图 带有要呈现的视图名称列表的参数，用逗号分隔。如果您不指定- 视图 论点, 核武器 渲染所有视图。
