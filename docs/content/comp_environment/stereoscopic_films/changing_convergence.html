改变收敛

重新收敛节点允许您转移 收敛 (眼睛或相机的向内旋转)，以便使用 3D 眼镜观看时，图像中的任何选定点都显示在屏幕深度。这一点被称为 收敛点。 这是两个摄像机的视线相遇的地方。

通常，看起来最接近观众的图像元素被用作汇聚点。然而，要让图像中的一个元素跳出屏幕，你需要收敛到这个元素后面的一些东西上。

要计算收敛位移，重新收敛节点需要一个视差场，该视差场将一个视图中像素的位置映射到另一个视图中相应像素的位置。要创建视差字段，可以使用 O_DisparityGenerator 插件，这是 Ocula 插件集的一部分。或者，您可以在 3D 应用程序中创建视差字段。一旦你有了视差字段，你就可以把它存储在一个 。Exr 文件或使用 ShuffleCopy 节点在需要的数据流中添加视差通道。

请注意，重新收敛节点仅水平而不是垂直移动视图。

更改立体图像的收敛点的步骤

1。 确保图像序列上游有一个视差场，您要更改其收敛。如果图像序列是 。Exr 文件中，视差字段可以包含在其通道中。否则，您可以使用 ShuffleCopy 节点或 Ocula 的 O_DisparityGenerator 插件将其添加到数据流中。
2. 从工具栏中选择 视图 > 立体声 > 重新收敛 在要调整其收敛的图像序列之后插入重新收敛的节点。
3. 将查看器附加到重新收敛节点。
4. 要更好地查看重新收敛节点的效果，请插入一个浮雕节点 ( 视图 > 立体声 > 浮雕 ) 在重新收敛节点和查看器之间。

5. 确保 “重新收敛属性” 面板处于打开状态。您应该在查看器中看到收敛点覆盖。使用 3D 眼镜查看时，将点拖动到要显示在屏幕级别的点的顶部。收敛转移到这个位置。

您也可以将汇聚点输入点的 x 、 y 坐标, 收敛后 字段。

6. 默认情况下，重新收敛节点移动右侧视图以实现收敛移位。然而，如果你喜欢，你可以使用 模式 重新收敛控件中的下拉菜单改为移动左视图 (选择 左移 ) 或同时移动两个视图 (选择 转变两者 )。
7. 如有必要，在重新收敛控件中调整收敛的偏移 (以像素为单位)。要使图像的所有元素从屏幕级别向前移动，请在 收敛偏移 字段。要进一步移动所有元素，请输入负值。

也可以在整个图像序列中使用与收敛点相同的元素。例如，你可以让相同的演员总是出现在屏幕深度。要在整个序列中收敛于同一元素，请将重新收敛节点与跟踪器节点链接。

在整个图像序列中使用与收敛点相同的元素

1。 在要调整其收敛的图像序列后插入跟踪器节点。

2. 跟踪您希望在整个序列中显示在屏幕级别的点。有关如何使用跟踪器节点的详细信息，请参阅 跟踪和稳定 .
3. 当您有轨迹动画数据时，将其应用于重新收敛节点的 汇聚在 通过链接表达式控制。最简单的方法是 Ctrl / Cmd + 拖动旁边的动画按钮 轨道 列表到动画按钮旁边 汇聚在 控制。
