复制对一个视图所做的更改

当对位、创建 paint effects 或执行依赖于图像位置的其他操作时，您可以在另一个视图中自动复制对一个视图所做的更改。这适用于 RotoPaint 节点、 Roto 节点以及具有 x 和 y 坐标控件的任何节点、组或小控件。

要重现对上述节点、组或小控件所做的更改, 您需要一个视差字段，将一个视图中像素的位置映射到另一个视图中相应像素的位置。您可以使用 O_DisparityGenerator 插件 (包含在 Ocula 插件集中) 或 3D 应用程序创建视差字段。一旦你有了视差字段，你就可以把它存储在一个。Exr 文件或使用 ShuffleCopy 节点在需要的数据流中添加视差通道。

有关再现绘制笔划、贝塞尔形状或 b样条形状的更多信息，请参见 在其他视图中再现笔画/形状 .

再现 X 和 Y 值

每当在任何 x 或 y 控件中有值时 核武器 对于一个视图，可以自动为另一个视图生成相应的值。这对节点和小发明都是如此。例如，您可以使用跟踪器节点在一个视图中跟踪某个内容，然后为另一个视图自动生成轨迹的 x 和 y 位置。

为一个视图生成 x 和 y 值，并为另一个视图自动生成它们

1。 确保在您操作的图像序列上游有一个视差场。如果图像序列是 。Exr 文件中，视差字段可以包含在其通道中。否则，您可以使用 ShuffleCopy 节点或 Ocula 的 O_DisparityGenerator 插件将其添加到数据流中。
2. 在正在操作的图像序列后插入一个具有 x 和 y 控件的节点。
3. 将查看器附加到您在上一步中添加的节点，并在一个视图中进行更改。
4. 从旁边的 “视图” 菜单中 x 和 y 控件，选择 使用差异从 [视图] 关联 [视图] ,例如 使用视差从左向右关联 。这将为其他视图生成相应的 x 和 y 值。

如果您安装了 Foundry 的 Ocula 插件，您也可以选择 将 [视图] 从 [视图] 与 Ocula 关联 。这样，在创建相应的 x 和 y 值时会进行额外的改进，结果可能会更准确。

5. 如果要进一步调整 x 和 y 值，需要独立调整这两个视图。对一个视图所做的调整不会自动为另一个视图生成。
