翘曲剪辑

翘曲 指在剪辑中减慢、加速甚至反转选择帧，而不一定改变其总长度。否则，扭曲会在剪辑内的一系列帧中添加、减去或反转时间细节。例如，下图描绘了一个我们可能想要扭曲的雪地摩托剪辑 (向下采样到只有十帧以便于表示)。

规划时间扭曲。

一种方式 -- 事实上，一种经典的方式 -- 扭曲这个片段是在碰撞前以原始速度播放帧, 慢动作中涉及碰撞的帧，快速动作中涉及碰撞后的帧。

你可以通过雕刻曲线来实现这样的扭曲 核武器 的时间扭曲曲线是重定时节点参数的一部分，看起来类似于下面描述的曲线。

编辑扭曲曲线。

编辑扭曲曲线的基本 “规则” 如下:

• 要减慢运动，请减小曲线的坡度。

• 要加快运动速度，请增加曲线的坡度。

• 要反转运动，请在曲线上创建一个向下倾斜的部分 (换句话说，是一个倾斜)。

扭曲剪辑

1。 单击 时间> 重定时 将重定时节点插入脚本。
2. 单击 时间扭曲 选项卡显示时间扭曲曲线。
3. 将查看器附加到此节点，以便您可以看到更改的效果。
4. 根据上述规则雕刻时间扭曲曲线。( Ctrl/Cmd + Alt 单击以在曲线上插入关键帧节点; Ctrl/Cmd + 拖动以重新定位关键帧结; Ctrl/Cmd + 拖动以旋转关键帧结控制手柄。)
5. 如果要在输出上启用帧混合，请在重定时节点的 快门 参数，或者在重定时节点之前插入帧混合节点。
