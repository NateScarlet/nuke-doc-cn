时间操作

本章解释了时间或基于时间的操作 Nuke 。了解如何扭曲时间 (即减速、加速或反转剪辑) 、应用运动模糊以及执行编辑操作，如滑动、剪切、拼接和冻结帧。

快速启动

以下是工作流的快速概述:

1。 如果你想加快或减慢你的剪辑:

• 您可以使用重定时节点 ( 时间 > 重定时 )。如果你愿意，你也可以添加一个框架混合 ( 时间 > 框架混合 ) 在重定时之前创建更平滑的重定时结果的节点。有关更多信息，请参见 简单的重新计时 和 插值 .

• 对于更高级的重新计时操作和添加运动模糊，您可以使用 OFlow 节点 ( 时间 > OFlow )，或者如果你有 NukeX 的许可证，Kronos 节点 ( 时间 > Kronos )。有关更多信息，请参见 OFlow 重新计时 和 Kronos .

• 有时你会想在剪辑中创建加速和慢动作，而不改变其实际长度。这称为扭曲，您可以使用重定时节点 ( 时间 > 重定时 ) 去做。有关更多信息，请参见 翘曲剪辑 .

重新计时剪辑后，您可能需要调整脚本的全局帧范围和播放速度 (每秒帧)。有关更多信息，请参见 全球帧范围和速度 .

2. 如有必要，可以使用 “时间模糊” 节点将运动模糊应用于跟踪到快速移动特征的垃圾蒙版。有关更多信息，请参见 应用时间模糊过滤器 .
3. 如果你想执行编辑操作:

• 您可以使用时间偏移 ( 时间 > 时间偏移 ) 和 TimeClip ( 时间 > TimeClip ) 节点来滑动剪辑。有关更多信息，请参见 偏移和滑动剪辑 .

• 帧率范围 ( 时间 > 框架范围 ) 节点允许您剪切剪辑。有关更多信息，请参见 切割剪辑 .

• 您可以使用 AppendClip ( 时间 > AppendClip ) 节点来拼接剪辑。有关更多信息，请参见 拼接剪辑 .

• 使用框架保持 ( 时间 > 框架保持 ) 节点允许您将剪辑或场景的输出设置为特定的帧或帧间隔。有关更多信息，请参见 冻结框架剪辑 .

注意: 大多数临时操作 Nuke 只能应用于 2D 合成，但帧间距、时间偏移和帧保持节点适用于 3D 节点，如几何图形。
