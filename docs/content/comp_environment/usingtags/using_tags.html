Using Tags

Tags are used by Nuke Studio to quickly sort or filter clips and track items for better visibility, organization, and export.

Introduction

Tags are used to mark shots of a particular type or content as you organize your project. The default tags supplied include Approved, Notes, Reference, and other general purpose tags. You can also create custom tags by right-clicking in the Tags tab or by pressing Ctrl/Cmd + Y. You can apply tags to clips, track items, individual frames, sequences, and tracks.
