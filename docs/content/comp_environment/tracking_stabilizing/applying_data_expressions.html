使用链接表达式应用跟踪数据

核武器 的 CornerPin2D 和 Stabilize2D 节点专门设计用于通过链接表达式接收跟踪数据，但是您可以以这种方式将跟踪数据应用于几乎任何 核武器 节点。例如，可以通过将链接表达式输入到 RotoPaint 节点的变换参数中，为带有跟踪数据的 Bezier 或 B-spline 形状设置动画。您还可以将跟踪数据应用于各个点。

本节介绍通过链接表达式将跟踪数据应用于任何节点的基本过程。

创建链接表达式

跟踪器节点的跟踪器面板显示与每个轨迹锚的位置相关的数据 ( 轨道 x 和 y 字段)。这是您最典型地应用于其他节点的数据。

要拖放跟踪数据:

1。 显示跟踪器参数和您希望应用跟踪数据的参数 (目标控件-例如，RotoPaint 节点的 翻译 控制)。
2. 从中选择所需的轨迹 轨道 列表。每个控件只能链接一个轨道。
3. Ctrl + 拖动 ( Cmd + 从源代码控制动画按钮拖动 Mac) 到目标控制动画按钮。

释放时，目标控件将变为蓝色，指示已应用表达式。在这种情况下，拖放操作创建了类似以下示例的链接表达式:

Tracker1.track.1.track_x

提示: 您还可以将跟踪 (或其他变换) 数据应用于单个 RotoPaint 、 SplineWarp 或 GridWarp 点 (这有时称为逐顶点跟踪)。要这样做, Ctrl / Cmd 将轨道的动画按钮拖放到查看器中的 RotoPaint 、 SplineWarp 或 gridavoppoint 上。

您可以根据需要将其他组件添加到此链接表达式中。例如，您可以通过减去初始帧的跟踪值来向链接表达式添加空间偏移，在这种情况下，最终表达式将类似于以下内容:

Tracker1.track.1.Track_x-Tracker1.tracks.1.track_x (1)

请参阅 表情 欲了解更多信息。输入链接表达式后，目标参数将变为蓝色。

将动画参数与跟踪器节点链接

您还可以连接控制节点的跟踪，如果你使用的 链接到 动画菜单中的选项。例如，要链接 翻译 使用跟踪器节点控制 Roto 节点，请执行以下操作:

1。 创建要链接到的跟踪器节点。
2. 转到跟踪器的变换选项卡并启用 实时链接转换 .

此控件使跟踪器能够随着表达式链接的更改而动态更新。

3. 在 变换 Roto 节点的 “属性” 面板的选项卡上，单击 翻译 动画菜单。
4. 选择 链接到 > 跟踪器链接对话框 .
5. 选择要在中使用的跟踪器节点 跟踪器节点 下拉列表和 链接到 下拉列表中，选择是否要链接到 位置 的轨道, 翻译 轨迹的值，或将转换值视为偏移。
6. 通过选中选择要使用的曲目 轨道 盒子。Expression 字段使用适当的表达式语法更新。如果选择多个轨迹，则轨迹是平均的，例如:

(Tracker1.track.1.track_x tracker1.track.2.track_x)/2

7. 然后单击 好 ,你的链接完成了。

当跟踪器值更改时，您的 Bezier 形状的平移值现在会更改。
