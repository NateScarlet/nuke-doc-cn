使用跟踪器的控制应用跟踪数据

将跟踪数据应用于输入图像或其他节点的最简单方法是使用跟踪器节点本身的控件。在这里，我们看看使用这些控制来稳定，匹配移动，消除或应用抖动。

稳定元素

跟踪器节点的控制允许您从节点的输入剪辑中删除运动，如不需要的相机抖动。可以使用单个轨迹来稳定 2D 平面上的水平和垂直运动，也可以使用两个或多个轨迹来消除旋转和缩放。

1。 创建要用于稳定素材的轨道:

• 如果您使用的是单个轨道，请确保 T 在轨迹列表中选择，因此跟踪器只计算平移。

在 变换 选项卡，选择 变换 > 稳定 1-pt 要锁定 过滤器 到 钥匙 。使用单个轨道时，此过滤器会产生最佳结果。

• 如果您在轨道列表中使用多个轨道，请选择您希望跟踪器在稳定图像时计算的转换, T Ranslation, R Otation，和/或 S 凯尔。

在 变换 选项卡，选择 变换 > 稳定 .

2. 设置 参考框架 如果您不想使用第一帧作为变换控制帧。
3. 使用 平滑 控制将帧平均在一起以平滑应用的变换。

例如，如果你是稳定使用更多的运动，你可能平均帧一起为翻译和旋转进入的帧数, t 和 r 字段。

4. 选择所需的过滤器。请参阅 选择过滤算法 欲了解更多信息。

Nuke 稳定素材，将其元素锁定在复合材料中的相同位置。

注意: 可以导出转换信息的节点相连或烤变换选择所需要的类型, 出口 下拉列表并单击 创建 .

匹配移动元素

您可以使用跟踪器节点的控件将跟踪的运动应用于另一个图像，即匹配-移动图像。

1。 使用跟踪器节点跟踪要匹配的功能。
2. 复制跟踪器节点，并在要匹配移动的镜头后粘贴它。
3. 在第二个跟踪器节点的控件中，转到 变换 标签。
4. 从 变换 下拉菜单，选择 匹配-移动 .
5. 设置 参考框架 如果您不想使用第一帧作为变换控制帧。
6. 使用 平滑 控制将帧平均在一起以平滑应用的变换。

举例来说，如果你使用一个以上的运动，你可能平均帧一起为翻译和旋转进入的帧数, t 和 r 字段。

7. 选择所需的过滤器。请参阅 选择过滤算法 欲了解更多信息。

移除或添加抖动

跟踪器可用于消除序列中的高频抖动、放大现有抖动或将抖动添加到另一个序列以获得一致的外观。

要消除抖动:

1。 创建要用于消除抖动的轨迹。
2. 在跟踪器节点的控制中，转到 变换 标签。
3. 从 变换 下拉菜单，选择 消除抖动 .
4. 设置 参考框架 如果您不想使用第一帧作为变换控制帧。
5. 使用 抖动周期 将帧平均在一起，调整抖动以达到所需的稳定性。
6. Use the smooth controls to average frames together to smooth the transforms applied.

例如，如果您使用多个轨道来消除抖动，则可以通过在 t 和 r 字段。

7. Select the required filter. See Choosing a Filtering Algorithm for more information.
8。 Nuke 从素材中移除抖动，将其元素锁定到复合材料中的相同位置。

夸大或增加抖动:

1。 创建要用于所需抖动操作的轨迹。
2. In the Tracker node’s controls, go to the Transform tab.
3. 从 变换 下拉菜单，选择 添加抖动 .
4. Set the reference frame if you don’t want to use the first frame as the transform control frame.
5. 使用 抖动周期 将帧平均在一起，调整抖动以实现所需的不稳定。
6. 使用 平滑 控制将帧平均在一起以平滑应用的变换。

举例来说，如果你添加的抖动使用更多的运动，你可能平均帧一起为翻译和旋转进入的帧数, t 和 r 字段。

7. Select the required filter. See Choosing a Filtering Algorithm for more information.

Nuke 夸大镜头中的跟踪抖动。

注意: 如果要将抖动传输到另一个序列，请复制跟踪器节点并将其粘贴到要添加抖动的镜头后，然后按照上面的步骤 3-7 操作。
