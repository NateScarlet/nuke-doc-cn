连接 ProjectionSolver 节点

要开始使用 ProjectionSolver，首先需要一个 3D 对象和要投影到对象上的 2D 图像。然后:

1。 通过单击创建 ProjectionSolver 节点 3D > ProjectionSolver .
2. 将您的 3D 模型连接到 Geo 输入。
3. 将您的 2D 镜头连接到 源 ProjectionSolver 节点的输入。

ProjectionSolver 节点的输出是一个与 2D 镜头的相机移动相匹配的相机节点。您还可以创建卡片来匹配每个不同的相机位置。
