Connecting the ProjectionSolver Node

To start using ProjectionSolver, you first need a 3D object and the 2D images you want to project onto the object. Then:

1.   Create a ProjectionSolver node by clicking 3D > ProjectionSolver.
2.   Connect your 3D model to the Geo input.
3.   Connect your 2D footage to the Source input of the ProjectionSolver node.
