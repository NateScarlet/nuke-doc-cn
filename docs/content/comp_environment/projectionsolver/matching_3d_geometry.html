将 3D 几何图形与 2D 素材匹配

首先，你需要告诉 ProjectionSolver 2D 镜头中的哪些点与 3D 模型上的对应点一致。执行以下操作:

1。 使用 3D 顶点选择工具 在查看器中选择 3D 模型上的顶点。
2. 转到您的 2D 素材并单击鼠标右键 添加 定位器 在顶点的 2D 点上。

在 3D 中选择顶点。 在 2D 中添加定位器。
3. 重复步骤 1 和 2，直到框架上至少有 6 个定位器。
4. 为下的定位器创建关键帧 定位器 标签。单击动画菜单 对于每个 2D 定位器并选择 设置 关键 。您也可以单击 关键 所有 定位器 按钮同时为所有定位器键。请注意，您不必创建密钥来解决单个帧。
5. 转到下一帧，将 2D 定位器移动到查看器中的正确位置，从而创建 2D 轨迹。

你应该添加至少 6 个定位器点来匹配 2D 镜头和 3D 模型。不过，如果你创建了 8 个定位器点，你会得到更好的结果，如果你想确保获得更好的结果，你会想创建大约 10-12 个定位器点。

提示: 当求解多个帧时，定位器不必在每一帧处设置键控。只有键控定位器用于求解每个帧。
