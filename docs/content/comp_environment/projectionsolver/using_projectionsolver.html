使用 ProjectionSolver

ProjectionSolver 帮助您将 2D 与 3D 点对齐。例如，您可以将 2D 图像与 3D 模型对齐。ProjectionSolver 节点的输出是一个与 2D 镜头的相机移动相匹配的相机节点。

快速启动

以下是工作流的快速概述:

1。 连接 ProjectionSolver 、 Read 、 ReadGeo 和其他可选节点。有关更多信息，请参见 连接 ProjectionSolver 节点 .
2. 将你的 2D 素材和 3D 模型与定位器 (至少 6 个) 相匹配，并通过你的素材帧为它们的运动创建关键帧。有关更多信息，请参见 将 3D 几何图形与 2D 素材匹配 .
3. 单击 解决 相机 。请参阅 解决您的相机 .
4. 在观众中细化你的场景，并使用 精炼 输出 和 现场直播 更新 控件来查看更改。有关更多信息，请参见 提炼解决方案 .
5. 当你对你的解决方案满意时，点击 创建 相机 .
6. 您也可以通过单击创建卡片节点 创建 卡片 帮助你在时间线中不擦洗的情况下可视化你的场景。有关更多信息，请参见 创建卡片 .
