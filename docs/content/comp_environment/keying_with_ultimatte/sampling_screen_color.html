采样屏幕颜色

使用 Ultimatte 键控的第一步是采样屏幕颜色，或者告诉 Ultimatte 你的蓝色或绿色屏幕是什么颜色。执行以下操作:

1。 连接完最后一个节点后，前景图像将显示在查看器中。
2. 单击 屏幕 在查看器上方的 “最后期限” 工具栏中，按住键选择屏幕颜色 Alt + Ctrl / Cmd 并在查看器中单击。

您应该在绿色屏幕上选择一个重要主题细节附近的区域，该区域不会以任何方式被遮挡。渲染图像并显示合成。如果需要进一步调整，请使用下面描述的控制。

调整 “最终” 选项卡上的控件

对屏幕颜色进行采样后，您可以开始调整结果。与上的控制 最后时刻 选项卡您可以选择要在其他 “属性” 面板选项卡上启用的不同控件集。还有一个 启用 每个选项卡上的复选框，可用于激活该选项卡上的控件和查看器中相应的工具。

• 电影 -检查这个以减少青色底切对的影响 电影 标签。

• 屏幕 正确 -检查此以使用上的控件 屏幕 正确 标签。

• 影子 -检查此以使用上的控件 影子 标签。

• 泄漏 抑制 -检查此以使用上的控件 泄漏 标签。

• 清理 -检查此以使用上的控件 清理 标签。

• 颜色 一致性 -检查此以使用上的控件 颜色 标签。
