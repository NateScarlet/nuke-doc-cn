使用覆盖工具和屏幕正确

屏幕校正补偿背衬区域中的异常，如不均匀的照明、污渍、接缝、背衬颜色的变化、绿色布景以及布景、吊杆臂投射的不必要阴影, 等等。这种技术假设有问题的绿色屏幕元素的精确副本省略了主题,通常称为干净的盘子，供应。

尽管这种技术通过区分前景元素和背景缺陷产生了很好的结果，但你通常没有必要的参考材料。在这种情况下，您可以使用 Ultimatte 创建合成清洁板。

为了达到最佳效果，除了允许最终生成合成板之外，还可以使用参考清洁板作为最终板的输入。这样，参考板允许最好的阴影分离, 而合成板用于减少诸如胶片颗粒或视频噪音等人工制品 (在主要摄影期间拍摄干净板时，这种人工制品很少准确再现)。将视图切换到 屏幕 通过单击 覆盖 下拉菜单并选择 屏幕 .

与 屏幕正确 选中，使用 添加 覆盖 工具来擦洗代表绿色屏幕的区域，包括阴影。通常，最好是按 Alt Ctrl / Cmd 在擦洗时，这样你就可以在输入图像中选择该像素的颜色，而不是处理后图像的颜色。

覆盖在最终过程中使用，以从最终复合材料中完全消除这些区域。以这种方式继续，直到前景主题及其阴影被覆盖包围。确保覆盖不覆盖任何要保留的前景细节。如果覆盖确实覆盖前景主题，则使用 移除 覆盖 工具来擦洗那些不应该被覆盖的区域。重复这两个过程，直到覆盖覆盖绿色屏幕区域，没有主题。在查看器中查看复合材料，以查看移除了背景区域中异常的屏幕修正复合材料。要了解 Ultimatte 自动设置了哪些控件，请单击 屏幕 正确 选项卡并记下控件的位置。

注意: 当擦洗图像时使用 添加覆盖 ,选定点的 RGB 值将累积在保留列表中，并 移除 覆盖 点在下拉列表中累积。如果两个列表都有相同的值，则可能会发生冲突，导致覆盖中没有可见的更改。如果发生冲突，请尝试使用撤销 ( Ctrl/Cmd X ) 刷新添加到相应列表中的最后一组点。如果没有帮助，你也可以按 重置 按钮上 屏幕 正确 选项卡以清除列表并重新开始。

在某些情况下，可能很难完全覆盖屏幕区域。这并不意味着不能在该区域删除屏幕，但可能需要调整其他控件，以帮助减少覆盖区域未覆盖的异常。这可能有利于重采样的屏幕使用 屏幕 在覆盖屏幕的区域中，观察者上方的工具不成功，但是如果最终的复合材料需要保留阴影，请小心选择阴影区域。

此外，在某些情况下，可能很难从前景细节的非常暗的区域中移除覆盖，这些区域可能接近屏幕黑暗或阴影区域中的值。如果覆盖确实包括一些黑暗的前景区域，这些区域可以通过启用 阴影 控制。

注意: 由于 Ultimatte 使用插值过程生成合成清洁板，因此排除图像的非视频区域很重要, 例如视频中可能出现的黑色消隐线或信箱图像中的黑色区域。可以使用其他节点 (如裁剪) 从覆盖和最终复合中排除这些区域。否则，合成清洁板可能含有黑色 (或其他颜色)，这可能会导致产生不准确的清洁板。

调整覆盖控件

您可以在属性面板中选择覆盖视图、颜色和输出模式 最后时刻 标签。

• 覆盖-覆盖控件用于在查看器上显示计算的覆盖，这对调试有帮助。它有助于调整屏幕正确的控件以及添加/删除覆盖工具。通过这种方式，你可以立即看到是否有问题，需要调整。在覆盖下拉菜单中，更改

• 关闭 -不查看覆盖。

• 屏幕 -以清晰的方式查看主题，初步磨砂区域与覆盖颜色混合。

• 主题 -显示与叠加颜色混合的主题，以及清晰的初步磨砂区域。

• 显示图像为单色 -选中此选项，使输入图像以灰度显示，以便更容易区分覆盖区域。

• 覆盖 颜色 -使用此更改覆盖的颜色。您可以调整 alpha 通道来修改覆盖的不透明度。

调整屏幕正确的控制

• 屏幕 宽容 -这是用来调整颜色范围或公差，包括或排除在屏幕覆盖。

• 收缩 -使用此控件缩小屏幕覆盖。

• 黑暗 ( 红色 较小 )-使用此控件可以在蓝色值 (使用绿色屏幕时) 大于原始前景图像中红色值的区域中包括或排除屏幕覆盖中的暗区域。

• 黑暗 ( 红色 更大 )-使用此控件可以在原始前景图像中红色值大于蓝色值 (当使用绿色屏幕时) 的区域中包括或排除屏幕覆盖的暗区域。

• 亮灯 ( 红色 较小 )-使用此控件可以在蓝色值 (使用绿色屏幕时) 大于原始前景图像中红色值的区域中包括或排除屏幕覆盖的亮区。

• 亮灯 ( 红色 更大 )-使用此控件可以在原始前景图像中红色值大于蓝色值 (当使用绿色屏幕时) 的区域中包括或排除屏幕覆盖的亮区。

• 孤儿 -使用此从屏幕覆盖中包括或排除相邻的 “流氓” 像素。
