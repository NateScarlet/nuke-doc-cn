扭曲图像

核武器 的 GridWarp 和 SplineWarp 节点允许您扭曲图像中的元素，并将一个图像转化为另一个图像。还有其他类型的翘曲可在 核武器 ,但在这一章中，我们将重点放在 gradwarp 和 SplineWarp 节点上。

快速启动

以下是工作流的快速概述:

1。 对于翘曲和变形，您可以使用 GridWarp 节点 ( 变换> 砂粒 ) 或 SplineWarp ( 变换> SplineWarp )。与 GridWarp 一起工作有时会稍微快一点，而 SplineWarp 节点允许更精细的调整。使用这两个节点，您首先将源图像连接到 Src 或 一个 输入，如果变形，目标图像到 Dst 或 B 输入。GridWarp 还允许您将额外的背景图像连接到 Bg 输入。
2. 根据您使用的节点，您可以继续设置源和目标点，并调整它们，直到您对结果满意。有关更多信息，请参见 使用 GridWarp 节点扭曲图像 和 使用 SplineWarp 节点扭曲图像 .
3. 如果你想把一个图像变成另一个图像，你也可以用两个 warper 节点来做。有关更多信息，请参见 使用 GridWarp 节点将一个图像变形成另一个图像 和 使用 SplineWarp 节点将一个图像变形为另一个图像 .
4. 如果你正在扭曲或变形移动图像，而不是静止图像，看看 变换和动画扭曲和变形 .
