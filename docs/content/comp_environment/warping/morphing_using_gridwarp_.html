使用 GridWarp 节点将一个图像变形成另一个图像

要使用 GridWarp 节点将一个图像变形为另一个图像:

1。 选择 图像 > 阅读 要导入要一起变形的两个图像。
2. 如果图像的分辨率不同，请在它们之后插入重新格式化节点。有关更多信息，请参见 重新格式化图像序列 或 重新格式化图像 .
3. 选择 变换 > 灰矮星 将 GridWarp 节点插入到脚本中。
4. 将源图像 (要转换为其他图像的图像) 连接到 Src Gridupp 节点的输入，以及目标图像 (您希望源图像变成的图像) 到 Dst 输入。将查看器连接到 GridWarp 节点。

5. 要使网格与输入图像的大小相同，请单击 调整到 图像 两个按钮下 源网格 和 目的地网格 .

6. 在砂砾中 设置 控件，设置 输出 到 源 显示源图像和网格。
7. 调整网格以遵循上的关键特征 源 图像。例如，如果你正在变形动物，你可能会考虑眼睛、鼻子和嘴巴。

8。 在砂砾中 设置 控件，设置 输出 到 目的地 显示目标图像和网格。

注意: 如果 源 和 目的地 图像相似，你可以复制 源 网格到 目的地 网格使用复制和粘贴 按钮。

9. 调整网格以遵循上的关键特征 目的地 图像。

10. 在砂砾中 设置 控件，设置 输出 到 变形 显示两个网格并激活 翘曲 和 混合 滑块。
11。 擦洗到你想要变形开始的框架。把 翘曲 向下滑动到 0 (源图像)。单击动画按钮 并选择 设置密钥 。然后，擦洗到你想要变形结束和设置的框架 翘曲 至 1 (目标图像)。
12. 动画的 混合 以同样的方式控制。擦洗到变形开始的框架，并设置 混合 到 0 (源图像)。单击动画按钮并选择 设置密钥 。擦洗到变形结束的帧，并输入 1 (目标图像) 作为 混合 价值。

在观众中播放序列，你会看到 源 图像被变形成 目的地 图像。
