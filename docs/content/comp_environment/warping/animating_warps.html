动画扭曲

在 GridWarp 和 SplineWarp 节点的 “属性” 面板中，有用于为源网格或形状和目标网格或形状设置动画的控件。在这里，我们首先看 GridWarp 节点，然后看 SplineWarp 节点。说明假设您知道如何使用这些节点扭曲静止图像 (如果不知道，请参考 使用 GridWarp 节点扭曲图像 和 使用 SplineWarp 节点扭曲图像 )。

使用动画制作扭曲 GridWarp 节点

1。 查看源图像时，根据需要调整源网格 (请参见 使用 GridWarp 节点扭曲图像 有关如何做到这一点的信息)。

如果 自动密钥 如果启用，则每次调整网格时都会添加关键帧。否则，单击 添加密钥 按钮 下 源网格 。这将当前网格保存为关键形状。

2. 移动到新框架并相应地调整源网格。自动设置新的关键图形。
3. 根据需要重复前一步。如果需要删除关键形状，请擦洗到设置它的框架并单击 删除密钥 下 源网格 .
4. 隐藏源网格并显示目标网格。
5. 在查看 GridWarp 节点的输出时，调整目标网格，直到您对扭曲感到满意。

如果 自动密钥 如果启用，则每次调整网格时都会添加关键帧。否则，单击 添加密钥 按钮 下 目的地网格 。这将当前网格保存为关键形状。单击 设置 按钮在 目的地网格 。当前网格保存为关键形状。

6. 移动到新框架并再次调整目标网格。修改后的网格将自动设置为关键形状。
7. 重复上一步，直到您对动画扭曲感到满意。

使用动画制作扭曲 SplineWarp 节点

1。 按照下所述创建所需曲线 使用 SplineWarp 节点扭曲图像 .

如果 自动密钥 如果启用，则每次调整形状时都会添加关键帧。否则，单击 添加密钥 按钮 与 扭曲的 已选择。这将当前形状保存为关键帧。

2. 移动到新框架并相应地调整形状。新的关键形状会自动设置。

根据需要重复前一步。如果需要删除关键形状，请擦洗到设置它的框架并单击 删除密钥 与 扭曲的 已选择。

使用 SplineWarp 节点链接变换

应用在属性面板中的变换 变换 选项卡可以是与其他形状链接的表达式 曲线 列表使用 变换链接 切换。

1。 在源形状上创建所需的变换。
2. 单击 变换链接 在曲线列表中切换。

菜单显示可用的形状和变换类型。

可以链接与源形状关联的所有变换类型，也可以选择，例如 除了额外的矩阵 .

3. 在这个例子中，任何 规模 添加到的关键帧 L_left_nose 形状平等地应用于 L_head 形状。
4. 如果要删除链接，请单击 变换链接 切换所需的形状，选择链接，然后单击 删除链接 .
