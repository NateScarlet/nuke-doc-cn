使用 SplineWarp 节点扭曲图像

SplineWarp 节点基于创建的多条 Bezier 或 b样条曲线变形图像。源曲线定义扭曲的位置，而目标曲线定义将源图像扭曲到的位置。与 GridWarp 节点不同，您可以在图像的任何地方绘制这些曲线，而不仅仅是在现有网格线上添加点，然后将它们连接起来以创建源和目标。用于添加和修改点的控件类似于 RotoPaint 节点控件。

Splininewarp 选项卡

裁剪到格式

禁用时，输入图像不会裁剪到 项目设置 > 完整格式大小 。因此，扭曲可以从图像中引入格式大小之外的像素，而不是黑色。

注意: 禁用 裁剪到格式 可以影响性能 核武器 必须计算整个图像的扭曲，而不仅仅是格式大小内的区域。

Bbox 边界曲线

启用后，将在输入格式周围添加边界曲线，从而限制图像边缘的扭曲。

注意: 在 AB 变形 模式，输入的格式 一个 作为边界。

渲染选项卡

曲线分辨率

调整扭曲/样条匹配的精度。较高的值会提高精度，但会牺牲速度，反之亦然。例如，对于低曲线分辨率的开放样条线，可能会发生图像撕裂。你可以提高 曲线分辨率 值来补偿撕裂，但渲染时间增加。

注意: 如果将对应点调高会导致性能问题，则可以使用对应点来提高曲线特定部分的翘曲精度。

边界曲线分辨率

调整上的插值点数 边界 和 硬边界 曲线。较高的值会阻止扭曲通过边界曲线进行过滤，但会牺牲速度，反之亦然。

预览分辨率

提高了较高值时预览的准确性和较低值时的渲染速度。

经典翘曲

设置要使用的扭曲函数类型:

• 残疾人 -使用更新的二次扭曲函数，可以很好地处理重叠的控制点，但具有更多的局部扭曲效果。

• 已启用 -采用了以前版本的翘曲功能 核武器 这具有更全局的扭曲效应，但不能很好地处理重叠的控制点。

过滤器

使用边界形状和销限制扭曲

您可能会注意到，扭曲周围区域的像素也会被移动以适应变化，但是从重新定位的点处得到的失真会进一步减少。如果你不希望附近的区域扭曲，你可以:

• 将曲线或形状指定为边界，以控制或消除扭曲。

• 将销放置在要限制扭曲的区域中。
