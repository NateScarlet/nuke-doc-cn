更改渲染帧的编号

默认情况下，当渲染图像序列时, 核武器 假设当前处理的帧与写出的帧编号之间存在精确关系。因此，如果选择渲染帧 15，则会相应地对生成的文件进行编号，例如 image.0015.rgb。然而, 框架 “写入” 节点上的参数允许您更改此行为。例如，如果您有一个从 image.0500.rgb 运行到 image.1000.rgb 的序列，您可能希望呈现该序列，以便生成的文件中的帧编号从 image.0001.rgb 运行到 image.0501.rgb。您可以通过表达式、指定的开始帧和常数偏移来执行此操作。每种方法如下所述。

使用表达式

1。 选择 图像 > 写 将写入节点插入脚本。
2. 在 “写入属性” 面板中，单击 文件 文件夹图标 ,然后导航到要保存渲染图像序列的目录路径。输入图像序列的名称。
3. 设置 框架 到 表达 。在右侧的字段中输入表达式。表达式将更改当前处理的帧和写出的帧编号之间的关系。当前帧的结果文件名显示在节点图的 “写入” 节点上。

例如，如果剪辑从帧 500 开始，并且要将该帧命名为 image.0001.rgb 而不是 image.0500.rgb，则可以使用表达式 框架-499 。这样，从当前帧中减去 499 帧，以获得写出的帧的编号。帧 500 以 image.0001.rgb 形式写出，帧 501 以 image.0002.rgb 形式写出，依此类推。

表达式的另一个例子是 框架 * 2 。此表达式将当前帧乘以 2，以获得写出的帧的数量。在帧 1，image.0002.rgb 被写出; 在帧 2，image.0004.rgb 被写出; 在帧 3，image.0006.rgb 被写出; 依此类推。

为剪辑中的第一帧指定帧编号

1. Select Image > Write to insert a Write node into your script.
2. In the Write properties panel, click the file folder icon , then navigate to the directory path where you want to save the rendered image sequence. Enter a name for the image sequence.
3. 选择 从开始 从 框架 下拉菜单。在右侧的字段中输入起始帧编号。这指定给序列中的第一帧的帧编号。其余帧的编号会相应地偏移。

例如，如果您的序列从帧 500 开始，并且您输入 1 在该字段中，帧 500 被写出为 image.0001.rgb，帧 501 被写出为 image.0002.rgb，依此类推。同样，如果您输入 100 在该字段中，帧 500 以 image.0100.rgb 的形式写出。

将所有帧编号偏移一个常数值

1. Select Image > Write to insert a Write node into your script.
2. In the Write properties panel, click the file folder icon , then navigate to the directory path where you want to save the rendered image sequence. Enter a name for the image sequence.
3. 从 框架 下拉菜单，选择 偏移 。在右侧的字段中输入恒定偏移。将此常数值添加到当前帧中，以获取写出的帧的编号。

例如，如果剪辑从帧 500 开始，并且您希望将第一帧渲染为 image.0001.rgb 而不是 image.0500.rgb，则可以使用 499 作为常数偏移。这样，从当前帧中减去 499，以获得写出的帧的编号。在第 500 帧，image.0001.rgb 被写出; 在第 501 帧，image.0002 被写出，依此类推。
