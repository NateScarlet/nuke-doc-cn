渲染农场

核武器 几乎所有第三方和专有渲染排队软件都支持。通过整合 核武器 有了这样的系统，渲染负载可以分布在所有的 核武器 -网络上的许可机器，无论是基于 Windows 、 Mac 还是基于 Linux 的。

注意: 如果你有一个 核武器工作室 许可证，您可以利用内部帧服务器，它允许您设置外部从机来更快地处理渲染。请参阅 在外部机器上使用帧服务器 欲了解更多信息。

您的安装 核武器 可以配置为将作业发送到网络渲染场，该场通常在 渲染 菜单 (即, 渲染> 渲染 )。但是，由于必须为每个工作室自定义此选项，因此您应该向系统管理员咨询如何发送 核武器 网络渲染脚本。

提示: 如果你试图强迫 核武器 要重试操作而不是失败，例如在渲染过程中出现许可证问题，您可能会发现 核武器 的退出代码很有帮助:
0 = 成功
1 = 渲染错误
100 = 许可证失败
