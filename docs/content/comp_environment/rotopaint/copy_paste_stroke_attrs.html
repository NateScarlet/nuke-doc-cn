复制、剪切和粘贴描边属性

复制、剪切和粘贴笔触/形状属性的实现方式与影响位置的方式类似:

复制笔触/形状属性的步骤

1。 随着 选择全部 , 选择点 ,或 选择羽毛点 工具 激活时，在查看器中选择笔触/形状。
2. 确保 隐藏点 在工具设置中禁用。
3. 在查看器中，围绕要复制的点 (或整个描边/形状) 拖动选框。如果 隐藏变换句柄 如果禁用，将出现一个变换框。
4. 在变换框内单击鼠标右键，然后选择 复制 > 曲线 (属性值)。

笔触/形状的属性值 (包括位置和形状) 将复制到剪贴板。值仅适用于特定关键帧。

切割描边/形状属性

1。 随着 选择全部 , 选择点 ,或 选择羽毛点 工具激活时，在查看器中选择笔触/形状。
2. 确保 隐藏点 在工具设置中禁用。
3. 在查看器中，围绕点拖动选框，或单击要切割的点的描边/形状。如果 隐藏变换句柄 如果禁用，将出现一个变换框。
4. 在变换框内单击鼠标右键，然后选择 切 > 曲线 (属性值)。

笔触/形状的属性值 (包括位置和形状) 将从查看器中剪切并放置在剪贴板中。值仅适用于特定关键帧。请注意，切割笔触/形状的形状不会从查看器中消失。

粘贴描边/形状属性

1。 随着 选择全部 , 选择点 ,或 选择羽毛点 工具 激活时，在查看器中选择笔触/形状。
2. 确保 隐藏点 在工具设置中禁用。
3. 在查看器中，围绕点拖动选框，或单击要粘贴的点的描边/形状。如果 隐藏变换句柄 如果禁用，将出现一个变换框。
4. 在变换框内单击鼠标右键，然后选择 粘贴 > Spline (属性值) .

这将粘贴剪贴板上复制的笔触/形状属性值。例如，如果复制了另一种类型的曲线, (值) 或 (动画) ,则禁用除该选项之外的所有选项。
