使用工具栏

在 RotoPaint 节点中，可以使用两个工具栏来定义要开始绘制的笔画/形状的类型。这些工具栏放置在查看器中。垂直 RotoPaint 工具栏用于选择要使用的工具，水平工具 RotoPaint 工具设置用于在绘制新笔划/形状之前调整当前选定工具的设置。

注意: 不能使用 RotoPaint 工具设置来调整已经存在的笔触/形状。对于在创建笔画/形状后要对其进行的任何更改，可以使用 RotoPaint 属性面板中的控件。

RotoPaint 工具栏和工具设置。

在 RotoPaint 工具栏中，可以选择笔触/形状工具。工具分组在工具栏图标下。您可以单击任何工具以使其处于活动状态，并通过右击 (或左键单击并按住) 图标查看工具组。当前选定的工具将被突出显示。

提示: 当查看器具有鼠标悬停焦点时，您也可以使用 S 通过当前选定工具的模式循环的键盘快捷键。

在查看器顶部的 RotoPaint 工具设置中，您可以定义所选工具的设置。此工具栏中的控件会根据您在任何给定时间选择的工具而更改。

提示: 通过单击工具栏旁边的隐藏按钮，可以隐藏工具栏。你也可以按 [ (方括号) 隐藏 RotoPaint 工具栏和 { (卷曲括号) 隐藏工具设置 (和查看器的整个顶部工具栏)。
