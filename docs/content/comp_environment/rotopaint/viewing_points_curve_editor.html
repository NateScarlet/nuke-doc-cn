查看曲线编辑器和摄影表中的点

您可以在 “曲线编辑器” 和 “摄影表” 中编辑笔触/造型点、其羽毛点和切线。若要阅读有关使用曲线编辑器和摄影表的更多信息，请参见 动画参数 .

要在曲线编辑器中查看点:

1。 选择您的笔触/形状，然后在一点上单击鼠标右键。
2. 选择 曲线编辑器… 并选择:

• 点 -将点添加到曲线编辑器

• 点 羽毛 -将您的点及其羽毛点添加到曲线编辑器中

• 点 切线 -将点及其切线添加到曲线编辑器

• 所有 -将点、羽毛点和切线添加到曲线编辑器。

3. 在涂料表上查看你的点非常相似，只需右键单击你的点，然后在下面选择一个相同的选项 涂料片.
