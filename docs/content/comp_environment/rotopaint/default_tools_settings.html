设置默认 RotoPaint 工具和设置

您可以选择让 RotoPaint 始终打开并选择特定的工具，或者使用您想要的设置打开特定的工具。

要将工具设置为默认工具:

1。 创建一个名为 Menu.py 在你的插件路径目录中，如果一个已经不存在。有关插件路径目录的更多信息，请参见 加载 Gizmos 、 NDK 插件以及 Python 和 Tcl 脚本 .
2. 从以下列表中选择您的默认工具，并在您的 Menu.py :

• 选择全部

Nuke.knobDefault (“rotopaint.Toolbox”，“selectall”)

• 选择样条

Nuke.knobDefault (“rotopaint.Toolbox”，“selectcurves”)

• 选择点

Nuke.knobDefault (“rotopaint.Toolbox”，“selectpoints”)

• 选择羽毛点

Nuke.knobDefault (“rotopaint.Toolbox”，“selectfeatherpoints”)

• 添加点

Nuke.knobDefault (“rotopaint.Toolbox”，“addpoints”)

• 移除点

Nuke.knobDefault (“rotopaint.Toolbox”，“removepoints”)

• 尖点

Nuke.knobDefault (“rotopaint.Toolbox”，“尖点”)

• 平滑点

Nuke.knobDefault (“rotopaint.Toolbox”，“曲线点”)

• 去除羽毛

Nuke.knobDefault (“rotopaint.Toolbox”，“removefeather”)

• 打开/关闭曲线

Nuke.knobDefault (“rotopaint.Toolbox”，“closcurve”)

• 贝塞尔

Nuke.knobDefault (“rotopaint.Toolbox”，“createbezier”)

• 尖头贝塞尔

Nuke.knobDefault (“rotopaint.Toolbox”，'createBezierCusped')

• B-样条

Nuke.knobDefault (“rotopaint.Toolbox”，“createbspline”)

• 椭圆

Nuke.knobDefault (“rotopaint.Toolbox”，“createellipse”)

• 矩形

Nuke.knobDefault (“rotopaint.Toolbox”，“createrectangle”)

• 尖头矩形

Nuke.knobDefault (“rotopaint.Toolbox”，'createRectangleCusped')

• 画笔

Nuke.knobDefault (“rotopaint.Toolbox”，“brush”)

• 橡皮擦

Nuke.knobDefault (“rotopaint.Toolbox”，“橡皮擦”)

• 克隆

Nuke.knobDefault (“rotopaint.Toolbox”，“克隆”)

• 揭示

Nuke.knobDefault (“rotopaint.Toolbox”，“reveal”)

• 道奇

Nuke.knobDefault (“rotopaint.Toolbox”，“dodge”)

• 燃烧

Nuke.knobDefault (“rotopaint.Toolbox”，“burn”)

• 模糊

Nuke.knobDefault (“rotopaint.Toolbox”，“blur”)

• 锐化

Nuke.knobDefault (“rotopaint.Toolbox”，“锐化”)

• 涂抹

Nuke.knobDefault (“rotopaint.Toolbox”，“smear”)

3. 重新启动 核武器 .

要设置默认工具属性:

1。 创建一个名为 Init.py 在你的插件路径目录中，如果一个已经不存在。有关插件路径目录的更多信息，请参见 加载 Gizmos 、 NDK 插件以及 Python 和 Tcl 脚本 .
2. 定义 RotoPaint 工具和要设置的默认设置。要获取特定的工具和设置名称，您可以将 RotoPaint 节点复制到 核武器 并将其粘贴到文本编辑器中。“工具箱” 后面带有弯曲括号的线条表示您当前的工具设置。

例如:

• 要设置 刷子大小 对于油漆 30 ,和 克隆 到 280 ,将此 Python 行添加到您的 Init.py :

Nuke.knobDefault (“RotoPaint.toolbox”，克隆 {

{刷 bs 30}

{克隆 bs 280}

} ')

OR

• 要设置 刷子硬度 对于油漆 1.0 默认情况下，添加此 Python 行:

Nuke.knobDefault (“RotoPaint.toolbox”，'brush {

{刷 h 1}

}''')

• 要设置 源变换 克隆轮到 上 默认情况下，添加此 Python 行:

nuke.knobDefault("RotoPaint.toolbox", '''clone {

{克隆 str 1}

}''')

3. Restart Nuke .

注意: 从上面的示例中可以看到，不同的 RotoPaint 工具有自己的默认工具设置。如果您尝试设置默认值，但没有指定哪个工具，则当您切换到其他工具时，默认值将被忽略。例如，nuke.knobDefault ('rotopaint.toolbar_brush_hardness'，'1.0') 集 刷子硬度 到 1.0 最初，但是一旦你激活了一个不同的工具，你就会得到该工具的默认值。

注意: 无法通过 RotoPaint.toolbox 设置多个默认值。例如，如果运行以下三个命令，则只为最后一个 (锐化) 设置默认值:
Nuke.knobDefault (“RotoPaint.toolbox”，'' 克隆 {克隆 ltt 0} ')
Nuke.knobDefault (“RotoPaint.toolbox”，'blur {blur ltt 0}')
Nuke.knobDefault (“RotoPaint.toolbox”，'锐化 {锐化 ltt 0}')
要设置克隆和模糊的默认值，你应该使用这个:
Nuke.knobDefault (“RotoPaint.toolbox”，克隆 {
{刷 ltt 0}
{克隆 ltt 0}
} ')
