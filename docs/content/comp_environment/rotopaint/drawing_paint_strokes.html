绘制油漆笔画

任何给定的 RotoPaint 节点都可以容纳许多绘画笔触和形状。您可以使用以下任何工具应用绘制笔划 (另请参见 绘制形状 )。

图标

工具

键盘快捷键

函数

画笔

N (在刷子和橡皮擦之间切换)

在当前板上应用颜色，或将颜色与当前板混合。您也可以从周围的框架克隆。

橡皮擦

N

从现有笔划中删除像素并恢复背景。

克隆

C (在克隆和显示之间切换)

将当前板的一个区域中的像素应用于当前板的另一个区域。

揭示

C

将源板中的像素应用于相应位置的目标板。

模糊

X (在模糊、锐化和涂抹之间切换)

模糊笔刷笔划区域中的图像。

锐化

X

在笔刷笔划区域中锐化图像。

涂抹

X

涂抹涂抹笔刷笔划的区域，将选定像素拉伸到其周围区域。

道奇

D (在道奇和烧伤之间切换)

使画笔笔划区域上的背景颜色变亮，以反映画笔笔划。在黑色上使用这个工具不会产生任何变化。中风区没有一部分是黑暗的。

燃烧

D

使笔刷描边区域上的背景色变暗以反映笔刷描边。中风区域的任何部分都没有变亮。

提示: 您可以选择让 RotoPaint 始终打开，并选择特定的工具。例如，如果要在选择画笔工具的情况下打开它，请执行以下操作:
1。创建一个名为 Menu.py 在你的插件路径目录中，如果一个已经不存在。有关插件路径目录的更多信息，请参见 加载 Gizmos 、 NDK 插件以及 Python 和 Tcl 脚本 .
2。要选择画笔工具作为默认工具，请在您的 Menu.py :
Nuke.knobDefault (“rotopaint.Toolbox”，“brush”)
3.重启 Nuke .

有关默认工具的更多信息，请参见 设置默认 RotoPaint 工具和设置 .

提示: 如果您使用的是平板电脑，你可以将一个新的行程的不透明度，大小，或硬度的笔压按相应 压力改变 RotoPaint 工具设置中的复选框。您也可以在以后更改该属性的压力将更改为 中风 属性面板的选项卡。

RotoPaint 工具栏中的单独选择工具允许您在绘制笔画/形状后选择它们，之后您可以使用 “属性” 面板控件对它们进行更改 (请参见 选择现有笔划/形状进行编辑 ,和 编辑现有笔触/形状属性 )。
