Bezier 和 Paint 节点在哪里？

Bezier 和 Paint 节点 pre-6.0 已经被否决，而 RotoPaint 节点则被否决。有了 RotoPaint，你可以添加更多的笔画和形状，对它们进行分组，等等。然而，Bezier 和 Paint 仍然在应用程序中，以向后兼容旧脚本。如果你找到了需求 (或者只是感到怀旧)，你可以用几个简单的方法创建 Paint 和 Bezier 节点:

• 新闻 X 在节点图上，确保 Tcl 在打开的对话框中选中，请输入 油漆 或 贝塞尔 在 命令 字段，然后单击 好 。您的节点显示在节点图中。

• 您可以使用您的工具栏菜单中的语句向工具栏菜单中添加 Paint 或 Bezier 节点 Menu.py 文件如下所示:

• 对于贝塞尔:

Tb = nuke.toolbar (“节点”)

Tb.addCommand ("Draw/Bezier"，"nuke.createNode (\" Bezier \ ")"，图标 = "Bezier.Png")

• 对于油漆:

tb = nuke.toolbar("Nodes")

Tb.addCommand (“Draw/Paint”，“nuke.createNode”，图标 = “Paint.Png”)
