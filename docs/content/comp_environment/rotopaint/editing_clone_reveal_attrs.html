编辑克隆或显示属性

使用克隆或显示工具时，可以调整下的控件 克隆 选项卡转换正在克隆或显示的输入。调整:

• 翻译 -在 x 轴和 y 轴上移动源图像。

• 旋转 -围绕枢轴点旋转源图像。

• 规模 -通过添加或删除像素来调整源图像的大小。使用 中心 (或 Ctrl / Cmd 拖动变换插孔) 以定位枢轴点。

• 倾斜 X -沿 X 轴从枢轴点倾斜源图像。使用 中心 (或 Ctrl / Cmd 拖动变换插孔) 以定位枢轴点。

• 倾斜 Y -将源图像沿 Y 轴从枢轴点倾斜。使用 中心 (或 Ctrl / Cmd 拖动变换插孔) 以定位枢轴点。

• 斜阶 -设置其中的顺序 倾斜 X 和 倾斜 Y 应用:

• XY - 倾斜 X 在之前应用 倾斜 Y .

• YX - 倾斜 Y 在之前应用 倾斜 X .

• 舍入到像素 -检查这个来绕过 翻译 金额为最接近的整数像素。这可以帮助您避免在使用过滤时软化。如果 源 设置为 颜色 在 中风 选项卡，此控件被禁用。

• 过滤器 -选择合适的过滤算法。有关更多信息，请参见 选择过滤算法 .

• 黑色外面 -当旋转或翻译克隆源时，图像区域的一部分可能会被裁剪。要用黑色填充裁剪部分，请检查 黑色外面 。要通过扩展图像的边缘来填充裁剪部分，请取消选中 黑色外面 .

• 时间偏移 -从不同的帧克隆或显示像素。时间偏移相对于当前帧 (-1 是当前帧之前的帧) 或绝对 (1 是剪辑中的第一帧)。

• 视图 -在立体项目中选择要克隆的视图。
