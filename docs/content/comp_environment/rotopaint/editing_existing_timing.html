编辑现有描边/形状计时

编辑现有描边/形状时，可以编辑在属性中可见描边/形状的帧范围 寿命 标签。笔画/形状/组的寿命在 生活 笔触/形状列表中的列。默认情况下，形状在上可见。 所有帧 ,而中风仅在 单帧 ,它被画在框架上。

图标 终身类型 描述

所有帧

单击以使所有帧的描边/形状可见 (默认)。

帧到结束

单击以使描边/形状从当前帧到最后一帧可见。

单帧

单击使描边/形状仅在当前帧上可见。

开始帧

单击以使笔触/形状从第一帧到当前帧可见。

帧范围

使用 “帧” 对话框或单击在指定帧范围内使笔画/形状可见 从 和 到 控制。
