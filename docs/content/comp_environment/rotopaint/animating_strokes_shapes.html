动画笔画/形状

出现在多个帧上的所有笔触/形状都可以设置动画。默认情况下, 自动密钥 RotoPaint 工具设置中的选项处于启用状态，这意味着对笔触/形状的更改会自动创建关键帧并对笔触/形状进行动画处理。还可以访问 “曲线编辑器” 和 “摄影表” 中的所有曲线和形状。

使用自动键设置笔画/形状动画

1。 绘制出现在多个帧上的描边/形状。默认情况下, 自动密钥 选择 RotoPaint 工具设置中的选项，并在您的描边/形状出现的第一帧中自动创建关键帧。
2. 移动到新框架。
3. 使用其中一个选择工具，选择要设置动画的点或描边/形状。
4. 根据需要调整笔划/形状位置中的点，或更改笔划/形状的属性。自动设置新的关键帧。时间轴上的帧标记变为蓝色，表示选定的笔触/形状已设置动画。
5. 对要设置为关键帧的所有帧重复步骤 2 、 3 和 4。

提示: 请注意，如果你把整个行程/造型使用 变换 Tab，RotoPaint 还绘制了笔画/形状动画位置的轨迹。例如，您可以通过以下方式在另一个节点 (如跟踪器或 CameraTracker) 中使用轨迹 Cmd / Ctrl 将值从 “平移动画” 按钮拖动到另一个节点中的适当字段。

手动设置笔画/形状动画

如果您选择切换 自动密钥 功能关闭，您仍然可以手动创建关键帧。可以将关键帧设置为整个描边/形状，或描边/形状的样条曲线、变换或属性。

1。 移动到要创建关键帧的帧，然后选择您的描边/形状。
2. 执行以下操作之一:

• 如果要创建一个设置为动画整个描边/形状的关键点，请在描边/形状上单击鼠标右键并选择 设置键> 全部 .

• 如果要创建设置为动画位置的关键点，请在描边/形状上单击鼠标右键，然后选择 设置关键点> 样条曲线。

• 如果要创建设置为动画变换的关键点，请在描边/形状上单击鼠标右键，然后选择 设置键> 变换 .

• 如果要创建设置为动画属性的关键点，请在描边/形状上单击鼠标右键，然后选择 设置键> 属性 .

如果你有 自动密钥 禁用该选项后，只能在关键帧处调整笔触/形状中的点。换句话说，为了对一个点进行更改，您要么必须移动到时间线上的现有关键帧，要么需要先创建一个新的关键帧。
