选择输出格式和通道

在 RotoPaint 属性面板中，可以选择一个或多个输出通道，以指示应存储更改结果的通道。

1。 从 输出 字段中，选择包含要使用的通道的层。默认情况下, Rgba 选中，并在右侧选中红色、绿色、蓝色和 alpha 通道。
2. 取消选中您不想处理的任何频道。该节点处理所有您退出选中的节点。有关选择通道的详细信息，请参阅 呼叫渠道 .
3. 如果你想，你可以使用 输出 面具 下拉菜单选择一个频道，RotoPaint 输出一个遮罩来显示它所呈现的内容。默认情况下，通道是 没有 ,但是如果您在菜单中选择了一个频道, 输出 面具 框将自动选中。

例如，如果你需要将纹理应用于你画的区域，但是你不想在其他区域将纹理加倍，面具会很有用。

默认情况下, 输出掩模 控件是隐藏的，但是你可以通过点击上面的黑色三角形来显示它 颜色。

4. 如有必要，选择您的 预乘 价值。

预乘 将选定的输入通道与表示绘制笔划和形状的遮罩相乘。例如，如果没有绘制笔划或形状 (绘制遮罩为黑色或空)，输入通道将设置为黑色, 如果油漆笔划或形状不透明 (油漆哑光是白色或全色)，输入通道将保持其全值。

请注意，选择 Rgba 将 alpha 与自身 (a * a) 相乘。如果你不想发生这种情况，设置 预乘 到 Rgb 相反。

提示: 您可以使用 链接菜单 将通道控件与其他控件链接。有关更多信息，请参见 使用链接菜单链接通道 .

5. 从 剪辑到 下拉菜单中，选择要如何限制输出图像:

• 没有 剪辑 -不要限制输出图像。

• Bbox -将输出图像限制到传入的边界框。

• 格式 -将输出图像限制在传入格式区域 (默认)。

• 联盟 Bbox 格式 -将输出图像限制为边界框和传入格式区域的组合。

• 相交 Bbox 格式 -将输出图像限制为边界框和传入格式区域的交集。

您也可以检查 替换 如果要在绘制通道之前将通道清除为黑色，请选中此框。你可能会发现 替换 例如，如果你在 alpha 通道中创建一个遮罩，但是传入的图像已经有一个 alpha 通道，你想扔掉它。

6. 如果您没有将输入连接到 RotoPaint，请选择您的 格式 价值。这是在没有任何可用输入格式的情况下节点应该输出的格式。如果连接了输入，则此控制无效。

默认情况下, 格式 控件是隐藏的，但是你可以通过点击上面的黑色三角形来显示它 颜色 .
