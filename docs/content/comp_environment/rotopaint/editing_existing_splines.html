编辑现有描边/形状样条线

要编辑笔画/形状的位置，首先需要在查看器或笔画/形状列表中选择笔画/形状。然后，可以修改组成笔划/形状位置的点。

向描边/形状添加点的步骤

1。 选择 添加点 工具 从 RotoPaint 工具栏。
2. 确保 隐藏点 在工具设置中禁用。
3. 在查看器中，单击选定的描边/形状以添加新点。

移动一个点

1。 随着 选择全部 工具 或 选择点 工具 在 “查看器” 或 “描边/形状” 列表中选择描边/形状。
2. 确保 隐藏点 在工具设置中禁用。
3. 在查看器中，拖动要移动到新位置的点。
4. 你也可以 使用数字键盘上的箭头轻推一个点。轻推一个点会将其移动到您选择的方向上一个像素。

提示: 如果你发现很难选择一个点，你可能想增加 2D/3D 手柄尺寸 或的 手柄拾取尺寸 在 核武器 首选项对话框 (偏好 > 面板 > 查看器句柄 ) .

一起移动几个点

1。 随着 选择全部 工具 或 选择点 工具 在 “查看器” 或 “描边/形状” 列表中选择描边/形状。
2. 确保 隐藏点 在工具设置中禁用。
3. 在查看器中，围绕要移动的点 (或整个描边/形状) 拖动选框。如果 隐藏变换句柄 如果禁用，将出现一个变换框。
4. 根据需要调整变换框。

注意: 移动形状 选择全部 活动将选定的点或羽毛点移动到新位置。擦洗时间线显示形状在关键帧之间穿过观察者。然而，移动形状 选择羽毛点 “活动” 仅将羽毛点移动到新位置，显示选定点的羽毛衰减。

删除点的步骤

1。 右键单击 添加点 工具和选择 移除点 工具 .
2. 在查看器或笔画/形状列表中选择笔画/形状。
3. 确保 隐藏点 在工具设置中禁用。
4. 在查看器中，单击要删除的点。

OR

在查看器中，右键单击要删除的点，然后选择 删除 .

删除整个描边/形状

1。 在笔触/形状列表中，或在具有 选择全部 工具 ,单击要删除的笔画/形状。
2. 在笔画/形状列表下方，单击减号按钮 (-),

OR

1。 激活 选择全部 工具 。在 观众 ,单击笔画/形状。
2. 在形状上单击鼠标右键并选择 删除 或按 删除 / 退格 关键。
