的 B-样条 工具允许您绘制 B-spline 形状。

绘制 b样条形状。

使用 b样条工具的步骤

1。 右键单击 贝塞尔 在 RotoPaint 工具栏中选择 B-样条 工具 .
2. 在查看器顶部的 RotoPaint 工具设置中，为形状选择颜色、混合模式、不透明度和其他设置。(有关可用选项的信息，请参见 编辑现有笔触/形状属性 )
3. 或者，在 RotoPaint 工具设置中设置形状的寿命。(有关可用选项的信息，请参见 编辑现有描边/形状计时 )。
4. 在查看器中单击以创建构成形状的样条线。

• Ctrl / Cmd 拖动可自由绘制样条曲线。

• 单击向左或向右拖动光标以调整张力 (即样条线和点之间的距离) 关于前一点 .

• 调整的张力 任何点 在您正在绘制的 B-spline 形状中，选择点，按 Ctrl/Cmd Shift, 向左或向右拖动: 向右增加张力，向左减少张力。

5. 要关闭形状，请按 返回 。更改为不同的工具也会关闭形状。默认情况下，关闭形状会激活 “全选” 工具。
6. 使用 “选择全部” 工具，您可以 转变 + 单击多个形状点以打开变换框，您可以使用该框进一步变换形状或形状中的特定点。
7. 您也可以在 b样条形状中尖点和平滑点，以在形状上创建锐角或再次平滑。为此，请在 b样条形状中的一点上单击鼠标右键，然后选择 尖头/去光滑 或 平滑 .

提示: 创建 B-spline 形状后，可以将其转换为 Bezier 形状。使用 “全选” 工具选择 b样条形状，在其上单击鼠标右键并选择 将 bspline 转换为 bezier 。现在，您可以像编辑其他贝塞尔形状一样编辑形状。
