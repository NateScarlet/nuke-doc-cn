自动计算镜头失真

图像分析自动估计序列中的镜头失真 (仅序列，不能对静止图像执行此分析)。它通过序列跟踪特征，并找到最能描述相同 3D 结构投射到图像不同部分的方式的失真模型。要自动分析镜头失真，请执行以下操作:

1。 单击 图像分析 标签。
2. 如果你的序列很长，你可能 希望通过选择来选择要分析的帧范围 指定 范围 下 分析范围 。改变范围的另一个原因是选择序列中有一些强特征的部分进行跟踪, 或者它检测到的特征在图像中分布良好 (例如，而不是全部聚集在一个角落)。指定中的范围 分析开始 和 分析停止 字段。
3. 如果您不想分析图像的所有区域，例如，如果图像中有移动对象，您可以在 面具 的输入或 alpha 通道 源 输入。从 面具频道 下拉菜单中，选择哑光频道。
4. 最后，按 分析图像 开始估计。
5. 当分析结束时，失真参数被设置在 透镜失真 选项卡以保持输入不变。

图像分析参数

这些是上的参数 图像分析 标签:

• 面具 -如果您不想分析图像的所有区域，例如，如果图像中有移动对象，您可以在 面具 或 源 输入并使用此控件选择遮罩通道。

• 分析范围 -选择是否分析整个 源剪辑范围 或一个 指定范围 ,定义在 分析开始 和 分析停止 字段。

• 相机运动 -选择 仅旋转 如果适用，或 自由运动 对于所有其他类型的运动。

• 分析序列 -通过输入序列跟踪特征并估计镜头失真。分析完成后，将在 透镜失真 为了不扭曲序列。

• 特征覆盖 -分析后，您可以选择显示 特点 或 功能和 轨道 在当前帧上检测到的。
