计算一幅图像的失真并将其应用于另一幅图像

您可以使用透镜失真节点分析一个图像或网格上的失真，然后将该失真应用于另一个图像。执行以下操作:

1。 使用透镜失真节点分析图像或网格以获取失真参数。确保 输出类型 设置为 图像 在 透镜失真 透镜失真节点的选项卡。检查生成的图像是否没有失真。
2. 设置 输出类型 到 位移 .

在这种模式下，透镜失真节点单独离开输入图像，并将与计算的失真相对应的像素位移写入 UV 通道。这些显示为向前 运动 观众的频道下拉列表中的频道。置换采用 STMap 节点使用的格式。

3. 取消选中 不失真 应用失真，而不是删除它。
4. 选择 变换 > STMap 在 Nuke 用于在节点图中插入 STMap 节点的工具栏。连接 STMap 节点的 Stmap 输入到透镜失真节点。
5. 使用步骤 1 中计算的失真导入要扭曲的图像。将该图像连接到 Src STMap 节点的输入。
6. 在 STMap 控件中，选择 运动 从 UV 通道下拉菜单。现在，这应该将从 lensdistraxed 节点的输入到 STMap 节点的失真 Src 输入。
