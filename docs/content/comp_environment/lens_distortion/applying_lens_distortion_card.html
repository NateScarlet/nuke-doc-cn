将镜头失真应用于卡节点

您可以使用计算出的镜头失真值来扭曲投影到 3D 卡片上的图像。执行以下操作:

1。 打开 卡 参数 “镜头失真” 选项卡上的文件夹。
2. 按住 Ctrl / Cmd 并拖动 x , y 和 z 值从 规模 参数的值上的相应栏目, 卡 卡节点中的选项卡 ( 3D > 几何 > 卡 ) 属性面板。这为卡上的再现失真设置了正确的比例。
3. 按住 Ctrl/Cmd 并拖动 一个 , B, 和 c 值放在相应的字段上 镜头 失真 卡节点属性面板中的选项卡。这些值现在用于不扭曲附加到 Img 卡节点的输入。

注意: 卡节点的镜头失真参数是估计的，不一定与镜头失真结果完全匹配，特别是如果计算出的镜头是不对称的, 换句话说，在 x 和 y轴上有不同数量的失真。
