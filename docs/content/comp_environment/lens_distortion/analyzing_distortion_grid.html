使用网格分析失真

网格分析估计棋盘或细线网格的失真，以获得更高的精度。一般来说，如果你有一个网格可以用来计算镜头失真，你应该使用网格分析来做到这一点。要分析网格，请执行以下操作:

1。 要估计网格的失真，请转到 网格分析 标签。
2. 将网格连接到透镜失真节点 源 输入并选择适当的 网格类型 .
3. 新闻 分析网格 估计失真和不失真的网格。
4. 分析网格后，您可以检查 对齐网格 纠正由于网格没有被直接拍摄到相机上而导致的透视失真。
5. 如果要将网格的失真应用于图像输入，请断开网格与 源 输入并将图像连接到它。

网格分析参数

这些是上的参数 网格分析 标签:

• 网格类型 -选择要使用的网格类型, 棋盘 或 细线 网格 .

• 分析网格 -计算镜头失真。分析完成后，将在 透镜失真 为了不扭曲序列。

• 对齐 网格 -检查以纠正由于网格没有被拍摄到相机上而导致的透视失真。换句话说，您可以将网格线与图像边缘对齐。

• 网格覆盖 -分析后，您可以选择显示 原始 或 不失真 网格 图像上的线条。
