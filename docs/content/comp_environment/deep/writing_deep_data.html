编写深层数据

您可以在扫描线 OpenEXR 中写出深层图像 2.2 使用 DeepWrite 节点进行格式化，该节点与标准写入节点共享许多控件。执行以下操作:

1。 选择 深 > DeepWrite 将 DeepWrite 节点插入到脚本中。
2. 在 “属性” 面板中，单击 文件 或 代理 字段的文件夹图标，并浏览到要存储深层图像的目录。
3. 在路径后，键入深层图像的名称，包括。Exr 扩展，然后单击 好 。如果要渲染图像序列，请包括帧号变量 (例如, #### ) 的名字。
4. 使用 数据类型 下拉菜单选择渲染文件的位深度: 16 位一半 或 32 位浮动 .
5. 设置 压缩 应用于渲染文件的压缩类型。
6. 从 元数据 下拉菜单中，选择渲染文件中包含的元数据:

• 没有元数据 -不创建自定义属性，只写出填充所需标题字段的元数据。

• 默认元数据 -可选的时间码、边缘码、帧速率和曝光头字段也使用元数据值填充。

• 默认元数据和 exr/*

• 除 input/* 之外的所有元数据

• 所有元数据

7. 默认情况下，未知元数据键具有前缀 核武器 当它们写入文件时附加到它们上。如果你想把它们写在文件 “按原样”，没有前缀，请检查 不附加前缀 .
8。 根据需要调整其余的控件。有关它们的更多信息，请参见 输出 (写) 节点 .
9. 单击 渲染 按钮。

核武器 提示输入帧范围，默认为 帧范围 字段。

10. 如有必要，更改开始和结束帧，然后单击 好 .

核武器 将深层数据写入扫描线 OpenEXR 2.2 或以上文件 (不支持平铺 OpenEXR 文件)。
