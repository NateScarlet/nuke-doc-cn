裁剪、重新格式化和转换深层图像

您可以使用相应的深层节点，以与常规图像相同的方式裁剪、重新格式化和变换深层图像。

注意: 请记住，由于每个像素上的样本可以位于任意深度，因此在转换过程中重新采样可能会产生意外的结果，因为相邻像素上可能没有相同深度的样本。

裁剪深层图像

您可以使用 DeepCrop 节点来剪辑深层图像，这很像正常的裁剪节点:

1。 将 DeepCrop 节点连接到要裁剪的深层图像。
2. 在 X 和 Y 方向调整查看器中的裁剪框以定义裁剪区域。此外，确定你的作物面积使用 Bbox 属性面板中的字段。如果你想将深度样本保持在裁剪框之外，你可以检查 保持在 bbox 之外 盒子。
3. 使用 Znear 和 Zfar “属性” 面板中的控件可对样本进行深度裁剪。如果您不想使用这些控件中的任何一个，可以通过取消选中 使用 他们旁边的盒子。如果要将深度样本保持在这些控件定义的 z 范围之外，应检查 保持在 zrange 之外 盒子。

重新格式化深层图像

DeepReformat 是深层数据的重新格式化节点。您可以使用它来设置深层图像的尺寸、比例等。要重新格式化深层图像:

1。 将 DeepReformat 节点连接到要调整大小的深层图像。
2. 在 类型 下拉菜单，选择:

• 到 格式 -将输出宽度和高度设置为选定格式。在中选择格式 输出 格式 下拉列表。如果格式尚不存在，您可以选择 新 从头开始创建新格式。默认设置, 根.格式 ,将图像大小调整为 项目设置 对话框。

• 到 盒子 -将输出宽度和高度设置为以像素为单位定义的尺寸。在中输入值 宽度 , 高度 和 像素 方面 用于指定尺寸的字段。

• 规模 -将输出宽度和高度设置为输入大小的倍数。使用 规模 滑块来定义因子。比例因子稍微舍入，使输出图像在下面选择的方向上是整数像素 调整大小类型 .

3. 您可以指定哪些大小你想要的 调整大小 类型 下拉列表。选择:

• 没有 -不调整原始大小。

• 宽度 -缩放原始图像，直到其宽度与输出宽度匹配。然后以保持原始纵横比的方式缩放高度。

• 高度 -缩放原始图像，使其填充输出高度。然后以保持原始纵横比的方式缩放宽度。

• 适合 -缩放原始图像，使其最小边填充输出宽度或高度。然后以保持原始纵横比的方式缩放最长的边。

• 填充 -缩放原始图像，使其最长的边填充输出宽度或高度。然后以保持原始纵横比的方式缩放最小边。

• 扭曲 -缩放原始尺寸，以便两侧填充输出尺寸。此选项不保留原始纵横比，因此可能会发生扭曲。

4. 检查 中心 用于定义输入像素是应重新采样到新大小还是在输出中居中。如果不居中，输入和输出的左下角将对齐。
5. 要进一步调整图像的布局，您可以选中相应的框:

• 翻转 -交换图像的顶部和底部。

• 翻牌 -交换图像的左侧和右侧。

• 转弯 -将图像旋转 90 度。

• 黑色 外面 -将像素设置在黑色格式之外。

• 保存 边界 盒子 -保留输出格式之外的像素，而不是将它们剪掉。

转化深层样品

可以使用 DeepTransform 节点重新定位深度样本。

1。 将节点连接到要转换的深度镜头。
2. 使用 翻译 x , y ,和 z 用于翻译样本的控件。
3. 使用缩放样本的 z 深度 Zscale 控制。高于 1 的值会增加深度，而低于 1 的值会减少深度。
4. 如果将遮罩连接到节点的 面具 输入，您可以使用它来调节深度转换在框架的不同部分的效果。
