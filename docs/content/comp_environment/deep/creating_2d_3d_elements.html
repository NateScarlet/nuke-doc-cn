从深层图像创建 2D 和 3D 元素

可以从深层图像创建二维图像或三维点云。

从深层图像创建 2D 图像

可以使用 DeepToImage 节点来展平图像，换句话说，将 deep image 中的所有样本合并为常规 2D 图像。

1。 将节点连接到要展平的深层图像 (或带有合并深层数据的深度合并)。
2. 在 “属性” 面板中, 体积 作文 默认情况下选中该框，但如果取消选中该框, 核武器 仅计算每个样本的前深度，并假设样本不重叠。如果取消选中此选项，计算所需的时间会更少，但是如果您的深层图像中有重叠的样本，结果图像可能不会像预期的那样代表每个像素。

从深层图像创建点云

您可以使用 DeepToPoints 节点将深度像素样本转换为 3D 空间中的点，您可以在 核武器 它的 3D 视图，很像点云。此节点对位置参考很有用。

1。 连接 DeepToPoints 节点 深 输入到要在 3D 中查看的深层图像。如果你有一个相机，你想通过它来观察点云，把它连接到 相机 输入。
2. 更改为三维视图 (按 标签 ) 查看结果。
3. 在 “属性” 面板中，可以使用 点 细节 滑块来调整云的密度。
点细节 设置为 0.005。 点细节 设置为 0.05。
4. 调整 点 尺寸 更改点的大小。您还可以在点云上使用三维对象选择，例如，在 DeepToPoints 结果上进行捕捉。(有关 3D 选择的更多信息，请参见 3D 选择工具 )。
点大小 设置为 2。 点大小 设置为 6。
