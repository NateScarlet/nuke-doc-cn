修改深层数据

Nuke 允许您为深度图像着色，以及使用表达式修改它们。

颜色校正深层图像

DeepColorCorrect 节点将颜色校正应用于每个像素的每个样本。

有用于调整阴影、中间色调和高光的控制集，以及 大师 设置一次调整所有这些。您可以使用上的查找曲线 范围 选项卡，用于控制被认为位于阴影、中间色调和高光中的图像的范围。有关颜色校正基础知识的更多信息，请参见 使用颜色 .

提示: 通过设置来制作遮罩 增益 Alpha 通道的值为 0，并设置 偏移 Alpha 通道的值为 1，在您想要遮罩的范围内。

调整深颜色校正的效果

在 掩蔽 选项卡您可以在深样本中设置颜色校正效果开始和结束的点。

1。 检查 Limit_z 框来激活 Zmap 工具。
2. 调整梯形，以便 一个 定界符标记您希望颜色校正开始的深度, B 和 C 标记完整效果和分隔符的长度 D 指示效果停止的位置。 Zmap 因此，工具的 y 轴指示效果的量，x 轴是深度样本的范围。
3. 使用 混合 控件来调整颜色校正结果和原始图像之间的整体混合。零值是原始图像，值 1 是全色校正结果。

用表情修改你的深层图像

可以使用 DeepExpression 节点运行 Nuke 深层数据上的表达式。使用 “属性” 面板中的控件:

1。 临时表达式有四个字段，就像正常表达式节点一样。如果您需要在多个字段中使用长表达式，并希望将该表达式临时分配给变量，则这些表达式将非常有用。在等号的左边输入变量名 ( = ) 标志，右侧的表达式。然后，可以使用变量名称在通道的表达式字段中表示整个表达式。
2. 在 Chans0 - Chans3 下拉列表然后可以指定要为其创建表达式的通道。这将添加或删除下面的表达式字段。
3. 然后，可以在 “通道表达式” 字段中输入不同通道的表达式。

有关表达式的更多信息，请查看 表情 .
