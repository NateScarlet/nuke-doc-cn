脚本编辑器和 Python

Nuke 附带一个全面的 Python 应用程序编程接口 (API)，使您能够使用 Python 脚本执行用户界面操作。这一章描述了如何使用脚本编辑器来执行 Python 脚本，并将您引导到关于 Python 的更多信息的来源。

快速启动

以下是工作流的快速概述:

1。 在中输入 Python 语句 Nuke 用于执行所需操作的脚本编辑器。
2. 用扩展保存脚本 。Py 在 sys.path 变量中包含的目录中。
3. 稍后，如果要执行相同的语句序列，请导入。Py 文件到 Nuke 的脚本编辑器。 Nuke 按指定顺序执行语句。

如果您需要更多关于在中使用 Python 的信息 Nuke ,你总是可以转向 Nuke Python 开发人员指南 ( 帮助 > 文档 )。

提示: 您可以导入 Nuke 作为第三方 Python 的模块 2.7.3 翻译。请参阅 Nuke 作为 Python 模块 欲了解更多信息。

提示: 您还可以在命令行上运行交互式 Python 会话 Nuke -t .
