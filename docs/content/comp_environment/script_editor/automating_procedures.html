在 Python 模块中保存语句

在 Python 模块中保存语句:

1。 在脚本编辑器的顶部，单击 保存脚本 按钮 .
2. 使用扩展保存脚本 。Py (例如 Firstmodule.py ) 在 sys.path 变量中包含的目录中。(要查看这些目录，请输入 打印 sys.path 在脚本编辑器中。要将目录添加到 sys.path 变量中，请输入 Sys.path.append ( "目录 ”) 在哪里 目录 表示要添加的目录。)

您现在已经创建了第一个 Python 模块。

在脚本编辑器中打开 Python 脚本

1。 单击 加载脚本 按钮 。在脚本编辑器的顶部。的 要打开的脚本 对话框打开。
2. 导航到包含要打开的脚本的 Python 模块，然后单击 打开 .
