在本用户指南的范围内，不可能详细介绍 Python 和所有可用的脚本。然而，如果您需要使用 Python 的帮助，您可能会发现更多信息的几个来源非常有用。

您可能还想阅读 Nuke 下的 Python API 参考文档 帮助 > 文档 > 巨蟒 脚本 参考 .

查看更多示例

在本章中，我们只描述了几个 Python 脚本的例子，但还有更多。您可以在以下位置找到它们:

• 在窗户上 :

驱动器号 : \ 程序文件 \ Nuke 10.5v8 \ Plugins \ nukescripts 或

驱动器号 : \ 程序文件 (x86) \ Nuke 10.5v8 \ Plugins \ nukescripts

• 在 Mac OS X 上 :

/应用程序/ Nuke 10.5v8 / Nuke 10.5v8 。应用程序/内容/MacOS/插件/nukescripts

• 在 Linux 上 :

/Usr/本地/ Nuke 10.5v8 /插件/nukescripts

要查看示例，请选择 。Py 文件，并在任何文本编辑器中打开它。

使用帮助语句

获取可用命令帮助的最快方法可能是在脚本编辑器中输入以下内容:

帮助 ( Nuke )

该语句按字母顺序列出了许多可用的 Python 命令及其描述。

您还可以获得更具体的帮助。例如，以下语句为您提供了对 SetValue () 方法:

帮助 (nuke.Knob.setValue)
