使用实用谷物

虽然 Nuke 的 ScannedGrain 节点提供了创建合成谷物的控制 (与刚才讨论的相比)，它的主要用途是读取和应用扫描谷物 -- 也就是说, 从实际胶片中提取的颗粒。如果您的设备有这样的序列，您可以在中读取并使用 ScannedGrain 节点应用它们。创建纹理文件如下所述，以及将生成的纹理文件与 ScannedGrain 节点一起使用。

创建胶片库存序列

1。 拍摄一张灰色卡片。只需要大约 50 帧。
2. 把胶片扫描进去。
3. 选择 图像 > 阅读 将扫描图像加载到 Nuke .
4. 添加模糊节点 ( 过滤器 > 模糊 ) 在图像之后模糊图像，直到看不到任何纹理。然后，将图像模糊一点。
5. 选择 合并 > 合并 在脚本中插入合并节点。将合并节点的 A 输入连接到原始图像，将 B 输入连接到模糊节点。然后，打开合并控件并选择 减去 从 操作 下拉菜单。从原始图像中减去模糊图像。

这一步和上一步的目的是从颗粒中减去任何整体灰度级，这样就只剩下颗粒了。

6. 选择 颜色 > 数学 > 添加 在减号节点后插入添加节点。在 “添加节点” 控件中，输入 0.5 价值 字段。这将为所有通道添加 0.5 的值。

这一步是必要的，因为 ScannedGrain 节点在读取 grain 文件时会从通道中减去 0.5 (需要减法来存储大多数文件格式的负数)。

7. 选择 图像 > 写 在添加节点后插入写节点。渲染输出。任何文件格式都可以 (例如，我们使用了.Rgb 扩展在我们网站上的谷物文件中)。

使用 ScannedGrain 节点添加扫描纹理

1。 单击 绘制 > ScannedGrain 在脚本中的适当位置插入 ScannedGrain 节点。
2. 将查看器连接到 ScannedGrain 节点的输出，以便您可以看到更改的效果。
3. 单击的文件夹图标 谷物 字段并导航到适当的胶片库存序列。选择 打开 .
4. 如有必要，请检查 调整大小 框向上或向下缩放谷物序列以匹配您的工作分辨率。
5. 在 最小宽度 字段，定义图像必须具有的最小宽度 (以像素为单位) 以接收纹理。
6. 在红色、绿色和蓝色中输入值 金额 场在每个通道的基础上增加或减少颗粒的密度。(粗略地说，这是通过增加或减少谷物序列的增益来实现的。)

现在你已经准备好微调谷物和背板之间的混合。

混合谷物和背板

1。 拖动到 饱和 滑块可在所有通道上增加或减少颗粒色调的强度。
2. 如有必要，还可以使用提供的曲线编辑器编辑纹理序列的颜色曲线。通过这种方式，您可以在每个通道的基础上改变增益、伽马、对比度等。(这些曲线的功能与中描述的相同。 使用 ColorLookup 曲线 )。
3. 要设置一个低阈值，根据输入图像，在该图像下不减去颗粒，请调整 Low_clip 滑块。
