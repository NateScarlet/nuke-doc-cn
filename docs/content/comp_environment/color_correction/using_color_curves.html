使用 ColorLookup 曲线

如果您更喜欢使用颜色曲线，您可以使用 ColorLookup 节点进行对比度、伽马、增益和偏移调整 (事实上还有许多其他调整)。 颜色曲线 参考给定颜色通道亮度的折线图。水平轴表示通道的原始值或输入值，垂直轴表示通道的新值或输出值。

如图所示, 您可以编辑 ColorLookup 节点的颜色曲线，以通过 ColorCorrect 节点进行所有类型的修正，而且您通常可以以比可能的更灵活和更精确的方式进行这些修正带滑块。

使用 ColorLookup 节点进行基本更正

1。 单击 颜色 > ColorLookup 在脚本中的适当位置插入 ColorLookup 节点。
2. 将查看器连接到 ColorLookup 节点的输出，以便您可以看到更改的效果。
3. 在 ColorLookup 属性面板中，单击 红色 , 绿色 , 蓝色 ,或 阿尔法 如果要将后续操作限制到特定通道。

可以选择多条曲线，以便参照另一条曲线编辑一条曲线。否则，选择 大师 曲线 (代表所有通道)。

4. 为了加快颜色计算，ColorLookup 节点使用介于 0 和在 范围 字段。您可以调整 范围 值，或取消选中 使用预计算表 如果需要，框可告知 ColorLookup 不使用预计算表。
5. 在查看器中，将光标拖动到要为校正采样的像素上。在 ColorLookup 属性面板中，按 Ctrl + Alt ( Cmd + Alt 在 Mac 上)，同时点击曲线，在红色、绿色和蓝色线与颜色曲线相交的地方设置点。

从采样颜色查看值。
6. 编辑点的位置并调整切线控制柄以调整颜色校正的曲线形状。

作为步骤 4 和 5 的替代方案，您可以使用 源 控件来拾取用于添加点的源颜色。然后，使用 目标 选择目标颜色。最后，执行以下操作之一:

• 单击 设置 RGB 要在红色、绿色和蓝色曲线上添加点，请映射 源 到 目标 .

• 单击 设置 RGBA 要在红色、绿色、蓝色和 alpha 曲线上添加点，请映射 源 到 目标 .

• 单击 设置一个 要在 alpha 曲线上添加点，请映射 源 到 目标 .

例如，可以使用这些控件来匹配两个板上的阴影、中间色调和高光。设置 源 在一个中阴影 rgb, 目标 要在另一个中阴影 rgb，然后按 设置 RGB 。中间色调和高亮区域也是如此。
