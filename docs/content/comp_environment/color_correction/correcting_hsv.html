纠正 HSV

Nuke 的 HSVTool 节点允许您从单个属性面板同时调整色调、饱和度和值分量。它还具有一个颜色替换工具。这个节点的主要优势是它在将修正限制在狭窄的颜色区域时提供的精度。

例如，假设你想通过多样化屋顶色调来给滨水场景增加一点冲击力。为此，您可以通过采样几个像素来限制对屋顶赭色色调的校正，然后移动它们的值。因为您限制了颜色范围，所以周围的图像通常不会受到移位的影响。

使用 HSVTool 节点进行 HSV 修正

1。 单击 颜色 > HSVTool 在脚本中的适当位置插入 HSVTool 节点。
2. 将查看器连接到 HSVTool 节点的输出，以便您可以看到更改的效果。
3. 适当限制您希望后续更正影响的颜色范围:

• 在 HSVTool 属性面板中，单击 Srccolor 色样。 Ctrl / Cmd 单击查看器以对显示的单一颜色进行采样，或 Ctrl / Cmd 转变 拖动可对一系列颜色进行采样。要在查看节点输出时从节点的输入中采样单一颜色, Ctrl / Cmd Alt 单击查看器。要从输入中采样区域, Ctrl / Cmd Alt 转变 在查看器上拖动。

提示: 您可以通过 Ctrl / Cmd 在查看器中单击鼠标右键。

• 的 范围 滑块打开 色调 , 饱和 ,和 亮度 钳位到采样范围。

• 对于任何颜色组件，拖动 范围 滑块根据需要扩大颜色范围。

• 对于任何颜色组件，拖动 范围 Rolloff 滑块微调颜色范围。这样做，调整允许超过范围滑块定义的限制的衰减量。

4. 进行必要的 HSV 修正:

• 对于色调校正，请拖动 旋转 滑块可输入 0 到 360 之间的色轮值。

• 要进行饱和修正，请拖动 饱和调整 滑块之间输入值 -1 (完全去饱和到一些灰色) 和 1 (完全饱和)。

• 对于值更正，请拖动 亮度调整 滑块之间输入值 -1 (黑色) 和 1 (白色)。

您还可以使用 srccolor 和 dstcolor 参数进行颜色替换: 首先对您希望用 srccolor 色块替换的颜色进行采样, 然后采样您希望使用的颜色作为 dstcolor 色块的替代品。Dstcolor 中的颜色在整个图像中替换 srccolor 中的颜色。

此外，请记住，HSVTool 节点是一个优秀的键盘手。您可以使用其色调、饱和度和亮度范围滑块来精确选择一系列颜色, 然后使用对话框底部的通道输出下拉列表将此选择输出为哑光通道。此下拉菜单允许您指定将哪些颜色组件 (色调、饱和度、值等) 添加到遮罩中。
