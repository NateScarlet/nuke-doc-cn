使用重新格式化节点

您可以将 “重新格式化” 节点用于三种不同的目的:

1。 根据分辨率和像素纵横比 (格式中单个像素的宽高比) 生成与所需图像格式匹配的图像序列。
2. 创建缩略图 (低分辨率帧，您可以将其发布到网络，以便在故事板上创建序列)。节点缩放框架，直到它适合您指定尺寸的矩形。它还将像素纵横比设置为 1 (平方)。
3. 缩放图像。比例因子稍微舍入，以便输出图像在重新格式化节点的控件中选择的方向上具有整数像素。

将图像转换为所需的图像格式

当你在元素中阅读时, Nuke 存储其格式设置并使其可用 到 “重新格式化” 节点。然后，您可以将现有格式之一应用于图像，或者自己创建、编辑和删除格式。

从头开始创建新格式时，可以定义整体分辨率、裁剪分辨率 (可选) 和像素纵横比。当您定义这些参数时，“重新格式化” 运算符会以下图所示的方式为您显示这些参数。

要创建新的输出格式:

1。 单击 变换 > 重新格式化 在脚本中的适当位置插入重新格式化节点 (通常在写入节点之前)。
2. 将查看器连接到 “重新格式化” 节点的输出，以便您可以看到更改的效果。
3. 选择 新 从 输出格式 下拉菜单。的 新格式 出现对话框。
4. 在中输入新格式的名称 名称 字段。
5. 在 文件大小 字段中，键入完整输出分辨率 (以像素为单位)。
6. 如果要裁剪完整的输出分辨率 (例如，创建一个字母框):

• 检查 图像区域 .

• 增加 x 用于定义裁剪左边界的字段。(显示将更新，以显示相对于全尺寸输入的裁剪左边界。)

• 增加 y 用于定义裁剪底部边界的字段。

• 增加 r 字段来定义裁剪的右边界。

• 增加 t 用于定义裁剪顶部边界的字段。

7. 如果图像序列的目标显示设备使用非正方形像素，请在 像素方面 字段 (例如，如果您的目的地是数字视频显示器，请键入 1.1 )。

注意: 您也可以将格式添加到 Nuke 通过条目到 Menu.py 文件:
1.打开 Menu.py 文件 (位于与您的 Nuke 可执行文件)。
2。添加一个类似于以下示例的条目:
Nuke.addFormat (“720 486 0 0 720 486 0.9 NTSC_video”)
其中数字分别指定格式的完全水平分辨率、完全垂直分辨率、左裁剪位置、底部裁剪位置、右侧裁剪位置、顶部裁剪位置和像素纵横比; 最后的文本字符串指定格式的名称。
3.保存并关闭 Menu.py 文件。下次你发射的时候 Nuke 格式可用于从 项目设置 对话框、重新格式化节点属性面板和其他位置。

要编辑格式:

1。 从中选择要编辑的格式 输出格式 下拉菜单。
2. 从同一下拉菜单中，选择 编辑 。的 编辑格式 出现对话框。
3. 编辑 名称 , 文件大小 , 图像区域 ,和 像素方面 必要的字段。
4. 单击 好 保存对格式的更改。

要删除格式:

1。 选择要从中删除的格式 输出格式 下拉菜单。
2. 从同一下拉菜单中，选择 删除 。格式将从菜单中删除。

要应用格式:

1。 如有必要，单击 变换 > 重新格式化 在脚本中的适当位置插入重新格式化节点 (通常在写入节点之前)。
2. Connect a Viewer to the output of the Reformat node so you can see the effect of your changes.
3. 从 类型 下拉菜单，选择 格式化 .
4. 从中选择要应用的格式 输出格式 下拉菜单。
5. 从 调整大小类型 字段中，选择要保留或替代原始纵横比的方法。选择:

• 宽度 缩放原始文件，直到其宽度与格式的宽度相匹配。然后以保持原始纵横比的方式缩放高度。

• 高度 缩放原始图像，直到其高度与格式的高度相匹配。然后以保持原始纵横比的方式缩放宽度。

• 适合 缩放原始图像，直到其最小边与格式的最小边相匹配。原始的较长的边然后以这样一种方式缩放，以保持原始的纵横比。

• 填充 缩放原始图像，直到其最长边与格式的最长边相匹配。然后以保持原始纵横比的方式对输入的较短侧进行缩放。

• 扭曲 缩放原始图像，直到其所有边与格式指定的长度匹配。此选项不保留原始纵横比，因此可能会发生扭曲。

6. 裁剪输出时，检查 中心 将裁剪区域定位在框架的中心。
7. 从中选择适当的过滤算法 过滤器 下拉菜单 (请参见 选择过滤算法 )。
8。 当使用 Key 、 Simon 和 Rifmen 滤镜缩放图像时，您可能会看到这些滤镜使用的像素锐化导致的光晕效果。如有必要，请检查 夹子 来纠正这个问题。

创建缩略图

1。 单击 变换 > 重新格式化 在脚本中的适当位置插入重新格式化节点 (通常在写入节点之前)。
2. Connect a Viewer to the output of the Reformat node so you can see the effect of your changes.
3. 从类型下拉菜单中，选择 到框 .
4. 在 宽度 和 高度 字段中，键入输出尺寸。单位是像素。
5. 使用 调整大小类型 下拉菜单选择保留或覆盖原始像素纵横比的方法。选择:

• 宽度 缩放原始值，直到其宽度与 宽度 字段。然后以保持原始纵横比的方式缩放高度 (这意味着您在中指定的输出 高度 可能与结果不匹配)。

• 高度 缩放原始值，直到其高度与 高度 字段。然后以保持原始纵横比的方式缩放宽度 (这意味着您在中指定的输出 宽度 可能与结果不匹配)。

• 适合 缩放原始值，直到其最小边与中的相应值相匹配 宽度/高度 。然后以保持原始纵横比的方式缩放较长的边。

• 填充 缩放原始值，直到其最长的边与中的相应值相匹配 宽度/高度 。然后以保持原始纵横比的方式缩放最小边。

• 扭曲 缩放原始值，直到其边与 宽度/高度 字段。此选项不保留原始纵横比，因此可能会发生扭曲。

6. 从中选择适当的过滤算法 过滤器 下拉菜单 (请参见 选择过滤算法 )。
7. 当使用 Key 、 Simon 和 Rifmen 滤镜缩放图像时，您可能会看到这些滤镜使用的像素锐化导致的光晕效果。如有必要，请检查 夹子 来纠正这个问题。

缩放图像序列

1. Click Transform > Reformat to insert a Reformat node at an appropriate place in your script (generally before a Write node).
2. Connect a Viewer to the output of the Reformat node so you can see the effect of your changes.
3. 从类型下拉菜单中，选择 规模 .
4. 在 规模 字段中，输入宽度和高度的比例因子。要使用不同的比例因子分别缩放每个方向，请单击 2 按钮。
5. Use the resize type dropdown menu to select the method by which you preserve or override the original pixel aspect ratio. Select:

• 宽度 缩放原始图像，使其填充输出宽度。然后以保持原始纵横比的方式缩放高度。

• 高度 缩放原始值，使其填充输出高度。然后以保持原始纵横比的方式缩放宽度。

• 适合 缩放原始图像，使其最小的边填充输出宽度或高度。然后以保持原始纵横比的方式缩放最长的边。

• 填充 缩放原始图像，使其最长的边填充输出宽度或高度。然后以保持原始纵横比的方式缩放最小边。

• 扭曲 缩放原始尺寸，以便两侧填充输出尺寸。此选项不保留原始纵横比，因此可能会发生扭曲。

6. Select the appropriate filtering algorithm from the filter dropdown menu (see Choosing a Filtering Algorithm ).
7. When scaling an image with Key, Simon, and Rifmen filters, you may see a haloing effect which is caused by pixel sharpening these filters employ. If necessary, check clamp to correct this problem.
