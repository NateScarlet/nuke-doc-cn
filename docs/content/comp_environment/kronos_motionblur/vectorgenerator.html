矢量发生器

VectorGenerator 允许您生成包含 运动矢量场。序列中图像的向量场与图像具有相同的尺寸，但每像素包含 (x，y) 偏移。这些偏移显示了如何将相邻图像扭曲到当前图像上。显然，由于序列中的大多数图像都有两个邻居，因此每个邻居都可以有两个向量场:

1。 的 向后 向量场: 每像素的 x 和 y 偏移，当应用于序列中的上一帧时，允许您重建当前帧的近似值。
2. 的 向前 向量场: 将下一帧转换为当前帧的近似值所需的 x 和 y 偏移。

VectorGenerator 的输出存储在 矢量通道。下面的图像显示剪辑的不同矢量图像。

查看 “向前” 或 “向后” 运动矢量时，查看器将 x 值表示为红色量，y 值表示为绿色量。运动向量可以是正的或负的，其中零代表没有运动。

一般来说，一旦你生成了一系列运动矢量场，很好地描述了特定片段中的运动，它们就适合用于任何可以接受矢量输入的节点。这些包括 Kronos 和 MotionBlur。如果你打算在你的项目中使用不止一个这样的效果，那么用 VectorGenerator 预先生成向量场可能是值得的，这样它们就可以被重复使用。
