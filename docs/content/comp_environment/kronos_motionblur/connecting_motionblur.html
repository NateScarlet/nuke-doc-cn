连接 MotionBlur

要连接 MotionBlur:

1。 选择 过滤器 > MotionBlur 在要添加运动模糊的片段后插入运动模糊节点。
2. 如果您的序列由在背景上移动的前景对象组成，则运动估计可能会在两者之间的边缘混淆。要解决此问题，请将前景区域的遮罩添加到 哑光 输入。然后，使用 哑光通道 在 MotionBlur 控件中选择要使用的遮罩组件。

这将强制前景的运动与背景的运动分开计算，因此在运动模糊中产生的伪影会更少。

3. 如果输入序列中的运动之前已经估计过 (例如，使用 矢量发生器 或第三方软件)，您可以向背景向量提供运动向量 ( BgVecs ) 和前景向量 ( FgVecs ) 输入以节省处理时间。

如果背景和前景有单独的向量，您应该将它们连接到适当的输入，并将用于生成它们的遮罩提供给 哑光 输入。如果你有一组向量，你应该把它连接到 FgVecs 输入。

注意: 的 BgVecs 只有连接了所有其他输入后，才会显示输入。

4. 将查看器附加到 MotionBlur 节点的输出。
5. 继续到 调整 MotionBlur 控件 下面。
