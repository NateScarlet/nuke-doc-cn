本地缓存媒体

Nuke 工作室 可以单独或通过设置自动本地化文件夹在本地缓存文件，以帮助保证播放稳定性。本地缓存最初在 偏好 对话框，然后在一个剪辑的基础上。

注意: 出口期间, Nuke 工作室 引用原始源文件，而不是本地化的播放副本。这主要是为了避免 Nuke 试图定位文件时媒体的本地化状态出现问题。
作为一种变通方法，您可以将 Nuke 指向与 Nuke 工作室 使用 Nuke 偏好 > 本地化到 字段。

默认情况下，文件被缓存到与缩略图数据相同的位置，尽管路径因平台而异:

• Mac 和 Linux - /Var/tmp/hiero

• 窗户 -系统临时目录，通常 C: \ temp

要设置初始缓存首选项:

1。 打开 偏好 按对话框 转变 + , (逗号)。
2. 输入的文件路径 自动 从本地化 ,如果需要。

任何驻留在该目录中的文件在符合或摄入时都会自动缓存 Nuke 工作室 .

3. 输入的文件路径 本地化到 ,如果需要。

将此字段留空将在 临时目录 作为本地缓存。

4. 输入的值 最大本地化文件 ,以兆字节为单位，用于控制缓存目录中有多少可用磁盘空间。

注意: 此字段中的负值始终保留指定的空间量。例如，-2000 停止用于缓存的 2gb 内存。

要缓存单个剪辑并控制本地化:

1。 在 bin 视图中选择剪辑。
2. 右键单击并选择 本地化政策 要显示可用选项:

• 总是 -选择被缓存，无论其位置如何，只要 最大本地化文件 限制没有违反。

• 从不 -无论选择的位置如何，都不会缓存该选择。

• 自动 -如果所选内容位于 自动本地化 目录，只要 最大本地化文件 限制没有违反。

剪辑缓存时，缩略图中将显示一个进度条。完全缓存的剪辑在缩略图顶部标有橙色条。

3. 导航到 缓存 > 本地化 > 编辑设置 要显示 本地化选项 .

注意: 这个对话框还显示，当你到达 最大本地化文件 限制。

4. 本地化缓存默认为 10000 MB，但您可以使用 商店大小 字段，如果需要。

NOTE: Negative values in this field reserve the specified amount of space at all times. For example, -2000 stops 2 GB of memory being used for caching.

5. 您可以通过单击启用或禁用本地缓存 启用本地化 .

你也可以将它本地缓存开启或关闭的 缓存 > 本地化 菜单。

6. 单击 清除所有未使用的文件 清除当前会话中未使用的所有剪辑的本地化缓存。

您还可以使用 缓存 > 本地化 菜单。

7. 检查 当 localiser 空间不足时，不要通知我 在本地化缓存空间不足时停止显示此对话框。
8。 单击 应用 确认更改而不关闭对话框或单击 好 关闭对话框并接受更改。

Nuke 工作室 还具有播放缓存功能，允许帧缓存在 RAM 内存中，而不是本地磁盘上。请参阅 缓存播放缓存中的帧 欲了解更多信息。
