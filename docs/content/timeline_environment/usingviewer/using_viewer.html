使用 时间线 观众

Nuke 工作室 的 时间轴环境 支持两种不同的查看器类型: 剪辑和序列。这一章描述了两者之间的区别以及如何使用它们。

剪辑查看器，有时称为源查看器，用 图标和专门处理源剪辑。您可以设置输入和输出点，并将标记应用于查看器，但源剪辑不受影响。

序列查看器，也称为记录查看器，用 图标并处理时间轴上的序列和剪辑实例。您也可以在这里设置和应用标记，但也可以通过修剪、重新计时等方式在时间轴上编辑剪辑实例。请参阅 时间线编辑工具 欲了解更多信息。

的 编辑 默认情况下，workspace 将剪辑和序列查看器结合在一起，使您能够使用插入和覆盖功能将源剪辑添加到时间轴。请参阅 插入、覆盖和三点编辑 欲了解更多信息。

查看您的媒体的观众，只需拖放一个剪辑或排序 项目 选项卡上的查看器输入，或双击该项目将其发送到相应的查看器。

注意: 查看器当前将所有 alpha 通道视为预乘，这可能会导致查看器背景被 “添加” 到图像中。如果您正在处理未预乘的图像，请将查看器背景设置为 黑色 .

请参阅 附录 A: 偏好 欲了解更多信息。
