重新格式化剪辑实例

默认情况下，时间轴上的剪辑实例保留源剪辑的分辨率。您可以使用时间轴强制剪辑实例为序列格式 属性 标签:

1。 在时间轴上选择目标剪辑实例。
2. 单击 属性 选项卡显示剪辑实例选项。

注意: 的 音量 控件仅在选定音频剪辑实例时可用。

3. 单击 剪辑重新格式化 下拉菜单并选择 到序列分辨率 .
4. 剪辑实例被重新格式化为 输出分辨率 指定在 序列 标签。
5. 根据源剪辑，您可能需要更改 调整大小类型 :

• 宽度 -缩放原始文件，直到其宽度与格式的宽度相匹配。然后缩放高度以保留原始纵横比。

• 高度 -缩放原始文件，直到其高度与格式的高度相匹配。然后缩放宽度以保留原始纵横比。

• 适合 -缩放原始图像，直到其最小边与格式的最小边相匹配。然后缩放原始的较长的边以保留原始的纵横比。

• 填充 -缩放原始图像，直到其最长边与格式的最长边相匹配。然后对输入的较短侧进行缩放，以保持原始纵横比。

• 扭曲 -缩放原始图像，直到其所有边与格式指定的长度匹配。此选项不保留原始纵横比，因此可能会发生扭曲。

提示: 当 剪辑重新格式化 是 到序列分辨率 ,禁用 中心 把夹在左下方 输出分辨率 ,而不是中心。
