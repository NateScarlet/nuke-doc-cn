添加过渡

Nuke 工作室 支持基本的视频和音频淡入淡出转换，以及在同一轨道上的剪辑实例之间进行溶解。过渡有三种口味:

• 淡入 -在单个剪辑实例上从黑色中淡入。

• 淡出 -在单个剪辑实例上淡出为黑色。

• 溶解 -通过合并帧，从一个剪辑实例淡出并进入下一个剪辑实例。

提示: 一次转型是在地方，它可以以同样的方式作为一个编辑的推动使用 , (逗号) 和 . (句点) 键盘快捷键，提供所需的句柄。

要添加淡入淡出过渡:

1。 在目标剪辑实例上单击鼠标右键，然后选择 编辑 > 添加过渡 > 淡入 , 淡出 , 音频淡入 或 音频淡出 添加淡入淡出图标。
2. 通过使用拖动淡入淡出图标来调整淡入淡出 多工具 或 移动/修剪 工具。

要添加溶解过渡:

注意: 只有当剪辑实例在同一轨道上并且在过渡的两侧都有足够的可用句柄时，才能在它们之间添加溶解。

1。 选择 多工具 或 滚动编辑 工具并将鼠标指针悬停在两个剪辑实例之间的编辑上。

提示: 单击并按住编辑点将可用控制柄显示为红色覆盖。

2. 右键单击并选择 编辑 > 添加过渡 > 溶解 或 音频交叉渐变 ,或使用 Ctrl / Cmd + T 键盘快捷键，将溶解图标添加到编辑中。
3. 通过拖动图标来调整溶解的两侧，类似于使用 多工具 或 移动/修剪 工具。
