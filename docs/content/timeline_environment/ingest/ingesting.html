摄取媒体

添加媒体就像从文件浏览器中拖放或选择 文件 > 导入文件 (s) 或 进口夹 (s) 。应用程序将媒体导入 bin 视图，为您提供所有剪辑的缩略图，并保留原始文件夹和文件层次结构。

媒体是软导入的，创建到磁盘上位置的符号链接。请参阅 使用复制导出器 有关如何快速整合媒体和项目的信息，或 本地缓存媒体 帮助稳定播放。

注意: 包含大量电影文件的项目 (例如 .R3d 和 。Mov ) 可能会超过每个进程可用文件句柄的数量，导致打开新文件或项目并导出时出现问题。

通过从终端输入以下命令，然后从同一会话运行应用程序，可以增加默认限制 1024:
Ulimit-Sn 2048

没有固有帧速率信息的剪辑在摄取时被分配一个帧速率，如 偏好 .

1。 打开 偏好 按对话框 转变 + S .
2. 选择 行为 > 时间码 从左侧的子菜单中。

3. 使用 红色文件时间码 下拉菜单确定 R3D 剪辑行为:

• 默认自文件 -使用有问题的 R3D 文件设置的默认设置。

• 绝对时间码 -强制使用剪辑元数据中指定的绝对时间码。

• 边缘时间码 -强制使用剪辑元数据中指定的边缘时间码。

4. 使用 其他媒体时间码 下拉菜单确定所有其他剪辑的剪辑行为:

• 文件头 -文件元数据头用于派生时间码 (如果存在)。此选项默认为 帧号 如果缺少标题。

• 帧号 -忽略元数据头，即使存在，并从剪辑中的帧中导出时间码。

5. 设置 最大有效时基 允许从图像标题，上面的值被钳制。

图像文件通常在标题描述中用特定于应用程序的时基值创建。这可能会导致伪高帧率的读取，夹具旨在防止这种情况发生。

如果您的剪辑确实具有极高的帧速率，请根据需要增加此值以避免夹持。

6. 启用或禁用 EDL 风格的电子表格时间码 :

• 禁用时, SrcOut 和 DstOut 值使用电影惯例，代表 最后 切割的框架。

• 启用时, SrcOut 和 DstOut 值使用视频约定，直接表示帧 之后 削减。

7. 单击 好 保存您的设置。
