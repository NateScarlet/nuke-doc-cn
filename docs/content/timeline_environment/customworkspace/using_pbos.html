使用像素缓冲区对象

Nuke 工作室 还包括使用像素缓冲区对象 (PBOs) 将纹理上传到 GPU 的选项，这可以减少从 RAM 缓存上传到 Windows 和 Linux 显卡的时间, 根据您的硬件和驱动程序组合。默认情况下禁用此选项，但您可以在 偏好 > 性能 通过检查子菜单 启用 PBO 纹理上传 .

注意: 必须重新启动 Hiero 才能使此选项生效。

根据您的硬件和驱动程序组合，启用 PBOs 的好处可以通过启用或禁用进一步增加 将图像从每个像素 3 到 4 个通道展开 ,也在 偏好 > 性能 子菜单。

注意: 结果可能会因平台、显卡、驱动程序和源图像格式而异。
