Audio Preferences

These preferences control how audio behaves in Nuke Studio and (Undefined variable: Custom.ProductPlayer). See Audio and the Timeline for more information.

Preference

Default Value

Description

Disable audio

 disabled

When enabled, audio output is muted throughout the application.

NOTE:  You must restart Nuke Studio or (Undefined variable: Custom.ProductPlayer) for this option to take effect.

Audio device

 Speakers

Sets the audio output device or driver.

Default latency
adjustment (ms)

 0 ms

Sets the default audio latency for all new Viewers in milliseconds. Audio latency allows you to correct audio and video sychronization by changing the point at which audio playback starts.

Positive values cause the audio track to start earlier in relation to the video track, and vice versa.

Default volume

 50%

Sets the default audio volume for all new Viewers.
