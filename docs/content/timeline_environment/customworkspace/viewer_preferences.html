Viewer Preferences

These preferences determine the default settings for the Viewer at startup as well as all new Viewers that you add to the workspace - you can still adjust the settings of individual Viewers.

Preference

Default Value

Description

Playback Mode

 Play All Frames

Sets the default playback mode for new Viewers. See Timeline Viewer Tools for more information.

OpenColorIO config file

 none

Sets the OpenColorIO configuration to use if you don’t intend to use the default settings.

Enter the filepath of the configuration file or click Choose to use the browser.

NOTE:  Nuke Studio also includes an environment variable method for setting a config file. See Environment Variables for more information.

Guides

 disabled

Selects the default Guides to apply to Viewers. Choose from Title Safe, Action Safe, and Format. See Timeline Viewer Tools for more information.

Fullscreen Display

 Primary Monitor (1)

Controls where the Viewer expands to when in Fullscreen mode.

See through missing media

 enabled

When enabled, any offline media on a timeline is treated as transparent, allowing the Viewer to display the track underneath.

Crop to format

 disabled

When enabled, clips or track items that extend outside the format guide are cropped.

Background

 Black

Sets the Viewer background color, including sliders to control brightness. Choose from Black, Gray, or Checkerboard.

NOTE:  Nuke Studio and (Undefined variable: Custom.ProductPlayer) currently treat all alpha channels as premultiplied, which can result in the Viewer background being “added” to the image. If you’re working with un-premultiplied images, set the Viewer background to Black.

Frame Increment

 12

Sets the frame increment amount for Skip and Nudge More operations. This control only affects new Viewers.

Filtering mode

 Auto

This allows you to specify the Viewer behavior when filtering your sequences for display.

• Nearest neighbor and Linear - standard OpenGL filtering options.

• Auto - mimics the previous Viewer filtering, which was based on Nuke’s behavior.
