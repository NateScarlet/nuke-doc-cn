使用帮助线程

Nuke 工作室 和 (未定义变量: Custom.ProductPlayer) 还包括两个专用于 OpenEXR 和 ARRI 文件的额外帮助线程设置。这些帮助线程帮助读者线程提高性能 。Exr 和 。阿里 文件。

您可以使用向上和向下箭头更改可用助手的数量，或者只需在提供的字段中输入所需助手的数量。

通常，默认设置 0 应该自动正确分配帮助线程。然而，如果你的项目是 。Exr / 。阿里 你可能会考虑增加帮助线程的数量。

注意: 必须重新启动应用程序才能将更改应用于 ARRI helper 设置。
