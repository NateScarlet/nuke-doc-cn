使用脚本编辑器

核武器工作室 和 (未定义变量: Custom.ProductPlayer) 包括一个 Python 脚本编辑器 ,可从用户界面访问，允许您直接输入 Python 语句。

要访问脚本编辑器:

1。 导航到 窗口 > 脚本 编辑 .

的 脚本编辑器 显示。

2. 在编辑器的下半部分输入脚本，然后按 Ctrl/Cmd + 输入 执行脚本。

结果显示在编辑器的顶部。一个简单的例子可能是:

从 hiero.core import *

Bin = 项目 () [-1]。clipsBin ()

Bin.addItem (Bin (“盘子”)

Bin [“盘子”]。importFolder (“/素材/Hiero/finalshots”)

哪些进口 Hiero.core ,定义 Bin 在上次打开的项目中，创建一个名为 盘子 在根级别，并将指定的文件夹导入 盘子 .
