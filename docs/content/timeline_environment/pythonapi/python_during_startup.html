启动期间运行 Python

在启动时, 核武器工作室 扫描各种目录，寻找自定义应用程序行为的 Python 脚本。这些脚本可以从在线资源下载并放置在磁盘上的特定位置，具体取决于您使用的平台。

如果您熟悉 Python 脚本，您可以将自定义启动脚本放在相同的位置，但请记住，您应该小心，因为错误会产生性能问题。

以下目录中的应用程序包中包含一些示例脚本:

在 Mac 上:

• /应用程序/ 核武器工作室 9.0v1 / 核武器工作室 9.0v1 。应用程序/内容/插件/网站-包/核武器/示例/

在 Linux 上:

• /Usr/本地/ 核武器工作室 9.0v1 /Plugins/site-packages/nuke/examples/

在 Windows 上:

• 驱动器号: \ 程序文件 \ 铸造 \ 核武器工作室 9.0v1 \ Plugins \ site-packages \ nuke \ 示例

在启动时, 核武器工作室 搜索 <Path>/Python/Startup 和 <Path>/Python/StartupUI 对于任何 Python 。Py 模块或包件 _ _ Init _ _.py 。扫描分两次完成 -- 搜索所有启动文件夹，然后搜索所有启动文件夹。

您还可以通过创建指向自定义位置的 Python 文件来添加自己的启动文件夹，这些文件夹在标准路径之后进行扫描。例如:

导入 hiero.core

Hiero.core.addPluginPath (“/mycustoplugins”)

搜索 /Mycustoplugins/Python/Startup 在标准路径之后。

发现的每个包或模块都被导入并添加到内置包中 Hiero.plugins .

注意: Python 启动脚本按字母顺序导入，因此如果需要，您可以强制排序。
