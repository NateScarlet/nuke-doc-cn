使用标签

标签由使用 核武器工作室 快速排序或过滤剪辑和剪辑实例，以获得更好的可见性、组织和导出。

导言

标记用于在组织项目时标记特定类型或内容的快照。提供的默认标记包括 批准 , 注意 , 参考 ,和其他通用标签。你也可以用鼠标右键单击创建自定义标签的 标签 标签或按 Ctrl / Cmd Y 。可以将标记应用于剪辑、剪辑实例和 Comp 剪辑实例、单个帧、序列和轨迹。

可以使用刻录功能将剪辑和剪辑实例标记和注释添加到导出中。请参阅 将刻录文本添加到导出中 欲了解更多信息。

在导出到 XML 的过程中，标记也可以转换为 Final Cut Pro 或 Premiere 标记。请参阅 导出 EDLs 和 xml 欲了解更多信息。
