Using Tags

Tags are used by Nuke Studio to quickly sort or filter clips and clip instances for better visibility, organization, and export.

Introduction

Tags are used to mark shots of a particular type or content as you organize your project. The default tags supplied include Approved, Note, Reference, and other general purpose tags. You can also create custom tags by right-clicking in the Tags tab or by pressing Ctrl/Cmd+Y. You can apply tags to clips, clip instances and Comp clip instances, individual frames, sequences, and tracks.

Clip and clip instance tags and notes can be added to exports using the burn-in feature. See Adding Burn-in Text to Exports for more information.

Tags can also be converted to Final Cut Pro or Premiere markers during export to XML. See Exporting EDLs and XMLs for more information.
