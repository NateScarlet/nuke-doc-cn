轨迹、范围和手柄设置

中的轨迹、标记和句柄控件 出口 对话框允许您选择要导出的帧范围或剪辑实例。

1。 设置 版本 出口编号 (如果适用)。使用箭头增加版本号，使用 +/-按钮增加或减少填充。也可以直接在数字字段中键入。

注意: 请参阅 使用版本 有关版本控制如何在中工作的更多信息 核武器工作室 .

2. 选择 此出口的轨迹 通过启用或禁用列表中的轨迹。 核武器工作室 出口 所有轨道 默认情况下。
3. 使用启用或禁用标记 按标签过滤 面板。单击复选框以循环浏览可用的标记状态。
4. 如果要导出序列，请设置 范围 根据需要控制:

• 选择 整个序列 或 输入/输出点 仅导出选定的帧。

• 设置如何剪辑 开始帧 使用下拉菜单导出:

• 序列 -使用序列的开始帧。

• 自定义 -使用右侧的字段为所有剪辑指定开始帧。

5. 如果您要导出快照，请设置 手柄 根据需要控制:

• 夹子长度 -导出可用的完整剪辑长度，就像剪辑作为查看器打开一样。

• 切割长度 -仅导出时间表上包含的削减。

注意: 选择 切割长度 允许您向每个剪辑添加手柄，直到最大可用源剪辑长度。

• 检查 应用 Retimes 导出时间轴上存在的任何 retimes。

注意: 当 应用 Retimes 已禁用，这是默认状态 创建 Comp 但是，任何时间扭曲软效果都不包括在生成的 Nuke 脚本中。通过创建新剪辑实例时 创建 Comp 或 从导出标记生成跟踪 ,时间扭曲软效果将从原始剪辑实例复制到新实例。

• Set how clip Start Frames are derived using the dropdown menu:

• 源 -使用源剪辑的开始帧。

• Custom - specify a start frame for all clips using the field to the right.

6. 设置如何 核武器工作室 要使你的出口, 渲染与 下拉列表。 核武器工作室 提供以下选项:

• 帧服务器 -使用多个核弹进程来加快渲染时间 并与时间线中的任何 Nuke Comp 渲染共享渲染队列 ,改善资源管理。

请参阅 在外部机器上使用帧服务器 欲了解更多信息。

• 单一渲染过程 -使用一个单一的核弹过程来呈现你的出口。渲染 QuickTimes 回到此设置，但当帧服务器检测到问题时也会使用它。

• 自定义渲染过程 -使用自定义渲染过程。 核武器工作室 需要一个 Python 脚本将导出传递到您选择的渲染场。脚本必须位于特定目录中，取决于平台，如中所示 加载 Gizmos 、 NDK 插件以及 Python 和 Tcl 脚本 。如果不存在脚本，则下拉列表仅包含默认渲染进程。

7. 单击 出口 .

的 导出队列 窗口显示每个导出组件预计需要多长时间的估计。

导出完成后，将创建快照模板中指定的文件结构。然后，当核弹剪辑准备就绪时，您可以在单独的轨道上导入它们。

提示: 单击放大镜图标可在浏览器窗口中显示文件结构。

当剪辑从 核武器工作室 此外，它们还标有核弹标签，标记哪些剪辑有出口历史。以这种方式标记的剪辑可用于快速构建 VFX 轨道，如中所述 构建 VFX 轨道和 Comp 剪辑 .
