About This Online Help

This Online Help works the same way as most standard help systems. To find what you're looking for, click on a topic in the Contents pane on the left or enter search terms in the Search field.

In some topics, you can:

• click to expand the topic to display hidden content. If you then click , the content becomes hidden once more.

• click on Related Topics to display a list of topics that are related to the current topic.

• watch video tutorials related to the current topic. To find all the video tutorials dotted throughout the Online Help, select Tutorials > Video Tutorials in the Contents pane on the left.

NOTE:  Watching video tutorials requires an internet connection.

• click on an example Nuke script link to load the example .nk script in Nuke. To find all the example scripts included in the Online Help, select Example Nuke Scripts in the Contents pane on the left.

NOTE:  Loading example scripts only works if you launched the help from Nuke and have set documentation source to local in the Behaviors > Documentation tab of the Preferences.

See Using Script Links for more information.

You can also use the following buttons at the top of each help topic:

Open the Print dialog, so you can send the current topic to the printer.

Toggle expanding and collapsing all elements, such as dropdown effects and expanding text effects, in the current topic.

Open the previous topic in the table of contents.

Open the next topic in the table of contents.

NOTE:  The help content may differ depending on what you've selected as the documentation source on the Behaviors > Documentation tab of the Preferences. For more information, see Getting Help.
