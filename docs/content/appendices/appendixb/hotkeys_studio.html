附录b: 键盘快捷键

键盘快捷键，或热键，提供快速访问的功能 Nuke 。下表显示了这些击键。

公约

以下约定适用于鼠标点击和按键的说明。

• LMB 表示单击或按下鼠标左键。

• MMB 表示单击或按下鼠标中键

• 人民币的意思是点击或按下鼠标右键。

• 当你看到鼠标按钮缩写后的 “拖动” (即, “MMB 拖动” )，这告诉您在拖动鼠标指针时按住鼠标按钮。

• 按键组合与 Ctrl , Alt ,和 转变 键告诉您按住键，然后键入指定的字母。

例如，“按 Ctrl S ”意味着按住 Ctrl 键，按 S ,然后释放两个键。

注意: 在 Mac OS X 上，替换 Ctrl 配钥匙 Cmd 关键。

注意: 表格中的击键显示为大写，但您不会将其键入为大写。如果 转变 修饰符不会出现在字母之前，只需单独按下字母键。

注意: 本节假设您使用的是默认的键盘和鼠标按钮指定。如果鼠标按钮不适合您，请尝试将鼠标控制类型重置为标准 Nuke 设置 ( 偏好 > 面板 > 查看器句柄 > 3D 控制类型 > Nuke )。
