光波说唱

LightWrap 可以通过 “包装” 或将光线从背景扩散到前景对象来帮助将对象合成到非常明亮的背景中。节点通过混合背景中的任何物体，在前景对象的边缘周围创建光的反射。

在使用合并节点在背景上合成前景之前，您应该在前景元素上应用 LightWrap。
