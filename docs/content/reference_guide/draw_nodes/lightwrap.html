LightWrap

LightWrap can help composite objects into a very bright background by ’wrapping’ or spilling light from the background onto the foreground object. The node creates a reflection of light around the edges of your foreground object by blending in whatever is in the background.

You should apply LightWrap on your foreground element before you composite the foreground over the background with the Merge node.

LightWrap Tab

Diffuse

size

15

Adjusts the spread of the reflections on the foreground element. This and Intensity need to be balanced out together. You may want to start by bringing Diffuse all the way down to better see what you are blending in from the background. Then, adjust Intensity before going back to the Diffuse slider and, if necessary, Intensity again until you are happy with the result.

Intensity

Intensity

0

Adjusts the brightness of the reflections on the foreground element. This and Diffuse need to be balanced out together. You may want to start by bringing Diffuse all the way down to better see what you are blending in from the background. Then, adjust Intensity before going back to the Diffuse slider and, if necessary, Intensity again until you are happy with the result.
