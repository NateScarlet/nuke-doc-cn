谷物

“纹理” 节点允许您将合成纹理 (而不是从实际胶片库派生的纹理) 添加到图像中。这有助于确保组合中的所有元素，包括数字生成的元素，看起来都像是在同一胶片上拍摄的。

的 预设 下拉菜单包括预定义的谷物类型，如 柯达 5248 和 柯达 5218 。这些是 2k 扫描的正确大小。

您还可以调整谷物节点的控件以匹配谷物样本:

1。 找到一个背景相当恒定的样本。
2. 模糊样品以去除颗粒。
3. 将模糊样本连接到谷物节点的输入。
4. 在查看器中，在纹理节点和原始样本图像之间擦除，并调整纹理节点的控件以匹配纹理。它有助于分别查看和匹配红色、绿色和蓝色通道。

另见 ScannedGrain 和 F_ReGrain .

启用时，纹理仅限于图像 alpha 通道的非黑色区域。当您希望将纹理应用于预乘的 CG 图像而不污染背景黑色时，这很有用。
