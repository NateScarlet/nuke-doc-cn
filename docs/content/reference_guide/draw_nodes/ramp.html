Premult

没有

用代表渐变的遮罩预乘选定的输入通道。例如，如果没有渐变 (哑光是黑色或空的)，输入通道将设置为黑色。当渐变不透明时 (哑光是白色或全白)，输入通道保持其全值。

格式

选择限制输出图像的方式:

• 没有剪辑 -使用整个图像。

• Bbox -将输出图像限制到传入的边界框。

• 格式 -将输出图像限制在传入格式区域。

• 联盟 Bbox 格式 -将输出图像限制为传入边界框和格式区域的组合。

• 相交 Bbox 格式 -将输出图像限制为边界框和传入格式区域的交集。

类型

类型

线性

设置要应用的渐变类型:

• 线性 -斜坡有一个线性梯度。

• 线性 -斜坡对肉眼来说似乎有更多的线性梯度。
