噪音

另见 谷物 , ScannedGrain ,和 F_ReGrain .

Premult

用表示噪声的遮罩预乘选定的输入通道。例如，没有噪音的地方 (哑光是黑色或空的)，输入通道被设置为黑色。当噪音是不透明的 (哑光是白色或全) 时，输入通道保持其全值。

颜色标签

坡道

坡道

没有

选择是否在噪声中创建颜色渐变:

• 没有 -不要创建颜色渐变。

• 线性 -渐变从一种颜色线性变化到另一种颜色。

• Smooth0 -渐变颜色逐渐缓和到点 0 端。这意味着点 0 端的颜色比点 1 端的颜色分布宽。

• Smooth1 -渐变颜色简化为点 1 末端。这意味着点 1 端的颜色比点 0 端的颜色分布宽。

• 平滑 -渐变颜色逐渐简化为两端。这意味着点 0 和点 1 末端的颜色分布比坡道中心的颜色宽。
