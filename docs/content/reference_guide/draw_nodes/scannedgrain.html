ScannedGrain

应用胶片颗粒的实际扫描来复制胶片的颗粒。从扫描的胶片颗粒循环中选择帧，并应用于输入图像。这允许你将谷物添加到 CG 创作中，以人工匹配原始板或 “年龄” 镜头。

另见 谷物 和 F_ReGrain .

全谷物

[文件 dir $ program_name]/FilmGrain/

设置扫描纹理图像的文件路径。

要创建胶片纹理图像:

1。 拍摄一张灰色卡片，扫描结果存档。
2. 模糊扫描图像。
3. 从原始图像中减去模糊图像。
4. 向 rgb 通道添加 0.5。
5. 将扫描的纹理图像保存到所需的文件类型 (例如.rgb)。
