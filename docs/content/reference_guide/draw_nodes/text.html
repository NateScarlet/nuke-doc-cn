文本

此节点允许您在图像上添加文本覆盖。您可以简单地键入要显示的文本，或者使用 Tcl 表达式或 Tcl 变量创建文本覆盖。文本覆盖也可以使用动画层在 组 标签，以便它们的属性 (如位置、大小和颜色) 随时间而变化。这些功能使文本节点非常有用，例如，用于创建石板或滚动信用。

输入要显示的文本、 Tcl 表达式、 Tcl 变量或它们的组合。在方括号中输入 Tcl 表达式，例如, [日期] 。要开始新的一行，请按 返回 .

要显示特殊的 Unicode 字符，如外语字符和版权标志，您可以:

• 使用 HTML 命名实体，如 & 复制; 要显示 ©

• 使用十六进制实体，如 & # Xa9; 要显示 ©

• 使用十进制实体，如 & #169; 要显示 ©

• 键入 Unicode 字符，如 © ,在键盘上，或者从其他应用程序中剪切和粘贴它们。UTF-8 字符编码用于将它们存储在控件的值和保存的 Nuke 脚本中。

只有当您使用的字体具有要在其中显示的字符时，上述操作才有效。

注意: 我们建议使用上述实体，而不是键入 < ,例如。这是因为文本节点的未来版本可能会解释 HTML 标记。在 HTML 中，某些字符 (如大于和小于符号) 被保留。如果你现在在你的文本中使用这些符号，未来的版本可能会把它们误认为是 HTML 标记。

请参阅 变量和实体示例 有关可在文本节点中使用的 Tcl 表达式、 Tcl 变量、 HMTL 命名实体、十六进制实体和十进制实体的列表。

提示: 要获得所有 Tcl 表达式的列表，您可以使用 日期 ,您也可以键入 X 在节点图上，将打开的脚本命令对话框设置为 Tcl ,输入 日期-h ,然后单击 好 .

Xjust

左

设置如何水平对齐文本:

• 左 -将文本沿屏幕文本框的左边缘对齐。这使得文本的右边缘变得粗糙。

• 中心 -从屏幕文本框的中心对齐文本。这使得文本的两个边缘变得粗糙。

• 右 -将文本沿屏幕文本框的右边缘对齐。这使得文本的左边缘变得粗糙。

• 证明 -沿屏幕文本框的左侧和右侧边缘对齐文本。这没有留下粗糙的边缘。理由是通过扩大字母之间的空格来完成的。如果没有空格，或者空格比它们宽三倍多，字母就会被扩大。

Yjust

顶部

设置如何垂直对齐文本:

• 顶部 -将文本与屏幕文本框的上边缘对齐。

• 中心 -从屏幕文本框的中心对齐文本。

• 底部 -将文本与屏幕文本框的底部边缘对齐。

坡道

坡道

没有

选择是否在文本中创建颜色渐变:

• 没有 -不要创建颜色渐变。

• 线性 -渐变从一种颜色线性变化到另一种颜色。

• Smooth0 -渐变颜色逐渐缓和到点 0 端。这意味着点 0 端的颜色比点 1 端的颜色分布宽。

• 平滑 -渐变颜色逐渐简化为两端。这意味着点 0 和点 1 末端的颜色分布比坡道中心的颜色宽。
