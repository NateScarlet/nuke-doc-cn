RotoPaint

RotoPaint 是一个基于矢量的节点，用于帮助完成旋转对位、钻机移除、垃圾铺垫和除尘等任务。您可以使用单个和层组属性绘制 Bezier 和 B-Spline 形状，包括逐点和全局羽毛、运动模糊、混合模式以及单个或分层 2D 变换。

RotoPaint 节点为您提供了比 Roto ,尽管许多控件在两个节点之间共享。与 Roto 节点一样，您应该使用查看器工具创建形状和绘制笔画，然后使用控制面板编辑它们。

查看器左侧的工具栏具有与 Roto 节点相同的点选择和操作以及形状创建工具。此外，RotoPaint 节点还具有画笔、克隆/显示、模糊和道奇/刻录工具。单击并按住或右键单击工具栏按钮以打开子菜单，选择其任何可用工具类型。与当前工具相关的选项显示在查看器顶部的工具栏中。单击工具栏项可循环浏览该类工具的可用选项。

作为一般工作流指南:

1。 从左侧查看器工具栏中选择您的工具。
2. 使用查看器上方的 RotoPaint 工具设置来调整工具的属性。
3. 绘画时，您可以:

• 转变 在查看器上拖动以更改画笔大小。

• 克隆工具: Ctrl / Cmd 单击并拖动以设置克隆偏移。要重置克隆偏移, Ctrl / Cmd 转变 单击并拖动。

4. 绘制形状时，您可以:

• 在查看器中单击以放置点。您可以在单击时拖动以拉出 Bezier 控制柄或调整 b样条张力。

• Ctrl / Cmd 拖动可自由绘制形状。

• 单击第一个点或按 返回 关闭形状。要使形状保持打开状态，请按 Esc .

• Ctrl / Cmd Alt 单击可向现有形状添加点。

• 选择一个点并按 Z 增加点的平滑度。

• 选择点并按 转变 Z 达到顶点。

• 选择点并按 删除 删除一个点。

• 贝齐尔人: 转变 在切线控制柄上拖动，将相反的控制柄捕捉到相同的长度。 Ctrl / Cmd 在切线控制柄上拖动，使其独立于相反的控制柄移动。

• B-spline: Ctrl / Cmd Alt 拖动一个点以调整其张力。

5. 使用选择形状或描边 选择 工具或控制面板中的形状/描边列表。
6. 使用控制面板来调整或微调形状或描边。

也可以通过按创建此节点 P 在节点图上。

另见 Roto , Project3D , F_RigRemoval ,和 F_WireRemoval .

设置当前形状 (A) 中的颜色如何与基础图像和形状 (B) 合并:

• 颜色烧伤 (使 B 向 A 变暗)-通过增加对比度使 B 变暗以反映 A。图像的任何部分都不会变亮。

• 颜色-道奇 (使 B 向 A 变亮)-通过减小对比度使 B 变亮以反映 A。图像的任何部分都没有变暗。

• 差异 (Abs (A-B))-从 A 中减去 B，反之亦然，这取决于哪个更亮。与白色混合会反转 A，而与黑色混合不会产生任何变化。类似的颜色将返回黑色像素。

• 排除 (B-2AB)-创建类似于 差异 模式，但相比之下较低。喜欢与 差异 ,与白色混合反转。与黑色混合不会产生任何变化。

• 从 (B-A)-从 B 中减去 A。

• 硬光 (如果 A <乘以。5，如果 A> 屏幕。5)-点亮高光并使阴影变暗。如果 B 比 50% 的灰色轻，结果就像被筛选一样变亮。如果 B 比 50% 的灰色更暗，结果就会变暗，就像它被乘以一样。

• 最大 (Max (A，B))-选择两种颜色中的浅色作为结果颜色。只有比 B 暗的区域被替换，而比 B 轻的区域不改变。

• Min (Min (A，B)-选择两种颜色中较暗的颜色作为结果颜色。任何比 B 轻的零件都被替换。图像中比 B 暗的任何部分都不会改变。

混合模式 (续)

• 减去 (A-B)-从 A 中减去 B。

• 乘 (AB，如果 A <0 和 B <0)-将 A 乘以 B。结果总是更暗。与黑色混合会产生黑色，与白色混合会使颜色保持不变。

• 结束 (A 超过 B)-两幅图像的颜色不会以任何方式相互作用，Nuke 将在 A 中显示颜色的全部价值

• 覆盖 (如果 B <。5，屏幕如果 B>。5)-根据 A，将颜色相乘或屏幕。B 照亮 A，同时保留高光和阴影。

• 加 (A B)-两种颜色的总和。增加亮度以照亮 A 和反射 B。Plus 类似于 屏幕 混合模式，但产生更极端的结果。

• 屏幕 (A B-AB，如果 A 和 B 在 0-1 之间，否则 A 如果 A> B，否则 B)-这是一个软 加 让一切变得更明亮，但却远离白人。浅色比深色有更多的效果。结果总是颜色变浅。与黑色混合会使像素保持不变，与白色混合总是返回白色。其结果类似于将多个幻灯片相互突出。

• 软光 -根据 A，使颜色变暗或变亮。不那么极端的 硬光 混合模式。
