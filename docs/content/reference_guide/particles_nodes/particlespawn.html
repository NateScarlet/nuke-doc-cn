粒子典当

如果你想让你现有的粒子发射更多的粒子，你应该转向粒子棋子。此节点从现有粒子发射粒子。

连接名称

输入

粒子

要作为粒子发射的图像或几何体-连接此输入时，将创建另一个粒子输入，允许您混合粒子类型。

合并

另一个粒子系统，包括另一个粒子发射器，用于生成粒子。

旋钮 (脚本)

ParticleSpawn 选项卡

显示

显示

不变

调整衍生粒子的显示特征。这些设置不会影响场景的渲染输出; 这些设置仅用于在 3D 查看器中显示。

• 关闭 -隐藏粒子。

• 线框 -仅显示粒子的轮廓。

• 固体 -用纯色显示所有粒子。

• 实心线框 -将粒子显示为带粒子轮廓的纯色。

• 纹理 -仅显示表面纹理。

• 纹理线框 -显示线框和表面纹理。

• 不变 -不会更改粒子显示模式。查看器可以覆盖此设置。

可选

可选

已启用

启用后，您可以通过单击查看器中的衍生粒子来选择它们。

渲染

Render_mode

不变

设置粒子的渲染方式。此控件独立于 显示 选择，但具有相同的设置。

通道

通道

b

衍生的粒子仅存在于选定的通道中。

排放率

费率

10

设置每帧产生的粒子的确切数量，并受 速率通道 粒子发射器的控制。如果你 速率通道 全程小于 1，产卵率降低。

速率变化

Rate_variation

0

通过将高斯分布的随机数添加到 排放率 值乘以 速率变化 价值。

粗略地说，0 等于 排放率 1 是非常随机的变化。

寿命

寿命

10

设置每个衍生粒子存在的帧数。

寿命范围

Lifetime_variance

0

通过将高斯分布的随机数添加到 寿命 值乘以 寿命范围 价值。

粗略地说，0 等于 寿命 1 是非常随机的变化。

半衰期

半衰期

0

设置一个 半衰期 导致产生的粒子的一小部分在每一帧中随机死亡，这样平均来说，在给定时间存在的粒子的一半将在 半衰期 (以与放射性衰变相同的方式)。

例如，在两次 半衰期 值 (以帧为单位)，将只保留原始生成粒子的四分之一，依此类推。

速度

速度

1

设置粒子产卵时每帧的速度 (单位)。

速度范围

Velocity_variation

0

通过将高斯分布的随机数添加到 速度 值乘以 速度范围 价值。

粗略地说，0 等于 速度 1 是非常随机的变化。

旋转速度

Rotation_velocity

0

设置在 3D 空间中产生的粒子围绕其局部 y 轴旋转的速度。

Y 轴指向粒子最初产生的方向，但随后保持不变。您可以使用 ParticleMotionAlign 节点将粒子沿其运动方向对齐，或 ParticleLookAt 节点使所有粒子在特定方向对齐。

旋转速度范围

Rotation_velocity _ variation

0

产生随机旋转速度 通过将高斯分布的随机数添加到 旋转速度 值乘以 旋转 速度范围 价值。

粗略地说，0 等于 旋转 速度 1 是非常随机的变化。

尺寸

尺寸

0.1

设置每个衍生粒子的大小。如果 粒子 输入是几何图形，每个粒子上的几何图形实例受此值影响。

尺寸范围

Size_variation

0

产生颗粒大小的随机变化 通过将高斯分布的随机数添加到 尺寸 值乘以 尺寸范围 价值。

粗略地说，0 等于 尺寸 1 是非常随机的变化。

质量

质量

0.1

设置施加力时每个衍生粒子的质量，例如使用 ParticlePointForce 节点。

质量范围

Mass_variation

0

产生随机的粒子质量 通过将高斯分布的随机数添加到 质量 值乘以 质量范围 价值。

粗略地说，0 等于 质量 1 是非常随机的变化。

传输速度

Transfer_velocity

1

以滑动比例将速度从初始发射器传输到生成的粒子-所有继承速度为 1，0 继承速度为 0。

传播

传播

0

将扩散应用于粒子-默认情况下，围绕发射方向的圆锥体形状。

质量守恒

Conserve_mass

已启用

启用后，将从原始粒子中减去任何衍生粒子的质量。如果始发粒子的质量在帧的末尾为零，则会将其删除。

动量守恒

保守动量

已启用

当启用时，任何衍生粒子的动量都从原始粒子的动量中减去，这与牛顿第三运动定律一致。

将速度与运动方向对齐

Velocity_direction

已启用

启用后，生成的粒子继承父粒子的速度，但运动方向由 ParticleSpawn 控件决定。

禁用时，生成的粒子继承父粒子的运动方向，但速度会增加。

颜色

颜色

1

设置生成粒子的初始颜色时 粒子 输入未连接。

继承颜色

Inherit_color

禁用(disabled)

启用后，生成的粒子的颜色从原始粒子中提取。禁用时，生成的粒子从 粒子 输入，如果连接，或 颜色 控制。

输入顺序

Input_order

随机

设置的顺序 粒子 用于表示的输入:

• 随机 -为发射的每个粒子选择随机输入。

• 在秩序 -每个发射的粒子都循环输入。

从开始

Start_frame_animation

首先

确定新发射粒子的表示从哪个帧开始:

• 首先 -对每个新粒子使用输入的第一帧。

• 在秩序 -为每个新粒子使用连续帧。

• 当前 -使用粒子发射的帧。

• 随机 -在表示的帧范围内使用随机帧。

限制范围

Start_wrap

禁用(disabled)

启用后，粒子开始帧将强制在表示输入的范围内。这导致 从开始 > 在秩序 和 当前 选项，在连续循环中，一旦超过帧范围，将其绕回帧范围的开始。

前进

Frame_advance

在步骤中

确定粒子表示在发射后是否动画:

• 常数 -粒子没有动画，并且在其整个生命周期中保持相同的表示。

• 在步骤中 粒子的表示逐帧推进。

• 随机 -每次都选择不同的随机帧。

最大夹子长度

Max_clip_length

100

根据表示输入设置动画中使用的帧数。

条件选项卡

概率

概率

1

设置此节点影响粒子的概率。如果将其设置为零，节点将不会影响任何粒子，如果该值为 1，节点将影响每个粒子。

最小年龄

Min_age

0

将此节点的效果限制为粒子超过此最小年龄。粒子的年龄是其寿命在 0 到 1 之间标准化。

最大年龄

Max_age

1

将此节点的效果限制为粒子低于此最大年龄。粒子的年龄是其寿命在 0 到 1 之间标准化。

随机种子

种子

0

设置整数以更改粒子中生成的随机性的结果。你可以通过改变这个数字来获得稍微不同的效果。

通道

通道

所有

指定应将此节点的效果应用于哪些粒子通道。通道 一个 和 b 通道有任意名称，如果希望不同的粒子发射器节点或其他粒子力节点对单独的通道产生影响，这些名称非常有用。

区域选项卡

区域

区域

没有

设置要用于限制粒子效果的区域。例如，如果您选择 球体 因此，只有球形区域内的粒子才会受到粒子效果的影响。

• 没有 -所有粒子都正常受到影响。

• 球体 , 盒子 , 半空间 ,和 气缸 -控制区域的边界形状。

反转区域

Region_invert

禁用(disabled)

启用后，区域外部的粒子将受到影响，而不是区域内部的粒子。

File_menu

N/A

选择导入或导出通道文件:

• 导入 chan 文件 -导入通道文件，并根据通道文件中的转换数据转换区域标记。通道文件包含给定镜头中每个动画帧的一组笛卡尔坐标。您可以使用 Nuke 或 3D 跟踪软件 (如 3D 均衡器、 Maya 或 Boujou) 创建和导出它们。

• 导出 chan 文件 -将应用于区域标记的转换参数导出为通道文件。这是艺术家之间共享设置的有用方法。

Snap_menu

N/A

• 匹配选择位置 -区域标记将根据选定的点捕捉到新位置。

• 匹配选择位置，方向 -区域标记将根据选定的点捕捉到新的位置和方向。

• 匹配选择位置、方向、大小 -区域标记将根据选定的点捕捉到新的位置、方向和大小。

SRT

设置缩放 (S) 、旋转 (R) 和平移 (T) 的操作顺序。可能的操作组合是 SRT 、 STR 、 RST 、 RTS 、 TSR 、 TRS。

旋转顺序

ZXY

设置旋转顺序。可能的轴向组合是 ZXY，XYZ，XZY，YXZ，YZX，ZXY，ZYX。

翻译

翻译

0,0，0

用于沿 x 、 y 和 z 轴平移区域标记。也可以通过在 3D 查看器中单击并拖动轴来调整平移值。

旋转

旋转

0,0，0

用于围绕 x 、 y 和 z 轴旋转区域标记。可以通过按住旋转来调整旋转值 Ctrl / Cmd 并在 3D 查看器中拖动。

规模

缩放

1 、 1

用于在 x 、 y 和 z 轴上缩放区域标记。

均匀刻度

Uniform_scale

1

允许您在 x 、 y 和 z 轴上同时缩放区域标记。

倾斜

倾斜

0,0，0

用于倾斜 x 、 y 和 z 轴上的区域标记。

枢轴

枢轴

0,0，0

当您对区域标记的位置、缩放、倾斜和旋转进行更改时，这些更改将从对象的原点或轴的位置发生。枢轴 x 、 y 和 z 控件允许您偏移枢轴点，并将其移动到您喜欢的任何位置 -- 您甚至可以将其移动到对象的外部。随后应用的转换将相对于新的轴点位置发生。

你也可以按住 Ctrl / Cmd Alt 并将枢轴点拖到 3D 查看器中的新位置。
