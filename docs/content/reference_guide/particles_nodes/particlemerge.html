ParticleMerge

If you have more than one set of particle nodes, and you want to combine them into one stream, ParticleMerge is your node. Attach your particle streams to ParticleMerge’s numbered inputs and you’re all set.

Inputs and Controls

Connection Type

Connection Name

Function

Input

numbered inputs

The first particle system you intend to merge. Adding a particle system spawns another numbered input.
