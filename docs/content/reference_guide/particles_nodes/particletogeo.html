ParticleToGeo

使用 ParticleToGeo，可以为特定通道中的粒子创建 3D 几何图形。如果在粒子流中的任何位置连接了三维几何对象，ParticleToGeo 节点将在每个粒子处复制该几何。

显示

display

不变

调整粒子的显示特征。这些设置不会影响场景的渲染输出; 这些设置仅用于在 3D 查看器中显示。

• 关闭 -隐藏粒子。

• 线框 -仅显示粒子的轮廓。

• 固体 -用纯色显示所有粒子。

• 固体 线框 -将粒子显示为带粒子轮廓的纯色。

• 纹理 -仅显示表面纹理。

• 纹理 线框 -显示线框和表面纹理。

• 不变 -不会更改粒子显示模式。查看器可以覆盖此设置。

渠道

渠道

所有

设置通过通道指定几何图形的粒子。例如，默认 所有 将几何图形指定给所有通道中的粒子。

对齐模式

对齐

没有

设置精灵 (非几何图形) 粒子的对齐方式:

• 没有 -粒子不对齐。

• 旋转 -粒子根据其旋转排列。

• 速度 -粒子根据它们的速度排列。
