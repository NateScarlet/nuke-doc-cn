ParticleSpeedLimit 选项卡

显示

display

不变

调整粒子的显示特征。这些设置不会影响场景的渲染输出; 这些设置仅用于在 3D 查看器中显示。

• 关闭 -隐藏粒子。

• 线框 -仅显示粒子的轮廓。

• 固体 -用纯色显示所有粒子。

• 固体 线框 -将粒子显示为带粒子轮廓的纯色。

• 纹理 -仅显示表面纹理。

• 纹理 线框 -显示线框和表面纹理。

• 不变 -不会更改粒子显示模式。查看器可以覆盖此设置。

可选

可选

已启用

启用后，可以通过单击查看器中的粒子来选择它们。

渲染

Render_mode

不变

设置粒子的渲染方式。此控件独立于 显示 选择，但具有相同的设置。

最低

最低

最大

设置粒子可以移动的最大速度。

条件选项卡

概率

概率

1

设置此节点影响粒子的概率。如果将其设置为零，节点将不会影响任何粒子，如果该值为 1，节点将影响每个粒子。

最小年龄

Min_age

0

将此节点的效果限制为粒子超过此最小年龄。粒子的年龄是其寿命在 0 到 1 之间标准化。

最大年龄

Max_age

1

将此节点的效果限制为粒子低于此最大年龄。粒子的年龄是其寿命在 0 到 1 之间标准化。

随机种子

种子

0

设置整数以更改粒子中生成的随机性的结果。你可以通过改变这个数字来获得稍微不同的效果。

通道

通道

所有

指定应将此节点的效果应用于哪些粒子通道。通道 一个 和 b 通道有任意名称，如果希望不同的粒子发射器节点或其他粒子力节点对单独的通道产生影响，这些名称非常有用。

区域选项卡

区域

区域

没有

设置要用于限制粒子效果的区域。例如，如果您选择 球体 因此，只有球形区域内的粒子才会受到粒子效果的影响。

• 没有 -所有粒子都正常受到影响。

• 球体 , 盒子 , 半空间 ,和 气缸 -控制区域的边界形状。

反转区域

Region_invert

禁用(disabled)

启用后，区域外部的粒子将受到影响，而不是区域内部的粒子。

File_menu

N/A

选择导入或导出通道文件:

• 导入 chan 文件 -导入通道文件，并根据通道文件中的转换数据转换区域标记。通道文件包含给定镜头中每个动画帧的一组笛卡尔坐标。您可以使用 Nuke 或 3D 跟踪软件 (如 3D 均衡器、 Maya 或 Boujou) 创建和导出它们。

• 导出 chan 文件 -将应用于区域标记的转换参数导出为通道文件。这是艺术家之间共享设置的有用方法。

Snap_menu

N/A

• 匹配选择位置 -区域标记将根据选定的点捕捉到新位置。

• 匹配选择位置，方向 -区域标记将根据选定的点捕捉到新的位置和方向。

• 匹配选择位置、方向、大小 -区域标记将根据选定的点捕捉到新的位置、方向和大小。

旋转顺序

Rot_order

设置旋转顺序。可能的轴向组合是 ZXY，XYZ，XZY，YXZ，YZX，ZXY，ZYX。

倾斜

倾斜

枢轴

枢轴

0,0，0

当您对区域标记的位置、缩放、倾斜和旋转进行更改时，这些更改将从对象的原点或轴的位置发生。枢轴 x 、 y 和 z 控件允许您偏移枢轴点，并将其移动到您喜欢的任何位置 -- 您甚至可以将其移动到对象的外部。随后应用的转换将相对于新的轴点位置发生。

你也可以按住 Ctrl / Cmd Alt 并将枢轴点拖到 3D 查看器中的新位置。
