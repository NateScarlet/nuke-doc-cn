重新点亮

“重点亮” 节点拍摄包含法线和点位置路径的 2D 图像，并允许您使用 3D 灯光重新点亮该图像。基本上，Relight 绕过了返回到 3D 应用程序并重新渲染照明的需要，提供了一种在 2D 环境中重新点亮 3D 场景的快速交互式方式。

Relight 的工作原理是使用存储在单独图像通道中的法线和点位置通道将 3D 着色器应用于 2D 图像, 并允许您附着和操作 3D 灯光 (或通过场景节点的多个灯光)。
