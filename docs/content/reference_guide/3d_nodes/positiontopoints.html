PositionToPoints

从 xyz 图像 (如位置通道) 生成三维点云。如果所有数据都在输入图像的多个通道中，则设置 表面点 输入图像中包含 xyz 数据的通道，并可以选择设置 表面正常 指向具有点法线的通道。如果在单独的图像流中有 xyz 和法线，则可以将 xyz 流连接到 Pos 输入并可选地将法线连接到 规范 输入。

输入

未命名

从中生成点云的图像 (如果所有 xyz 数据都位于一个图像中)。这也是获取点云颜色信息的地方。

Pos

包含位置数据的图像。请注意，在连接未命名输入之前，此输入是隐藏的。

规范

包含法线数据的图像。请注意，这个输入是隐藏的，直到 Pos 输入已连接。

显示

显示

固体

调整 3D 对象的显示特征。这些设置不会影响场景的渲染输出; 这些设置仅用于在 3D 查看器中显示。

• 关闭 -隐藏三维几何对象。

• 线框 -仅显示对象几何图形的轮廓。

• 固体 -用纯色显示所有几何图形。

• 实心线框 -使用对象的几何轮廓将几何图形显示为纯色。

• 纹理 -仅显示表面纹理。

• 纹理线框 -显示线框和表面纹理。
