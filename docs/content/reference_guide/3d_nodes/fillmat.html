FillMat

FillMat 节点允许您用恒定颜色替换选定的材质通道。通常，您可以使用此节点使一个对象容纳其他对象，类似于使用黑色常数节点作为输入纹理。然而，使用 FillMat 节点的优点是，除了 rgb 通道之外，您可以轻松地将其应用于 alpha 通道，并且它不会破坏着色序列, 因此，您可以将其插入节点树中的其他材质节点之后。
