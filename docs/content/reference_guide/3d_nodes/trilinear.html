三线性

通过三线性节点，您可以使用从对象的边界框到给定目标框的三线性插值将三维几何对象整体扭曲。例如，可以使用此节点创建动画对象变形，如弹跳球的挤压。

三线性选项卡

显示

display

不变

调整 3D 对象的显示特征。这些设置不会影响场景的渲染输出; 这些设置仅用于在 3D 查看器中显示。

• 关闭 -隐藏三维几何对象。

• 线框 -仅显示对象几何图形的轮廓。

• 固体 -用纯色显示所有几何图形。

• 固体 线框 -使用对象的几何轮廓将几何图形显示为纯色。

• 纹理 -仅显示表面纹理。

• 纹理 线框 -显示线框和表面纹理。

• 不变 -不会更改对象的显示模式。查看器可以覆盖此设置。

可选

可选

已启用

启用后，您可以在查看器中正常进行选择。禁用时，无法选择或更改点。

不变

N/A

复制

Copy_shape

N/A

复制边界框角点的当前位置 ( P0-p7 ) 进入临时缓冲区。

粘贴

Paste_shape

N/A

粘贴先前复制的边界框角的位置 ( P0-p7 )。

设置密钥

Set_key_shape

N/A

为所有边界框角的位置设置关键帧 ( P0-p7 ) 在当前帧上。这与选择相同 设置 关键 从每个控件旁边的动画菜单中。

删除密钥

Del_key_shape

N/A

删除所有边界框角点位置上的关键点 ( P0-p7 ) 在当前帧上。

N/A
