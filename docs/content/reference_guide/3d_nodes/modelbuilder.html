ModelBuilder

ModelBuilder 节点 ( NukeX 和 Nuke 工作室 仅) 提供了一种为 2D 快照创建 3D 模型的简单方法。您可以通过创建形状然后编辑模型来构建模型，并通过将顶点拖动到相应的 2D 位置来对齐模型。

为了能够对齐模型，ModelBuilder 需要一个跟踪摄像头和一个用于视觉参考的输入图像。如果场景中已经有其他 3D 几何图形和点云，也可以使用它们作为参考。

如果您没有连接相机、图像序列或参考几何图形，您仍然可以创建和编辑 3D 模型，只是无法在对齐模式下执行任何操作。

使用 ModelBuilder 创建和编辑模型需要 NukeX 许可证，但生成的几何图形也可以在 Nuke 中使用。

所以，这是几何编辑的基础。让我们看看如何用它来为这张照片做一些干净的盘子工作。我想建立一些几何和做一些预测。我有一个 ModelBuilder 节点连接到我的镜头和一个匹配移动的相机，所以我想为我的镜头设置它。让我们回顾一下我的相机移动。所以，你可以看到这里的比例是 100 左右。我将不得不进去更新我的 远 裁剪平面 (在查看器属性中); 让我们在那里使用 10,000。我还需要设置 距离 ModelBuilder 节点上的参数。我喜欢使用 10 倍的刻度作为我在这里使用 so 1,000 所需的最大距离。

我想做的第一件事是给后墙建模，所以我会使用 对齐模式 放置一张卡片并锁定在照片上。对齐教程将告诉你如何做到这一点。我们要做的是放几个角，然后向前跳，所以我们已经对齐了这些角，现在我们有一张卡锁定在镜头上, 但那是在窗格上。所以，我要做的是把一个点作为 定位器 在墙上，然后我将在窗玻璃上拿起我的卡片，放大它，让它进入拍摄的位置。然后我将切换到 3D，然后将卡从窗玻璃移动到墙的前面。所以，现在我在墙上有一张卡片可以使用，我可以编辑这张卡片来模拟墙的形状。我要做的第一件事是将卡片的边缘与窗框对齐。你可以使用 拖动选择 或按住 转变 要执行多个选择，您也可以使用 Alt 转变 如果需要，从选择中删除。不过，这里有一个方便的技巧，就是选择一条边，然后使用 选择边循环 快速拾取边缘，这样你就可以放置它们。现在，你看，我的卡上没有足够的边来存放这里的窗框，所以我将通过细分这些边来插入一些顶点，然后我将使用 雕刻 将新顶点连接在一起。现在我有了一个新的优势，我可以把它捡起来，移到正确的位置。最后，我要把这个边缘放在窗台上。要创建插入窗口，我可以切换到 选择面 ,选择这些面并 合并 所以我可以把每一个都捡起来, 挤出 它返回，并再次定位在后窗口窗格上。所以现在我们有了嵌入的窗户，我们有了后墙所需的几何形状。

接下来，我想为床创建一些几何图形。所以我首先用对齐方式锁定我的几何图形，在这里我将从床上的一张卡片开始。我定位角落，现在我在墙上有一张卡片，我可以切换到编辑这张卡片，以创建我需要的形状。我拿起前面的边缘, 挤出 它下来，然后我拿起角落边缘 斜面 它创建一个光滑的形状。这给了我床的表面。

最后，我要在床上做一个风扇模型。我将使用对齐方式将立方体放置在风扇的正面。所以，再次定位角，一旦我们有一个立方体，我会进去编辑，选择边缘并使用 斜面 在这里创造角的圆度。现在我们有了风扇。

现在我们有了所有的几何图形，让我们看看 3D 的外观。您可以从 ModelBuilder 中选择并点击 烘焙 (选定几何图形) 创建单独的节点。您也可以使用 烘焙 (投影) 为特定帧创建投影组。这是我以前为第一帧创建的一个。所以我们要做的是把我们烘焙的场景几何图形挂起来，用 ScanlineRender 节点渲染出来。

所以你可以看到，在短短几分钟内，我们已经对齐和编辑了为这张照片设置投影所需的几何图形，我们使用 ModelBuilder 节点在 Nuke 中完成了这一切。谢谢你的收听，我希望你喜欢这个教程。
