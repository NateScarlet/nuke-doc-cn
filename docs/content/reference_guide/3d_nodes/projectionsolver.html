ProjectionSolver

ProjectionSolver (仅限 NukeX) 可帮助您将 2D 与 3D 点对齐。例如，您可以将 2D 图像与 3D 模型对齐。ProjectionSolver 节点的输出是 相机 与 2D 镜头的相机运动相匹配的节点。您也可以创建 卡 节点来匹配每个不同的相机位置。

镜头

镜头失真

Lensdistimultiontype

无镜头失真

选择预期的失真类型:

• 无镜头失真 -禁用所有镜头失真控制，并将镜头视为没有失真。这个选项给出了处理原始镜头的最佳解决方案。计算的焦距将补偿镜头失真。

• 已知镜头 -允许您手动指定镜头失真，并在进行相机解算时使用该信息。当您有可用于定义镜头失真的网格时，请使用此选项。你可变形参数的计算 透镜失真 首先节点，然后将值粘贴到 ProjectionSolver 上。

• 精炼已知镜头 -给你一个机会，给一个近似失真，使用，但试图完善它在相机解决。当使用网格时，这是不必要的，但是如果使用替代方法或手动设置来计算镜头失真，这可能会改善解决方案。

• 未知镜头 -自动从序列中计算镜头失真 (与 图像分析 中的选项 透镜失真 节点)，然后在相机求解中细化失真。这样，您就不必使用透镜失真节点分别计算镜头失真。

过滤器

filter

设置要在何时使用的过滤算法 不失真输入 已启用。这允许您避免图像质量问题，特别是在帧的高对比度区域 (高度锯齿或锯齿, 如果没有过滤像素并保留其原始值，则可能会出现边缘)。

• 冲动 -重新映射的像素携带其原始值。

• 立方 -重新映射的像素接收一些平滑。

• 钥匙 -重新映射的像素会得到一些平滑，加上轻微的锐化 (如曲线的负 y 部分所示)。

• 西蒙 -重新映射的像素会获得一些平滑，加上中等锐化 (如曲线的负 y 部分所示)。

• 里夫曼 -重新映射的像素获得一些平滑，加上显著的锐化 (如曲线的负-y 部分所示)。

• 米切尔 -重新映射的像素接收一些平滑，加上模糊来隐藏像素化。

• 帕森 -重新映射的像素接收所有过滤器的最大平滑。

• 缺口 -重新映射的像素接收平坦平滑 (倾向于隐藏莫尔图案)。

• Lanczos4 , Lanczos6 ,和 Sinc4 -重新映射的像素接收锐化，这对于缩小缩放很有用。 Lanczos4 提供最少的锐化和 Sinc4 最多的。
