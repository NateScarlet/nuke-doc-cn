显示

display

纹理

调整 3D 对象的显示特征。这些设置不会影响场景的渲染输出; 这些设置仅用于在 3D 查看器中显示。

• 关闭 -隐藏三维几何对象。

• 线框 -仅显示对象几何图形的轮廓。

• 固体 -用纯色显示所有几何图形。

• 固体 线框 -使用对象的几何轮廓将几何图形显示为纯色。

• 纹理 -仅显示表面纹理。

• 纹理 线框 -显示线框和表面纹理。

Nuke 并不局限于 2D 空间，事实上，它有一个完整的 3D 环境。例如，这里有一艘 3D 船和一个 3D 球体。要查看 3D 环境，请转到 视图 菜单上写着 2D 并切换到 3D ,还有环境。为了改变视图，这是默认的相机，您可以使用 Alt 或者你的 选项 键，以及鼠标按钮。例如, Alt 和鼠标左键滚动, Alt 和鼠标中键缩放，和 Alt 和鼠标右键轨道。

让我们看看现场有什么。背景中有 3D 相机、聚光灯、点光源、原始球体、进口宇宙飞船和一张大型原始卡片。让我们看看节点网络，你可以看到我们需要做一个 3D 场景。连接最多的节点是场景节点。场景节点将灯光和几何体组合在一起，以便将它们传递到渲染节点。为了渲染场景，使其成为 2D，您需要某种渲染节点。在这种情况下，有一个 ScanlineRender 节点。连接到 ScanlineRender 的是一个 3D 相机。连接到场景节点的是两个灯光 -- 有聚光灯和点光源。如果我打开聚光灯下的属性，你可以看到常见的选项，比如 颜色 和 强度 而且，在聚光灯下, 锥角 。这里还有两个原始几何图形 -- 有球体和卡片。这将是一个很好的时间来注意 3D 节点有一个圆形的，像药丸一样的形状，而不是矩形的 2D 节点。

您可以创建一个或一个简单的几何形状的 3D 菜单。你可以把你的观点或你的位置，加上一个直接的和一些专门的灯，像一个叫做光，你可以用它从其他程序，如玛雅导入灯。也有 几何 菜单，它有原语，如 卡 ,或其他形状，如 立方体 和 气缸 。可以变换灯光和几何图形。例如，如果我打开球体，你会看到有一个 翻译 , 旋转 ,和 规模 财产。一旦打开，您还将看到有一个变换句柄。如果您单击 “沿轴拖动” 控制柄，可以沿该方向移动它，例如 Y。当然，您也可以在 “属性” 面板中输入值。灯光也有自己的变换集。现在，一个新功能是灯光可以在 3D 环境中投射阴影。例如，如果我去聚光灯下，去 阴影 标签，你会看到有一个地方可以点击 投射阴影 。让我们回到 2D 视图。你可以在宇宙飞船上看到球体的阴影。现在，除了阴影，当然，你可以动画所有这些属性。您可以设置灯光动画，随时间变化，以及几何图形。所有这些属性旁边还有动画按钮。你可以像使用 Nuke 中的任何其他节点一样键入这些。

你会注意到这两块几何有着色器连接到他们的 Img 管道。这些是正确点亮曲面所必需的。球体有一个 Phong，它类似于像 Maya 这样的程序中的一个。该卡具有发射着色器，该着色器具有发射组件或环境颜色组件。现在，就宇宙飞船而言，它必须通过 ReadGeo 节点导入。ReadGeo 节点有一个引入文件的位置，这支持 。Fbx 文件，或 。Obj 文件，或 alembic 文件, 。Abc 。如果文件中有动画，Nuke 会识别它。例如，使用 。Fbx 文件，它可能有多个需要。Nuke 会认识到这一点，你可以选择动画拍摄。所以，如果我回到 3D 视图，擦洗时间线，我们会看到船是预先动画的，这个动画是在玛雅中创建的。还有一个材料连接到 Img ReadGeo 的管道。现在，因为 UV 纹理空间通过 。Fbx 文件，为了映射几何图形，您只需要通过读取节点引入纹理位图，并连接到着色器。举例来说，这里是漫图接通 MapD ,或地图漫射。有一个镜面地图通过连接 地图 ,或地图高光。让我们回到 2D 视图。

现在，如果有任何动画，您也可以激活运动模糊。为此，您将转到渲染节点，例如，使用 ScanlineRender，转到 多样本 标签和更改 样品 到更高的数字，如 8 。在这一点上，运动模糊将出现，正如你在这里看到的。更高的 样品 数量，质量越高。

所以，对 Nuke 的 3D 环境有一个简要的介绍。请记住，您需要为此创建的任何节点都可以通过 3D 节点菜单。这包括所有的着色器、几何体、灯光、场景节点和摄像机。除了动画灯光和几何图形，您还可以自由地为相机制作动画。他们有自己的一套转变。无论如何，我建议探索 Nuke 的这个组成部分。
