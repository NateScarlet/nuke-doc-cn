传输

传输节点允许您通过模拟穿过表面的光的吸收来渲染半透明的彩色阴影。传输着色器必须与投射完整 alpha 阴影的灯光一起使用。

注意: 仅当 3D 场景包含投射完整 alpha 阴影并使用光线渲染的灯光时，传输节点才有效。

另见 光线渲染 .

透过率

0.18

当 吸收模式 设置为 常数 ,控制从纹理图像穿过的光量。0 等于没有光通过，1 是所有光通过。

强度

吸收模式

Absorption_mode

常数

设置用于计算阴影的吸收模式:

• 常数 -使用 透过率 控制以设置吸收量。

• 固体 -使用输入材料的固体成分来设置吸收量。

• 扩散 -使用输入材料的漫反射组件来设置吸收量。

• 地图 -使用 地图 输入设置吸收量。
