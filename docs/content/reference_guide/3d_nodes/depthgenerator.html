深度发生器

DepthGenerator 节点 ( NukeX 和 Nuke 工作室 仅) 允许您从您的镜头生成深度贴图。该节点使用跟踪摄像机的信息创建一个显示深度变化的通道。

深度贴图是一种图像，它使用每个像素的亮度来指定 3D 场景点和用于捕捉场景的虚拟相机之间的距离。

DepthGenerator 还允许您在法线和位置传递时输出深度，并创建 Card3D 节点位于 3D 空间中，并根据深度通道移位。

忽略遮罩

IgnoreMask

没有

设置遮罩类型以排除 源 深度计算期间的镜头。DepthGenerator 期望值为 0 (对于要使用的区域) 或 1 (对于要忽略的区域)。

深度只能在真实世界 3D 位置不变的情况下计算。如果你 源 镜头包括移动的前景对象，最好提供一个遮罩来防止移动区域的失真:

• 没有 -没有一个镜头被忽略。

• 源 Alpha -使用的 alpha 通道 源 剪辑以定义要忽略的区域。

• 掩模倒置亮度 -使用倒置的亮度 面具 输入以定义要忽略的区域。

• 面具阿尔法 -使用 面具 输入 alpha 通道以定义要忽略的区域。

• 面具倒阿尔法 -使用倒置 面具 输入 alpha 通道以定义要忽略的区域。

法线传递包含图像中每个像素的三个信息向量: X 方向、 Y 方向和 Z 方向。换句话说，它存储图像中每个点面对的方向。您可以使用法线通过 重新点亮 节点。

帧分离

框架分离

1

调整当前帧与用于计算输入深度的帧之间的偏移。

对于快速相机移动，使用较小的分离值，对于慢速相机移动，使用较大的分离值。

增加分离会给出更大的基线和更精确的深度。然而，高分离降低了图像匹配的质量，并给出了不正确的 3D 点。

动画的 框架 分离 以补偿可变的相机速度。

摄像机在 相机 必须为帧范围内的所有帧定义输入。

注意: 的 分析序列 和 分析框架 按钮尝试计算最佳分离。

计算精度

精度

N/A

显示分析帧分离时计算的深度精度。接近 1 的值被认为是准确的，接近 0 的值不准确。

在 3D 中放置元素时，使用高精度的框架 (例如，通过单击 创建卡片 或使用 PositionToPoints 节点)。

如果你没有得到准确的深度值，试着调整 帧分离 或使用 忽略面具。

深度生成

深度细节

VectorDetail

0.5

调整此选项以改变用于计算深度贴图的图像的分辨率。默认值 0.5 等于图像分辨率的一半。较低的值加快处理速度，并提供更平滑的结果。更高的值会获得更精细的细节，但也会增加处理时间。

正常细节

NormalDetail

0.25

调整曲面法线的分辨率。减少细节以平滑法线传递。

默认值 0.25 会导致以四分分辨率计算法线传递。

噪音

NoiseLevel

0.01

在计算深度贴图时，设置要在输入素材中忽略的噪波量。

尝试将其增加以平滑平面图像区域中的深度。

强度

强度

1

设置帧之间匹配像素的强度。增加这一点以强制匹配，例如在错过细节的地方，或者减少到平滑的深度。

在大多数情况下，您可以将此保留为默认值。

锐度

锐度

0.5

定义对象边界的不同程度。增加清晰度以分离对象，并将其降低到平滑深度。

平滑度

平滑度

0.5

调整视差计算的平滑度权重。这不会影响深度计算，而是对结果应用智能模糊。

高平滑度可能会错过许多局部细节，但不太可能产生嘈杂的深度值。低平滑度将集中在细节匹配上，即使生成的深度图参差不齐。

深度限制

近

0.1

设置最小 z 深度。低于此数字的所有深度值都将设置为此值。

远剪辑平面

远

10000

设置最大 z 深度。高于此数字的所有深度值都将设置为此值。

标记不良区域

MarkRegions

禁用(disabled)

启用后，深度计算不明确的区域将在深度贴图中标记为非常大的值。
