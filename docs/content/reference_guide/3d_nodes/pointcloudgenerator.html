PointCloudGenerator

使用 PointCloudGenerator 节点 ( NukeX 和 核武器工作室 仅) 可以根据生成的信息创建密集的点云 摄影师 并进一步使用密集的点云来创建 2D 镜头的 3D 网格。密集的点云是场景中三维建模和定位对象的有用起点。

注意: 分析必须在 NukeX 中进行，但结果可以在 Nuke 中使用。
