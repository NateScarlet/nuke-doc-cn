BlendMat

BlendMat 节点设置像素是如何被应用的材料着色的，以与后面对象的像素相结合。它就像 MergeMat 节点，但是它不是与另一个材质混合，而是与 3D 场景中后面渲染的任何材质混合。

注意: 同一层上的多个 BlendMat 操作可能会产生意想不到的结果。

操作

operation

结束

选择要如何组合 BlendMat 节点的输入材质和背景像素:

• 没有 -将材料设置为黑色。

• 替换 -显示材料和背景重叠的材料。

• 结束 -根据材料的 alpha 在背景像素上复合材料。

• 钢网 -显示背景像素，其中材质的 alpha 为黑色。如果材质的 alpha 为白色，则材质设置为黑色。要做到这一点，BlendMat 节点需要处理 alpha 通道，因此将通道设置为 rgba。此操作与 mask 相反。

• 面具 -显示背景像素，其中材质的 alpha 为白色。当材质的 alpha 为黑色时，材质也设置为黑色。要做到这一点，BlendMat 节点需要处理 alpha 通道，因此将通道设置为 rgba。此操作与模板相反。

• 加 -将背景像素添加到材质中。

• 最大 -如果材质的像素值大于背景像素，则使用该材质，否则使用背景像素。

• Min -如果材质的像素值小于背景像素，则使用该材质，否则使用背景像素。

表面混合

输入片段

确定输入纹理如何与应用着色器的几何体的顶点颜色混合:

• 输入纹理 -仅使用输入纹理。

• 顶点颜色 -仅使用顶点颜色。

• 调制 纹理 * 顶点颜色。

• 加 纹理顶点颜色。

• 结束 -(纹理 ((1-a) * 顶点颜色))。

• 减 -纹理-顶点颜色。
