径向扭曲

径向扭曲节点是顶点沿对象中心方向的非线性变换，产生桶形或销垫形扭曲。

显示

display

不变

调整 3D 对象的显示特征。这些设置不会影响场景的渲染输出; 这些设置仅用于在 3D 查看器中显示。

• 关闭 -隐藏三维几何对象。

• 线框 -仅显示对象几何图形的轮廓。

• 固体 -用纯色显示所有几何图形。

• 固体 线框 -使用对象的几何轮廓将几何图形显示为纯色。

• 纹理 -仅显示表面纹理。

• 纹理 线框 -显示线框和表面纹理。

• 不变 -不会更改对象的显示模式。查看器可以覆盖此设置。

可选

渲染

Render_mode

不变

设置对象的渲染方式。此控件独立于 显示 选择，但具有相同的设置。

失真

失真

0

设置失真量。低于 0 的值产生桶形失真，而高于 0 的值产生销垫形失真。如果将值设置为 0，3D 对象不会扭曲。

功率偏压

Power_bias

4

扭曲的幅度。值越高，对象就会变得越扭曲。

旋转中心

中心

0,0，0

通过在此处输入旋转的 X 、 Y 和 Z 坐标来移动旋转的中心点。

也可以通过在查看器中拖动中心点的控件来移动中心点。

规模

规模

1,1，1

X 、 y 或 z 方向的失真量。

您也可以通过在查看器中拖动控件来调整这一点。

保存中心

Preserve_center

已启用

选中以使对象在 3D 空间中的原始位置居中。取消选中以允许对象从其中心偏移。
