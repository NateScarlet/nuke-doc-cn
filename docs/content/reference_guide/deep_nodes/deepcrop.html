• 在某些平面前面或后面进行深度裁剪，或

• 在观众的裁剪框内部或外部裁剪深层图像 (很像普通的 作物 节点)。

Bbox

bbox

N/A

在查看器中设置裁剪框的大小。默认情况下，此框外的任何区域都将被裁剪。

您可以调整以下内容:

• x -图像左边缘和裁剪框左侧之间的距离 (以像素为单位)。

• y -图像底部边缘和裁剪框底部边缘之间的距离 (以像素为单位)。

• r -图像左边缘和裁剪框右侧之间的距离 (以像素为单位)。

• t -图像底部边缘和裁剪框顶部边缘之间的距离 (以像素为单位)。

• w -裁剪框的宽度。只有当您单击 Wh 按钮。

• h -裁剪箱的高度。只有当您单击 Wh 按钮。

也可以通过拖动裁剪框的边缘来调整查看器中的裁剪框。

使用

Use_bbox

已启用

设置是否在查看器中使用裁剪框。

保持在 bbox 之外

Outside_bbox

残疾人

启用后，裁剪框内的任何样本都将被裁剪。

禁用时，裁剪框外的任何样本都将被裁剪。
