残疾人

禁用时，Nuke 通过 unpremultiply 将颜色分配给每个样本 颜色 图像的 alpha，然后再乘以每个样本的 alpha。

然而，有时最终高质量平面渲染中的 alpha 可能与给定深度样本集所表示的 alpha 不匹配。这可能是压缩的结果，通常发生在磁盘上的深层文件，或者着色器的一些变化。

当 目标输入 alpha 启用后, 颜色 输入的 alpha 分布在样本中，以便深度数据平坦化后最终产生的 alpha 与 颜色 输入的 alpha。
