O_DisparityToDepth

O_DisparityToDepth 插件根据剪辑的视差场和立体相机设置，为立体剪辑的每个视图生成 z 深度贴图。

Z 深度贴图是一种图像，它使用每个像素的亮度来指定 3D 场景点和用于捕捉场景的虚拟相机之间的距离。例如，如果你想在镜头中引入雾和景深效果，你可能需要一个 z 深度贴图。在 Nuke 中，ZBlur 节点 ( 过滤器 > ZBlur ) 需要输入深度图。

O_DisparityToDepth 将最终的 z-depth 贴图存储在深度通道中，因此在首次计算深度贴图时可能看不到任何图像数据。要查看 Nuke 内部的输出，请从查看器左上角的通道控件中选择深度通道。
