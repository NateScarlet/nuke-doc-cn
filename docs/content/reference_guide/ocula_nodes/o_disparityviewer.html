O_DisparityViewer

O_DisparityViewer 插件允许您可视化 差距 在节点树的向量中，显示详细说明正负视差的柱状图，或者用视差冲突覆盖观察者。

注意: 如果在输出序列时启用了节点，则所有三种 O_DisparityViewer 模式都将烘焙到渲染中。这允许您将输出与差异一起进行审查。
