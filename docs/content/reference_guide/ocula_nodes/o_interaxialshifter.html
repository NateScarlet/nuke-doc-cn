O_InteraxialShifter

O_InteraxialShifter 插件允许您减少 轴间距离 立体图像; 即两个摄像机之间的距离。使用此插件，您可以生成 原始图像之间指定位置的两个新视图。

输入和控制

连接类型

连接名称

函数

输入

Fg

一个可选的面具 从背景描绘前景区域。面具确保差异被限制在每个划定的区域内。 请注意，遮罩应该存在于两个视图中，O_InteraxialShifter 期望 alpha 值为 0 (对于背景) 或 1 (对于前景)。

源

一对立体图像。

注意: 如果图像中没有嵌入视差通道，则应在图像序列后添加 O_Solver 和 O_DisparityGenerator 节点。

控制 (UI)

旋钮 (脚本)

默认值

函数

O_InteraxialShifter 选项卡

要使用的视图

视图对

依赖于源

设置要用于创建新视图的两个视图。这些视图将被映射到左眼和右眼。

向左插入位置

Leftinterpateposition

0.25

在要生成左视图的视图之间设置一个位置。位置表示为视图之间距离的一小部分。

插入正确位置

Rightinterpateposition

0.75

在要生成正确视图的视图之间设置一个位置。位置表示为视图之间距离的一小部分。

过滤

过滤

正常

设置过滤质量:

• 极端 -使用 sinc 插值滤波器提供更清晰的图片，但渲染需要更长的时间。

• 正常 -使用双线性插值，给出了良好的结果，比极端快得多。

扭曲模式

WarpMode

正常

设置用于创建新视图的方法:

• 简单 -这是最快的选择，但可能会在图像边缘和在一个视图中可见但在另一个视图中不可见的对象周围产生不太理想的结果。

• 正常 -这是标准选项，对图像边缘和遮挡进行更优化的处理。

• 闭塞 -这是一个高级选项，可以提高结果。

• 尖锐闭塞 -此选项类似于 闭塞 ,但产生更少的工件，其中视差字段是从 3D CG 源生成的。

前景组件

面具

没有

将要使用的通道设置为 从背景中描绘前景区域。面具确保差异被限制在每个划定的区域内。

• 没有 -使用整个图像区域。

• 源 Alpha -使用的 alpha 通道 源 剪辑。

• 源倒阿尔法 -使用的倒阿尔法通道 源 剪辑。

• 遮罩亮度 -使用的亮度 Fg 输入。

• 掩模倒置亮度 -使用倒置的亮度 Fg 输入。

• 面具阿尔法 -使用的 alpha 通道 Fg 输入。
