O_Solver

O_Solver 插件定义输入图像中两个视图之间的几何关系 (即相机关系或求解)。如果要在树中使用 DisparityGenerator 、 VectorGenerator 或 verticalalalaligner，这是必要的。

• 关键帧匹配 -在查看器覆盖中显示相机关系计算的功能和匹配。

对齐方法

对齐方法

垂直倾斜

设置的调整方法，用以调整的意见时, 显示 设置为 预览对齐 :

• 垂直倾斜 -使用倾斜沿 y 轴对齐特征。这不会沿 x 轴移动特征。

• 透视扭曲 -对图像进行四角扭曲，使其在 y轴上对齐。这可能会沿 x 轴略微移动特征。

• 旋转 -通过围绕一点旋转整个图像来垂直对齐特征。旋转中心由算法确定。

• 规模 -通过缩放图像垂直对齐特征。

• 简单转变 -通过向上或向下移动整个图像来垂直对齐特征。

• 缩放旋转 -通过同时缩放和围绕一点旋转整个图像来垂直对齐特征。旋转中心由算法确定。

• 相机旋转 -首先对两个相机进行 3D 旋转，使它们具有完全相同的方向和平行的观察轴，然后重新收敛视图以提供原始收敛，从而对齐特征。为了获得最佳结果，请使用 相机 为拍摄摄像机提供信息的输入。

匹配偏移

偏移

100

设置应用于对齐特征匹配的偏移 (以像素为单位)。您可以:

• 增加这个值可以人为地增加差异，这样就更容易看到特征匹配的水平程度。任何非水平的匹配都可以被认为是糟糕的匹配，并手动删除。

• 减小此值可将特定匹配项的差异设置为零，并检查每个特征处的垂直偏移。比赛应该放在一起。如果它们是垂直偏移的，你知道它们不好，可以手动删除它们。

注意: 的 匹配偏移 控件仅在 显示 设置为 预览对齐 .
