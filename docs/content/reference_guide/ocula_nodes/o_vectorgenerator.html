O_VectorGenerator

O_VectorGenerator 生成 立体图像中每个视图的运动矢量场。运动向量场将一帧上像素的位置映射到相邻帧中相应像素的位置。它与图像具有相同的尺寸，但每像素包含一个 (x，y) 偏移。这些偏移显示了如何将相邻图像扭曲到当前图像上。

输入和控制

连接类型

连接名称

函数

输入

Fg

可选遮罩，指定运动计算中仅包含的区域。可以使用它为前景元素创建运动层。要为不同元素创建层，请使用几个 O_VectorGenerator 节点。

如有必要，您也可以使用 忽略 遮罩以排除前景区域中的元素。请注意，遮罩应该存在于两个视图中，O_VectorGenerator 期望 alpha 值为 0 (对于背景) 或 1 (对于前景)。

忽略

一个可选遮罩，指定要从运动计算中排除的区域。您可以使用此输入来防止遮挡处的扭曲，或者通过忽略所有前景元素来计算背景层的运动。

注意: 遮罩应该存在于两个视图中，O_VectorGenerator 期望 alpha 值为 0 (对于要使用的区域) 或 1 (对于要忽略的区域)。

求解器

如果 源 序列不包含 o _ solver 能够很好地匹配的特征，您可以在具有相同相机设置的另一个序列快照上使用 o _ solver。如果这样做，将 O_Solver 连接到此输入。

源

一对立体图像。

除非您使用 求解器 输入，图像后面应该跟着一个 O_Solver 节点。如果图像中没有嵌入视差通道，则在此输入中还应该有一个 O_DisparityGenerator 节点。

忽略遮罩

IgnoreMask

没有

设置可选遮罩，指定要从运动计算中排除的区域。您可以使用此输入来防止遮挡处的扭曲，或者通过忽略所有前景元素来计算背景层的运动。

注意: 遮罩应该存在于两个视图中，O_VectorGenerator 期望 alpha 值为 0 (对于要使用的区域) 或 1 (对于要忽略的区域)。

• 没有 -不要使用忽略面具。

• 源 Alpha -使用的 alpha 通道 源 剪辑为忽略遮罩。

• 源倒阿尔法 -使用的倒阿尔法通道 源 剪辑为忽略遮罩。

• 遮罩亮度 -使用的亮度 忽略 作为忽略遮罩输入。

• 掩模倒置亮度 -使用倒置的亮度 忽略 作为忽略遮罩输入。

• 面具阿尔法 -使用的 alpha 通道 忽略 作为忽略遮罩输入。

• 面具倒阿尔法 -使用的倒阿尔法通道 忽略 作为忽略遮罩输入。

前景遮罩

ForegroundMask

没有

设置一个可选的遮罩 指定要包括在运动计算中的唯一区域。可以使用它为前景元素创建运动层。要为不同元素创建层，请使用几个 O_VectorGenerator 节点。

如有必要，您也可以使用 忽略 遮罩以排除前景区域中的元素。

注意: 遮罩应该存在于两个视图中，O_VectorGenerator 期望 alpha 值为 0 (对于背景) 或 1 (对于前景)。

• 没有 -不要使用前景遮罩。

• 源 Alpha -使用的 alpha 通道 源 剪辑作为前景遮罩。

• 源倒阿尔法 -使用的倒阿尔法通道 源 剪辑作为前景遮罩。

• 遮罩亮度 -使用的亮度 Fg 作为前景遮罩输入。

• 掩模倒置亮度 -使用倒置的亮度 Fg 作为前景遮罩输入。

• 面具阿尔法 -使用的 alpha 通道 Fg 作为前景遮罩输入。

• 面具倒阿尔法 -使用的倒阿尔法通道 Fg 作为前景遮罩输入。

噪音

NoiseLevel

0

设置在计算运动矢量时输入素材中 O_VectorGenerator 应忽略的噪波量。

值越高，运动矢量场越平滑。如果发现低对比度图像区域中的运动矢量场很嘈杂，您可能需要增加此值。

强度

强度

1

设置帧之间匹配像素的强度。更高的值允许您准确地将一幅图像中的相似像素与另一幅图像进行匹配，即使生成的运动场参差不齐，您也可以集中精力进行细节匹配。

较低的值可能会错过本地细节，但不太可能为您提供奇怪的伪向量，从而产生更平滑的结果。通常，有必要将这些品质中的一个与另一个交换。您可能希望增加此值以强制图像匹配，例如，在错过细节的地方，或者减少此值以平滑运动向量。

对齐

对齐

0.1

设置约束运动矢量以匹配上游 O_Solver 节点定义的水平对齐的程度。

值 0 使用无约束运动估计计算运动矢量。增加值会强制向量对齐。在大多数情况下，您希望此设置为 0 或默认值 0.1。

锐度

锐度

0

设置计算的运动矢量字段中对象边界的不同程度。增加此值可生成不同的边框和单独的对象。

减小该值可同时模糊运动层并最小化遮挡。为了更好地使用 O_Retimer 构建图片，您可以将此值设置为 0。

平滑度

平滑度

0

将应用于运动矢量场的额外平滑量设置为图像匹配后的后期处理。

值越高，结果越平滑。您可以将其与 锐度 参数分别为快照中的不同对象平滑运动矢量场。
