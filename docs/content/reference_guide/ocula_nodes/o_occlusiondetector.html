O_OcclusionDetector

O_OcclusionDetector 为每个视图中被遮挡的像素生成一个遮罩: 也就是说，像素在一个视图中可见，但在另一个视图中不可见。

深度遮挡

DisparityError

0.1

设置在深度变化时标记遮挡的阈值 (即，在视差急剧变化的地方)。

增加此值可将更多区域标记为遮挡。如果图像中有极端的遮挡，请尝试使用值 0.3 或 0.5。

颜色阈值

ColourError

0.05

设置在视图之间的颜色差异处标记遮挡的阈值 (即，视图之间的图像内容不同)。

减小此值以将更多区域标记为遮挡。如果图像中的颜色变化是平坦的，请尝试使用值 0.02 来获取更多的遮挡。如果有很多高度纹理区域，如树叶，尝试将该值增加到 0.1。

细化选项

遮挡大小

遮挡大小

小

为不同的遮挡大小选择预设。选择在一个视图中可见但在另一个视图中不可见的区域的大小将设置 闭塞扩张 , 遮挡填充 ,和 遮挡模糊 相应地控制。

默认为 小 ,但是如果图像中有很多视差，视图之间可能会有很多图像内容被遮挡或显示，因此您可以切换到 大 或 极端 创建更大的遮挡区域。

闭塞扩张

闭塞扩张

1

设置要展开被视为遮挡的区域的数量 (以像素为单位)。

遮挡填充

遮挡填充

4

设置要填充被遮挡区域内的孔的大小 (以像素为单位)。

增加此值以填充较大的孔。使用 O_ColourMatcher 或 O_FocusMatcher 时，孔可能会出现并消失，导致闪烁。

遮挡模糊

遮挡模糊

8

设置模糊遮挡边界的量 (以像素为单位)。

使用 O_ColourMatcher 或 O_FocusMatcher 时，增加此值以平滑遮挡区域处的过渡。
