UDIM 导入

通过 UDIM Import，您可以导入一组遵循 UDIM 编号方案的纹理块，并将它们快速应用于 3D 对象的曲面。点击 UDIM 导入打开标准 读取文件 (s) 文件浏览器。浏览并选择包含有效 UDIM 补丁的序列将打开 UDIM 导入 对话框。的 UDIM 导入 对话框按顺序列出所有 UDIM 修补程序，并提供一些管理它们的选项。如果选择的文件不包含有效的 UDIM 编号，将显示一个解释问题的对话框。

UDIM Import 为序列中的每个补丁创建一个读取节点，并将一个 UVTile 节点 (允许您修改补丁在 UV 空间中的坐标) 附加到每个补丁。有关 UVTile 节点的更多信息，请参见 UVTile

的 UDIM 导入 对话框允许您禁用序列中的单个修补程序，添加其他文件，并设置读取节点 (针对每个单个修补程序) 在节点图中显示的方式。启用 邮票 在 “读取” 节点中显示修补程序的缩略图视图。检查 组节点 为清晰起见，将节点添加到单个组节点。
