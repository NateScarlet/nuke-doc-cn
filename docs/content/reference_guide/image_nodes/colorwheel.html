旋转

旋转

填充格式

FillFormat

已启用

启用时，色轮将选定格式填充到 x 轴或 y 轴，这取决于哪个轴较小-色轮始终保持 1:1 的纵横比。

禁用时，使用区域控件指定色轮的区域。
