从左下角开始设置棋盘内方框的颜色:

• 颜色 0 -方框在 0,0，然后是每个备用方框和行。

• 颜色 1 -方框在 1,0，然后是每个备用方框和行。

• 颜色 2 -盒子在 1,1，然后是每个备用盒子和行。

• 颜色 3 -盒子在 0,1，然后是每个备用盒子和行。
