寻求帮助

Nuke 在不同的地点提供多种形式的帮助。

查看工具提示

大多数控件以工具提示的形式提供简洁的说明。要显示工具提示，请将鼠标指针移动到界面控件或节点参数上。

要查看节点的工具提示，请将鼠标指针移动到 ? 节点属性面板中的图标。

查看上下文相关的在线帮助

上下文相关的在线帮助包括每个节点及其参数的全面描述。要在默认的 web 浏览器中显示这些描述，请单击 ? 节点属性面板中的图标。

选择帮助源

在 Nuke ,上下文相关的帮助内容可能会有所不同，这取决于您在 偏好 .

要选择帮助源:

1。 新闻 转变 + S 在节点图上打开 偏好 对话框。
2. 转到 行为 > 文档 标签。
3. 设置 文档来源 至:

• 本地 -使用 Nuke 它内置的帮助系统。这包括在 Nuke 安装和不需要互联网连接。

注意: 当你点击一个 ? 节点上的按钮, Nuke 在以下位置搜索与请求的节点同名的 HTML 文件 (例如，blur.html):

1.NUKE_PATH 环境变量中包含的目录。

2. /$ HOME/.nuke 和 /$ HOME/。nuke/文档 目录。

你当地的 插件 目录。

在 Windows 上，这是 C: \ Program Files \ Common Files \ Nuke \ 10.5 \ Plugins \ 或 C: \ Program Files (x86) \ Common Files \ Nuke \ 10.5 \ 插件 \

在 Mac OS X 上，这是/库/应用程序支持/ Nuke / 10.5 /插件/

在 Linux 上，这是/usr/local/ Nuke / 10.5 /插件/

4. Nuke 安装目录。

• 铸造 -使用我们网站上提供的帮助系统。这包含最新的信息，但需要互联网连接。

注意: 在 Windows 上，您可能必须添加防火墙程序或端口例外，才能查看我们网站上的最新帮助。如果连接被阻止, Nuke 回到了 本地 复制。

• 自定义 -使用自己的自定义帮助系统。

4. 如果你设置 文档来源 到 本地 ,您可以:

• 启用 汽车端口 自动分配自由文档服务器端口，或

• 使用 本地港口 手动指定本地文档服务器端口和 端口范围 定义要尝试的端口范围。通常, 本地港口 值应为 1024 或更高。将值设置为 0 会导致自动分配端口。

注意: 能够使用来自多个会话的在线帮助 Nuke 同时运行，您必须启用 汽车端口 .

5. 如果你设置 文档来源 到 铸造 :

• 的 铸造主机 字段显示文档服务器主机地址。

• 使用 铸造港 指定文档服务器端口。

6. 如果你设置 文档来源 到 自定义 :

• 设置 自定义主机 到您的文档服务器主机地址。这必须是 Internet 协议版本 4 (IPv4) 地址。

• 使用 本地港口 指定本地文档服务器端口。通常, 本地港口 值应为 1024 或更高。将值设置为 0 会导致自动分配端口。

• 使用 自定义端口 指定自定义文档服务器端口。

使用帮助菜单

您可以单击主 帮助 访问以下内容的菜单:

• 关键任务 -热键的列表。

• 文档 -The Nuke 在线帮助, Nuke 开发工具包 (NDK)，以及在中使用 Python 、 TCL 和表达式的文档 Nuke .

• 发行说明 -功能和错误修复的重要更新。

• 培训和教程 - Nuke 教程视频和其他培训资源列表。

• Nukepedia -包含所有有用信息的在线资源 Nuke .

• 邮件列表 -关于的信息 Nuke -相关电子邮件列表。

• 插件安装程序 -在我们的网站上打开 Nuke 插件页面，让您可以轻松访问大量的插件 Nuke .
