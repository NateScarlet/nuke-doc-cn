OneView

OneView 允许您单独处理一个视图。如果需要对不同的视图执行不同的操作，例如，如果要对一个视图进行颜色校正，而不是对另一个视图进行颜色校正，这将非常有用。使用后续节点所做的任何更改仅应用于提取的视图 (无论您在查看器中显示的视图是什么)。

如果要对两个视图执行相同的操作，但对每个视图使用不同的值，则可以在节点的控件中拆分视图。

要将视图合并到节点树的后面，请使用 JoinViews 节点。
