重新收敛

如果您正在进行立体或多视图项目，重新收敛节点允许您转移收敛 (眼睛或相机的向内旋转) 以便使用 3D 眼镜观看时，图像中的任何选定点都显示在屏幕深度。这一点被称为 收敛点 。这是两个摄像机的视线相遇的地方。

在收敛点，使用 3D 眼镜观看时，图像中的不同视图会对齐并显示在屏幕深度。汇合点后面的任何东西都会出现在屏幕后面，而它前面的任何东西似乎都会弹出屏幕。

要计算收敛位移，重新收敛节点需要一个视差场，该视差场将一个视图中像素的位置映射到另一个视图中相应像素的位置。要创建视差字段，可以使用 O_DisparityGenerator 插件，这是 Ocula 插件集的一部分。或者，您可以在 3D 应用程序中创建视差字段。一旦你有了视差字段，你就可以把它存储在一个。Exr 文件，或使用 洗牌 节点添加数据流中需要的视差通道。

也可以在整个图像序列中使用与收敛点相同的元素。为此，请将重新收敛节点链接到 跟踪器 节点。

要更好地查看重新收敛节点的效果，可以插入 浮雕 重新收敛节点和查看器之间的节点。

注意: 重新收敛节点仅水平而不是垂直移动视图。
