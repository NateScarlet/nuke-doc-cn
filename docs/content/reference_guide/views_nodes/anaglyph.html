浮雕

如果您正在处理立体或多视图项目，您可以使用此节点将输入转换为浮雕图像。默认情况下，左输入被过滤以删除蓝色和绿色，右视图以删除红色。

浮雕图像在使用双色浮雕眼镜观看时会产生 3D 效果。

另见 MixViews 和 侧边栏 .

要转换为浮雕图像的图像。这个输入需要包含至少两个视图，一个用于左眼，另一个用于右眼:

• 如果您的输入是 。Exr 文件，该文件可能已经包含两个视图。这是因为。Exr 文件支持单个文件中的多个视图。

• 对于任何其他文件类型，您需要为左视图和右视图分别设置文件。可以使用 JoinViews 节点将这些文件组合到单个输入中。

默认情况下，浮雕将输入图像转换为灰度图像。要向图像中添加颜色，请在此处选择介于 0 (灰度) 和 1 (彩色) 之间的值。

注意: 如果图像包括非常红色、绿色或蓝色的区域，在其中添加更多颜色可能不会产生最佳效果。
