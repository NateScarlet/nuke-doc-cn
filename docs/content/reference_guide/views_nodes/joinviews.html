JoinViews

如果您的视图中存在单独的文件 项目设置 (例如，一个文件用于左视图，另一个文件用于右视图)，JoinViews 允许您将这些文件合并到单个输出中。这可能是有用的:

• 您希望对所有视图执行相同的操作。默认情况下，Nuke 应用您对已处理节点的所有视图所做的任何更改。

• 您的后续节点 (例如, 分割和加入 ) 需要包含两个视图的单个输入。

另见 分割和加入 和 OneView .
