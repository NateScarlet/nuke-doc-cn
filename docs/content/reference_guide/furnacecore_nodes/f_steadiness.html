F_Steadiness

F_Steadiness 使用全局运动估计 (GME) 来计算四角引脚, 这样，单个镜头内的相机运动可以在一系列帧上平滑，也可以通过锁定特定帧来移除。

模式

模式

平滑

这控制 F_Steadiness 是在保持整体相机运动的同时平滑拍摄，还是将拍摄锁定到单个帧以完全移除相机运动。它可以设置为:

• 增量锁定 -计算将每个帧带到锁定帧的引脚。这将通过从锁定帧到每个帧的工作来计算 pin，以增量方式计算每个帧之间的 GME，并累积它以创建拐角 pin。

• 绝对锁定 -这还计算了将每个帧带到锁定帧的 pin。然而，它通过直接从相关帧到锁定帧进行 GME 来做到这一点。

• 平滑 -在这种情况下，镜头被平滑为一系列的帧描述 平滑 控制。使用 平滑 保持相机的整体运动，但要平滑尖锐的颠簸和踢。

精度

精度

0.6

控制时间/准确性的权衡。这越高，分析就越慢，但是你有更好的结果的可能性。

过滤

过滤

介质

设置过滤质量。

• 低 -低质量，但渲染速度快。

• 介质 -使用双线性过滤器。这给出了良好的结果，并且渲染速度比 高 过滤。

• 高 -使用 sinc 滤波器来插入像素，从而进行更清晰的修复。这给出了最好的结果，但需要更长的时间来处理。
