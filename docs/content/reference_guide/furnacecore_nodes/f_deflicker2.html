F_DeFlicker2

F_Deflicker2 用于消除闪烁，特别是局部闪烁，并依赖于场景的几何形状 (即，在整个图像中不存在的闪烁), 比如由镜头中不同步的荧光灯引起的。

注意: F_DeFlicker2 在执行计算时查看当前帧之外的输入帧，因此可能是一个计算昂贵的节点。因此，在节点树中使用两个以上的 F_DeFlicker2 实例将大大增加渲染时间。因此，强烈建议您分别渲染每个实例。
