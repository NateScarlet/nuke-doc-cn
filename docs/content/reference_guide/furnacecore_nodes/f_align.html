F_Align

F_Align 拍摄同一场景的两个序列，并在空间上排列它们。它使用全局运动估计 (GME) 来计算一个四角引脚，以便每帧一次拍摄 (源输入) 将在第二个镜头 (参考输入) 中与相应的帧对齐。结果是源图像，它已被重新定位以与参考图像对齐。

分析范围

范围

源剪辑范围

这控制了任何分析运行的框架范围:

• 指定范围 -读取 分析开始 和 分析停止 要分析的框架范围的字段。

• 源剪辑范围 -自动从连接到的剪辑的长度确定要分析的帧范围 Src 输入。

• 当前帧 -分析仅发生在当前帧上。这对于纠正分析整个剪辑时可能发生的任何错误非常有用。

过滤

过滤

介质

设置过滤质量:

• 低 -低质量，但渲染速度快。

• 介质 -使用双线性过滤器。这给出了良好的结果，并且渲染速度比 高 质量过滤。

• 高 -使用 sinc 滤波器来插入像素，从而进行更清晰的修复。这给出了最好的结果，但需要更长的时间来处理。
