F_ReGrain

F_ReGrain is used to add grain to a sequence. It is designed to sample an area of grain from one image and then to generate unlimited amounts of this grain with exactly the same statistics as the original. This new grain can then be applied to another image.

See also Grain and ScannedGrain.

Grain Type

grainType

Preset Stock

Selects whether the grain is sampled from the Grain input or from a set of standard stocks.

• Preset Stock - grain characteristics are sampled from the film stock specified in the Preset Stock field.

• From Grain Clip - samples and reconstructs the grain characteristics from the Grain input.

Preset Stock

presetStock

FUJIF250under 2K

The film stock that grain characteristics are sampled from when GrainType is set to PresetStock.

Grain Amount

amount

1

Adjusts the brightness of the grain. Setting this to 0 adds no grain.

Grain Size

size

1

Adjusts the size of the grain granules. The larger the value, the bigger and softer the granules.

Output

output

Result

Sets whether to render the result or a test image.

• Result - shows the Src input with the grain applied.

• Grain Plate - shows a test image with the grain applied. This test image is composed from a section of the input image surrounded by a uniform solid color sampled from the image with the grain applied. If the inner area is indistinguishable from the outer area, then you have a good grain sample.
