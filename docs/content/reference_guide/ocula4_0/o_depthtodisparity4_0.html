O_depthtopatiance

许多 Ocula 节点依靠视差图来产生它们的输出。通常，视差图是使用 O_Solver 和 o_dispararitygenerator 节点的组合创建的。

但是，如果您有一个具有立体相机信息和 z-深度贴图的 CG 场景，您也可以使用 o _ depthtoparity 节点生成视差贴图。如果相机信息和 z 深度图是正确的，这比使用 O_Solver 和 O_DisparityGenerator 节点更快、更准确。
