的 O_verticalalaligner 节点允许您垂直扭曲视图，以便其对应的特征水平对齐。的 垂直倾斜 和 局部对齐 选项允许您扭曲视图，同时保持每个像素的水平位置相同，以便收敛没有变化。

AlignWhat

两种观点

设置如何移动视图以对齐图像:

• 两种观点 -移动左视图和右视图，使它们对齐。

• 从左到右 -移动左视图与右对齐。

• 从右到左 -移动右侧视图与左侧对齐。

过滤器

filter

Lanczos6

设置将像素从原始位置重新映射到新位置时要使用的过滤算法。这有助于避免图像质量问题，特别是在帧的高对比度区域 (如果没有过滤像素并保留其原始值，则可能会出现高度锯齿或锯齿的边缘)。

• 冲动 -重新映射的像素携带其原始值。

• 立方 -重新映射的像素接收一些平滑。

• 钥匙 -重新映射的像素接收一些平滑，加上轻微的锐化。

• 西蒙 -重新映射的像素接收一些平滑，加上中等锐化。

• 里夫曼 -重新映射的像素接收一些平滑，加上显著的锐化。

• 米切尔 -重新映射的像素接收一些平滑，加上模糊来隐藏像素化。

• 帕森 -重新映射的像素接收所有过滤器的最大平滑。

• 缺口 -重新映射的像素接收平坦平滑 (倾向于隐藏莫尔图案)。

• Lanczos4 -重新映射的像素会接收一些锐化，这对于缩小缩放很有用。

• Lanczos6 -重新映射的像素获得适度的锐化，这对于缩小缩放很有用。

• Sinc4 -重新映射的像素接收锐化，这对于缩小缩放很有用。

对齐类型

全

设置要应用的全局对齐更正类型:

• 自定义 -使用复选框手动定义要包括的对齐类型。

• 变换 -执行 2D 校正，而不改变像素方面或倾斜。

• 匹配相机 -使用连接到上游 O_Solver 的匹配移动相机执行校正。

• 仅限 Keystone -正确的垂直对齐而不改变视差。

• 全 -应用除 垂直倾斜 。每个选项按从上到下的顺序应用。

AlignCamera

已启用

垂直移位

已启用

启用后，通过向上或向下移动整个图像来垂直对齐特征匹配。计算 2D 垂直偏移以更正全局偏移。

2D 旋转

对齐旋转

已启用

启用后，通过围绕点旋转整个图像来垂直对齐特征匹配。旋转中心由算法决定。这有助于纠正平面内相机滚动。

透视扭曲

AlignPerspective

已启用

启用后，对图像执行四角扭曲，使其在 y 轴上对齐。这可能会沿 x 轴略微移动特征。 透视扭曲 可以帮助纠正相机倾斜和滚动。

注意: 透视变化会改变像素纵横比并引入像素倾斜。

垂直倾斜

AlignVerticalSkew

禁用(disabled)

启用后，使用倾斜沿 y 轴对齐特征。这不会沿 x 轴移动特征。 垂直倾斜 允许您在不改变水平视差的情况下纠正键石刑。这将改变图像的垂直移动，而不改变像素的水平位置。然而，这也可能会引入像素倾斜。
