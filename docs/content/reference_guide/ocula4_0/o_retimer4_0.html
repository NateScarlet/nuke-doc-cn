O_Retimer

O_Retimer 被设计为重计时镜头，以便它以更快或慢动作回放。O_Retimer 使用由 O_VectorGenerator 节点生成的上游运动矢量来重新计时素材。这些运动矢量描述了每个像素如何从一帧移动到另一帧 (请参见 O_VectorGenerator )。有了精确的运动矢量，可以通过沿运动方向插值在序列时间线的任何点生成输出图像。

使用 GPU (如果可用)

UseGPUIfAvailable

已启用

启用时，渲染发生在 本地 GPU 指定 (如果可用)，而不是 CPU。GPU 和 CPU 之间的输出在 NVIDIA GPU 上是相同的，但是使用 GPU 可以显著提高处理性能。

Nuke 在 2013年底的 Mac Pro 6,1 和 2015年中的 Mac Pro 11,5 上支持 AMD gpu，运行 OS X 10.9.3 “mavericks” 或更高版本。虽然，在某些情况下，GPU 和 CPU 之间的比特级相等保持不变，但对于某些操作，这种配置可能存在精度限制。

注意: 在没有本地 GPU 的情况下启用此选项允许脚本在 GPU 上运行，只要脚本在具有 GPU 可用的机器上打开。
你应该选择这个，如果你想使从命令行的 -- Gpu 选项。

GPU 处理还需要某些 NVIDIA GPU 和 CUDA 驱动程序，或者在 2013年底运行 10.9.3 “mavericks” 或更高版本的 Mac Pro 6,1 和 2015年中 Mac Pro 11,5 的情况下，某些 AMD GPU 和驱动程序。有关最低要求的更多信息，请参考 帮助 > 发行说明 .

计时

计时

速度

设置如何控制剪辑的新计时:

• 速度 -如果您希望根据总持续时间来描述重新计时，请选择此选项: 双速将使剪辑的持续时间减半，或半速将使剪辑的持续时间增加一倍。

• 源帧 -如果你想用 “在输出剪辑的 100 帧，我想看到源剪辑的 50 帧” 来描述重新计时，请选择此选项。您需要为此设置至少两个关键帧，以重新计时剪辑。

速度

速度

0.5

根据播放速度和总持续时间定义重新计时: 双速将剪辑的持续时间减半，半速将剪辑的持续时间加倍。

注意: 当你重新定时录像使用 速度 方法，请确保在输入帧上设置关键帧，因为在输入帧上设置关键帧会改变输出帧的数量。
