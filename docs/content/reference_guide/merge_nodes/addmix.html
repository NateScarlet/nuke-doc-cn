AddMix

使用 AddMix，输入的 alpha 一个 用于索引两条颜色校正查找曲线 (LUTs)。第一个用于乘以输入 一个 ,第二个乘输入 B ,并将结果加在一起。AddMix 执行与合并节点类似的操作 结束 模式，但也使图像成倍。如果你有一个已经渲染过的 CG 图像，你最好不要使用 AddMix，因为你可能会因为你的图像被预乘两次而失去细节。

如果您的输入包含覆盖特定区域的掩码，并且您希望只在该特定区域合并, 例如，如果你已经旋转了一个特定的形状，并且想用旋转形状作为 alpha 来混合它。
