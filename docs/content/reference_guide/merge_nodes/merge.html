合并

合并允许您将多个图像分层在一起。使用此节点时, 您需要选择一个合成算法，该算法确定如何计算一个输入的像素值和另一个输入的像素值，以创建作为合并图像输出的新像素值。的 操作 下拉菜单包含大量不同的合成算法，在构建合成时给你很大的灵活性。

当使用大多数可用的合并算法时, Nuke 期望预乘输入图像。然而，随着 哑光 操作您应该使用未预乘的图像。

现在我们有了一些基础知识，我们可以继续制作更复杂的节点网络。这就是基于层的合成和基于节点的合成之间的区别真正变得明显的地方。为了节省一些时间，我已经读了两段镜头: 一段是峡谷的静止图像，另一段是 CG 宇宙飞船的图像序列。

让我们建立一个更复杂的网络。我要把观众和峡谷联系起来。现在，合成的一个常见任务是应用效果或过滤器来改变图像。例如，我可以模糊这个峡谷。如果在节点图中没有选择节点，我创建了一个新节点 -- 例如，我可以转到 过滤器 > 模糊 使用鼠标右键菜单 -- 新节点本身就进来了，它没有连接。实际上有几种方法可以在节点之间建立这些连接。当然，一个是简单地点击拖动管道，并通过连接查看器将其放在另一个节点的顶部。如果你有一个预先存在的连接线，例如，Viewer1 管道连接到 Read1，我可以简单地点击拖动我的新节点，并将其放在当前连接上。现在，在我们进一步讨论之前，我想谈谈这些管道的含义。您可以查看管道和箭头，从中可以确定信息的流动方向。所以，我可以告诉信息是从 Read1 流出，通过 Blur1，然后进入 viewer1。

现在，还有其他方法可以将这些连接在一起。如果我断开这些管道，单击拖动末端，然后断开它们，您也可以简单地连接输入和输出。例如，模糊节点有一个输入管道。我可以告诉它是一个输入管道，因为它流向这里顶部的节点。查看器也有一个输入管道。一些节点也有输出。在 Blur1 上有一个输出，在 Read1 上有一个输出管道 -- 这意味着信息从这些节点中流出。所以，我可以将 Blur1 的输入连接到 read1 的输出。现在让我们抓住输入管道并将其放在 Read1 上，这意味着 Read1 现在正在向 blur1 发送信息。您还可以获取 Blur1 的输出并将其放在 Viewer1 上，这意味着 Viewer1 的输入来自 Blur1。有第三种方法可以做到这一点，如果我再断开这些。将观众重新连接到峡谷，然后选择峡谷。如果选择了一个节点，并且我创建了一个像模糊这样的新过滤器，它会自动插入下游。让我删除另一个模糊，因为我现在不需要它。所以，有一个小网络。现在没有模糊发生，一些过滤器节点会发生的一件事是默认情况下它们是关闭的。如果我增加 尺寸 在模糊的属性，它变得模糊。

合成的另一个重要任务是组合多个图像或片段。现在，在基于层的系统中，您只需堆栈层。在像 Nuke 这样的程序中，你必须使用一个特殊的节点来合并输入。让我们试一试。我将删除模糊节点。我将引入一个合并节点，这是在 合并 > 合并 或的 M 关键。合并节点具有 一个 和 B 输入，加上输出。你可以把它与基于层的合成相关联 B 输入作为下层，你可以想到 一个 作为上层输入。因此，您可以将查看器连接到 merge1 的输出。在这种情况下，连接 B 到峡谷和连接 一个 去宇宙飞船。如果我放大，我可以看到我的宇宙飞船现在被合成在我的峡谷顶部。现在，工作的原因是宇宙飞船周围有阿尔法透明度。这是在玛雅中渲染的。Nuke 会自动识别 alpha 通道，所以这个黑色被转换为透明。你可以看出核弹已经认出了阿尔法，因为这里有一条小白线，就在红线、绿线和蓝线旁边。这些是通道线。所以，在这里它识别 rgb alpha，而在峡谷上它只看到 rgb。在这种情况下，这是你的基本合并。现在，关于合并有几件有用的事情需要知道。一件事是你可以拥有不止一个 一个 输入。事实上，一旦你联系上 一个 ,该 A2 出现在左边。如果你连接 A2 是的，你会有 A3 等等。最高的 一个 数字相当于最高层，所以 A3 是在顶部 A2 ,等等，用 B 在最底部。

现在，有一个 面具 然而，在右边的输入，我们将在后面谈到对位。合并节点也有一个 混合 这里最底部的滑块。混合滑块控制的影响 一个 输入。如果我减少了, 一个 开始淡出。合并节点的另一个重要方面是 操作 在这里的顶部。在基于层的合成中，你可能有类似混合模式的东西。现在，混合模式决定了上层如何与下层结合 -- 某种类型的数学公式。在 Nuke 中，它是 操作 这默认为 结束 。结束类似于 正常 里面的 After Effects。您可以将此菜单更改为其他样式，例如 加 。Plus 添加 一个 和 B 一起。在这种情况下，它给了我一个半透明、更亮的结果。我会把它变成 结束 现在。如果你好奇这些操作是做什么的，让你的鼠标悬停在菜单上。您将看到所有不同操作的所有不同数学公式。在这种情况下，资本 a 和 B 代表的 rgb 值 一个 输入和 B 输入，而小写字母 a 和 b 表示的 alpha 值 一个 和 B 输入。
