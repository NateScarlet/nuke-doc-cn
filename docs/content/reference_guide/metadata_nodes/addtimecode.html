AddTimeCode

向从输入节点向下传递的元数据添加时间代码。默认设置将时间代码 01: 00 添加到第一帧。这段时间的代码根据输入剪辑的播放速度在整个帧范围内更新，这反过来又由 Fps (每秒帧数) 中的控制 项目设置 。如果你改变 Fps 中的值 项目设置 ,元数据中的时间代码将更新以反映更改。
