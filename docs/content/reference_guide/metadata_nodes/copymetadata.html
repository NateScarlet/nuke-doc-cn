来自的元数据

MergeMode

图像元

设置要向下传递的图像和元数据组合:

• 图像元 -将元数据从元输入添加到图像输入的元数据。如果输入共享任何公共元数据键，则从元输入获取的值将覆盖从图像输入获取的值。

• 元仅 -仅使用元输入中的元数据。

• 元图像 -将图像输入的元数据添加到元输入的元数据中。如果输入共享任何公共元数据键，从图像输入获取的值将覆盖从元输入获取的值。

• 仅图像 -仅使用图像输入中的元数据。这产生的结果与根本不使用 CopyMetaData 相同: 图像和元数据都是从图像输入中获取的。如果要筛选向下传递的元数据，此选项很有用。
