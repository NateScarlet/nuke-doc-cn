模糊型

模糊型

缩放

选择要应用的模糊类型:

• 缩放 -从中心点向所有方向涂抹像素。您可以使用它来模拟光线，或者在一个剪辑和另一个剪辑之间过渡。

• 径向 -在中心点周围涂抹像素。可以使用它来模拟旋转对象上的运动模糊。

• 线性 -在给定方向上涂抹像素。可以使用它来模拟移动对象上的运动模糊。
