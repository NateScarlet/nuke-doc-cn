风格

RGB 亮度

根据可用的输入通道设置要创建的光线样式:

• RGB 亮度 -根据输入图像亮度创建光线。

• 阿尔法边缘 -根据输入图像 alpha 的边缘创建光线。此设置对全帧阿尔法没有可见效果，因为光线是在格式之外创建的。

• 阿尔法固体 -根据输入图像的整个 alpha 创建光线。

• 影子制作 -使用输入图像的 alpha 创建光线来投射阴影。

RGB 选项

亮度公差

Luma_tol

0

设置基亮度，低于该亮度时不会生成光线。增加此值可消除图像低亮度区域产生的光线。

Alpha 选项

边缘大小

Edge_size

0

调整 alpha 边缘的厚度，控制受 alpha 影响的光线量。正值减少了允许超过边缘的光线数量，反之亦然。

边缘模糊

3

控制应用于 alpha 边缘光线的模糊量。
