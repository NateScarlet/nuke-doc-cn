侵蚀 (模糊)

类似于 侵蚀 (过滤器) ,但是更平滑，输入像素相对于 尺寸 控制。负值导致较亮的区域扩展到较暗的区域，正值导致较暗的区域扩展到较亮的区域，这对于遮罩特别有用。此外，您可以添加模糊的投入使用, 模糊 和 质量 控制。

注意: 除 0 或 1 之外的任何颜色的实心区域会更改其颜色，因为它们被视为抗锯齿边缘。这可以通过使用不同的算法来纠正，例如 侵蚀 (过滤器) 节点。
