矩阵

将用户定义的矩阵添加到输入中，允许您将所有或特定通道乘以矩阵值。可以使用 “动画” 菜单复制和粘贴现有矩阵，也可以手动调整各个字段。

生成边缘检测过滤器 (请参见 EdgeDetect ) 将中心像素乘以 8，将周围像素乘以-1，然后将九个值相加，以计算中心像素的新值。对源图像中的每个像素重复此过程，以创建输出图像。
