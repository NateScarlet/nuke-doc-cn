Erode (fast)

Also known as the Dilate node, Erode (fast) controls the relative amount of dark (negative size values) and light (positive size values) pixels in the input. This can be used to grow or shrink mattes, for example, an alpha matteâ€™s pixels can be given a positive size value to cover more area.

Erode (fast) is similar to Erode (filter), but computationally less expensive.
