NoOp

NoOp 通过未更改的输入图像。它不会以任何方式改变图像，但可以用作控制其他节点的接口。为此，在 NoOp 控制面板上单击鼠标右键，选择 管理用户旋钮 ,并添加所需的控件 (例如，复选框)。然后, Ctrl Cmd 将此控件拖到另一个节点中的控件的顶部 (例如, 禁用 模糊节点的复选框)。这将创建一个链接 Tcl 表达式，该表达式支持模糊节点的 禁用 选中 NoOp 控件中的复选框，允许您使用 NoOp 控件禁用模糊节点。

您也可以使用 NoOp 节点来更改 文件 使用单个控件的多个读取节点的路径。

有关 Tcl 表达式的更多信息，请参见 表情 Nuke 或选择中的一章 帮助 > 文档 > 旋钮数学表达式 在 Nuke 。
