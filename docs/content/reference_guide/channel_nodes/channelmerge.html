ChannelMerge

ChannelMerge 节点允许您将每个输入的一个通道合并在一起，并将结果保存在选定的 输出通道 。所有其他通道都将从 B 输入。

如果没有 一个 输入连接后，要合并的两个通道都从 B 输入。

默认情况下，ChannelMerge 合并输入的阿尔法。

operation

联盟

设置输入像素值的方式 一个 用输入的像素值计算 B 要创建作为合并图像输出的新像素值，请执行以下操作:

• 缺席 Abs (A-B)-像素有多少不同。

• B 如果不是 a 答: B-显示一个存在的地方; 否则显示 B。

• 划分 A/B，如果 A <0 和 B <0，则为 0-除以值，但阻止两个负值成为正数。

• 从 (B-A)-从 B 中减去 A。

• 在 仅 Ab 显示图像 A 与 B 的 alpha 重叠的区域。

• 最大 (Max (A，B))-选择两种颜色中的浅色作为结果颜色。只有比 B 暗的区域被替换，而比 B 轻的区域不改变。

• Min (Min (A，B)-选择两种颜色中较暗的颜色作为结果颜色。任何比 B 轻的零件都被替换。图像中比 B 暗的任何部分都不会改变。

• 减去 (A-B)-从 A 中减去 B。

• 乘 (AB，如果 A <0 和 B <0)-将 A 乘以 B。结果总是更暗。与黑色混合会产生黑色，与白色混合会使颜色保持不变。

• 出去 A (1-b)-仅显示图像 A 中与 B 的 alpha 不重叠的区域。

• 加 B-两种颜色的总和。增加亮度以照亮 A 并反射 B。

• 钢网 B (1-a)-这是 out 操作的反面。仅显示图像 B 中与 A 的 alpha 不重叠的区域。

• 联盟 B-AB-显示图像 A 和 B。

• Xor B-2AB-显示图像 A 和 B，其中图像不重叠。
