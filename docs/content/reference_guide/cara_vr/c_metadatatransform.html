C_MetaDataTransform

CaraVR 元数据最初是由 C_CameraSolver 创建的，但是 C_MetaDataTransform 可以修改流，而无需添加对图像质量的过滤命中，以执行某些操作，例如使用 C_Tracker 稳定拍摄。

看到商队 在线帮助 有关 C_MetaDataTransform 的更多信息。
