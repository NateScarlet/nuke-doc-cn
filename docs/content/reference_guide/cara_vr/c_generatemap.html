C_GenerateMap

C_GenerateMap 允许您生成 Stitch_map 和 Ppass_map 通道。的 Stitch_map UV 通道表示图像的绝对像素位置，标准化在 0 和 1 之间，而 Ppass_map XYZ 通道表示 latlong 图像的标准化 3D 坐标，范围在-1 和 1 之间。

C_GenerateMap 有一个可选输入来设置地图的格式。如果没有连接到 Src 输入, 项目设置 > 全尺寸格式 被使用。

使用 GPU (如果可用)

UseGPUIfAvailable

已启用

启用时，渲染发生在 本地 GPU 指定 (如果可用)，而不是 CPU。GPU 和 CPU 之间的输出是相同的，但是使用 GPU 可以显著提高处理性能。

注意: 在没有本地 GPU 的情况下启用此选项允许脚本在 GPU 上运行，只要脚本在具有 GPU 可用的机器上打开。
你应该选择这个，如果你想使从命令行的 -- Gpu 选项。

GPU 处理还需要某些 NVIDIA GPU 和 CUDA 驱动程序。有关最低要求的更多信息，请参见 帮助 > 发行说明 .
