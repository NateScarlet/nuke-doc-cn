C_CameraSolver

CaraVR 的 C_CameraSolver 节点为多摄像机钻机创建了相对于节点点或球体的近似摄像机几何。这定义了每个相机的位置和方向预览，以便图像可以由 C_Stitcher 缝合以进行输出。

看到商队 在线帮助 有关 C_CameraSolver 的更多信息。

相机

设置要添加到节点图中的导出节点:

• 相机 -为装备中的每个相机创建附加到轴和场景的相机节点。

• 变换 (分割) -创建单个 C_SphericalTransform，其中包含钻机中每个相机的旋转数据。

• 变换 (单独) -使用钻机中每个相机的 OneView 和 C_SphericalTransform 节点为每个视图创建单独的旋转数据。

• 手动 2D 缝合 -为每个视图创建一个包含 C_SphericalTransform 的 RigWorkflow2D 组，并传递给 JoinViews 和 C_Blender 节点，以将视图重新组合成一个连贯的整体。

• 手动 3D 缝合 -为每个视图创建一个包含 C_SphericalTransform 和 Camera 的 RigWorkflow3D 组，投影到球体上并通过 C_RayRender 渲染。

• 摄入 (加入) -为所有输入创建 ShuffleView 节点，允许您手动将输入分配给视图，然后将视图连接在一起。

• 摄入 (单独) -为所有输入创建 ShuffleView 和 OneView 节点，允许您手动将输入分配给视图。

此出口类似于 摄入 (加入) ,但是视图被分成单独的流。

• 接触表 -创建一个 ContactSheet，其中包含所有已解决的视图以供查看。
