HueShift

HueShift 变换输入图像的色彩空间。它的工作原理是将颜色转换为 CIE XYZ 空间，然后在空间上进行简单的缩放和旋转。在 XYZ 空间中，X 和 Z 控制颜色，Y 是亮度。
