表达

允许您使用类似 C 的语法表达式将复杂的数学公式应用于通道的值。如有必要，可以将不同的表达式应用于不同的通道集。

• x 或 y 是像素坐标。如果输入是代理图像，则这些图像将被缩放并转换为全尺寸图像中的坐标。

• Cx 和 赛 可以提供更有用的坐标系统。在这个系统中，0,0 是图片的中心。

-1,0 是左边缘的中心，1,0 是右边缘的中心。1,1 是从右侧边缘的原点向上 45 度的点 (如果长宽比大于 1，则在图片顶部之外)。

• 您可以通过名称引用任何输入通道，例如 r 为红色通道。

• 您可以通过名称引用任何节点上的任何控件。例如, Blur1.size 返回模糊的大小。您还可以计算动画控件，例如 Blur1.size (t) .

• 空白表达式与零相同。

• 所有的数学都是用 32 位浮点数完成的。

另见 MergeExpression .
