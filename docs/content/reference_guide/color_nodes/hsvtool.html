HSVTool

HSVTool 允许您从单个控制面板同时调整色调、饱和度和值分量。它还具有一个颜色替换工具，使一个很好的键控。

这个节点的主要优势是它在限制对窄样本颜色的修正方面提供的精度。您可以通过采样几个像素，然后移动它们的值，将校正限制在某些色调。因为您限制了颜色范围，所以周围的图像通常不受移位的影响。
