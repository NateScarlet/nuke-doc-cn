快船测试

ClipTest 将斑马条纹应用于指定范围之外的所有像素。这可以帮助您确保输入的最黑和最白的白色在预期的显示设备上可见。默认情况下，ClipTest 突出显示 0 到 1 范围之外的任何像素。

注意: 此节点影响最终输出，而不仅仅是查看器。
