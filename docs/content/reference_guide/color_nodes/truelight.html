Truelight

Truelight 是一个颜色管理系统，可以让你在电脑显示器上预览电影的最终外观。预览基于胶片记录器、投影系统和显示器的校准数据。

该技术获得 FilmLight 的许可，FilmLight 还为显示校准和轮廓生成提供额外的工具。

原始 Truelight 输出

Output_raw

已启用

启用后，不执行 Truelight 输出回线性颜色空间的转换。

Truelight 通常用于在显示 (监视器) 颜色空间中生成图像，在这种情况下，这应该是活动的。如果关闭，输出将使用反向 sRGB LUT 进行线性化。

如果您使用 Truelight 作为查看器输入过程，您通常希望将其输出直接发送到屏幕，而无需进一步的 LUT 转换, 因此，激活它，并将查看器 LUT 设置为线性，这样查看器就不会进行进一步的转换。

如果您想将 Truelight 输出直接写入文件，请激活它，并将写入节点颜色空间设置为线性，以防止它将输入从线性转换为另一个颜色空间。
