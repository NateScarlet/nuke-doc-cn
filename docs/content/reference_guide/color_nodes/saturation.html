饱和

当你只想校正图像的饱和度 (颜色强度) 并且不需要将校正限制在任何特定的色调范围时，使用饱和度节点。

如果您确实想调整一系列色调的饱和度 (例如，减少绿色、蓝色或红屏溢出)，请使用 HueCorrect 而不是节点。
