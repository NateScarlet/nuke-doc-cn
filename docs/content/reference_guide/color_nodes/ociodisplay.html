OCIODisplay

使用 OpenColorIO 库将颜色空间转换应用于图像序列，以便可以在特定的显示设备上准确地表示它。OCIODisplay 节点基于 OpenColorIO 库。有关更多信息，请参见 Http://opencolorio.org .

用作遮罩的可选图像。默认情况下，颜色空间转换仅限于遮罩的非黑色区域。
