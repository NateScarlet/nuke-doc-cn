MatchGrade

MatchGrade ( NukeX 和 Nuke 工作室 仅) 允许您自动计算等级，以匹配 源 输入到中的颜色 目标 输入。您可以使用 MatchGrade:

• 提取一个烘焙等级，如果 目标 夹子，你想匹配包含相同的帧, 源 剪辑,

• 匹配两个不同剪辑之间的等级以创建相同的外观,

• 屏蔽等级，使其仅匹配 源 和 目标 剪辑，和

• 导出 LUT 或 CDL 文件以在其他地方重复使用计算的等级。

匹配分级源

选择是否要匹配 源 使用同一剪辑或不同剪辑的分级版本输入:

• 匹配分级源 -匹配的 源 带有同一剪辑的分级版本的剪辑 (连接到 目标 输入)。这允许您提取和导出应用于原始剪辑的烘焙级。

在这种模式下，MatchGrade 的两个输入都需要相同的帧。

• 匹配不同的剪辑 -匹配一个剪辑的颜色 ( 源 ) 与另一个 ( 目标 )。这允许您创建一个等级来重现 目标 夹在 源 剪辑。

在这种模式下，MatchGrade 需要不同的帧 源 和 目标 输入。

变换

变换

3D LUT

将改造用高档的 源 输入:

• 3D LUT -将等级计算为 3D 查找表 (LUT)。如有必要，您可以使用 出口 控件将此写入 。Csp 格式化，然后单击 创建 OCIOFileTransform 创建应用等级的 OCIOFileTransform 节点。

• CDL -计算等级作为颜色决策列表 (CDL)。如有必要，您可以单击 创建 OCIOCDLTransform 创建应用坡度的 OCIOCDLTransform 节点。

注意: CDL 变换是有限的，不能为所有类型的颜色变换建模。在大多数情况下，选择 3D LUT 给出最好的结果。

注意: 此控件仅在设置了至少一个参照帧时可用。

LUT 分辨率

Lutres

32

设置匹配存储颜色转换的 LUT 的分辨率。

值为 16 会加快处理速度。值为 64 可以改善结果，但也会增加处理时间。

如果两个非常相似的颜色 源 图像是非常不同的 目标 ,尝试将其设置为 64，这是最大值。
