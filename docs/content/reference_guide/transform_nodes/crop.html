裁剪标签

盒 x，y，r，t

(或 x，y，w，h)

盒子

N/A

要保留的输入图像区域。这个盒子外面的任何东西都是裁剪好的。

您可以调整以下内容:

• x -原始图像的左边缘和裁剪框的左侧之间的距离 (以像素为单位)。

• y -原始图像的底部边缘和裁剪框的底部边缘之间的距离 (以像素为单位)。

• r -原始图像的左边缘和裁剪框右侧之间的距离 (以像素为单位)。

• t -原始图像的底部边缘和裁剪框的顶部边缘之间的距离 (以像素为单位)。

• w -裁剪框的宽度。只有当您单击 Wh 按钮。

• h -裁剪箱的高度。只有当您单击 Wh 按钮。

也可以通过拖动裁剪框的边缘来调整查看器中的裁剪框。

柔软

柔软

0

允许您对裁剪部分的边缘进行小插曲。值越大，边缘周围的区域就越褪色为黑色。

值为 0 不会产生渐晕。

重新格式化

重新格式化

禁用(disabled)

启用后，图像输出格式将更改为与裁剪的图像匹配。

禁用后，将使用原始图像输出格式。

相交

相交

禁用(disabled)

启用后，输出边界框是裁剪边界框和传入边界框的交集。

禁用时，输出边界框与裁剪边界框匹配，并且可以延伸到传入边界框之外。

黑色外面

作物

已启用

这将呈现为图像边界外的黑色像素，使元素更容易在另一个元素上分层。如果取消选中此控件，外部区域将用图像序列的最外层像素填充。

在大多数情况下，你应该保持 黑色外面 检查过了但是，您可能希望关闭此选项以进行相机抖动，或者希望使用类似的形状对输出进行纹理映射或相交。

注意: 启用 黑色外面 如果没有 alpha，还会添加一个覆盖输入图像区域的实心 alpha。
