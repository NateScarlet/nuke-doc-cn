AdjustBBox

AdjustBBox 节点允许您将边界框的边缘展开或裁剪指定数量的像素。边界框定义了 Nuke 认为具有有效图像数据的帧区域。例如，如果你有一个带有大量黑色 (0,0，0,0) 的图像, 您可以调整边界框，使其只包含有用的区域，这样 Nuke 就不会在没有变化的地方浪费时间计算结果。
