PointsTo3D

使用 PointsTo3D 节点使用两个或三个跟踪点之间的视差计算图像序列中 2D 点的 3D 位置。

将图像及其相应的跟踪相机连接到 PointsTo3D 节点输入。选择要计算的 2D 点，放置 一个 在特定的框架上点击 设置框架 。现在，将帧更改为不同的相机角度，并指定相同的 2D 特征 B 并单击 设置框架 。为点重复 C ,再次在不同的框架上。单击 计算 生成 3D 空间中的点以及转换回屏幕空间的点。

自由移动

选择跟踪的相机移动类型:

• 自由移动 -这将图像中的 2D 点变成 3D 空间中的点。这需要相机移动，你可以在三帧中找到 2D 点。选择要计算的 2D 点，放置 A点 在特定的框架上点击 设置框架 。现在，将帧更改为不同的相机角度，并指定相同的 2D 特征 B 点 并单击 设置框架 。重复 for 点 C ,再次在不同的框架上。单击 计算 生成 3D 空间中的点以及转换回屏幕空间的点。您也可以单击 生成轴 以创建具有此位置的 3D 轴对象。

• 节点 -这忽略了 B 点 和 点 C 。相反，它只是把 A点 相机前面的固定距离，使用相机的焦距来决定距离。
