IDistort

IDistort warps the input image based on the values in the imageâ€™s UV channels. The U and V values are offsets for where a pixel will come from. For example, if pixel 51, 23 has a U and V value of -1, 5, the pixel's value will come from 50, 28 of the input channels.

You can use the Copy node to merge the two distortion channels in with your image channels, and then select the two channels from the UV channels dropdown menu.
