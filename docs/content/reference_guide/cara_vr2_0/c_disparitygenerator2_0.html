C_DisparityGenerator

C_DisparityGenerator 节点用于为立体图像创建视差图。视差图描述一个视图中像素的位置，相对于另一个视图中相应像素的位置。它包括两组视差向量: 一组将左视图映射到右，另一组将右视图映射到左。

注意: C_DisparityGenerator 不同于 Ocula 版本，它可以在 latlong 空间中映射，这意味着当图像环绕时，视差向量通过帧边缘时，将被映射。

看到商队 在线帮助 有关 C_DisparityGenerator 的更多信息。

向量生成

矢量细节

VectorDetail

0.3

调整计算的视差向量的密度。更高的细节可以获得更好的视差变化，但需要更长的时间来计算。

强度

强度

1.5

设置在左视图和右视图之间匹配像素时应用的强度。更高的值允许您准确地将一幅图像中的相似像素匹配到另一幅图像，即使生成的视差图是锯齿状的，您也可以专注于细节匹配。较低的值可能会错过本地细节，但不太可能为您提供奇怪的伪向量，从而产生更平滑的结果。

通常，有必要将这些品质中的一个与另一个交换。你可能想增加 强度 强制视图匹配错过的细节，或者减少细节以平滑视差图。

一致性

一致性

1

设置左视图和右视图中的相同点相互映射的精确程度。增加该值以鼓励左、右视差向量匹配。

平滑度

平滑度

1

控制图像边缘的深度变化。在高对比度图像中，增加平滑度以减少图像边缘的深度分离。在低对比度图像中，降低平滑度以拾取图像边缘的深度变化。
