C_双边

C_双边是一个平滑过滤器，它根据邻近的源像素的空间距离和颜色相似性来混合它们。该滤波器特别擅长保留边缘，尽管它的计算成本很高。

它类似于标准的 Nuke 双边节点，但是增加了 GPU 加速和可选的 指南 在过滤时计算颜色相似性的输入。

看到商队 在线帮助 有关 C_AlphaGenerator 的更多信息。

指南

过滤时计算颜色相似性的可选图像。

连接 指南 输入将过滤器的输出转换为与引导图像相同的格式，从而产生联合双边重采样过滤器。如果 Src 和 指南 图像具有相同的格式，节点充当交叉双边过滤器。

如果 指南 输入未连接，输出格式等于 Src 输入和作为标准双边滤波器。

面具

用作遮罩的可选图像。默认情况下，过滤器效果仅限于遮罩的非黑色区域。

起初，遮罩输入在节点的右侧显示为三角形，但是当您拖动它时，它会变成一个标记为遮罩的箭头。如果看不到遮罩输入，请确保 面具 控件已禁用或设置为 没有 .

使用 GPU (如果可用)

UseGPUIfAvailable

已启用

启用时，渲染发生在 本地 GPU 指定 (如果可用)，而不是 CPU。GPU 和 CPU 之间的输出是相同的，但是使用 GPU 可以显著提高处理性能。

注意: 在没有本地 GPU 的情况下启用此选项允许脚本在 GPU 上运行，只要脚本在具有 GPU 可用的机器上打开。
你应该选择这个，如果你想使从命令行的 -- Gpu 选项。

GPU 处理还需要某些驱动程序。有关最低要求的更多信息，请参见 帮助 > 发行说明 .

尺寸 (w，h)

尺寸

3

设置比较像素的区域的高度和宽度以计算过滤器。较高的值会扩大比较区域，产生更大的效果。

位置西格玛

PositionalSigma

0.4

控制附近像素影响整体滤镜的量。

低值仅混合彼此接近的像素。更高的值也混合了更遥远的像素，只要它们的颜色相似。

颜色西格玛

ColorSigma

0.4

控制过滤过程中像素颜色的影响。

低值仅混合颜色非常相似的像素，有助于保留图像中的边缘。较高的值混合具有较大颜色差异的像素，导致边缘更平滑，颜色不均匀的区域更大。
