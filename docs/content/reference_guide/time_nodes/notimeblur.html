NoTimeBlur

NoTimeBlur 防止下游节点请求小数帧或多帧。可以使用此节点来避免不必要的计算。例如，您可能会发现使用 时间模糊 在合成中的所有上游节点上，可能是不必要的，而且非常耗时。在这些情况下，可以使用 NoTimeBlur 节点来限制要应用 TimeBlur 的节点数量。只需在 TimeBlur 上方的节点树中插入 NoTimeBlur 节点和您希望 TimeBlur 节点处理的任何节点。

还可以使用 NoTimeBlur 节点强制动画对象与输入图像对齐。

• Rint -将帧舍入到最接近的整数值。

• Ceil -将帧舍入到最接近的整数值。
