框架混合

FrameBlend 节点通过生成之前和之后的帧的附加复合来插入帧，而不是仅仅在现有帧之间创建副本。这种方法在所有快速移动的特征周围创建 “重影”。当作为静止帧的一部分观看时，结果可能看起来很奇怪，但在实际播放期间会有助于更平滑的运动。

您应该在要影响的时间效果之前插入此节点 (例如，在重定时节点之前)。
