移除 3:2 下拉

从输入素材中移除 3:2 下拉。

如果您使用的镜头最初是电影，但被转换为视频，您可能希望在将镜头导入 Nuke 时使用 Remove 3:2 下拉节点。这允许您在添加效果之前将镜头转换为其原始状态 (24 fps 非隔行电影)。如果你以后想再次将镜头渲染回视频，你可以使用 添加 3:2 下拉 节点重新引入 3:2 下拉。

3:2 下拉过程的结果是:

• 包含同一胶片帧中的两个字段的三个完整帧，以及

• 包含来自两个不同胶片帧的字段的两个分割域帧。两个分裂场帧总是彼此相邻。

相位是两个分割场帧发生在镜头的前五帧内的点，相对于第一个输入帧。这里的值应该与添加 3:2 下拉时使用的阶段相匹配。

反转场优势

Field_dom

禁用(disabled)

视频帧通常分为两个字段。每个字段只包含一半的图像信息，绘制为水平扫描线:

• 帧的第一个字段包含所有其他扫描线。

• 帧的第二个字段包含剩余的扫描线。

当这两个字段以快速顺序显示时，它们似乎会产生一个正常的完整帧。

反转场优势 确定首先显示哪个字段:

• 启用时，奇数行首先出现。

• 禁用时，偶数行首先出现。
