• 输入 -默认值，将帧范围设置为附加到 Src 输入。

• 全球 -将跟踪帧范围设置为 项目设置 > 帧范围 控制。如果没有定义帧范围，您读取的第一个图像的帧范围将用作 全球 帧范围。

• 自定义 -设置由描述的帧范围 从 和 到 字段。

矢量细节

VectorDetailReg

0.3

设置矢量质量。默认值 0.3 对于低细节和移动的序列来说已经足够了，但是在某些情况下，您可能需要增加细节来提高矢量质量。

注意: 高细节矢量渲染需要更长的时间，但可以改善从矢量扭曲节点获得的结果。
