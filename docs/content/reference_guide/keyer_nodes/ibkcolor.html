IBKColor

IBK 键盘器不同于许多其他键盘器，它使用输入图像 (一个只有背景颜色变化的干净板)，而不是使用单一的颜色选择器开钥匙。当使用不均匀的蓝色或绿色屏幕时，这通常会给你带来好的结果。

IBK 键盘器由两个节点组成: IBKColor 和 IBKGizmo 。IBKColor 从蓝色或绿色屏幕图像创建干净的盘子，IBKGizmo 拉动钥匙。

注意: 当图像用作 IBKGizmo 的输入时，从 IBKColor 到某些格式的预渲染输出可能会导致颜色信息丢失和结果不正确。我们建议 。Exr Format 用于预渲染图像，因为它支持全浮动数据，减少信息丢失。

作为一般工作流指南:

1。 将 IBKColor 的输出连接到 c IBKGizmo 的输入。
2. 附加 IBKColor 的输入，以及 Fg IBKGizmo 的输入，原始的蓝屏或绿屏图像。
3. 附上 Bg 将 IBKGizmo 输入到背景图像。
4. 创建两个查看器，并排查看两个节点的输出。
5. 调整 IBKColor 的控制，直到你得到一个干净的盘子，除了背景颜色。
6. 调整 IBKGizmo 的控制，直到你满意的关键。
7. 将 IBKGizmo 的输出连接到合成节点 (如 Merge)，以在背景上合成前景。
