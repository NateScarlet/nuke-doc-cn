灵长类动物

Primatte 键是通过从蓝色或绿色屏幕图像中采样单个像素或一系列像素来逐步创建的。这将控制 colorspace 中的多面体，该多面体确定遮罩内或外的像素。要使用 Primatte，请单击 自动计算 自动感知背衬屏幕颜色，消除它，甚至消除一些前景和背景噪音。或者，您可以设置 操作 您希望如何使用颜色样本，或者使用左箭头和右箭头按钮来逐步完成所有操作。然后，使用 Ctrl 单击或 Ctrl 转变 拖动以从查看器中选择一种颜色或一系列颜色。Primatte 然后使用这些来调整多面体。Nuke 在此过程中提供了无限的撤销。

算法

algorithm

灵长类动物

选择 Primatte 用来计算键控结果的算法:

• 灵长类动物 -Primatte 算法提供最佳结果，并支持纯色和补色溢出抑制方法。它是使用三个多面多面体 (如本章中进一步描述的) 来分离 3D RGB 颜色空间的算法。它也是默认的算法模式，因为它是计算密集型的，渲染可能需要更长的时间。

• 灵长类动物 RT -是最简单的算法，因此是最快的。它只使用一个平面来分离 3D RGB 颜色空间 (如本章中进一步描述的)，因此, 没有像上面那样仔细地将前景从支持屏幕中分离出来的能力 灵长类动物 算法。的其他缺点 灵长类动物 RT 算法是，它在不太饱和的背衬屏幕颜色下工作不好，也不支持补色溢出抑制方法。

• 灵长类动物 RT -这是在上述两个选项之间。它使用六平面表面颜色分离算法 (如本文档中进一步描述)，并在质量和性能方面在另外两者之间提供结果。的其他缺点 灵长类动物 RT 算法是，它在不太饱和的背衬屏幕颜色下工作不好，也不支持补色溢出抑制方法。

合成的一个重要任务是去除蓝屏或绿屏。Nuke 为这项任务提供了各种各样的键控。我将非常简短地介绍其中的几个，这样你就可以介绍了。我带了两段录像来试试这个。有一个我们以前使用过的人的绿色屏幕，还有一个房子的静止图像。它只是有一个明亮的天空，即使在这张特定的照片中没有绿色或蓝色，你仍然可以使用 keyer 工具攻击天空并移除它。你可以到键工具, Keyer 菜单。

我将从最简单的一个开始，叫做 Keyer。现在，尽管它没有很多选择，但对于像这样的某些情况来说，这是非常好的。我要降下来了 一个 合并和读取节点之间的管道。Keyer 有几个操作，其中之一是 亮度键 ,这是默认的。您还可以将某些颜色 (如绿色) 或某些属性 (如饱和度) 作为目标。我们暂时让它保持亮度。这里还有一个图，它代表操作值。例如，此图表表示亮度，因为它从 0 到 1 或最大值。这里还有四个黄色条: 一个 , B , C ,和 D 。您可以通过交互方式单击并拖动这些对象。现在 B 和 C 一开始是重叠的，但是你可以把它们分开。一旦我移动这些，阿尔法哑光就会开始形成。让我们去看看。我将进入观众并按下 一个 关键; 这是阿尔法频道。这意味着任何亮度值在 B 和 C 之间的像素都变得不透明。任何亮度在 C 和 D 之间或 B 和 a 之间的像素值在不透明和透明之间有一个逐渐变细的值。所以，在这种情况下，我能做的是将 B 和 A 移动到最左边，然后调整 C 和 D，使天空变得透明。例如，这意味着亮度值超过 0.9 的任何像素都将变成 100% 透明，或者亮度值介于 0.8 和 0.9 之间的像素具有逐渐变细的透明度。
