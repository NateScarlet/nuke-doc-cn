ChromaKeyer

ChromaKeyer 是一种绿色和蓝屏键控器，可以利用本地 GPU。使用 屏幕颜色 选择器从选择颜色 源 输入用作蓝色/绿色屏幕颜色。要从前景对象中删除蓝色/绿色溢出，请使用 绝望的偏见 控制从源拾取肤色。使用 哑光 参数来提高哑光。

使用 GPU (如果可用)

UseGPUIfAvailable

已启用

启用时，渲染发生在 本地 GPU 指定 (如果可用)，而不是 CPU。GPU 和 CPU 之间的输出在 NVIDIA GPU 上是相同的，但是使用 GPU 可以显著提高处理性能。

核武器 在 2013年底的 Mac Pro 6,1 和 2015年中的 Mac Pro 11,5 上支持 AMD gpu，运行 OS X 10.9.3 “mavericks” 或更高版本。虽然，在某些情况下，GPU 和 CPU 之间的比特级相等保持不变，但对于某些操作，这种配置可能存在精度限制。

注意: 在没有本地 GPU 的情况下启用此选项允许脚本在 GPU 上运行，只要脚本在具有 GPU 可用的机器上打开。
你应该选择这个，如果你想使从命令行的 -- Gpu 选项。

GPU 处理还需要某些 NVIDIA GPU 和 CUDA 驱动程序，或者在 2013年底运行 10.9.3 “mavericks” 或更高版本的 Mac Pro 6,1 和 2015年中 Mac Pro 11,5 的情况下，某些 AMD GPU 和驱动程序。有关最低要求的更多信息，请参考 帮助 > 发行说明 .

屏幕颜色

_ ScreenColor

0,0，0

将屏幕颜色设置为透明，通常为蓝色或绿色背景。

单击旁边的色样 屏幕颜色 激活滴眼器，然后 Ctrl / Cmd 转变 在查看器中的绿色像素上单击并拖动矩形区域。这将平均选定区域中的像素以产生更好的关键点。

屏幕增益

_ ScreenGain

1

调整屏幕颜色的去除量，使屏幕变得哑光。增加此值更多键。

屏幕平衡

_ 屏幕平衡

0.5

将图像的平衡点设置为关键帧。如果密钥在默认设置下工作不太好，请尝试将余额设置为大约 0.05 、 0.5 和 0.95，看看什么最有效。

哑光

色度增益

_ ChromaGain

0

控制使用源图像和屏幕颜色之间的色度差异来校正哑光。

增加 色度增益 生成前景中透明度较低的遮罩，但会产生较硬的边缘。

白点

_ WhitePoint

1

设置用于更正前景的错误阈值。高于该点的所有 alpha 值都设置为 1。

黑点

_ BlackPoint

0

设置用于更正背景的错误阈值。此点以下的所有 alpha 值都设置为 0。

阿尔法偏倚

_ 字母

0.5

调整 阿尔法 偏见 如果你的屏幕颜色不是纯粹的蓝色或绿色，并且导致前景图像的部分变得透明。从受影响的前景部分拾取颜色。

内面具

_ InsideMask

忽略

如果您使用多个 ChromaKeyer 节点在图像的不同部分拉几个不同的键 (多道键控)，请设置 内面具 到 源 alpha 将以前的 ChromaKeyer 节点的结果相加。

当设置为 忽略 ,任何上游密钥都不考虑。

绝望

使用自定义 despill 偏见

_ Overridedepesillbias

残疾人

可以使用 绝望的偏见 控制。

绝望的偏见

DespillBias

0.5

调整 绝望的偏见 从前景图像周围删除任何剩余溢出。挑选最有用的颜色 绝望的偏见 通常是头发的颜色和肤色。

注意: 这种控制是残疾人除外 使用自定义 despill 偏见 已启用。

替换模式

_ 替换

忽略

设置用于修改遮罩 alpha 的像素的颜色方法:

• 忽略 -如果 alpha 被修改，鄙视图像将保持不变。

• 边缘硬颜色 -被鄙视的图像有相应数量的 替换颜色 增加阿尔法的任何增加。

• 边缘线性颜色 -如果 alpha 改变，图像有相应数量的原始像素 (屏幕颜色和全部) 重新引入/删除。

• 边缘柔和的颜色 -被鄙视的图像有相应数量的 替换颜色 然而，对于 alpha 的任何增加，它会尝试调节结果像素的亮度，使其与原始像素相匹配。这产生了比 边缘硬颜色 选项。

外接程序磨砂修复

_ ReplaceAlphaDiff

残疾人

启用后，添加 替换颜色 进入使用哑光校正的区域 黑点 和 白点 控制以增强绝望效果。

替换颜色

_ ReplaceColor

1 、 1

当 替换模式 设置为 边缘硬颜色 或 边缘柔和的颜色 ,设置遮罩增加哑光 alpha 的地方要添加的颜色。

替换金额

_ ReplaceAmount

1

控制的数量 替换颜色 添加到最终结果中。

预乘

_ 预乘

已启用

启用后，将 ChromaKeyer 生成的遮罩与前景预乘。

预乘磨砂可以从背景中去除噪音，但在某些区域 (如头发) 也可能过于激进。
