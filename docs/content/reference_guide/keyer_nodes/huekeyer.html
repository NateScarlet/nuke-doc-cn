HueKeyer

此节点是色度键控器。它从传入的红色、绿色和蓝色通道计算色调，并使用 金额 曲线以确定在 alpha 通道中输出的值。在图中，x 轴表示色调，y 轴表示该色调的 alpha 值。要键入特定的色调，请在 x轴上找到色调 Ctrl / Cmd Alt 单击以在该位置的曲线上创建点。然后在 y 轴上将该点拖动到值 1，以便在 alpha 通道中将该色调设置为黑色。
