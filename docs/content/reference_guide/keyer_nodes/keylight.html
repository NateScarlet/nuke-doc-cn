键光

Keylight 是一个行业成熟的色差键控。使用 屏幕颜色 选择器从选择颜色 源 输入用作蓝色/绿色屏幕颜色和 视图 下拉菜单判断键。要从前景对象中删除蓝色/绿色溢出，请使用 绝望的偏见 控制从源拾取肤色。使用 屏幕磨砂 参数来提高哑光。

Keylight 的核心算法是由计算机电影公司 (现在是 Framestore) 开发的，并由 Foundry 进一步开发。

合成的一个重要任务是去除蓝屏或绿屏。Nuke 为这项任务提供了各种各样的键控。我将非常简短地介绍其中的几个，这样你就可以介绍了。我带了两段录像来试试这个。有一个我们以前使用过的人的绿色屏幕，还有一个房子的静止图像。它只是有一个明亮的天空，即使在这张特定的照片中没有绿色或蓝色，你仍然可以使用 keyer 工具攻击天空并移除它。你可以到键工具, Keyer 菜单。

我将从最简单的一个开始，叫做 Keyer。现在，尽管它没有很多选择，但对于像这样的某些情况来说，这是非常好的。我要降下来了 一个 合并和读取节点之间的管道。Keyer 有几个操作，其中之一是 亮度键 ,这是默认的。您还可以将某些颜色 (如绿色) 或某些属性 (如饱和度) 作为目标。我们暂时让它保持亮度。这里还有一个图，它代表操作值。例如，此图表表示亮度，因为它从 0 到 1 或最大值。这里还有四个黄色条: 一个 , B , C ,和 D 。您可以通过交互方式单击并拖动这些对象。现在 B 和 C 一开始是重叠的，但是你可以把它们分开。一旦我移动这些，阿尔法哑光就会开始形成。让我们去看看。我将进入观众并按下 一个 关键; 这是阿尔法频道。这意味着任何亮度值在 B 和 C 之间的像素都变得不透明。任何亮度在 C 和 D 之间或 B 和 a 之间的像素值在不透明和透明之间有一个逐渐变细的值。所以，在这种情况下，我能做的是将 B 和 A 移动到最左边，然后调整 C 和 D，使天空变得透明。例如，这意味着亮度值超过 0.9 的任何像素都将变成 100% 透明，或者亮度值介于 0.8 和 0.9 之间的像素具有逐渐变细的透明度。

让我们回到 rgb。按下 一个 钥匙又来了，我们走了。现在，它看起来不像是正在发生的任何事情。你要做的是预乘阿尔法。所以，我将选择 Keyer 节点，点击鼠标右键, 合并 > Premult 。Premult 将 alpha 值乘以 rgb 值。一旦我添加了这个，天空就被移除了。现在我可以进一步测试了。我可以把东西钩到 B 合并的管道。例如，我可以用鼠标右键单击，然后转到 图像 > 常数 。常数会产生纯色 -- 我会把它钩到 B 管道。然后转到该节点的色轮，选择一种颜色，如浅蓝色。天空消失了，现在常数出现在天空区域。

Let’s move on to some other keyers. I am going to go back to the greenscreen footage. Now, in fact, you can have more than one Viewer in any project. You can make a new Viewer at any time by going to the 观众 > 创建新查看器 ( Ctrl / Cmd + I ). If you have more than one Viewer, you can switch between them by clicking the tabs. So now, we are going to work on this greenscreen. The first keyer we will try here is Keylight, so Keyer > 键光 . Foundry writes Keylight and, in fact, it is available in other compositing packages like After Effects, the functionality is the same. It looks a little intimidating at the start because of all the inputs. What you have to do is plug the 源 into the greenscreen, in this case, the A pipe into the output. Now there are many options you can adjust. I am going to adjust the basic ones you need to remove the greenscreen. The first thing to do is select the 屏幕颜色 . You can click the swatch here, get the eyedropper, and go back to the Viewer. There are several ways to sample pixels with the eyedropper. You can Ctrl / Cmd +click or Ctrl / Cmd +drag your mouse. I will try the drag, so click+drag, let go - the screen color is sampled and you will see it right here in the swatch, and also beside the Screen Colour property. Let’s take a look at the alpha. Now, Keylight offers this 视图 menu, which you can change from 最终结果 to 组合磨砂 . That’s what the alpha channel looks like, so right now there is some gray in the transparent area. What you can do is raise the 剪辑黑色 to erode that. What the Clip Black does is it looks for any pixel less than the slider and it makes it 100% transparent. There is also some gray in the white area.
