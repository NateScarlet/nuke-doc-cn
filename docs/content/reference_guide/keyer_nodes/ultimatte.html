最后时刻

Ultimatte AdvantEdge 技术是一个色差键控。这是 Ultimatte 公司超过 25 年的蓝屏成像经验的高潮。最终的结果是一个复合材料，它保留了头发、烟雾和运动模糊边缘等精细细节，并且妥协程度最低。

作为一般工作流指南:

1。 使用 屏幕 控件，在要保留的细节附近对屏幕 (背衬) 颜色进行采样。
2. 激活 覆盖 显示和使用 覆盖 + 和 - 选择器，以获得覆盖尽可能多的屏幕，而不会侵入前景主题区域。覆盖表示绝对不在复合材料中的区域。
3. 关闭覆盖视图，查看 alpha 通道，并使用 哑光 + 和 - 拾取器将更坚固的区域带入哑光，或从哑光中去除噪音或其他畸变。
4. 使用 Hold shadow 滴管 (仅在 屏幕正确 和 影子 被启用) 来擦洗你想保留的阴影。这些阴影最好在前景图像中看到。这允许将阴影与前景主题区域分开 (更准确地) 处理
5. 最后，您可以使用 泄漏 和 匹配 选择器，以改善应用于 RGB 通道的颜色校正。

Ultimatte 是 Ultimatte 公司的商标。Ultimatte 专利由 Ultimatte 公司持有。

Screen_correct

已启用

启用时，覆盖 屏幕校正 控件被激活。

有几种情况下，您可以通过禁用屏幕更正来获得更好的结果:

• 当没有清晰的屏幕视图时。例如，它完全被烟雾覆盖，或者通过带有大量反射的窗户拍摄。

• 当你碰巧有噪音/纹理和屏幕颜色的组合，使得找到产生可用结果的控件设置不切实际或不可能。

屏幕正确算法在内部生成一个合成清洁板，但是为了获得最佳结果，如果你有清洁板，也提供一个清洁板作为输入。

注意: 重要的是要预先裁剪出任何非图像区域，如信箱消隐区域，以便它们不会插入屏幕区域。
